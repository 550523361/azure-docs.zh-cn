# <a name="azure-container-service-documentation"></a>Azure 容器服务文档

Azure 容器服务让创建、配置和管理预配置来运行容器化应用程序的虚拟机群集变得更简单。 容器服务使用热门开源计划和业务流程工具的优化配置。 通过此服务，可以使用现有技能或利用不断增加的大量社区专业知识，在 Microsoft Azure 上部署和管理基于容器的应用程序。
