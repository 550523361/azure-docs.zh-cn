# <a name="technical-prerequisites-for-creating-a-solution-template-for-the-azure-marketplace"></a>为 Azure 应用商店创建解决方案模板的技术先决条件
在开始之前，请仔细阅读过程，并了解执行每个步骤的位置和原因。 在开始产品/服务创建过程前，应尽可能多地准备公司信息和其他数据、下载必需的工具和/或创建技术组件。  

## <a name="developing-building-blocks"></a>开发构建基块
必须先决定解决方案的构建基块，然后再继续使用发布门户上的解决方案模板。 下面是一些指向文档的链接，可在该过程中起到帮助作用：

1. [设计 Azure Resource Manager 模板的最佳实践](../best-practices-resource-manager-design-templates.md)
2. [创作 Azure Resource Manager 模板](../resource-group-authoring-templates.md)
3. Azure 快速入门模板：
   * [https://azure.microsoft.com/zh-cn/documentation/templates/](https://azure.microsoft.com/documentation/templates/)
   * [https://github.com/azure/azure-quickstart-templates](https://github.com/azure/azure-quickstart-templates)

## <a name="next-steps"></a>后续步骤
现在已查看先决条件并已完成必需的任务，可以继续创建解决方案模板产品/服务，如[创建解决方案模板指南](marketplace-publishing-solution-template-creation.md)中所详述。 若要返回到发布者任务的主页，请参阅[入门：如何将产品/服务发布到 Azure 应用商店](marketplace-publishing-getting-started.md)。
