# <a name="cloudsimple-node-quota-limits"></a>云简单节点配额限制

在为 CloudSimple 服务启用订阅时，四个节点是可供购买的默认数量。  可以从 Azure 门户购买任何[节点类型](cloudsimple-node.md)。  创建私有云至少需要三个同一 SKU 的节点。  如果您购买了节点，则当您尝试购买其他节点时，您可能会看到错误。

## <a name="quota-increase"></a>配额增加

您可以通过提交支持请求来增加节点配额。 服务操作团队将评估请求，并与您合作增加节点配额。  打开新票证时选择以下选项：

* 问题类型：**技术**
* 订阅：**您的订阅 ID**
* 服务类型 **：VMware 解决方案（按云简单）**
* 问题类型：**专用节点配额**
* 问题子类型：**增加专用节点的配额**
* 主题：**配额增加**

在支持票证的详细信息中，提供所需的节点数和节点 SKU。

* 节点 SKU
* 请求增加配额的其他节点数
