# <a name="view-the-list-of-avs-portal-users"></a>查看 AVS 门户用户的列表

用户首次访问 AVS 门户时，会被添加到用户列表中。 若要查看有权通过 Azure 访问 AVS 门户的用户的列表，请[访问 avs 门户](access-cloudsimple-portal.md)，在侧边菜单中选择 "**帐户**"，然后在 AVS 门户中选择 "**用户**"。

* 若要显示用户详细信息（包括 Azure 订阅、租户和用户 Id），请单击 "**用户**" 页上的条目。

* 若要查看用户的活动审核日志，请选择 "**审核日志**" 选项卡。
* 若要锁定或解锁用户帐户，请在显示用户详细信息时单击 "**锁定**" 切换。 当帐户解锁时，用户可以访问 AVS 门户。 帐户锁定后，将阻止对该门户的访问。
