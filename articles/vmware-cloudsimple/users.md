# <a name="view-the-list-of-cloudsimple-portal-users"></a>查看云简单门户用户列表

用户首次访问 CloudSimple 门户时，将添加到用户列表中。 要查看通过 Azure 访问云简单门户的用户列表，[请访问云简单门户](access-cloudsimple-portal.md)，选择侧菜单上的**帐户**，然后在云简单门户中选择 **"用户**"。

* 要显示用户详细信息（包括 Azure 订阅、租户和用户 ID），请单击 **"用户"** 页上的条目。

* 要查看用户的审核活动日志，请选择 **"审核日志"** 选项卡。
* 要锁定或解锁用户帐户，请在显示用户详细信息时单击 **"锁定**切换"。 当帐户解锁时，用户可以访问 CloudSimple 门户。 当帐户被锁定时，对门户的访问将被阻止。
