# <a name="account-management-overview"></a>帐户管理概述

创建 CloudSimple 服务时，它在 CloudSimple 上创建一个帐户。  该帐户是与你的 Azure 订阅相关联的服务所在的位置。  具有的所有用户**所有者**并**参与者**订阅中的角色有权访问 CloudSimple 门户。  上找到的 Azure 订阅 ID 和租户 ID 与 CloudSimple 服务相关联[帐户页](https://docs.azure.cloudsimple.com/account/)。

## <a name="additional-alert-emails"></a>其他警报的电子邮件

可以在 CloudSimple 接收警报配置电子邮件 Id:

* 与你的服务
* 用于自动处理

## <a name="cloudsimple-operator-access"></a>CloudSimple 操作员访问权限

您可以控制对服务操作人员 CloudSimple 门户访问。  服务操作人员登录到门户时提交支持票证。  服务操作将修复报告的任何问题，可在审核日志中所执行的操作。

## <a name="users"></a>用户

所有用户都具有**所有者**并**参与者**订阅中的角色有权访问 CloudSimple 门户。  当访问门户时，CloudSimple 帐户上创建用户。  可以禁用对特定用户在 CloudSimple 门户的帐户页面访问。
