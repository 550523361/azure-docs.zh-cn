# <a name="account-management-overview"></a>帐户管理概述

当您创建云简单服务时，它会在 CloudSimple 上创建一个帐户。  该帐户与服务所在的 Azure 订阅相关联。  订阅中具有**所有者**和**参与者**角色的所有用户都有权访问 CloudSimple 门户。  与 CloudSimple 服务关联的 Azure 订阅 ID 和租户 ID 将在["帐户"页上](account.md)找到。

## <a name="additional-alert-emails"></a>其他警报电子邮件

您可以在云简单中配置电子邮件 ID 以接收警报：

* 与您的服务相关
* 用于自动处理

## <a name="cloudsimple-operator-access"></a>云简单操作员访问

您可以控制对服务操作人员的 CloudSimple 门户的访问。  当您提交支持票证时，服务操作人员登录到门户。  服务操作将修复报告的任何问题，所执行的操作可在审核日志中提供。

## <a name="users"></a>用户

订阅中具有**所有者**和**参与者**角色的所有用户都有权访问 CloudSimple 门户。  访问门户时，用户是在 CloudSimple 帐户上创建的。  您可以在"帐户"页中禁用特定用户的云简单门户访问。
