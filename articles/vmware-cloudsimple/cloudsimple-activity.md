# <a name="activity-management-overview"></a>活动管理概述

CloudSimple 跟踪可能影响私有云环境功能的所有活动。 活动包括警报、事件、任务和审核活动。 ["活动"页面](monitor-activity.md)汇总了所有当前活动，并允许您向下钻取以获取其他详细信息。

## <a name="events"></a>事件

事件跟踪云简单门户上的用户和系统活动。 事件显示与特定资源关联的活动以及影响的严重性。  您可以在云简单门户中查看事件。

## <a name="alerts"></a>警报

警报是云简单环境中任何重要活动的通知。 影响计费或用户访问的事件显示为警报。  您可以确认来自云简单门户的警报。

## <a name="tasks"></a>任务

任务跟踪任何需要 30 秒以上才能完成的用户操作。 您可以通过 CloudSimple 门户监视任务的进度。  对于已完成的任务，该信息包括完成的总时间。

## <a name="audit"></a>审核

审核日志跟踪用户操作。 审核日志包含用户为操作提供的参数。  您可以使用审核日志监视所有用户的用户活动。
