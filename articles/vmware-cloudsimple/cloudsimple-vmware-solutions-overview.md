# <a name="what-is-vmware-solution-on-azure-by-cloudsimple"></a>CloudSimple 通过在 Azure 上的 VMware 解决方案是什么

**CloudSimple 通过在 Azure 上的 VMware 解决方案**是完全托管的服务，可在 Azure 中运行的 VMware 平台。 此解决方案包括 vSphere、 vCenter、 vSAN、 NSX-T 和相应的工具。
VMware 环境本机 Azure 裸机基础结构上运行 Azure 云位置。 该服务包括高效安全地使用 VMware 平台所需的所有功能。

![CloudSimple 概述在 Azure 上的 VMware 解决方案](media/azure-vmware-solution-by-cloudsimple.png)

## <a name="features"></a>功能

* 按需自助服务配置的 VMware 云环境。 添加和删除按需调整容量
* VMware 平台部署、 升级、 管理平面备份、 运行状况/容量监视、 警报、 故障排除，和修正。
* 是启用 VMware，包括 L2/L3 服务和防火墙规则管理所需的网络服务。
* 边缘类型网络服务，包括 VPN、 公共 IP 和 Internet 网关。 这些边缘服务在 Azure 上运行，并执行相应的安全和 Azure 的 DDoS 保护。
* 保留容量，以降低成本。
* 高速、 低延迟连接到 Azure 和本地。
* 解决方案体系结构的客户可以使用 Azure 服务以集成方式，充分利用此唯一的"在公有云中的 VMware 云"体系结构。 这些 Azure 服务包括 Azure AD 中，存储、 应用程序网关和其他人。
* 基础结构完全专用于您和物理上独立于其他客户的基础结构。
* 此类活动管理、 使用情况、 计费/计数和用户管理的管理功能。
* 全天候客户支持。

## <a name="benefits"></a>优点

* **操作连续性：** CloudSimple 提供本机访问 VMware 平台。 CloudSimple 体系结构是与您的现有兼容：
  * 应用程序
  * 操作
  * 网络
  * 安全
  * 备份
  * 灾难恢复
  * 审核
  * 符合性工具
  * 进程
* **不重新训练：** VMware 平台兼容性，可使用现有技能和知识。
* **基础结构灵活性：** 您不再需要预测所有容量需求，并最终会有浪费的容量或基础结构短缺。 CloudSimple 传递作为云服务，并且可以添加或减少容量，在任何时间
* **安全性：** 通过 Azure CloudSimple 环境的访问权限提供内置的 DDoS 保护和安全监视。
* **降低成本：** CloudSimple 平台高度工程，并提供高级别的自动化、 运营效率和规模效益。 此外，CloudSimple 发布能够利用存在 VMware 在公有云来降低成本的解决方案体系结构。 示例包括 Azure AD 中，备份到 Azure 存储中，应用程序网关、 负载均衡器和其他人。
* **一个新的混合平台：** 该服务使高速、 低延迟访问 Azure 的其余部分的权限。 此外，CloudSimple 管理实现了统一的管理的 VMware 虚拟机和 Azure 中使用的相同 UI 和 API 的其余部分。 您的开发团队可以充分利用公共和专用平台集成的、 一致的方式。
* **基础结构监控、 疑难解答和支持：** CloudSimple 作为服务运行基本的基础结构。 失败将自动替换硬件。 而 CloudSimple 确保顺利运行环境，您可以集中精力消耗。
* **策略兼容性：** 保留基于 VMware 的工具、 安全过程、 审核实践和符合性认证。

## <a name="scenarios"></a>方案

* **数据中心停用或迁移：** 当达到限制的现有数据中心或刷新硬件时，获取更多的容量。 很容易，从而在云中，添加所需的容量和消除管理硬件刷新的难题。 降低风险和成本相比耗时转换或重新组织了架构的云迁移。 使用熟悉的 VMware 工具和技术来加速云迁移。 一次在云中使用 Azure 服务来更新您的步调应用程序。
* **按需展开：** 扩展到云，以满足未预料到的需要，如新的开发环境或季节性容量突发情况。 您可以轻松地根据情况下，需要创建新的容量和只要您需要使其保持。 减少您的先期投资，加快速度的预配，并跨本地和云降低具有相同的体系结构和策略的复杂性。
* **灾难恢复和在 Azure 云中的虚拟桌面：** 建立到数据、 应用和在 Azure 云中的桌面的远程访问。 具有高带宽连接时，将上传 / 下载数据快，若要从事件中恢复。 低延迟网络提供快速响应时间，用户期望从桌面应用。 使用 CloudSimple，很容易地复制所有策略和网络在云中使用 CloudSimple 门户和熟悉的 VMware 工具。 工作量和风险的创建和管理灾难恢复和 VDI 实现，极大地减少了恢复和复制的易用性。
* **高性能应用程序和数据库：** CloudSimple 提供设计运行要求最高的 VMware 工作负荷的超聚合体系结构。 运行 Oracle、 Microsoft SQL server、 中间件系统和高性能无 SQL 数据库。 为你自己的高速 25 Gbps 网络连接时，允许您运行跨本地、 在 Azure 和 Azure 的 VMware 的混合应用的数据中心体验云专用工作负荷，而影响性能。
* **真正的混合：** 在 VMware 和 Azure 之间统一 DevOps。 优化 Azure 服务和可以应用跨所有工作负荷的解决方案的 VMware 管理。 访问公有云服务而无需展开你的数据中心或重新构建您的应用程序。 集中管理标识、 访问控制策略，日志记录和监视的 VMware 在 Azure 上的应用程序。
