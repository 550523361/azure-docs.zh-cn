# <a name="what-is-azure-vmware-solution-by-cloudsimple"></a>什么是 Azure VMware 解决方案（按云简单）

**云简单 Azure VMware 解决方案**是一项完全托管的服务，使您能够在 Azure 中运行 VMware 平台。 此解决方案包括 vSphere、vCenter、vSAN、NSX-T 和相应的工具。 VMware 环境在 Azure 云位置的 Azure 裸机基础结构上本机运行。 该服务包括高效、安全地使用 VMware 平台所需的所有功能。

![按云简单概述在 Azure 上的 VMware 解决方案](media/azure-vmware-solution-by-cloudsimple.png)

## <a name="features"></a>功能

* VMware 云环境的按需自助服务调配。 能够按需添加和删除容量。
* VMware 平台部署、升级、管理平面备份、运行状况/容量监控、警报、故障排除和补救。
* 支持 VMware 所需的网络服务，包括 L2/L3 服务和防火墙规则管理。
* 边缘类型的网络服务，包括 VPN、公共 IP 和互联网网关。 这些服务在 Azure 上运行，并带有 Azure 的安全和 DDoS 保护。
* 容量预留以降低成本。
* 高速、低延迟连接到 Azure 和本地。
* 客户以集成方式使用 Azure 服务并充分利用这种独特的"公共云中的 VMware 云"体系结构的解决方案体系结构。 Azure 服务包括 Azure AD、存储、应用程序网关和其他。
* 完全专用且与其他客户的基础架构隔离的基础架构。
* 管理具有活动管理、使用情况、计费/计量和用户管理等功能。
* 24x7 客户支持。

## <a name="benefits"></a>优点

* **操作连续性**。 CloudSimple 提供对 VMware 平台的本机访问。 CloudSimple 架构与您现有的架构兼容：
    * 应用程序
    * 操作
    * 安全性
    * 备份
    * 灾难恢复
    * 审核
    * 合规性工具
    * 进程
* **没有再培训**。 VMware 平台兼容性允许您使用现有的技能和知识。
* **基础设施敏捷性**。 您不再需要预测所有容量需求，最终导致容量浪费或基础设施短缺。 CloudSimple 作为云服务提供，您可以随时添加或减少容量
* **安全性。** 通过 Azure 访问云简单环境可提供内置的 DDoS 保护和安全监视。
* **成本更低**。 CloudSimple 平台经过高度设计，提供高水平的自动化、运营效率和规模经济。 此外，CloudSimple 还发布解决方案体系结构，利用 VMware 在公共云中的存在来降低成本。 示例包括 Azure AD、Azure 存储备份、应用程序网关、负载均衡器等。
* **一个新的混合平台**。 该服务支持对 Azure 其余部分的高速、低延迟访问。 此外，CloudSimple 管理允许使用相同的 UI 和 API 统一管理 VMware 虚拟机和 Azure 的其余部分。 您的开发团队可以以集成、一致的方式利用公共和私有平台。
* **基础结构监视、故障排除和支持**。 CloudSimple 将底层基础架构作为服务进行运营。 自动更换故障硬件。 您可以专注于消费，而 CloudSimple 可确保环境平稳运行。
* **策略兼容性**。 保留基于 VMware 的工具、安全程序、审核实践和合规性认证。

## <a name="scenarios"></a>方案

* **数据中心停用或迁移**。 达到现有数据中心或刷新硬件的限制时，获取其他容量。 您可以轻松地在云中添加所需的容量，并消除管理硬件更新的麻烦。 与耗时的转换或重新架构相比，降低云迁移的风险和成本。 使用熟悉的 VMware 工具和技能加快云迁移。 在云中，使用 Azure 服务以您的速度实现应用程序的现代化。
* **按需扩展**。 扩展到云以满足意外需求，例如新的开发环境或季节性容量突发。 您可以轻松地按需创建新容量，并且仅在需要时保留它。 通过本地和云中的相同体系结构和策略，减少前期投资，加快预配速度，并降低复杂性。
* **Azure 云中的灾难恢复和虚拟桌面**。 在 Azure 云中建立对数据、应用和桌面的远程访问。 使用高带宽连接，您可以快速上传和下载数据以从事件中恢复。 低延迟网络为您提供用户期望从桌面应用快速响应时间。 借助 CloudSimple，使用 CloudSimple 门户和熟悉的 VMware 工具轻松在云中复制所有策略和网络。 易于恢复和复制可大大减少创建和管理 DR 和 VDI 实现的工作和风险。
* **高性能应用程序和数据库**。 CloudSimple 提供了一种超融合架构，旨在运行您最苛刻的 VMware 工作负载。 运行 Oracle、微软 SQL 服务器、中间件系统和高性能无 SQL 数据库。 通过高速 25 Gbps 网络连接将云体验为您自己的数据中心，从而在不影响性能的情况下运行跨本地、Azure 上的 VMware 和 Azure 专用工作负荷的混合应用。
* **真正的混合**。 跨 VMware 和 Azure 统一 DevOps。 针对可应用于所有工作负荷的 Azure 服务和解决方案优化 VMware 管理。 访问公共云服务，而无需扩展数据中心或重新构建应用程序。 集中 Azure 上 VMware 应用程序的标识、访问控制策略、日志记录和监视。
