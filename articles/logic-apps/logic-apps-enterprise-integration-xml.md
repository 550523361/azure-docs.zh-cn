# <a name="xml-processing"></a>XML 处理
通过 Enterprise Integration Pack 可以方便地验证和处理与业务合作伙伴交换的 XML 文档。 以下是使用逻辑应用处理这些 XML 消息的方式：  

* [XML 验证](logic-apps-enterprise-integration-xml-validation.md "了解 XML 消息验证") - 通过 XML 验证可以针对特定架构验证源自源终结点的消息。
* [XML 转换](../logic-apps/logic-apps-enterprise-integration-transform.md "了解 XML 消息转换和映射") - 通过 XML 转换可以基于目标终结点的要求转换 XML 消息。   
* [平面文件编码和平面文件解码](logic-apps-enterprise-integration-flatfile.md "了解平面文件编码/解码") - 通过平面文件编码/解码可以对平面文件进行编码或解码。 
* [XPath](https://msdn.microsoft.com/library/mt643789.aspx) - 可用于扩充消息以及从消息中提取特定属性。 提取的属性随后可以用于将消息路由到目标或中间终结点。
