# <a name="learn-about-agreements-and-enterprise-integration-pack"></a>了解协议和 Enterprise Integration Pack
## <a name="overview"></a>概述
协议是企业对企业 (B2B) 通信的基石，允许企业实体使用行业标准协议进行无缝通信。  

## <a name="what-is-an-agreement"></a>什么是协议？
就 Enterprise Integration Pack 而言，协议是 B2B 贸易合作伙伴之间的通信约定。 协议基于合作伙伴希望实现的通信，是特定的协议或传输。

企业集成支持三种协议/传输标准：  

* [AS2](logic-apps-enterprise-integration-as2.md)
* [X12](logic-apps-enterprise-integration-x12.md)
* [EDIFACT](logic-apps-enterprise-integration-edifact.md)

## <a name="why-use-agreements"></a>为什么要使用协议
使用协议的一些常见好处包括：

* 使不同的组织和企业能够以熟知的格式交换信息。  
* 执行 B2B 业务时可提高效率  
* 创建企业集成应用时可轻松地创建、管理和使用它们  

## <a name="how-to-create-agreements"></a>如何创建协议
* [创建 AS2 协议](logic-apps-enterprise-integration-as2.md)   
* [创建 X12 协议](logic-apps-enterprise-integration-x12.md)   
* [创建 EDIFACT 协议](logic-apps-enterprise-integration-edifact.md)  

## <a name="how-to-use-an-agreement"></a>如何使用协议
创建协议后，可通过 Azure 门户使用协议，以便使用 B2B 功能创建[逻辑应用](logic-apps-what-are-logic-apps.md "了解逻辑应用")。

## <a name="how-to-edit-an-agreement"></a>如何编辑协议
可以按照以下步骤编辑任何协议：  

1. 选择包含想要修改的协议的集成帐户。  
2. 选择“协议”磁贴  
3. 在“协议”边栏选项卡上选择想要修改的协议  
4. 从上述菜单中选择“编辑”。   
5. 在打开的“编辑”菜单中进行一些更改，然后选择“确定”按钮以保存更改  

## <a name="how-to-delete-an-agreement"></a>如何删除协议
可以按照以下步骤从包含想要删除的协议的集成帐户中删除任何协议：   

1. 选择“协议”磁贴  
2. 在“协议”边栏选项卡上选择想要删除的协议  
3. 从上述菜单中选择“删除”。  
4. 确认确实要删除该协议  
5. 请注意，“协议”边栏选项卡中不再列出该协议
