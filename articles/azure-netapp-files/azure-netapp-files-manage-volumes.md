# <a name="manage-volumes-by-using-azure-netapp-files"></a>使用 Azure NetApp 文件管理卷
可以通过查看、更新或删除卷来管理卷。 

如果要删除卷，必须先删除卷的所有子对象。 

1.  从“管理容量池”边栏选项卡中单击“卷”边栏选项卡。**** 
2.  使用“浏览卷”**** 视图来查看、更新或删除现有卷。
