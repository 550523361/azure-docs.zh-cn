# <a name="introduction-to-azure-netapp-files"></a>Azure NetApp 文件简介
Azure NetApp 文件服务是一种企业级高性能计量式文件存储服务。 Azure NetApp 文件支持任何工作负荷类型，并且默认具有高可用性。 你可以选择有保证的服务和性能级别，并通过该服务设置快照。 

Azure NetApp 文件文档提供了有关使用 Azure NetApp 文件创建和管理卷的说明。 

> [!IMPORTANT] 
> 本文档中所述的所有任务都需要你已加入公共预览版计划且已已列入允许列表，以便访问 Microsoft.NetApp Azure 资源提供程序。 若要请求加入“公共预览版”计划，请参阅[注册 Azure NetApp 文件](azure-netapp-files-register.md)。
