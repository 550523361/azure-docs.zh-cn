# <a name="azure-netapp-files-documentation"></a>Azure NetApp 文件文档

了解如何使用 Azure NetApp 文件来创建和管理 NetApp 帐户、容量池和卷，如何配置导出策略，如何为虚拟机装载卷，以及如何管理快照。

> [!IMPORTANT] 
> 本文档中所述的所有任务都需要你已加入公共预览版计划且已已列入允许列表，以便访问 Microsoft.NetApp Azure 资源提供程序。 有关加入公共预览版计划的详细信息，请参阅 [Azure NetApp 文件公共预览版注册页](https://aka.ms/nfspublicpreview)。
