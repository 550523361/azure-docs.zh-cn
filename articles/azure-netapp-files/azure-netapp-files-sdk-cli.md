下表列出了支持的 Sdk。  可在 GitHub 中的已发布位置找到有关受支持的 Sdk 的详细信息。

|------------------|--------------------------------------------------------------|
|    .NET  |    [Azure/azure-适用于网络的 sdk](https://github.com/Azure/azure-sdk-for-net/tree/master/sdk/netapp)    |
