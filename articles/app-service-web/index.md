# <a name="azure-web-apps-documentation"></a>Azure Web 应用文档

应用服务 Web 应用是一个完全托管的计算平台，非常适合用来托管网站和 Web 应用程序。 使用 Microsoft Azure 提供的这个平台即服务 (PaaS) 产品，可以在 Azure 维护用于运行和扩展应用的基础结构时重点关注业务逻辑。
