# <a name="fedramp-overview"></a>FedRAMP 概述

[联邦风险与授权管理计划 (FedRAMP)](https://www.fedramp.gov) 是美国政府范围的计划，它提供一种标准化方法来对云产品和服务进行安全评估、授权和持续监视。 FedRAMP 基于一组从[美国国家标准与技术研究院 (NIST) 特殊发布 SP 800-53](https://csrc.nist.gov/publications/detail/sp/800-53/rev-4/final) 安全控制派生的标准化要求定义安全评估过程。

浏览下面的控件系列，了解此 Azure 安全性和合规性蓝图自动化如何与控制要求保持一致。
