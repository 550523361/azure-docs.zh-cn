# <a name="personnel-security-ps"></a>人员安全 (PS)

> [!NOTE]
> 本文档所述的控制措施由美国国家标准技术研究所 (NIST) 和美国商务部定义，作为 NIST 特别出版物 800-53 修订版 4 的一部分。 请参阅 NIST 800-53 修订版 4，了解每项控制措施的测试过程和指南。

## <a name="nist-800-53-control-ps-1"></a>NIST 800-53 控制措施 PS-1

#### <a name="personnel-security-policy-and-procedures"></a>人员安全政策和程序

**PS-1** 组织制定、记录并向 [赋值：组织定义的人员或角色] 分发一项人员安全政策，该政策提出目标、范围、角色、职责、管理承诺、组织实体间的协调和符合性；设定用于实施人员安全政策及相关的人员安全控制的程序；按 [赋值：组织定义的频率] 评审并更新当前人员安全政策并按 [赋值：组织定义的频率] 评审并更新人员安全程序。

**责任：**`Customer Only`

|||
|---|---|
| **客户** | 客户的企业级人员安全政策和程序可足以满足这项控制要求。 |
| **提供商 (Microsoft Azure)** | 不适用 |
