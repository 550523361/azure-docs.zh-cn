# <a name="azure-security-best-practices-and-patterns"></a>Azure 安全最佳实践和模式
我们当前提供了以下 Azure 安全最佳实践和模式文章。 请务必定期访问此网站，我们的 Azure 安全最佳实践和模式列表会持续更新：  

* [Azure 网络安全最佳实践](azure-security-network-security-best-practices.md)
* [Azure 数据安全与加密最佳实践](azure-security-data-encryption-best-practices.md)
* [标识管理和访问控制安全最佳实践](azure-security-identity-management-best-practices.md)
* [物联网安全最佳实践](azure-security-iot-best-practices.md)
* [Azure IaaS 安全最佳实践] (azure-security-iaas.md)
* [有关 Microsoft Azure IaaS 上软件更新的最佳实践](azure-security-best-practices-software-updates-iaas.md)
* [Azure 边界安全最佳实践](../best-practices-network-security.md)
* [在 Azure 中实现安全的混合网络体系结构](../guidance/guidance-iaas-ra-secure-vnet-hybrid.md)
* [Azure PaaS 最佳实践] (https://docs.microsoft.com/en-us/azure/security/security-paas-deployments)

Azure 提供了一个安全的平台，你可以基于其构建你的解决方案。 我们还提供了一些服务和技术，使你的解决方案在 Azure 上更加安全。 由于有许多选项可供使用，许多人对 Microsoft 推荐的用于提高安全性的最佳实践和模式表示了兴趣。

我们理解你的兴趣，并已创建一个文档集（其中描述了你可以做的事情），给出了正确的上下文，以提高 Azure 部署的安全性。

在这些最佳实践和模式文章中，我们讨论了一个针对特定主题的最佳实践和有用模式的集合。 这些最佳实践和模式衍生自我们在这些技术上的经验和客户经验。

对于每项最佳实践，我们将尽可能进行仔细的说明：

* 最佳实践是什么
* 为何要启用该最佳实践
* 如果你无法启用该最佳实践，可能的结果是什么
* 最佳实践的可能替代方案
* 如何学习启用最佳实践

我们期待能提供更多有关 Azure 安全体系结构和最佳实践的文章。 如果你希望我们提供什么主题，请在本页面底部的讨论区提出来。
