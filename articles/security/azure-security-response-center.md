# <a name="microsoft-security-response-center"></a>Microsoft 安全响应中心
Microsoft 安全响应中心 (MSRC) 由全球一些最有经验的安全专家主持。 这些专家每天每时识别、监视、响应和解决安全事件以及本地和云的漏洞。

除了 MSRC 在后台的连续工作，MSRC 团队还拥有大量的可用资源，以便用户可以了解如何更有效地保护自己的 Azure 资产和部署。

## <a name="the-msrc-blog"></a>MSRC 博客
可以从 [MSRC 博客](https://blogs.technet.microsoft.com/msrc/)获取有关 MSRC 执行何种操作来帮助保护用户免遭云威胁的最新新闻。

## <a name="white-papers"></a>白皮书
MSRC 已发布了大量[白皮书](https://technet.microsoft.com/library/bb969102.aspx)以帮助用户了解他们所做的工作及其工作方式。 一些白皮书深入介绍我们如何保护 Microsoft 云的安全，并包括有关用户如何采用相同的安全配置的有用信息。

## <a name="security-researcher-engagement-and-bounty-programs"></a>安全研究人员参与和奖励计划
MSRC 支持与全球安全研究人员的协作和合作以提高 Microsoft 产品的安全性。

Microsoft 奖励计划对研究新型漏洞技术、减少新型漏洞的防范理念和识别 Microsoft 本地和云的软件中的严重漏洞的研究人员给予奖励。

若要了解有关这些计划的详细信息，请访问 [MSRC Bug 奖励](https://technet.microsoft.com/security/dn425036)页和 [MSRC 博客](https://blogs.technet.microsoft.com/msrc/)。

若要了解有关 MSRC 的详细信息，请访问 [MSRC 主页](https://technet.microsoft.com/library/dn440717.aspx)。
