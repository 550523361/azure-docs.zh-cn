# <a name="azure-security-mvp-program"></a>Azure 安全 MVP 计划
Microsoft 最有价值人员 (MVP) 是社区领导者，他们是帮助其他人充分利用他们在 Microsoft 技术上的经验的最佳典范。 他们和社区及 Microsoft 分享了他们的热情、实用知识以及专业技术。

Microsoft Azure 现已任命在 Azure 安全方面具有特殊专业知识的社区专家。 Microsoft MVP 可被授予 Microsoft Azure 中 Azure 安全贡献领域的 MVP。

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

是否能成为 MVP 并没有任何评判标准，部分原因在于相关标准因技术及其生命周期而异，其中一些标准包括被提名人对在线论坛（例如 Microsoft Answers、TechNet 和 MSDN；Wiki 和在线内容；会议和用户组；播客；网站；博客和社交媒体；以及文章与书籍）的贡献所产生的影响。

是否是 Azure 安全方面的专家？ 你知道谁是吗？ 立即[提名自己或他人](https://mvp.microsoft.com/Nomination/nominate-an-mvp)成为 Azure 安全 MVP！
