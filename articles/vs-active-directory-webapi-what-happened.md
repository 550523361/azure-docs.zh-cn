#####代码文件已添加到您的项目 

身份验证启动类 **App_Start/Startup.Auth.cs**（包含 Azure AD 身份验证的启动逻辑）已添加到你的项目。 

#####启动代码已添加到您的项目 

如果你的项目中已经有一个 Startup 类，**Configuration** 方法将进行更新，以包括对添加到该方法的 ConfigureAuth(app) 的调用。否则，Startup 类已添加到您的项目。 


#####您的 app.config 或 web.config 文件具有新的配置值。

已添加以下配置条目。 
	<pre>
	`<appSettings> 
    		<add key="ida:ClientId" value="ClientId from the new Azure AD App" /> 
    		<add key="ida:Tenant" value="Your selected Azure AD Tenant" /> 
    		<add key="ida:Audience" value="The App ID Uri from the wizard" /> 
	</appSettings>` </pre>

###已创建 Azure AD 应用 

已在您在向导中选定的目录内创建一个 Azure AD 应用程序。
