# <a name="what-is-the-azure-database-migration-service-preview"></a>什么是 Azure 数据库迁移服务预览版？
Azure 数据库迁移服务是一项完全托管的服务，旨在实现从多个数据库源到 Azure 数据平台的无缝迁移，并且最小化停机时间。 该服务目前在公共预览版中提供，开发工作的重点为：

- 可靠性和性能。
- 源目标对的迭代添加。
- 无摩擦迁移中的持续投资。

## <a name="use-familiar-tools"></a>使用熟悉的工具
Azure 数据库迁移服务集成了一些现有工具和服务的功能。  它为客户提供高度可用的综合解决方案。 Azure 数据库迁移服务使用[数据迁移助手](http://aka.ms/dma)提供评估报表。  这些建议将指导你完成执行迁移之前所需的更改。 你可以自己选择执行所需的修正。 准备好开始迁移过程时，Azure 数据库迁移服务会执行所有相关步骤。 该过程利用了 Microsoft 确定的最佳做法，因此你开始迁移后便可高枕无忧。
