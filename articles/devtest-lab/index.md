#<a name="devtest-labs-documentation"></a>开发测试实验室文档

Azure 开发测试实验室是一项可帮助开发人员和测试人员在 Azure 中快速创建环境，同时尽量减少浪费并控制成本的服务。  可使用可重用模板和项目通过快速预配 Windows 和 Linux 环境来测试应用程序的最新版本。
