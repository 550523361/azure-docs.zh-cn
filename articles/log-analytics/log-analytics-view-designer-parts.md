# <a name="log-analytics-view-designer-visualization-part-reference"></a>Log Analytics 视图设计器可视化部分参考
Log Analytics 中的视图设计器允许你在 OMS 控制台中创建自定义视图，以包含 OMS 存储库中数据的多种可视化效果。 本文针对可在自定义视图中使用的每个可视化部分提供了设置参考。

可用于视图设计器的其他文章：

* [视图设计器](log-analytics-view-designer.md) - 概要介绍视图设计器和用于创建和编辑自定义视图的过程。
* [磁贴参考](log-analytics-view-designer-tiles.md) - 可在自定义视图中使用的每个磁贴的设置参考。 

下表描述视图设计器中可用的不同类型的磁贴。  以下各节详细描述每个磁贴类型及其属性。

| 视图类型 | 说明 |
|:--- |:--- |
| [查询列表](#list-of-queries-part) |显示日志搜索查询的列表。  用户可以单击每个查询以显示其结果。 |
| [编号和列表](#number-amp-list-part) |标头具有单个数字，用于表示来自日志搜索查询中的记录数。  列表显示查询的前十个结果，其中的图形指示数值列的相关值或其随时间的变化。 |
| [两个数字和列表](#two-numbers-amp-list-part) |标头具有两个数字，用于表示来自单独日志搜索查询中的记录数。  列表显示查询的前十个结果，其中的图形指示数值列的相关值或其随时间的变化。 |
| [圆环图和列表](#donut-amp-list-part) |标头显示从日志查询中的值列汇总的单个数字。  此圆环图以图形方式显示了前三个记录的结果。 |
| [两个时间线和列表](#two-timelines-amp-list-part) |标头将一段时间内两个日志查询的结果显示为带有标注的柱形图，以显示从日志查询中的值列汇总的单个数字。  列表显示查询的前十个结果，其中的图形指示数值列的相关值或其随时间的变化。 |
| [信息](#information-part) |标头显示静态文本和可选的链接。  列表显示带有静态文本和标题的一个或多个项。 |
| [折线图、标注和列表](#line-chart-callout-amp-list-part) |标头显示一段时间内来自一个日志查询中的具有多个系列的折线图和具有汇总值的标注。  列表显示查询的前十个结果，其中的图形指示数值列的相关值或其随时间的变化。 |
| [折线图和列表](#line-chart-amp-list-part) |标头显示一段时间内来自一个日志查询中的具有多个系列的折线图。  列表显示查询的前十个结果，其中的图形指示数值列的相关值或其随时间的变化。 |
| [折线图的堆栈部分](#stack-of-line-charts-part) |显示一段时间内来自一个日志查询中的具有多个系列的三个独立折线图。 |

## <a name="list-of-queries-part"></a>查询列表部分
显示日志搜索查询的列表。  用户可以单击每个查询以显示其结果。  默认情况下，视图将包括单个查询，你可以单击“**+ 查询**”以添加其他查询。

![查询列表视图](media/log-analytics-view-designer/view-list-queries.png)

| 设置 | 说明 |
|:--- |:--- |
| **常规** | |
| 标题 |在视图顶部显示的文本。 |
| 新建组 |选择此项可在以当前视图开始的视图中创建新组。 |
| 预选择的筛选器 |当用户选择查询时左侧筛选器窗格中包含的以逗号分隔的属性列表。 |
| 呈现模式 |当选择查询时显示的初始视图。  打开查询后，用户可以选择任意可用视图。 |
| **查询** | |
| 搜索查询 |要运行的查询。 |
| 友好名称 |要向用户显示的查询的描述性名称。 |

## <a name="number--list-part"></a>编号和列表部分
标头具有单个数字，用于表示来自日志搜索查询中的记录数。  列表显示查询的前十个结果，其中的图形指示数值列的相关值或其随时间的变化。

![查询列表视图](media/log-analytics-view-designer/view-number-list.png)

| 设置 | 说明 |
|:--- |:--- |
| **常规** | |
| 组标题 |在视图顶部显示的文本。 |
| 新建组 |选择此项可在以当前视图开始的视图中创建新组。 |
| 图标 |标头中的结果旁边显示的图像文件。 |
| 使用图标 |选择此项可使图标显示。 |
| **标题** | |
| 图例 |在标头顶部显示的文本。 |
| 查询 |针对标头运行的查询。  将显示由查询返回的记录数的计数。 |
| **列表** | |
| 查询 |针对列表运行的查询。  将显示结果中前十个记录的前两个属性。  第一个属性应为文本值，第二个属性应为数值。  根据数值列的相关值自动创建条形图。<br><br>在查询中使用 Sort 命令可对列表中的记录进行排序。  用户可以单击“全部查看”以运行查询并返回所有记录。 |
| 隐藏图形 |选择此项可禁用数值列右侧的图形。 |
| 启用迷你图 |选择此项可显示迷你图而不是水平条。  有关详细信息，请参阅[通用设置](#sparklines)。 |
| 颜色 |水平条或迷你图的颜色。 |
| 名称和值分隔符 |如果想要将文本属性分析为多个值，则为单字符分隔符。  有关详细信息，请参阅[通用设置](#name-value-separator)。 |
| 导航查询 |当用户在列表中选择某个项时运行的查询。  有关详细信息，请参阅[通用设置](#navigation-query)。 |
| **列表** |**> 列标题** |
| Name |在列表第一列的顶部显示的文本。 |
| 值 |在列表第二列的顶部显示的文本。 |
| **列表** |**> 阈值** |
| 启用阈值 |选择此项可启用阈值。  有关详细信息，请参阅[通用设置](#thresholds)。 |

## <a name="common-settings"></a>通用设置
以下部分介绍了对几个可视化部件通用的设置。

### <a name="a-namename-value-separatorname--value-separatora"></a><a name="name-value-separator">名称和值分隔符</a>
如果要将列表查询中的文本属性分析为多个值，则使用单字符分隔符。  如果指定了分隔符，你可以在“名称”框中提供以该分隔符分隔的每个字段的名称。

例如，考虑一个名为 Location 的属性，该属性包括 Redmond-Building 41 和 Bellevue-Building12 等值。  可以指定“–”作为“名称和值分隔符”，“*City-Building*”作为“名称”。  这会将每个值分析为两个称作“*City*”和“*Building*”的属性。 

### <a name="a-namenavigation-querynavigation-querya"></a><a name="navigation-query">导航查询</a>
当用户在列表中选择某个项时运行的查询。  使用 *{选定的项}* 以包括用户选择的项的语法。

例如，如果查询具有一个名为“*计算机*”的列且导航查询为 *{选定的项}*，则当用户选择计算机时将运行“*Computer="MyComputer"*”之类的查询。  如果导航查询为“*Type=Event {选定的项}*”，则将运行查询“*Type=Event Computer="MyComputer"*”。

### <a name="a-namesparklinessparklinesa"></a><a name="sparklines">迷你图</a>
迷你图是一种小型折线图，表明一时间段内列表项的值。  对于具有列表的可视化部件，可选择是显示水平条（指示数值列的相关值）还是迷你图（指示其在一时间段内的值）。

下表描述了迷你图的设置。

| 设置 | 说明 |
|:--- |:--- |
| 启用迷你图 |选择此项可显示迷你图而不是水平条。 |
| 操作 |如果已启用迷你图，则对列表中每个属性执行操作以计算迷你图的值。<br><br>- 最后一个采样：时间间隔内序列的最后一个值。<br>- 最大值：时间间隔内序列的最大值。<br>- 最小值：时间间隔内序列的最小值。<br>- 总和：时间间隔内序列值的总和。<br>- 摘要：使用相同的度量命令作为标头中的查询。 |

### <a name="a-namethresholdsthresholdsa"></a><a name="thresholds">阈值</a>
阈值允许在列表中的每个项旁显示彩色图标，对超出特定值或处于特定范围内的项提供快速的视觉指示。  例如，对于值可接受的项，显示绿色图标；如果值处于指示警告的范围内，则显示黄色；如果超出错误值，则显示红色。

如果为部件启用阈值，则必须指定一个或多个阈值。  如果某项的值大于阈值并小于下一个阈值，则使用该颜色。  如果某项大于当时的最高阈值，则设置该颜色。   

每个阈值集都具有一个带**默认**值的阈值。  如果没有超过其他值，这就是颜色集。  可通过单击 **+** 或 **x** 按钮添加或删除阈值。

下表描述了阈值的设置。

| 设置 | 说明 |
|:--- |:--- |
| 启用阈值 |选择此项可在每个值的左侧显示颜色图标，指示其相对于指定阈值的运行状况。 |
| Name |用于确定阈值的名称。 |
| 阈值 |阈值的数值。  每个列表项的运行状况颜色设置为项目值超出的最高阈值的颜色。  如果没有超出阈值，则采用默认阈值的颜色。 |
| 颜色 |阈值的颜色。 |
