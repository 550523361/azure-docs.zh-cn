# 测试 Service Fabric 服务
可测试性是一套专为测试建立于 Microsoft Azure Service Fabric 基础之上的服务而设计的工具。这些工具让你能够轻松地引发有意义的故障及运行测试方案来执行和验证你的服务在整个生命周期内要经历的大量不同的状态和转换，所有一切都以受控且安全的方式进行。若要了解更多信息，请参阅：

- [可测试性概述](/documentation/articles/service-fabric-testability-overview) 
- [运行一个操作](/documentation/articles/service-fabric-testability-actions)  
- [运行一个方案](/documentation/articles/service-fabric-testability-scenarios) 
- 如何为你的服务编写测试

    - [服务到服务通信失败](/documentation/articles/service-fabric-testability-scenarios-service-communication)
