# <a name="sfctl-mesh"></a>sfctl mesh
删除和管理 Service Fabric 网格应用程序。

## <a name="subgroups"></a>子组
|子组|描述|
| --- | --- |
| [应用程序](service-fabric-sfctl-mesh-app.md) | 获取和删除应用程序资源。 |
| [code-package-log](service-fabric-sfctl-mesh-code-package-log.md) | 获取给定服务副本的指定代码包的容器日志。 |
| [部署](service-fabric-sfctl-mesh-deployment.md) | 创建 Service Fabric 网格资源。 |
| [gateway](service-fabric-sfctl-mesh-gateway.md) | 获取和删除网格网关资源。 |
| [网络](service-fabric-sfctl-mesh-network.md) | 获取和删除网格网络资源。 |
| [秘密](service-fabric-sfctl-mesh-secret.md) | 获取和删除网格机密资源。 |
| [秘密价值](service-fabric-sfctl-mesh-secretvalue.md) | 获取和删除网格机密值资源。 |
| [服务](service-fabric-sfctl-mesh-service.md) | 获取服务详细信息并列出应用程序资源的服务。 |
| [service-replica](service-fabric-sfctl-mesh-service-replica.md) | 获取副本详细信息并列出应用程序资源中给定服务的副本。 |
| [体积](service-fabric-sfctl-mesh-volume.md) | 获取和删除卷资源。 |
