# <a name="jenkins-plugins-for-azure"></a>适用于 Azure 的 Jenkins 插件

以下的 Jenkins 插件支持与 Azure 配合使用的各种功能。
  
| Jenkins 插件 | 描述                                   |
|------------------------------------------------------------------------------|
| [Azure 应用服务插件](https://plugins.jenkins.io/azure-app-service)     | 若要部署 Azure 应用服务 Jenkins 插件 （目前支持仅 Web 应用）。 | 
| [Azure AD 插件](https://plugins.jenkins.io/azure-ad)                       | 支持身份验证和授权通过 Azure Active Directory Jenkins 插件。 | 
| [Azure 容器代理插件](https://plugins.jenkins.io/azure-container-agents) | Azure 容器代理插件可帮助你在 Jenkins 中代理以运行容器 | 
| [Azure Container Service plugin](https://plugins.jenkins.io/azure-acs)（Azure 容器服务插件）       | 若要部署到 Azure 容器服务 (AKS) 的配置 Jenkins 插件。 | 
| [Azure 凭据插件](https://plugins.jenkins.io/azure-credentials)      | 若要管理的 Azure 凭据的 Jenkins 插件。 | 
| [Azure Function 插件](https://plugins.jenkins.io/azure-function)           | 若要使用此插件将部署到 Azure 函数，您首先需要 Jenkins 实例中有一个 Azure 服务主体。 | 
| [Azure Service Fabric 插件](https://plugins.jenkins.io/service-fabric)     | 适用于 Linux Azure Service Fabric 项目的 Jenkins 插件。 |
| [Azure 存储插件](https://plugins.jenkins.io/windows-azure-storage)     | 上传的一个插件，生成项目，或下载生成依赖项从 Microsoft Azure Blob 存储。 | 
| [Azure VM 代理插件](https://plugins.jenkins.io/azure-vm-agents)         | 若要在 Azure 虚拟机上 （通过 Azure 资源管理器模板） 中创建 Jenkins 代理 Jenkins 插件。 | 
| [Azure 虚拟机规模集插件](https://plugins.jenkins.io/azure-vmss)           | 若要将 VM 映像部署到 Azure 虚拟机规模的 Jenkins 插件设置。 |
