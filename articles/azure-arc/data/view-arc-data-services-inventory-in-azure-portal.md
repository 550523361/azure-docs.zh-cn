# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>查看 Azure 门户中的实例清单

上传 [指标、日志](upload-metrics-and-logs-to-azure-monitor.md)或 [使用情况](view-billing-data-in-azure.md)后，可以从 Azure 门户将实例作为 Azure 资源查看。 为此，请导航到特殊 URL [https://portal.azure.com](https://portal.azure.com) ，并按以下屏幕截图操作

1. 中转到所有服务
1. 搜索数据库实例类型
1. 将其添加到你的收藏夹
1. 单击左侧导航栏中的 "实例类型"
1. 查看与其他 Azure SQL 或 Azure PostgreSQL 超大规模资源相同的视图中的实例 (使用筛选器获取精细查看)
