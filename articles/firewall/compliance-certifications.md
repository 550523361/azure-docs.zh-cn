## <a name="icsa-labs-corporate-firewall-certification"></a>ICSA 实验室企业防火墙认证

![ICSA 认证](media/overview/icsa-cert-firewall-small.png)

ICSA Labs 是安全和健康 IT 产品以及网络连接设备的第三方测试和认证的领先供应商。 它们衡量大多数世界顶级技术供应商的产品合规性、可靠性和性能。

Azure 防火墙是第一个获得 ICSA 实验室企业防火墙认证的云防火墙服务。 有关 Azure 防火墙认证报告，请参阅[ICSA 实验室认证测试和审核报告](https://aka.ms/ICSALabsCertification)。 有关详细信息，请参阅[ICSA 实验室防火墙认证计划](https://www.icsalabs.com/technology-program/firewalls)页面。
