# <a name="azure-firewall-compliance-certifications"></a>Azure 防火墙符合性认证

Azure 防火墙是支付卡行业（PCI）、服务组织控制（SOC）、国际标准化组织（ISO）和 HITRUST 兼容。

以下认证适用于全球 Azure 和 Azure 政府。

以下 Azure 防火墙认证适用于全球 Azure：

- 23 NYCRR 500
- AFM 和 DNB （荷兰）
- AMF 和 ACPR （法国）

- 阿根廷 PDPA
- 澳大利亚 IRAP
- CDSA
- CFTC 1.31
- CSA STAR 证明

- 加拿大隐私法律
- DPP （英国）
- 欧盟 ENISA IAF
- 欧盟标准条款
- 欧洲银行机构
- FCA 和 PRA （英国）
- FERPA（美国）
- FFIEC （US）
- FINMA（瑞士）
- FSA（丹麦）
- GLBA（美国）
- 德国 C5
- GxP（FDA 21 CFR 第 11 部分）
- HITRUST
- ISO 20000-1:2011
- ISO 22301:2012
- ISO 27001:2013
- ISO 27017:2015
- ISO 27018:2014
- ISO 9001:2015
- 日本我的数字 Act
- K-ISM
- KNF （波兰）
- MA 和 ABS （新加坡）
- MPAA （US）
- NBB 和 FSMA （比利时）

- NHS IG 工具包（英国）
- 荷兰 BIR 2012
- OSFI （加拿大）
- PCI DSS 1 级
- POWE 和 IRDAI （印度）
