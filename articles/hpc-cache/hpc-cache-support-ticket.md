# <a name="open-a-support-ticket-for-azure-hpc-cache"></a>打开 Azure HPC 缓存的支持票证

使用 Azure 门户打开支持票证。 导航到缓存实例，然后单击侧边栏底部显示**的"新建支持请求"** 链接。

要在没有活动缓存时打开票证，请使用 Azure 门户中的主要帮助和支持页。 从屏幕左上角的控件打开门户菜单，然后滚动到底部，然后单击 **"帮助 + 支持**"。

选择 **"新建支持请求****"，然后选择"技术"** 以获得特定于 Azure HPC 缓存的帮助。

从列表中选择订阅。

要查找 Azure HPC 缓存服务，请单击 **"所有服务**"按钮并搜索 HPC。

![支持请求的屏幕截图 - 基础知识选项卡，部分填写，如所述](media/hpc-cache-support-request.png)

使用您的信息和首选项填写其余字段，然后在准备就绪时提交票证。

提交请求后，您将收到一封包含机票号码的确认电子邮件。 支持人员将联系您了解该请求。
