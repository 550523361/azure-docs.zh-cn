# <a name="azure-maps-routing-coverage"></a>Azure Maps 路由覆盖区域

本文提供了有关 Azure Maps 路由的覆盖区域信息。 搜索查询时，Azure 地图返回从位置 A 到位置 B 的最佳路由。您将获得准确的旅行时间、旅行信息的实时更新和路线说明。 您还可以添加其他搜索参数，如当前交通、车辆类型和避免的条件。 路线的优化取决于区域。 这是因为，Azure 地图具有不同区域的不同级别信息和准确性。 下表列出了区域以及您可以请求哪些类型的信息。
 
查看[**地理编码**](geocoding-coverage.md)的覆盖区域。       
查看[**流量**](traffic-coverage.md)的覆盖区域。  
查看[呈现](render-coverage.md)的覆盖区域****。

|--------------------|----------------------------------------|

| Ø                  | 国家/地区提供简化数据。 |
| 缺少国家/地区 | 未提供国家/地区数据。          |
