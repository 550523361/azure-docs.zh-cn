# <a name="provide-data-feedback-to-azure-maps"></a>向 Azure 地图提供数据反馈

Azure 地图自 2018 年 5 月起提供。 Azure 地图一直提供新的地图数据、易于使用的 REST API 和强大的 SDK，以便为企业提供不同类型的业务用例。 现实世界每时每一秒都在变化，我们为客户提供真实的数字表示至关重要。 计划开放或关闭设施的客户需要我们的地图及时更新。 因此，他们可以在正确的设施内高效地规划交付、维护或客户服务。 我们创建了 Azure 地图数据反馈网站，使我们的客户能够提供直接的数据反馈。 客户的数据反馈直接反馈给我们的数据提供商及其地图编辑器。 他们可以快速评估和将反馈纳入我们的地图产品。  

[Azure 地图数据反馈站点](https://feedback.azuremaps.com)为我们的客户提供了一种简单的方法来提供地图数据反馈，尤其是有关业务兴趣点和住宅地址。 本文将指导您如何使用 Azure 地图反馈网站提供不同类型的反馈。

## <a name="add-a-business-place-or-a-residential-address"></a>添加营业场所或住宅地址 

您可能需要提供有关缺少兴趣点或住宅地址的反馈。 可通过两种方法来执行此操作。 打开 Azure 地图数据反馈站点，搜索缺失位置的坐标，然后单击"添加位置"

  ![搜索缺失位置](./media/how-to-use-feedback-tool/search-poi.png)

或者，您可以与地图进行交互。 单击该位置在坐标处放置一个图钉，然后单击"添加位置"。

  ![添加引脚](./media/how-to-use-feedback-tool/add-poi.png)

单击后，您将被引导到表单中，以提供地点的相应详细信息。

  ![添加位置](./media/how-to-use-feedback-tool/add-a-place.png)

## <a name="fix-a-business-place-or-a-residential-address"></a>修复营业场所或住宅地址 

反馈网站还允许您搜索和定位营业场所或地址。 如果地址或引脚位置不正确，则可以提供反馈来修复它们。 要提供修复地址的反馈，请使用搜索栏搜索营业地或住宅地址。 从结果列表中单击您感兴趣的位置。 单击"修复此位置"。

  ![要修复的搜索位置](./media/how-to-use-feedback-tool/fix-place.png)

要提供反馈以修复地址，请填写"修复地点"表单，然后单击"提交"按钮。

  ![修复窗体](./media/how-to-use-feedback-tool/fix-form.png)

如果位置的引脚位置错误，请检查"修复位置"窗体上的复选框，该复选框显示"引脚位置不正确"。 将图钉移到正确的位置，然后单击"提交"按钮。

  ![移动销位置](./media/how-to-use-feedback-tool/move-pin.png)

## <a name="add-a-comment"></a>添加注释 

除了允许您搜索位置外，反馈工具还允许您添加与位置相关的详细信息的免费表单文本注释。 要添加注释，请搜索该位置或单击该位置。 单击"添加注释"，撰写注释，然后单击"提交"。

  ![添加注释](./media/how-to-use-feedback-tool/add-comment.png)

## <a name="track-status"></a>跟踪状态 

您还可以通过选中"我想跟踪状态"框并在发出请求时提供电子邮件来跟踪您的请求的状态。 您将在电子邮件中收到一个跟踪链接，该链接提供您的请求的最新状态。 

  ![反馈状态](./media/how-to-use-feedback-tool/feedback-status.png)
