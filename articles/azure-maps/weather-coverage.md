# <a name="azure-maps-weather-coverage"></a>Azure Maps 天气范围

本文提供 Azure Maps[天气服务](https://aka.ms/AzureMapsWeatherService)的覆盖范围信息。 Azure Maps 天气数据服务可以返回详细信息，如雷达磁贴、当前天气状况、天气预测和天气。

对于所有国家和地区，Azure Maps 没有相同级别的信息和准确性。

下表提供了可从每个国家/地区请求的天气信息类型的相关信息。

| 符号 | 含义 |
|--------|---------|
|*       |涵盖路线、当前条件、每小时预测、季度预测和每日预测 Api 的天气 |
