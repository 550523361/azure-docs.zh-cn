# <a name="azure-maps-community---open-source-projects"></a>Azure 地图社区 - 开源项目

这些开源、社区驱动的计划由 Azure 地图团队创建和维护。 它们不是标准产品或服务的一部分。

下面列出了一些最流行的 Azure 映射开源项目和示例。

| 项目名称 | 描述 |
|-|-|
| [Azure 映射代码示例](https://github.com/Azure-Samples/AzureMapsCodeSamples) | 用于在基于 Web 的应用中使用 Azure 映射的代码示例的集合。 |
| [Azure 映射 Gov 云代码示例](https://github.com/Azure-Samples/AzureMapsCodeSamples) | 用于通过 Azure 政府云使用 Azure 映射的代码示例的集合。 |
| [Azure 地图& Azure 活动目录示例](https://github.com/Azure-Samples/Azure-Maps-AzureAD-Samples) | 演示如何将 Azure 活动目录与 Azure 映射一起使用的示例集合。 | 
| [使用 Azure Maps 实现 IoT 空间分析](https://github.com/Azure-Samples/iothub-to-azure-maps-geofencing) | 跟踪和捕获时空中发生的相关事件是一个常见的 IoT 应用场景。 |
| [Azure 地图犹太笔记本示例](https://github.com/Azure-Samples/Azure-Maps-Jupyter-Notebook) | 使用 Azure 映射 REST 服务的 python 示例集合。 |
| [机器人框架 - 兴趣点技能](https://github.com/microsoft/botframework-solutions/tree/488093ac2fddf16096171f6a926315aa45e199e7/skills/csharp/pointofinterestskill) | 兴趣点技能为使用 Azure 地图与 Azure 机器人服务和机器人框架使用 Azure 映射的虚拟助理提供与 POI 相关的功能。 |

此处[提供了包含](https://github.com/microsoft/Maps/blob/master/AzureMaps.md)社区创建开源项目的较长列表

## <a name="supportability-of-open-source-projects"></a>开放源码项目的支持性

以下语句适用于创建和维护的所有 Azure 地图开源项目和示例：

- Azure 地图开源项目和示例由 Microsoft 和社区创建。
- Azure 地图开源项目和示例由 Microsoft 和社区维护。
- Azure 映射开源项目和示例使用支持和推荐的技术。
- Azure 地图开源项目和示例是一个社区计划 - 为他人谋福利的主动工作，并且也有正常工作的人员。
- Azure 地图开源项目和示例不是产品，因此它不受高级支持或其他官方支持渠道的支持。
- Azure 地图开源项目和示例的支持方式与 Microsoft 在社区支持下完成的其他开源项目类似。
