# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>如何管理 Azure Maps 帐户的定价层

可以通过 Azure 门户管理 Azure Maps 帐户的定价层。 创建[帐户](https://azure.microsoft.com/free/?WT.mc_id=A261C142F)后，可以查看或更改帐户的定价层。

有关选择适当定价层的详细信息，请参阅[选择适当的定价层](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier)

## <a name="view-pricing-tier"></a>查看定价层

若要查看选择的定价层，请导航到设置菜单中的“定价层”选项。

![查看选择的定价层](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>更改定价层

创建 Azure Maps 帐户后，可以升级或降级 Azure Maps 帐户的定价层。 若要升级或降级，请导航到设置菜单中的“定价层”选项，并选择不突出显示的定价层。 单击保存按钮以保存所选定价层选项。

![更改定价层](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> 如果升级或降级 Azure Maps 帐户的定价层，则无需生成新的订阅密钥。
