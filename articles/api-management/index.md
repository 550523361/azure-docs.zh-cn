# <a name="api-management-documentation"></a>API 管理文档

API 管理可帮助组织将 API 发布给外部、合作伙伴和内部开发人员，以充分发挥其数据和服务的潜力。  API 管理提供了核心能力，通过开发人员参与、商业洞察力、分析、安全和保护确保成功的 API 程序。
