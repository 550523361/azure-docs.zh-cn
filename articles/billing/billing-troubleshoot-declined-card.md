# <a name="troubleshoot-a-declined-card-at-azure-sign-up"></a>在 Azure 注册时对拒绝的卡进行故障排除

本文可帮助解决在 Azure 门户或 Azure 帐户中心的 Azure 注册中拒绝信用卡的问题。 在开始解决问题之前, 请检查以下几点:

- 确保为 Azure 帐户配置文件提供的信息 (例如联系人电子邮件、地址和电话号码) 正确无误。
- 确保信用卡信息正确无误。
- 确保尚未拥有包含相同信息的 Microsoft 帐户。
- 不接受借记卡

## <a name="issues"></a>问题

下面列出了可能会导致信用卡在 Azure 注册时被拒绝的常见问题。

### <a name="the-credit-card-provider-is-not-accepted-for-your-country"></a>你所在的国家/地区不接受信用卡提供商

选择卡时, Azure 会显示所选国家/地区中有效的卡选项。 请联系银行或信用卡颁发商, 验证是否已为你的信用卡启用了国际交易。 有关支持的国家/地区和货币的详细信息, 请参阅[Azure 购买常见问题解答](https://azure.microsoft.com/pricing/faq/)。

>[!Note]
>美国运通信用卡目前不支持作为印度支付方式。 我们没有时间范围, 因为它可能是接受的付款形式。

### <a name="youre-using-a-virtual-or-prepaid-card"></a>你使用的是虚拟卡或预付卡 

虚拟或预付信用卡或借记卡不能作为 Azure 订阅有效的付款选项。

### <a name="your-credit-information-is-inaccurate-or-incomplete"></a>信用信息不准确或不完整 

输入的姓名、地址和 CVV 代码必须与卡上打印的内容完全匹配。

### <a name="the-card-is-inactive-or-blocked"></a>该卡处于非活动状态或已被阻止 

联系银行以确保卡处于活动状态。

你可能会遇到其他注册问题 

有关如何排查 Azure 注册问题的详细信息, 请参阅以下知识库文章: 

[无法在 Azure 门户或 Azure 帐户中心注册 Azure](billing-troubleshoot-azure-sign-up.md)

### <a name="you-represent-a-business-that-doesnt-want-to-pay-by-card"></a>你代表不想按卡支付的企业 

如果你表示企业, 则可以使用发票付款方式, 如支票、整夜支票或电汇, 为你的 Azure 订阅付费。 设置帐户以通过发票付款后, 不能更改为其他支付选项, 除非你具有 Microsoft 客户协议, 并通过 Azure 网站注册 Azure。

有关如何按发票付款的详细信息, 请参阅[通过发票提交对 Azure 订阅付款的请求](billing-how-to-pay-by-invoice.md)。

### <a name="your-credit-card-information-is-outdated"></a>你的信用卡信息已过时 

有关如何管理卡信息, 包括更改或删除卡的信息, 请参阅[添加、更新或删除 Azure 信用额度](billing-how-to-change-credit-card.md)。
