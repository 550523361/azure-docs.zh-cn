# <a name="change-the-profile-information-for-your-azure-account"></a>更改 Azure 帐户的个人资料信息

Microsoft 使用帐户管理员个人资料中的电子邮件地址发送有关 Azure 帐户的重要计费、服务和建议相关通知。 建议指定定期检查的电子邮件地址。 作为帐户管理员, 你还可以更新配置文件, 以更改帐户的通讯地址和电话号码。

Microsoft 使用用户配置文件中的电子邮件地址, 该用户不是帐户管理员, 而是发送与服务相关的通知和帐户建议。 所有帐单通知都将发送给帐户管理员。 你还可以更新配置文件以更改你的邮件地址和电话号码, 但不会更改帐户的邮件地址或电话号码。

## <a name="change-your-profile-information"></a>更改你的个人资料信息

1. 登录到 [Azure 帐户中心](https://account.azure.com/subscriptions)。
1. 选择“编辑详细信息”按钮，并更新“个人资料”信息。

   ![Azure 配置文件信息示例](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>将通知发送到其他电子邮件地址

除了与帐户关联的帐户管理员电子邮件地址外, 还可以将服务通知和帐户建议发送到电子邮件地址。 帐户管理员的电子邮件地址将获取有关帐户下所有订阅的重要通知, 包括与计费相关的更新。

### <a name="change-the-account-administrators-email-address-in-your-profile"></a>更改配置文件中的帐户管理员电子邮件地址

帐户管理员是设置 Azure 帐户以及接收帐户下所有订阅相关重要电子邮件通知的人员。 建议指定帐户管理员定期收信的联系人电子邮件地址。

### <a name="update-your-sign-in-email-address"></a>更新登录电子邮件地址

更新个人资料电子邮件地址不会更新登录电子邮件地址。 若要更改登录电子邮件地址, 必须转移帐户的所有权。

### <a name="update-the-billing-address-for-your-credit-card"></a>更新信用卡的帐单地址

若要了解如何更新帐单信息，请查看[更改用于支付 Azure 订阅的信用卡](billing-how-to-change-credit-card.md)。

### <a name="update-your-countryregion"></a>更新你的国家/地区

由于技术限制, 我们无法更改现有帐户的国家/地区。 但你可以在所需的国家/地区创建帐户, 然后联系 Azure 支持人员将服务迁移到该帐户。

## <a name="service-and-marketing-emails"></a>服务和市场营销电子邮件

在[Azure 门户](https://portal.azure.com)中, 系统将提示您每90天验证或更新电子邮件地址。 Microsoft 会将电子邮件发送到你的个人资料电子邮件地址, 其中包含与 Azure 帐户相关的信息:

- 服务通知
- 安全警报
- 计费目的
- 支持
- 营销通信
- 基于 Azure 使用情况的最佳实践建议

输入要接收有关帐户的通信的电子邮件地址。 输入电子邮件地址后, 你将选择接收来自 Microsoft 的通信。

![更新联系人信息的提示示例](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>更改联系人电子邮件地址

您可以使用以下方法之一来更改您的联系人电子邮件地址:

* 登录到 [Azure 帐户中心](https://account.azure.com/subscriptions)。 选择 "**编辑详细信息**", 然后更新配置文件电子邮件地址。 更新个人资料电子邮件地址不会更新登录电子邮件地址。

* 中转到 Azure 门户中的 "[联系信息](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)" 区域, 并输入电子邮件地址。 提供的地址将接收服务和市场营销通信。 更新 Azure 帐户中心配置文件中的电子邮件地址以自动匹配。 更新个人资料电子邮件地址不会更新登录电子邮件地址。

* 在[Azure 门户](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)中, 选择包含你的姓名首字母的图标或图片。 然后, 选择上下文菜单 ( **...** )。接下来, 从菜单中选择 **"我的联系人信息**", 并输入电子邮件地址。 提供的地址将接收服务和市场营销通信。 更新 Azure 帐户中心配置文件中的电子邮件地址以自动匹配。 更新个人资料电子邮件地址不会更新登录电子邮件地址。

![在 Azure 中更新电子邮件地址的示例](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>选择退出市场营销电子邮件

选择不接受市场营销电子邮件:

1. 使用你的个人资料电子邮件地址, 前往[请求窗体](https://account.microsoft.com/profile/permissions-link-request)提交请求。 你将通过电子邮件收到用于更新首选项的链接。
2. 选择该链接, 打开 "**管理通信权限**" 页。 此页显示电子邮件地址所选择的营销通信类型。 清除想要退出的任何主题, 然后选择 "**保存**"。

![用于管理通信权限的页面示例](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

如果选择退出市场营销通信, 则仍将根据你的帐户收到服务通知。

## <a name="change-the-subscription-name"></a>更改订阅名称

1. 登录到[Azure 门户](https://portal.azure.com), 从左侧窗格中选择 "**订阅**", 然后选择要重命名的订阅。
1. 选择 "**概述**", 然后从命令栏中选择 "**重命名**"。
    ![重命名 Azure 订阅的示例](./media/billing-how-to-change-azure-account-profile/rename-sub.png)
1. 更改名称后, 请选择 "**保存**"。
