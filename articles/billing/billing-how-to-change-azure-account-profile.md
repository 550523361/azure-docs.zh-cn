# <a name="change-your-azure-account-profile-information"></a>更改你的 Azure 帐户个人资料信息

配置文件信息用于你的 Azure 帐户与你联系。 应将其更新，以便 Microsoft 可以提醒您注意的重要通知。

Microsoft 使用帐户管理员的配置文件中的电子邮件地址发送重要的帐单、 服务和有关帐户建议相关通知。 我们建议指定定期收信的电子邮件地址。 以帐户管理员身份，还可以更新配置文件以更改帐户的邮寄地址和电话号码。

在用户的配置文件，不是帐户管理员的帐户用户的电子邮件地址由 Microsoft 用于发送与服务相关的通知和帐户的建议。 所有计费通知发送给帐户管理员。 此外可以更新配置文件以更改你的邮寄地址和电话号码，但它不会更改帐户的邮寄地址或电话号码。

## <a name="change-your-profile-information"></a>更改个人资料信息

1. 登录到 [Azure 帐户中心](https://account.azure.com/Profile)。
1. 选择“编辑详细信息”按钮，并更新“个人资料”信息。

   ![显示你的 Azure 示例配置文件信息](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>将通知发送到其他电子邮件地址

您可以服务通知和帐户建议发送到除了与你的帐户关联的帐户管理员电子邮件地址的电子邮件地址。 帐户管理员的电子邮件地址获取帐户，包括与计费相关更新下的所有订阅的重要通知。

### <a name="change-the-account-administrator-email-address-in-your-profile"></a>更改你的配置文件中的帐户管理员电子邮件地址

帐户管理员是设置 Azure 帐户以及接收帐户下所有订阅相关重要电子邮件通知的人员。 建议指定帐户管理员定期收信的联系人电子邮件地址。

### <a name="updating-your-profile-email-doesnt-update-your-sign-in-email-address"></a>正在更新你的配置文件的电子邮件不会更新登录电子邮件地址

正在更新配置文件的电子邮件地址不会更新登录电子邮件地址。 若要更改登录电子邮件地址，必须转移帐户的所有权。

### <a name="updating-your-profile-address-doesnt-update-your-credit-card-billing-address"></a>正在更新配置文件地址不会更新你的信用卡的帐单地址

若要了解如何更新帐单信息，请查看[更改用于支付 Azure 订阅的信用卡](billing-how-to-change-credit-card.md)。

### <a name="you-cant-update-your-country"></a>不能更新国家/地区

由于技术限制，我们不能更改现有帐户的国家/地区。 但是，用户可以在所需的国家/地区创建帐户，然后联系 Azure 支持将服务迁移到该帐户。

## <a name="service-and-marketing-emails"></a>服务和市场营销电子邮件

Microsoft Azure 团队会发送与服务相关的电子邮件通知你关键警报。 团队还会发送电子邮件帐户建议，也称为市场营销电子邮件，以帮助您优化和缩放你的项目。

系统会提示[Azure 门户](https://portal.azure.com)若要验证或更新每隔 90 天的电子邮件地址。 通知的目的是确保 Microsoft 已更新发生任何服务相关的紧急通知时的联系信息。 Microsoft 将电子邮件发送到配置文件与 Azure 帐户相关信息的电子邮件地址：

- 服务通知
- 安全警报
- 计费目的
- 支持
- 市场营销通信
- 最佳做法建议，基于 Azure 的使用情况

输入你想要收到有关你的帐户的电子邮件地址。 通过输入电子邮件地址，您在选择加入以接收来自 Microsoft 的通信。

![显示更新你的联系信息框示例](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>更改联系人电子邮件地址

你可以使用以下方法在联系人电子邮件地址：

**选项 1**：登录到[Azure 帐户中心](https://account.azure.com/Profile)。 选择**编辑详细信息**，然后更新个人资料电子邮件地址信息。 正在更新你的配置文件的电子邮件地址不会更新登录电子邮件地址。

**选项 2**：转到[联系信息](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)区域在 Azure 门户中的，输入电子邮件地址。 你提供的地址会收到服务和市场营销的通信。 Azure 帐户中心配置文件中的电子邮件地址将更新以自动匹配。 正在更新你的配置文件的电子邮件地址不会更新登录电子邮件地址。

**选项 3**：在中[Azure 门户](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade)中，选择您的姓名缩写的图标或图片。 然后，选择上下文菜单 (**...**).接下来，选择**我的联系信息**菜单中，输入电子邮件地址。 你提供的地址会收到服务和市场营销的通信。 在 Azure 帐户中心配置文件中的电子邮件地址将更新以自动匹配。 正在更新你的配置文件的电子邮件地址不会更新登录电子邮件地址。

![在 Azure 中显示的联系人信息更新示例](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>选择退出的市场营销电子邮件

若要选择退出接收市场营销电子邮件：

1. 转到[请求窗体](https://account.microsoft.com/profile/permissions-link-request)提交使用你配置文件的电子邮件地址的请求。 通过电子邮件来更新您的首选项，将收到一个链接。
2. 此链接显示的市场营销宣传资料中选择的电子邮件地址的类型。 清除你想要选择退出，然后单击任何主题**保存**。


![显示管理通信权限选项示例](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

### <a name="you-still-receive-service-notifications-if-you-opt-out-of-marketing-communications"></a>如果你选择退出的市场营销宣传资料仍收到服务通知

如果已为通信提供的电子邮件地址，您将收到服务通知，根据你的帐户。 当您选择退出，你仅选择退出的营销通信-你仍收到服务通知。
