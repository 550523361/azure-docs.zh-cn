# <a name="your-debit-card-or-credit-card-is-declined-at-azure-sign-up"></a>借记卡或信用卡在注册 Azure 时被拒绝
如果在注册 Azure 时借记卡或信用卡被拒绝或不被接受，则可能需要修复以下问题之一。

* **用户的国家/地区不接受此信用卡或借记卡提供商。** 选择卡时，只能看到用户选择的国家/地区内的有效选项。 请联系银行或信用卡颁发机构，确认信用卡能够进行国际交易。 请参阅 [Azure 购买常见问题解答](https://azure.microsoft.com/pricing/faq/)中的“受支持的国家/地区和货币”。
* **信用卡或借记卡信息不准确或不完整。** 输入的姓名、地址和 CVV 代码必须与卡上打印的内容一致。
* **用户使用的是虚拟卡或预付卡。** 虚拟或预付信用卡或借记卡不能作为 Azure 订阅有效的付款选项。
* **卡未激活或已阻止使用。** 请联系银行并确保卡已激活且未受阻止。
* **尝试激活第二个免费试用版。** 每位用户只能激活一次 Azure 免费试用版的产品/服务。 如果以前曾使用此信用卡激活免费试用版，或曾拥有与此信用卡关联的 Azure 订阅，则不能激活免费试用版。 使用 [Microsoft Azure 产品/服务详细信息](https://azure.microsoft.com/support/legal/offer-details/)中的其他 Azure 计划。

更新付款信息后，请尝试再次注册。

## <a name="business-that-doesnt-want-to-pay-by-card-set-up-invoicing"></a>不想用卡支付的业务？ 设置发票
如果用户代表企业，则可以使用转账支付方式支付 Azure 订阅，如支票、隔夜支票或汇款。 设置为通过转账支付后，便不能更改为其他支付选项。 若要通过转账付款，请参阅 [Azure 账单 - 如何开具发票](https://azure.microsoft.com/pricing/invoicing/)。

## <a name="update-your-credit-card-or-debit-card-information"></a>更新信用卡或借记卡信息
注册后，[管理付款信息](billing-how-to-change-credit-card.md)以更改或删除卡。

## <a name="need-help-contact-support"></a>需要帮助？ 联系支持人员。
如果仍需帮助，请[联系支持人员](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)以快速解决问题。
