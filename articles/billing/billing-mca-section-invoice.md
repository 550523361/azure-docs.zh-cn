# <a name="create-sections-on-your-invoice-to-organize-your-costs"></a>在你开发票来组织你的成本创建部分

在你开发票来组织你的成本按部门，开发环境中，创建部分或根据组织的需求。 然后授予其他人创建的部分收费的 Azure 订阅的权限。 任何使用费用和购买的订阅则计费的部分。 可以查看发票，在 Azure 门户中上部分的总费用或在 Azure 成本分析中查看它们。 有关详细信息，请参阅[按发票部分查看交易记录](billing-mca-understand-your-bill.md#view-transactions-by-invoice-sections)。

本文适用于 Microsoft 客户协议向计费帐户。 [检查是否有权访问 Microsoft 客户协议](#check-access-to-a-microsoft-customer-agreement)。

## <a name="create-an-invoice-section-in-the-azure-portal"></a>在 Azure 门户中创建发票部分

若要创建发票部分，必须是**账单的配置文件所有者**或**计费的配置文件参与者**。 有关详细信息，请参阅[管理的计费配置文件的发票部分](billing-understand-mca-roles.md#manage-invoice-sections-for-billing-profile)。

1. 登录到 [Azure 门户](https://portal.azure.com)。

2. 搜索**成本管理 + 计费**。

   ![显示了 Azure 门户搜索的屏幕截图](./media/billing-mca-section-invoice/billing-search-cost-management-billing.png)

3. 选择**发票部分**从左侧窗格。 根据您的访问权限，可能需要选择计费的配置文件或计费帐户，然后选择**发票部分**。

   ![显示发票部分列表的屏幕截图](./media/billing-mca-section-invoice/mca-select-invoice-sections.png)

4. 在页面顶部选择“添加”。 

5. 输入的发票部分的名称并选择计费的配置文件。 你会看到此计费配置文件的专用于反映将每个订阅和购买已分配给部分中的使用情况的发票上的部分。 

   ![显示发票部分创建页的屏幕截图](./media/billing-mca-section-invoice/mca-create-invoice-section.png)

6. 选择“创建”  。
