# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>用于在 Azure Maps 中添加地图、空间分析和移动性解决方案的地理空间 Api

[Azure Maps](https://azure.microsoft.com/services/azure-maps/)在云中以本机方式提供位置平台，并提供一系列地理空间服务，使用最新的映射数据为 web 和移动应用程序提供准确的地理上下文。 开发人员、企业和 Isv 可以使用 Azure Maps 来创建位置感知应用程序和 IoT、移动性、物流和资产跟踪解决方案。

## <a name="azure-maps-features"></a>Azure Maps 功能
- **搜索**：提供有关邻近兴趣点的详细信息，在地图上呈现位置，并使用地理编码（或反向地理编码）地址获取相应的纬度和经度坐标。
- **流量**：改善旅行时间，有助于避免停滞不前。
- **Maps**：提供清晰、易于阅读的地图，以将其集成到你的解决方案中。
- **路由**：显示在特定时间可用于多个目标的最短或最快路由。 路由还提供适用于查看器、骑车和商业汽车的专用路由和事实。
- **时区**：便于查看在世界各地的时间。
- **空间操作**：增强你的位置智能。
- **地理位置**：查找 IP 地址的国家/地区。
- **数据服务**：上传和存储用于空间操作或图像组合的地理空间数据。
- **移动性**：对附近的公共交通服务提供实时位置智能。 将停止、路由信息和行程时间估算。
