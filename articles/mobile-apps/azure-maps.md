# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>地理空间 API，用于使用 Azure 地图添加地图、空间分析和移动解决方案

[Azure 地图](https://azure.microsoft.com/services/azure-maps/)在云中提供了本地位置平台，并提供地理空间服务组合，这些服务使用新的映射数据为 Web 和移动应用程序提供准确的地理环境。 开发人员、企业和 ISV 可以使用 Azure 地图创建位置感知应用程序和 IoT、移动性、物流和资产跟踪解决方案。

## <a name="azure-maps-features"></a>Azure 地图功能
- **搜索**： 提供有关附近兴趣点的详细信息，渲染地图上的位置，并使用地理编码（或反向地理代码）地址获取相应的纬度和经度坐标。
- **交通**：改善您的旅行时间，并帮助您避免交通堵塞。
- **地图**： 提供清晰、易于阅读的地图，以集成到解决方案中。
- **路由**：显示在特定时间可用于多个目标的最短或最快的路由。 路线还为步行者、骑自行车者和商用车辆提供专门的路线和事实。
- **时区**：很容易看出它在世界任何地方的时间。
- **空间操作**：增强您的位置智能。
- **地理位置**：查找 IP 地址的国家/地区。
- **数据服务**：上传和存储地理空间数据，以便与空间操作或图像合成一起使用。
- **移动性**：提供有关附近公共交通服务的实时位置情报。 包括停靠点、路线信息和行驶时间估计。
