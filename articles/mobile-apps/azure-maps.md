# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>用于在 Azure Maps 中添加地图、空间分析和移动性解决方案的地理空间 Api

[Azure Maps](https://azure.microsoft.com/services/azure-maps/)在 Azure 云中本地提供位置平台，它提供了一系列地理空间服务，使用最新的映射数据为 web 和移动应用程序提供准确的地理上下文。 它使开发人员、企业和 Isv 能够创建位置感知应用程序和 IoT、移动性、物流和资产跟踪解决方案。

## <a name="azure-maps-features"></a>Azure Maps 功能
- **搜索**提供有关邻近兴趣点（POI）、地图上的呈现位置和地理编码（或反向地理编码）地址的详细信息，以获取相应的纬度/经度坐标。
- 使用**流量**改善旅行时间并避免停滞不前。
- 将清晰、易于阅读的**映射**集成到你的解决方案中。
- **路由**以提供最短或最快的可用路由（一次多个目标），或为查看器、骑车和商业汽车提供专用的路由和事实。
- 通过**时区**，用户可以轻松查看在世界上任意位置的时间。
- 用于增强位置智能的**空间操作**。
- 查找 IP 地址所在国家/地区的**地理位置**。
- **数据服务**，用于上载和存储用于空间操作或图像组合的地理空间数据。
- **移动性**为附近的公共交通服务提供实时位置智能，包括停止、路由信息以及旅行时间估算。
