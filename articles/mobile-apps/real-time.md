# <a name="build-real-time-applications-with-azure-signalr-service"></a>使用 Azure SignalR 服务构建实时应用程序

使用[Azure SignalR 服务](https://azure.microsoft.com/services/signalr-service/)，可以轻松地向应用程序添加实时功能。 此实时功能允许服务将内容更新推送到连接的移动应用程序。 因此，应用程序会更新，而无需轮询服务器或提交新的 HTTP 更新请求。

## <a name="azure-signalr-service-features"></a>Azure 信号R 服务功能
- 为广泛的客户端（如 Web 和移动浏览器、桌面应用程序、服务器进程、IoT 设备和游戏机）提供广泛的客户端支持。
- 允许多个实例协同工作，以扩展到数百万个移动应用程序连接。 它还支持多个全局区域，用于分片、高可用性或灾难恢复目的。
- 为不同的消息模式提供丰富的 API，用于将消息发送到特定连接、所有连接或属于特定用户或放置在任意组中的连接子集。
