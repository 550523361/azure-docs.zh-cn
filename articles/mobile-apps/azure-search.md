# <a name="ai-powered-cloud-service-with-azure-search"></a>带 Azure 搜索的 AI 支持的云服务
[Azure 搜索](https://azure.microsoft.com/services/search/)是一种搜索即服务云解决方案，它为开发人员提供 api 和工具，用于在 web 应用程序、移动应用程序和企业应用程序中添加丰富的搜索体验。 自定义代码调用数据引入（索引）来创建和加载索引。 另一方面，应用程序代码发出查询请求并处理响应。 使用 Azure 搜索中的功能在客户端中定义搜索体验，通过在服务中创建，拥有并存储的持久索引执行查询。

## <a name="azure-search-features"></a>Azure 搜索功能
- 具有全文搜索、简单查询语法和 lucene 查询语法的**自由格式文本搜索**。
- **简单的计分配置文件**，用于将相关性作为文档本身中的值的函数进行建模。
- 利用**异地搜索**，用户可以基于搜索结果与物理位置的接近程度浏览数据。
- **用户体验功能**，包括自动完成、搜索建议、排序和分页。
- 可以对图像和文本分析应用**认知搜索**，以便从原始内容中提取文本信息。
- 用于保存在索引过程中创建的根据的**知识存储**。
- 作为**数据源**连接到 Azure SQL 数据库、Azure Cosmos DB 或 azure Blob 存储，以便将可搜索的内容提取到主数据存储中。
- 使用**导入数据向导**来配置索引器、**索引设计器**以创建索引，并使用**搜索资源管理器**来测试查询并优化计分配置文件。

查找有关 Azure 搜索[功能](/azure/search/search-what-is-azure-search#feature-descriptions)的详细说明。
