# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>AI 支持的云服务，具有 Azure 认知搜索功能
[Azure 认知搜索](https://azure.microsoft.com/services/search/)是一种搜索即服务云解决方案，它为开发人员提供 API 和工具，用于在 Web、移动和企业应用程序中添加丰富的搜索体验，而不是私有的异构内容。 自定义代码调用数据引入（索引）来创建和加载索引。 另一方面，应用程序代码发出查询请求并处理响应。 搜索体验通过使用 Azure 认知搜索的功能在客户端中定义，在服务中创建、拥有和存储的持久化索引上执行查询。

## <a name="azure-cognitive-search-features"></a>Azure 认知搜索功能
- **自由格式文本搜索**：提供全文搜索、简单的查询语法和 Lucene 查询语法。
- **简单的评分配置文件**：模型相关性作为文档中值的函数。
- **地理搜索**：根据搜索结果与物理位置的接近程度来浏览数据。
- **用户体验功能**：提供自动完成、搜索建议、排序和分页。
- **认知搜索**：可应用于索引管道，从原始内容中提取文本信息，用于图像和文本分析。
- **知识存储**：保存索引期间创建的扩充。
- **数据源**：连接到 Azure SQL 数据库、Azure Cosmos DB 和 Azure Blob 存储时，在主数据存储中提取可搜索的内容。
- **导入数据向导**：配置索引器。 
- **索引设计器**：站起来索引。 
- **搜索资源管理器**：测试查询并优化评分配置文件。

在 Azure 认知搜索[功能](/azure/search/search-what-is-azure-search#feature-descriptions)上查找详细说明。
