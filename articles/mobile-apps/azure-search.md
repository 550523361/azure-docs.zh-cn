# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>支持 AI 的云服务与 Azure 认知搜索
[Azure 认知搜索](https://azure.microsoft.com/services/search/)是一种搜索即服务云解决方案，它为开发人员提供 api 和工具，用于在 web 应用程序、移动应用程序和企业应用程序中对私有、异类内容添加丰富的搜索体验。 自定义代码调用数据引入（索引）来创建和加载索引。 另一方面，应用程序代码发出查询请求并处理响应。 通过使用 Azure 认知搜索中的功能在客户端中定义搜索体验，并在服务中创建、拥有和存储的持久索引上执行查询。

## <a name="azure-cognitive-search-features"></a>Azure 认知搜索功能
- **自由文本搜索**：提供全文搜索、简单查询语法和 Lucene 查询语法。
- **简单的计分配置文件**：模型相关性作为文档本身的值的函数。
- **地域**：基于搜索结果与物理位置的邻近性来浏览数据。
- **用户体验功能**：提供自动完成、搜索建议、排序和分页。
- **认知搜索**：可应用于索引管道，以便从图像和文本分析的原始内容中提取文本信息。
- **知识库**：保存在编制索引期间创建的根据。
- **数据源**：连接到 Azure SQL 数据库、Azure Cosmos DB 和 azure Blob 存储时，可在主数据存储中提取可搜索的内容。
- **导入数据向导**：配置索引器。 
- **索引设计器**：表示索引。 
- **搜索资源管理器**：测试查询和精炼计分配置文件。

查找有关 Azure 认知搜索[功能](/azure/search/search-what-is-azure-search#feature-descriptions)的详细说明。
