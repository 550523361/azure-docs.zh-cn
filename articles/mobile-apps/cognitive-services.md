# <a name="azure-cognitive-services-to-solve-business-problems-using-artificial-intelligence"></a>Azure 认知服务，使用人工智能解决业务问题
[Azure 认知服务](https://azure.microsoft.com/services/cognitive-services/)提供强大的服务，可通过智能算法注入你的应用程序，以查看、听取、说话、理解和解释用户需要通过 AI 转换你的业务。 你可以使你的移动应用程序与你的用户根据上下文。 利用 Api、Sdk 和服务，开发人员可以将 AI 添加到其产品/服务，并可帮助生成智能应用程序，而无需直接获得 AI 或数据科学技能。

## <a name="cognitive-services-capabilities"></a>认知服务功能
- **影像**
    - 图像标记，缩略图
    - OCR （手写识别）
    - 面部检测
    - 情感检测
    - 视频见解
    - 图像和视频审核
    - 自定义图像识别

- **语音**
    - 语音转换到文本（语音对话）
    - 自定义语音脚本（复杂字词，干扰环境）
    - 文本转语音
    - 发言人 ID 和身份验证
    - 实时语音翻译

- **语言**
    - 上下文语言理解-自定义意向分析
    - 情绪分析，关键短语检测
    - 文本翻译最多60种语言
    - 文本审查
    - 拼写检查

- **确定** 
    - 内容审查器
    - 个性化内容交付
    - 异常探测器

- **搜索**
    - 自动搜索建议 
    - 实体信息扩充
    - 全面的新闻、图像、视频、web 和本地业务实体搜索
    - 定制的搜索体验

- 支持认知服务的**容器**
