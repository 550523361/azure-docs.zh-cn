# <a name="cognitive-services-helps-to-solve-business-problems-by-using-ai"></a>认知服务通过使用 AI 帮助解决业务问题
[Azure 认知服务](https://azure.microsoft.com/services/cognitive-services/)提供强大的服务，为您的应用注入智能算法，以查看、收听、说话、理解和解释用户使用人工智能 （AI） 转变业务所需的信息。 您可以使移动应用程序智能化、吸引人且与用户上下文相关。 开发人员可以使用 API、SDK 和服务将 AI 添加到其产品中，并帮助构建智能应用程序，而无需直接掌握 AI 或数据科学技能。

## <a name="cognitive-services-capabilities"></a>认知服务功能
- **影像**
    - 图像标记、缩略图
    - OCR（手写识别）
    - 人脸检测
    - 情感检测
    - 视频见解
    - 图像和视频审核
    - 自定义图像识别

- **语音**
    - 语音到文本（语音转录）
    - 自定义语音转录（复杂单词、嘈杂环境）
    - 文本到语音转换
    - 扬声器 ID 和身份验证
    - 实时语音翻译

- **语言**
    - 上下文语言理解，定制意图分析
    - 情绪分析，关键短语检测
    - 文本翻译多达 60 种语言
    - 文本审查
    - 拼写检查

- **决定** 
    - 内容审查器
    - 个性化内容交付
    - 异常检测器

- **搜索**
    - 自动搜索建议 
    - 实体信息增强
    - 全面新闻、图片、视频、网络和本地业务实体搜索
    - 定制和定制的搜索体验

- 支持认知服务容器
