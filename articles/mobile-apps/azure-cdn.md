# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Azure 上具有 Azure 内容交付网络的内容传递网络
[Azure 内容交付网络](https://azure.microsoft.com/services/cdn/)为开发人员提供了一个全球解决方案，通过将视频、图像或静态文件等高带宽内容缓存到全球数百个存在点，从而快速向用户交付其内容。 内容交付网络还可以加速无法缓存的动态内容，以缩短加载时间、节省带宽并提高响应能力。 无论您是开发或管理网站或移动应用程序，还是对流媒体、游戏软件、固件更新或 IoT 终结点进行编码和分发，您都将获得这些优势。

## <a name="azure-content-delivery-network-features"></a>Azure 内容交付网络功能
  - 与 Azure 服务（如 Web 应用程序、媒体服务、存储和云服务）无缝协作。
  - 轻松处理交通高峰和重负载。
  - 与 Akamai 和 Verizon 集成，构建和定制内容交付战略。
  - 使用动态站点加速。
  - 使用缓存规则控制内容传递网络缓存行为。
  - 获取 HTTPS 自定义域支持。
  - 查看 Azure 诊断日志以查看核心分析并将其导出到 Azure Blob 存储、Azure 事件中心或 Azure 监视器。
  - 通过压缩文件来提高性能。
  - 使用地理筛选按国家或地区限制内容交付网络内容。
