# <a name="content-delivery-network-on-azure-with-azure-cdn-service"></a>Azure 上的内容交付网络与 Azure CDN 服务
[Azure CDN](https://azure.microsoft.com/services/cdn/)为开发人员提供了一个全局解决方案，通过在世界各地将视频、图像或静态文件的内容缓存到数百个位置（PoP）来向用户快速交付高带宽内容，例如视频、图像或静态文件。 Azure CDN 还可以加速动态内容，这种情况下，不能进行缓存，还可以减少加载时间、节省带宽和提高响应能力，无论是开发还是管理网站或移动应用程序，或者编码和分发流式处理媒体、游戏软件、固件更新或 IoT 终结点。

## <a name="azure-cdn-features"></a>Azure CDN 功能
  - 与 Azure 服务**无缝**协作，包括 Web 应用程序、媒体服务、存储和云服务。
  - 轻松**处理流量峰值和繁重负载**。
  - **与 Akamai 和 Verizon 集成**，以构建和定制内容交付策略。
  - 动态站点加速。
  - 控制**缓存规则**Azure CDN 缓存行为。
  - HTTPS**自定义域支持**。
  - 用于查看核心分析并将其导出到 Blob 存储、事件中心或 Azure Monitor 的**Azure 诊断日志**。
  - 通过**压缩文件**来提高性能。
  - **地域筛选**，按国家/地区限制 Azure CDN 内容。
