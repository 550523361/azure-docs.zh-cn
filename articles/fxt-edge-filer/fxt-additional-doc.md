# <a name="additional-documentation"></a>其他文档 

此docs.microsoft.com网站之外的某些资源可以帮助您了解和使用 Microsoft Azure FXT 边缘文件器混合缓存。

## <a name="avere-vfxt-for-azure-cache-documentation"></a>用于 Azure 缓存文档的 Avere vFXT

[Azure 的 Avere vFXT](https://docs.microsoft.com/azure/avere-vfxt/)是使用 Microsoft Azure 中的虚拟节点的类似群集缓存产品。 它是使用 Azure 门户创建的，但使用与 Azure FXT 边缘文件管理器硬件缓存相同的控制面板配置软件。 

## <a name="legacy-hardware-documentation"></a>旧硬件文档 

该技术以前用于 Avere 系统的产品。 这些旧产品的用户文档可在[Avere 旧文档网站上](https://azure.github.io/Avere/)找到。 

具体来说，这些文档可能有有用的详细信息：

* [群集配置指南](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html)- 包含有关常见群集设置和管理任务的概述信息，并详细说明群集设置页面 

* [仪表板指南](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html)- 说明控制面板**仪表板**选项卡的功能

* [FXT集群创建指南](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html)- 以前产品的集群创建指南
