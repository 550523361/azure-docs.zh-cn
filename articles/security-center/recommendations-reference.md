# <a name="security-recommendations---a-reference-guide"></a>安全建议 - 参考指南

本文列出了您可能会在 Azure 安全中心中看到的建议。 环境中显示的建议取决于您保护的资源和自定义配置。

要了解如何响应这些建议，请参阅[Azure 安全中心的"修正"建议](security-center-remediate-recommendations.md)。

您的安全分数基于您已完成的安全中心建议数。 要决定首先解决哪些建议，请查看每个建议的严重性及其对安全分数的潜在影响。

>[!TIP]
> 如果建议的描述显示"没有相关政策"，通常是因为该建议依赖于不同的建议*及其*政策。 例如，"应修复端点保护运行状况故障..."的建议依赖于检查是否甚至*安装了*端点保护解决方案的建议（"应安装端点保护解决方案..."）。 基本建议*确实有*一个策略。 将策略限制为仅基础建议简化了策略管理。
