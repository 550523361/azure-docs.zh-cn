# <a name="why-have-you-received-a-notification-that-your-azure-subscription-has-a-past-due-balance"></a>为什么会收到 Azure 订阅欠款到期未付的通知？
如果你是 Azure 订阅的帐户管理员且没有按时付款，则会收到一封内容为欠款到期未付的电子邮件通知，或者会在 [https://account.windowsazure.com](https://account.windowsazure.com) 或 [https://portal.azure.com](https://portal.azure.com) 上看到一个提醒。

> [!NOTE]
> 对于本文中的任何部分，如需更多帮助，请[联系支持人员](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)以快速解决问题。
> 
> 

如果出于某些原因，我们无法处理你的付款，你可能会收到一封含有类似于以下内容的电子邮件：

**我们未能通过你的信用卡收取你的订阅费用。为防止发生服务中断的情况，请更新你的付款信息。**

请确保你能够收到通知电子邮件。 如果你无法收到通知电子邮件，则可能是因为正在使用不同的电子邮件地址进行登录和帐户管理。 Microsoft 使用帐户管理员个人资料中的电子邮件地址来通知与订阅计费相关的最新重要信息。 建议你指定一个定期收信的联系人电子邮件地址。

## <a name="what-will-happen-if-you-forget-to-pay"></a>如果你忘记付款，会发生什么情况
服务会被取消，你的资源将不再可用。 任何数据都会在服务终止 90 天后被删除。

## <a name="what-can-you-do-to-resolve-the-issue"></a>解决问题的措施
全额支付欠款。

**场景 1**：如果采用发票付款方式，可将付款寄送到发票底部列出的地点。 如果需要帮助，请联系 [Azure 支持](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)。

**场景 2**：如果没有支付帐单是因为信用卡被拒绝或过期，则可针对订阅使用其他信用卡或付款方式，或者与银行联系以解决该问题。 如果你更新付款方式，针对该付款方式的所有待付款项就会立即自动进行结算， 其中包括针对 Azure 的待付款项，以及使用该卡的其他 Microsoft 服务。

有关如何在 Azure 中更改付款方式的说明，请参阅[如何更改 Azure 订阅付款时使用的信用卡](billing-how-to-change-credit-card.md)。 必须以帐户管理员身份登录才能进行这种更改。

**场景 3**：如果因帐户管理员离开了公司或更改了角色而导致帐单通知没有收到，请联系 [Azure 支持](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)以解决此问题。

> [!NOTE]
