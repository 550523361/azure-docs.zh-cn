# <a name="free-30-day-trial-of-azure-remoteapp"></a>Azure RemoteApp 30 天免费试用版
> [!IMPORTANT]
> Azure RemoteApp 将于 2017 年 8 月 31 日停用。 阅读 [公告](https://go.microsoft.com/fwlink/?linkid=821148) 了解详细信息。
> 
> 

如果具有 Azure 订阅，则可以试用 [Azure RemoteApp 30 天免费试用版](https://www.remoteapp.windowsazure.com/en/tour.aspx)。 只需登录到 Azure 管理门户，导航到 Azure RemoteApp 节点，然后创建一个集合即可开始使用。 这是在转换为 Azure RemoteApp 付费订阅前试用完整管理体验的一个好办法。  

在试用模式期间，可以创建两个集合，每个集合限制为 10 个用户。 无任何其他限制。 随时可以将试用版转换为付费订阅。 30 天后，如果未转换为付费订阅，则集合将转入禁用状态。

> [!NOTE]
> Azure SLA 不适用于在试用模式下创建的集合。  
>
