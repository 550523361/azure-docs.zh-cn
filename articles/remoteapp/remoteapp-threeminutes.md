# 在 3 分钟内在设备上试用 Azure RemoteApp

> [AZURE.IMPORTANT]
正在中断 Azure RemoteApp。请参阅[公告](https://go.microsoft.com/fwlink/?linkid=821148)，了解详细信息。

想要在设备上查看 Azure RemoteApp？ 只要有 Microsoft 帐户，即可访问演示部署 10 分钟。我们提供了 Office 和一些其他应用程序。这种方法非常好，可以在 3 分钟内亲身感受 Azure RemoteApp 的用户体验。

感兴趣吗？

1. 转到 [http://remoteapp.azure.com](http://remoteapp.azure.com)。
2. 单击右上角的“安装客户端”。
3. 系统可以自动检测到你的设备，并且会将你转到设备的客户端。安装客户端。
4. 返回 [http://remoteapp.azure.com](http://remoteapp.azure.com)，并使用 Microsoft 帐户登录。

现在，可以访问 Azure RemoteApp 上实时运行的应用程序。

你的意见很重要。在每个会话结束时，将看到一个对话框，便于你提供反馈。如果有时间，请告知我们你的想法，让我们知晓你是否希望 Microsoft 与你联系以了解更多信息。

>[AZURE.NOTE] 由于这是供演示使用的受控环境，我们没有启用从发布的演示应用程序对 Internet 进行外部访问的权限。因此，无法访问 OneDrive 等服务。此外，还禁用了剪切板共享（复制和粘贴）与 USB 和打印机重定向功能。
