# <a name="whats-coming-in-azure-remoteapp"></a>Azure RemoteApp 中即将推出的功能
> [!IMPORTANT]
> 正在中断 Azure RemoteApp。 阅读 [公告](https://go.microsoft.com/fwlink/?linkid=821148) 了解详细信息。
> 
> 

**由于 Azure RemoteApp 宣布停用，所有预览和路线图项目都已被取消和结束。**

| 功能 | 状态 |
| --- | --- | --- |
| 在 [Azure 经典门户](http://manage.windowsazure.com)和 [Azure 门户](https://portal.azure.com)中都提供 Azure Remoteapp - 个人预览版 |**已取消** |
| 将单个应用程序发布给特定用户的能力 - 个人预览版（[反馈](https://feedback.azure.com/forums/247748-azure-remoteapp/suggestions/6067043-allow-the-ability-to-publish-specific-apps-to-spec/)） |**已取消** |
| Windows 客户端可再发行 MSI 软件包（[反馈](https://feedback.azure.com/forums/247748-azure-remoteapp/suggestions/6627191-client-deployment-provide-an-msi-package-to-allo/)- 个人预览版 |**已取消** |
| Azure Resource Manager (ARM) 支持 |**已取消** |
| Azure v2 资源支持 |**已取消** |
| 云解决方案提供商 (CSP) |**已取消** |
| 在印度地区的可用性 |**已取消** |
| 在英国地区的可用性 |**已取消** |

今年年初增加的其他改进︰

* Azure RemoteApp 的基于 HTML5 的客户端
* 持续支持 Azure RemoteApp 的 Windows 10 通用应用程序 - 预览版
* 在不加入域（到 Azure VM 的云集合连接）的情况下部署到 VNET
* 对 RemoteApp 混合集合重复使用现有虚拟网络（还支持 ExpressRoute）
* ExpressRoute
* 用于部署和管理的 RemoteApp PowerShell cmdlet 和 SDK
* 从 Azure VM 创建和上载映像
* 将业务线 (LOB) 应用程序上载到云集合和混合集合中
* 对 Windows Phone 的支持
* 使用情况指标仪表板
* 通过使用 CSV 文件配置大量用户
* 在澳大利亚数据中心的可用性
