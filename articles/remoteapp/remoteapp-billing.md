# <a name="how-to-stop-being-billed-for-azure-remoteapp"></a>如何停止对 Azure RemoteApp 收费
> [!IMPORTANT]
> 正在中断 Azure RemoteApp。 阅读 [公告](https://go.microsoft.com/fwlink/?linkid=821148) 了解详细信息。
> 
> 

只要拥有 Azure RemoteApp 集合，即使未分配任何用户，仍会对 RemoteApp 收费。 若要停止此收费，需删除所有集合。 

若要删除集合：

1. 在 Azure 经典门户中，打开 RemoteApp 页。
2. 单击想要删除的集合，然后单击页面底部的“**删除**”。
3. 在你的订阅中为每个集合重复此操作。 

可能需要花费几分钟来删除每个集合。
