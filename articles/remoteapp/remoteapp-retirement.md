# <a name="azure-remoteapp-retirement"></a>Azure RemoteApp 停用
我们无法联系到你以确认你是否知道 Azure RemoteApp 即将停用，因此将你定向到这篇文章。 

为了核实你是否知道 Azure RemoteApp 将于 2017 年 8 月 31 日停用，我们一直尝试联系你和其他 Azure 订阅管理员。我们[于 2016 年 8 月 12 日宣布了 Azure RemoteApp 即将停用的消息](http://aka.ms/araretirement)。   我们迫切需要你回复 [arainfo@microsoft.com](mailto:arainfo@microsoft.com)，请简要说明：
1.  你和组织确认已了解 2017 年 8 月 31 日停用的消息
2.  请告知我们你的迁移计划 - 如果需要，我们可以提供帮助。 [迁出 Azure RemoteApp 的选项](http://aka.ms/aramigration)中详细介绍了迁移选项，如 Citrix XenApp Essentials、MyCloudIT、Frame 等等。 

Azure 管理门户中的 REMOTEAPP 节点可能已被阻止，这么做是因为我们无法与你取得联系，因此不得不强制你参与。  如果未接收到我们的电子邮件，可能是因为与 Azure 订阅关联的电子邮件地址未受监视或者无效。  我们强烈建议对与 Azure 订阅关联的电子邮件地址进行验证，确保其有效且受到监视，方法是导航到 Azure 门户并验证订阅管理员。  

仍有疑问或需要迁移支持？  请发送电子邮件至 [arainfo@microsoft.com](mailto:arainfo@microsoft.com)，或立即与我们的转换团队[预订会议](http://aka.ms/ericorman)。
