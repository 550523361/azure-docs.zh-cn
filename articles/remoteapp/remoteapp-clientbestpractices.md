# <a name="best-practices-for-azure-remoteapp-clients"></a>Azure RemoteApp 客户端最佳实践
> [!IMPORTANT]
> Azure RemoteApp 将于 2017 年 8 月 31 日停用。 阅读 [公告](https://go.microsoft.com/fwlink/?linkid=821148) 了解详细信息。
> 
> 

以下信息可帮助你使用 Azure RemoteApp 客户端：

* 始终使用最新的客户端。 这确保你运行的客户端版本具有最新的错误修复方案、改进和功能。 你可能需要进行注册以便在相应的应用商店中自动接收客户端的更新。
* 如果你在一段时间内处于非活动状态，RemoteApp 将自动注销。 为了防止数据丢失，我们建议你在完成服务使用后关闭应用程序。
