# 如何：检查作业进度

本文是 [Media Services 视频点播工作流](/zh-cn/documentation/articles/media-services-video-on-demand-workflow) 系列教程的一部分。 

## 使用门户检查作业状态

1. 在[管理门户](http://manage.windowsazure.cn)中，单击**Media Services**，然后单击 Media Services 帐户名。
2. 选择**作业**页。 

	在该页上，你可以看到所有作业的状态。你还可以按状态、时间或作业 ID 进行筛选。
