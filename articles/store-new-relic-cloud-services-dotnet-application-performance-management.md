# <a name="new-relic-application-performance-management-on-azure"></a>Azure 中的 New Relic 应用程序性能管理
可以在 Azure 托管的应用程序中添加 New Relic 提供的高级性能监视功能。 除了获得全面的 Azure 应用监视、故障排除和优化功能外，还能通过 Azure 享受 New Relic 产品的折扣价格。

## <a name="what-is-new-relic"></a>什么是 New Relic？
使用 [New Relic 产品](https://newrelic.com/products)，可以解决应用错误，提前发现潜在问题，了解整个环境的性能。 其目的在于缩短您识别和诊断性能问题的时间，并使您能够轻松获得解决这些问题所需的信息。

New Relic 可从服务器和用户的浏览器跟踪 Web 事务的加载时间和吞吐量。 它可显示您在数据库方面花费的时间，分析缓慢查询和 Web 请求，提供运行时间监视和通知，跟踪应用程序异常，还有其他更多功能。 

## <a name="special-pricing"></a>特价
New Relic Standard 可供 Azure 用户免费使用。 New Relic Pro 根据 Azure 云服务的实例大小供应。 有关定价信息，请参阅 Azure 应用商店中的 [New Relic 页](https://azure.microsoft.com/marketplace/partners/newrelic/newrelic/)，或联系 New Relic (sales@newrelic.com) 获取企业级定价。

Azure 客户在部署 New Relic 代理时可获得为期两周的 New Relic Pro 试用订阅。

## <a name="sign-up-for-new-relic-using-the-azure-store"></a>使用 Azure 应用商店注册 New Relic
New Relic 可与 Azure Web 角色和辅助角色无缝集成。 可以直接从 Azure 应用商店快速轻松注册 New Relic。 有关说明，请参阅 New Relic 提供的 [Azure 应用商店注册说明](https://docs.newrelic.com/docs/agents/net-agent/azure-installation/azure-cloud-services#signup)。

## <a name="view-your-data"></a>查看数据
注册后，可以利用 New Relic 的优异应用程序监视功能和数据驱动型分析。 若要在 New Relic 中检查应用的性能，请执行以下操作：

1. 在 Azure 门户中选择“管理”。
2. 使用你的 New Relic 帐户电子邮件地址和密码进行登录。
3. 从应用程序索引中选择应用，在 [APM 概述页](https://docs.newrelic.com/docs/apm/applications-menu/monitoring/apm-overview-page)中查看所有应用的数据。

## <a name="using-new-relic-with-azure"></a>将 New Relic 与 Azure 配合使用
有关将 New Relic 与 Azure 配合使用的详细信息，请参阅 [New Relic 的文档站点](https://docs.newrelic.com/docs/agents/net-agent/azure-installation)，其中包括： 

* [New Relic for .NET](https://docs.newrelic.com/docs/agents/net-agent/getting-started/new-relic-net)（用于 .NET 的 New Relic）

* [New Relic for the Microsoft Azure Cloud platform](https://docs.newrelic.com/docs/agents/net-agent/azure-installation/azure-cloud-services)（适用于 Microsoft Azure 云平台的 New Relic）
* [New Relic for the Microsoft Azure App Services](https://docs.newrelic.com/docs/agents/net-agent/azure-installation/azure-portal)（适用于 Microsoft Azure 应用服务的 New Relic）
* [New Relic/Azure troubleshooting](https://docs.newrelic.com/docs/agents/net-agent/azure-troubleshooting)（New Relic/Azure 故障排除）
