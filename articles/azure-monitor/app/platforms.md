## <a name="languages---community-supported"></a>语言 - 社区支持

有大量的社区支持 Application Insights Sdk，其中许多最初由 Microsoft 创作。 社区支持的 Sdk 不会正式保留由 Microsoft 和我们不能提供对不在受支持列表任何 SDK 的支持。 这些 Sdk 将被视为处于实验阶段，并不建议用于生产环境。
