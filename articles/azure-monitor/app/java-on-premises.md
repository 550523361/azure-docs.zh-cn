# <a name="java-codeless-application-monitoring-on-premises---azure-monitor-application-insights---public-preview"></a>本地无代码应用程序监视 - Azure 监视器应用程序见解 - 公共预览

Java 无代码应用程序监视是简单性的 - 没有代码更改，Java代理只能通过几个配置更改来启用。

## <a name="overview"></a>概述

启用后，Java 代理将自动从使用最广泛的库和框架收集大量请求、依赖项、日志和指标。

请按照所有环境（包括本地）的[详细说明进行](https://docs.microsoft.com/azure/azure-monitor/app/java-in-process-agent)操作。
