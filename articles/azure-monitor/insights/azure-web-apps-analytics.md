> [!NOTE]
> Azure Web 应用分析解决方案已移到社区支持。  
>- 该解决方案不再可从 Azure 市场获得，但可以从社区支持的 [Azure 快速入门模板](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/)安装。
