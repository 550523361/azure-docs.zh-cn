# <a name="prepare-data-with-the-azure-machine-learning-data-prep-sdk-for-python"></a>使用适用于 Python 的 Azure 机器学习数据准备 SDK 准备数据
 
数据准备是机器学习工作流的重要部分。 如果模型有权清理更易于使用的格式中的数据，那么它们将会更加准确和高效。 可使用 [Azure 机器学习数据准备 SDK](https://docs.microsoft.com/python/api/overview/azure/dataprep?view=azure-dataprep-py) 来加载各种格式的数据，将其转换使其更易于使用，并将该数据写入模型可访问的位置。 


## <a name="steps-for-data-preparation-with-azure-machine-learning"></a>使用 Azure 机器学习进行数据准备的步骤 
1. [加载数据](how-to-load-data.md )
2. [转换数据](how-to-transform-data.md )
3. [写入数据](how-to-write-data.md )
