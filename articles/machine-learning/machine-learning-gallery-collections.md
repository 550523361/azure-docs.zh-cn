# <a name="discover-and-share-collections-in-the-cortana-intelligence-gallery"></a>在 Cortana Intelligence 库中发现并共享收藏
[!INCLUDE [machine-learning-gallery-item-selector](../../includes/machine-learning-gallery-item-selector.md)]

## <a name="collections"></a>集合
使用**[收藏](https://gallery.cortanaintelligence.com/collections)**，可让你将提出特定解决方案或概念的实验、API 以及其他库项组合在一起。 这些项可以组合在一起供以后参考、使用或共享。

## <a name="discover"></a>发现
若要浏览库中的收藏，请打开“[库](http://gallery.cortanaintelligence.com)”，将鼠标指向库主页顶部的“**更多**”，然后选择“**收藏**”。

![从库主页中选择收藏](media/machine-learning-gallery-collections/select-collections-in-gallery.png)

 “**[收藏](https://gallery.cortanaintelligence.com/collections)**”
页显示最近添加的和最受欢迎的收藏列表。
单击“**查看所有**”，以查看所有收藏。
在本页中，可以浏览库中的所有集合。 还可在页面左侧选择筛选条件并在顶部输入搜索词进行搜索。

 单击任何收藏，可打开该收藏的详细信息页并了解关于该收藏的详细信息。
你可以在此页进行评论、提供反馈，或通过评论部分提问。 甚至可以使用 LinkedIn 或 Twitter 的共享功能将其与朋友或同事共享。 同时，你也可以通过电子邮件发送指向该实验的链接，邀请其他用户查看此页面。

![与朋友分享此项](media/machine-learning-gallery-how-to-use-contribute-publish/share-links.png)

![添加你自己的备注](media/machine-learning-gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>参与
你可以创建包含你所拥有的项或由社区提供的项的收藏。 它们可以是提出特定解决方案或概念的任意库项

例如，你可以使用收藏将有关特定主题的项组合在一起，或者你可以将解决复杂问题的多步骤实验组合在一起。
由 Microsoft 提供的初始收藏包含用于解决实际问题（例如在线欺诈检测、文本分类、零售预测和预防性维护）的多步骤机器学习实验模板。

### <a name="create-a-collection"></a>创建集合
请按照下列步骤为 Cortana Intelligence 库创建并提供收藏：

1. 使用你的 Microsoft 帐户登录到库

2.  单击窗口顶部的图像，然后单击你的姓名
  
    ![单击帐户名](media/machine-learning-gallery-collections/click-account-name.png)

3. 单击“新建项”
   
    ![单击“新建项”](media/machine-learning-gallery-collections/click-new-item.png)
4. 为“**项类型**”选择“**收藏**”，然后为收藏提供名称、简短摘要、说明，以及将有助于用户找到收藏的任何标记
   
    ![输入新收藏的信息](media/machine-learning-gallery-collections/create-collection-page-1.png)
5. 单击“下一步”，你可以上载图像文件，或选择将与收藏一起显示的库存图像；选择将有助于用户识别内容和收藏用途的内容
   
    ![输入新收藏的信息](media/machine-learning-gallery-collections/create-collection-page-2.png)
6. 单击“**下一步**”，你可以决定收藏是“**公共**”（任何人都可以查看）还是“**未列出**”（仅限具有直接链接的人可以查看收藏）
   
   > [!IMPORTANT]
   > 一旦将收藏设置为“**公共**”，就不可以再将其设置为“**未列出**”。
   > 
   > 
   
    ![选择“公共”或“未列出”](media/machine-learning-gallery-collections/create-collection-page-3.png)
7. 单击“创建” 

你的收藏现在是 Cortana Intelligence 库的一部分了。 它将在你帐户页的“**收藏**”选项卡下列出。

### <a name="add-items-to-a-collection"></a>向收藏添加项
可以通过打开收藏，单击“**编辑**” ，然后单击“**添加项**”向收藏添加项。

![通过详细信息页向收藏添加项](media/machine-learning-gallery-collections/add-to-collection-from-details-page.png)

你将会看到你向库中提供的项的列表，或者你可以在库中搜索要添加的项。 单击某个项将其选中。 你选择的每个项包含在要添加的项集中 -“**添加**”按钮指示已选择的项数。

![添加到收藏](media/machine-learning-gallery-collections/add-to-collection.png)

或者，如果在库中浏览要包括的项时发现某个项，只需打开该项，单击“**添加到收藏**”，并指定你想要将其添加到的收藏即可。

![通过详细信息页向收藏添加项](media/machine-learning-gallery-collections/add-to-collection-from-item-details.png)

可以通过打开收藏，并单击“**编辑**”更改收藏的摘要、说明或标记。
编辑你的收藏时，还可以通过使用项旁边的箭头按钮更改项在收藏中的顺序，从而在列表中移动该项。 通过单击项的左上角并选择“**添加/编辑备注**”向收藏中的项添加备注。 若要从收藏中移除一个项，请选择“**移除**”。

![更改收藏的详细信息](media/machine-learning-gallery-collections/change-collection-details.png)

**[将我转到库 >>](http://gallery.cortanaintelligence.com)**
