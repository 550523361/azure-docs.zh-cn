# <a name="discover-and-share-tutorials-in-cortana-intelligence-gallery"></a>在 Cortana Intelligence 库中发现并共享教程
[!INCLUDE [machine-learning-gallery-item-selector](../../includes/machine-learning-gallery-item-selector.md)]

## <a name="tutorials"></a>教程
Cortana Intelligence 库提供了多个[教程](https://gallery.cortanaintelligence.com/tutorials)，可向用户演示机器学习技术和概念。 教程还可能介绍用于解决机器学习问题的高级方法。

## <a name="discover"></a>发现
若要浏览[库中](http://gallery.cortanaintelligence.com)的教程，请在“更多”下选择“教程”。

![在库主页上选择“教程”](media/machine-learning-gallery-tutorials/select-tutorials-in-gallery.png)

“[教程](https://gallery.cortanaintelligence.com/tutorials)”页显示最近添加的教程和热门教程列表。 若要查看所有教程，请选择“查看全部”按钮。 若要搜索特定教程，请选择“查看全部”，然后选择筛选条件。 也可以在库页面顶部的“搜索”框中输入搜索项。

可在教程详细信息页上获取有关教程的详细信息。 若要打开某个教程详细信息页，请选择教程。 在教程详细信息页的“评论”部分中，可对教程进行评论、提供反馈或提问。 甚至可以在 Twitter 或 LinkedIn 上与好友或同事分享教程。 还可以通过电子邮件发送指向教程详细信息页的链接，来邀请其他用户查看该页面。

![与朋友分享此项](media/machine-learning-gallery-how-to-use-contribute-publish/share-links.png)

![添加你自己的备注](media/machine-learning-gallery-how-to-use-contribute-publish/comments.png)

## <a name="contribute"></a>参与
可以将教程添加到库，来帮助其他用户解决问题或学习概念。

### <a name="create-a-tutorial"></a>创建教程

1. 使用 Microsoft 帐户登录到库。

2. 选择页面顶部的图像，然后选择你的姓名。
  
    ![选择帐户名](media/machine-learning-gallery-tutorials/click-account-name.png)

3. 选择“新建项”。
  
    ![选择“新建项”](media/machine-learning-gallery-collections/click-new-item.png)

4. 在“说明”页上，为“项类型”选择“教程”。 输入教程的名称、简短摘要、详细说明和任何可能有助于其他用户找到教程的标记。
  
    ![输入新教程的信息](media/machine-learning-gallery-tutorials/create-tutorial-page-1.png)
5. 在“图像选择”页上，选择随收藏一同显示的图像。 可以上传自己的图像文件，或者选择内置图像。 选择可帮助用户标识教程内容和用途的图像。
  
    ![输入新教程的信息](media/machine-learning-gallery-tutorials/create-tutorial-page-2.png)

6. 在“设置”页上，针对“可见性”选择教程是“公开”（任何人都可以查看教程）还是“未列出”（仅具有指向教程的直接链接的人员才可以查看它）。
  
    > [!IMPORTANT]
    > 将教程设置为“公开”并选择“创建”后，以后不能将教程更改为“未列出”。
    > 
    > 
  
    ![选择“公开”或“未列出”](media/machine-learning-gallery-tutorials/create-tutorial-page-3.png)

7. 选择“创建” 。

教程现在位于 Cortana Intelligence 库中。 教程将在帐户页的“项”选项卡上列出。

**[转到库](http://gallery.cortanaintelligence.com)**
