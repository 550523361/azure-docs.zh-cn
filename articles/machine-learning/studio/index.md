# <a name="azure-machine-learning-studio-documentation"></a>Azure 机器学习工作室文档

Azure 机器学习工作室是一个协作型拖放式工具，可用于根据数据构建、测试和部署预测分析解决方案。
教程、视频和示例模型演示了如何使用工作室来生成及部署机器学习模型。
