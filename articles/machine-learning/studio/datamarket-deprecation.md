# <a name="datamarket-is-now-deprecated"></a>DataMarket 现已弃用

[Microsoft DataMarket](https://datamarket.azure.com/datasets) 现已弃用，因此托管在此处的机器学习 API 现也已弃用。

现可在[认知服务 API](https://www.microsoft.com/cognitive-services) 中找到这些功能（以及更多）。

## <a name="text-analytics-api"></a>文本分析 API

以前的 DataMarket 文本分析 API 客户应开始使用[认知服务文本分析 API](https://www.microsoft.com/cognitive-services/text-analytics-api)。
认知服务文本分析 API 支持早期 DataMarket 版本支持的所有方案，以及语言检测和主题检测等其他方案。
