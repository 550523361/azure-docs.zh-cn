# <a name="visualize-graph-data-stored-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>通过数据可视化解决方案将存储在 Azure Cosmos DB Gremlin API 中的图形数据可视化

可以使用各种数据可视化解决方案来可视化 Azure Cosmos DB Gremlin API 中存储的数据。 用于图形数据可视化的[Apache Tinkerpop 团体](https://tinkerpop.apache.org/#poweredby)建议使用以下解决方案。

## <a name="linkurious-enterprise"></a>Linkurious 企业

![Linkurious 徽标](./media/graph-visualization/linkurious-logo.jpg)

[Linkurious Enterprise](https://linkurio.us/product/)使用图形技术和数据可视化将复杂数据集转换为交互式视觉对象。 该平台连接到数据源，并使调查人员无缝地跨数十亿的实体和关系进行导航。 结果是检测可疑关系的新功能，而不使用查询或表杂耍。

Linkurious Enterprise 的交互接口提供了一种简单的方法来调查复杂数据。 您可以搜索特定实体，展开 "连接" 以发现隐藏的关系，并将所选的布局应用于解开复杂的网络。 Linkurious Enterprise 现在与 Azure Cosmos DB Gremlin API 兼容。 它适用于端到端图形可视化方案，并支持用户界面中的读取和写入功能。 你可以[使用 Azure Cosmos DB 来请求 Linkurious 演示](https://linkurio.us/contact/)

![Linkurious 演示](./media/graph-visualization/linkurious-demo.gif)

## <a name="cambridge-intelligence"></a>剑桥智能

![剑桥智能徽标](./media/graph-visualization/ci-logo.png)

[剑桥智能的](https://cambridge-intelligence.com/products/)图形可视化工具包现在支持 Azure Cosmos DB。 Azure Cosmos DB 支持以下两个可视化工具包：

- [适用于 JavaScript 开发人员的 KeyLines](https://cambridge-intelligence.com/keylines/)

- [重新绘制响应开发人员的图形](https://cambridge-intelligence.com/regraph/)

![剑桥智能演示](./media/graph-visualization/ci-demo-2.gif)

这些工具包使你可以为用例设计高性能图形可视化效果和分析应用程序。 它们充分利用了强大的 Web 图形库（WebGL）呈现和精心设计的代码，使用户能够快速、富有视觉的可视化体验。 这些工具与任何浏览器、设备、服务器或数据库兼容，并附带分步教程、完整记录的 Api 和交互式演示。

![剑桥智能演示](./media/graph-visualization/ci-demo-1.gif)
