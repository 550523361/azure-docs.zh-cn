# <a name="tan-azure-cosmos-db"></a>TAN (Azure Cosmos DB)
 返回指定表达式中指定角度（弧度）的正切值。  
  
## <a name="syntax"></a>语法
  
```sql
TAN (<numeric_expr>)  
```  
  
## <a name="arguments"></a>参数
  
*numeric_expr*  
   为数值表达式。  
  
## <a name="return-types"></a>返回类型
  
  返回数值表达式。  
  
## <a name="examples"></a>示例
  
  以下示例可计算 PI()/2 的正切值。  
  
```sql
SELECT TAN(PI()/2) AS tan 
```  
  
 下面是结果集：  
  
```json
[{"tan": 16331239353195370 }]  
```  

## <a name="remarks"></a>备注

此系统函数不会使用索引。
