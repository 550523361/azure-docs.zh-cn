# <a name="azure-resource-health-documentation"></a>Azure 资源运行状况文档

资源运行状况在 Azure 问题影响你的资源时帮助你进行诊断并获得支持。 它通知你有关资源的当前和过去运行状况的信息，并帮助你缓解问题。 在你需要有关 Azure 服务问题的帮助时，资源运行状况将提供技术支持。
