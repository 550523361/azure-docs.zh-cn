# <a name="integration-patterns"></a>集成模式

Azure FarmBeats 是一种企业对企业产品，可在 Azure 应用商店中提供。 FarmBeats 通过融合数据集，通过构建人工智能 （AI） 或机器学习 （ML） 模型，实现跨提供程序聚合农业数据集，并生成可操作的见解。

![FarmBeats 项目](./media/architecture-for-farmbeats/farmbeats-architecture-1.png)

以下各节介绍 Azure FarmBeats 的集成模式。

## <a name="why-integrate-with-azure-farmbeats"></a>为什么要与 Azure 服务器场节拍集成？

本节重点介绍希望将其数据系统集成到 Azure FarmBeats 中的合作伙伴。

Azure FarmBeats 是一种可扩展的产品，它使农业企业能够将不同的历史和实时农业数据集添加到单个平台中。 Azure FarmBeats 可帮助农业企业在服务器场的上下文中规范化、上下文化和聚合其数据。

通过成为 Azure FarmBeats 的数据合作伙伴，您可以打开系统以更广泛地采用，并通过数据产品联系更多客户。 Azure FarmBeats 提供了一个称为 Datahub 的可扩展 API 层，它可帮助您系统地将设备中的数据引入到标准化架构中。

在客户的 Azure FarmBeats 实例中提供数据后，您的客户可以在数据的基础上构建更丰富的分析和工具。
