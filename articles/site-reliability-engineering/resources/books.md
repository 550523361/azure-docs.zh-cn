# <a name="books-about-site-reliability-engineering"></a>关于现场可靠性工程的书籍

有关站点可靠性工程 （SRE） 的更多详细信息，最佳来源是已就此问题出版的三本书

- [网站可靠性工程：Google 如何运行生产系统__](https://shop.oreilly.com/product/0636920041528.do)（称为“SRE 书籍”）
- [网站可靠性工作簿：切实可行的 SRE 实现方法__](https://shop.oreilly.com/product/0636920132448.do)（称为“SRE 工作簿”）
- [探求 SRE：有关批量运行生产系统的对话__](https://shop.oreilly.com/product/0636920063964.do)

每本图书都提供一系列重要信息：

- SRE 书籍 - 详细说明了多年来 Google 是如何实现 SRE 的。

- SRE 工作簿 - 作为 SRE 书籍的配套指南，不仅更详细地说明了 Google 和其他一些地方实现的 SRE，还更详细地说明了实现方式和原因。

- 探求 SRE - 更全面地概览了 SRE 领域，除了介绍起源外，还介绍了其他环境是如何实现 SRE 的。

由于这些书描述了组织的经验、环境和文化，它们可能与您一样，也可能不类似您所从事，因此，以批判的眼光阅读这些书籍非常重要。 阅读时，尝试确定哪些做法将在组织中成功或不会成功。 请花些时间甄别你确信可带来一些积极价值的信息。 仔细想想组织文化和价值观中的哪些部分可能会支持 SRE 按规定运行，以及哪些部分可能会带来更多挑战。 仔细的、反复的SRE拥抱几乎总是比在这些书中读到的东西的彻底复制产生更好的结果。
