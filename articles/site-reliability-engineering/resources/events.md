# <a name="participate-in-the-global-site-reliability-engineering-community"></a>参与全球站点可靠性工程社区

虽然阅读 SRE 相关图书很重要，但与志同道合的人谈论 SRE 通常可能更为重要。 围绕 SRE 讨论挑战、成功与失败，对深入细致了解这个主题有着举足轻重的作用。

以 SRE 内容为主题的聚会和会议有很多。 也许最直接相关的是 USENIX 举办的全球分布[的 SREcon 会议](https://www.usenix.org/conferences)。

越来越多的SRE内容正在通过会议，如[LISA，](https://www.usenix.org/conferences/byname/5)[全天DevOps，DevOps](https://www.alldaydevops.com)[企业峰会](https://events.itrevolution.com)和本地DevOps会议，如[DevOps天](https://www.devopsdays.org)。 无论在哪里找到这个主题，都可以查找此内容以及对主题感兴趣的其他人。
