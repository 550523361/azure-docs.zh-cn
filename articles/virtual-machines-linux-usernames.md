# 在 Azure 上选择 Linux 的用户名

在 Azure 上创建针对 Linux 映像的虚拟机实例后，你便能选择用户名进行设置。

Linux 定义了一组你无法使用的用户名。**请避免使用这些名称**，这一点很重要。如果你使用这些用户名之一，则将无法设置映像。

此外，服务管理 API 将返回一个错误，指明：未能创建用户 *xxxxxx*。此外，如果你因未取消设置已在映像中创建的用户名的上一个捕获操作而使用映像中已有的用户名，则将出现此情况。
