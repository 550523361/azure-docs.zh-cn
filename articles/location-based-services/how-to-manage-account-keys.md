# <a name="how-to-manage-your-azure-location-based-services-account-and-keys"></a>如何管理 Azure Location Based Services 帐户和密钥

可以通过 Azure 门户管理你的 Azure Location Based Services 帐户和密钥。 获得帐户和订阅密钥后，可以在你的网站或移动应用程序中实现 API。

如果你还没有 Azure 订阅，可以在开始前创建一个 [免费帐户](https://azure.microsoft.com/free/?WT.mc_id=A261C142F)。

## <a name="log-in-to-azure"></a>登录 Azure 

通过 http://portal.azure.com 登录到 Azure 门户。

## <a name="create-a-new-account"></a>创建新帐户

1. 单击 Azure 门户左上角的“新建”按钮。

2. 搜索并选择“Location Based Services”，然后单击“创建”。

3. 输入 Location Based Services 帐户信息。 

![在门户中输入帐户信息](./media/how-to-manage-account-keys/new-account-portal.png)

## <a name="manage-keys-on-the-account-page"></a>在帐户页上管理密钥

在左侧菜单的“设置”下，单击“密钥”。

![在门户中管理帐户密钥](./media/how-to-manage-account-keys/account-keys-portal.png)

使用密钥可以针对 Azure Location Based Services 进行身份验证并开始开发基于位置的应用程序。

## <a name="delete-an-account"></a>删除帐户

你可以通过在顶部导航窗格上单击“删除”按钮删除帐户。

![在门户中删除帐户](./media/how-to-manage-account-keys/account-delete-portal.png)
