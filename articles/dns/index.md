#<a name="dns-documentation"></a>DNS 文档

Azure DNS（域名系统）是一个 DNS 域服务，它使用 Microsoft Azure 基础结构提供名称解析。  通过在 Azure 中托管你的域，你可以使用与其他 Azure 服务相同的凭据、API、工具和计费来管理你的 DNS 记录。
