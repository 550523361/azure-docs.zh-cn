# <a name="subscribe-to-an-offer"></a>订阅产品

*适用范围： Azure 堆栈集成系统和 Azure 堆栈开发工具包*

既然你已经[创建提议](azure-stack-create-offer.md)，测试你的用户可以创建订阅。

1. [登录](azure-stack-connect-azure-stack.md)至 Azure 堆栈用户门户 (https://portal.local.azurestack.external)，并选中**获取订阅**。

   ![](media/azure-stack-subscribe-plan-provision-vm/image01.png)
2. 在**显示名称**字段中，键入你的订阅的名称，单击**提供**，单击一个产品/服务**选择提议**边栏选项卡，，然后单击**创建**。

   ![](media/azure-stack-subscribe-plan-provision-vm/image02.png)
3. 若要查看你创建的订阅，请单击**更多的服务**，单击**订阅**，然后单击你的新订阅。  

在订阅产品/服务后，刷新门户后，若要确定哪些服务正在新订阅的一部分。

## <a name="subscribe-to-an-add-on-plan"></a>订阅外接程序计划
如果产品具有的外接程序计划，用户可以将它们添加到其订阅在任何时间。  

1. 在用户门户中，选择**更多的服务** > **订阅**。

2. 单击订阅 >**添加计划**按钮，然后选择外接程序计划。
