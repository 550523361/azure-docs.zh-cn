#<a name="documentdb-documentation"></a>DocumentDB 文档

Azure DocumentDB 是一个完全托管的 NoSQL 数据库服务，其构建目的是为了实现快速且可预测的性能，实现高度可用、弹性缩放、全局分发和简易开发。 作为一种无架构的 NoSQL 数据库，DocumentDB 提供丰富且流行的 SQL 查询功能，在传输 JSON 数据过程中可持续保持较低的延迟。
