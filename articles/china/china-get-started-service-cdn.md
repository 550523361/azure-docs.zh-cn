# <a name="azure-content-delivery-network"></a>Azure 内容交付网络
使用世纪互联运营的 Microsoft Azure（Azure 中国世纪互联）托管涉及流媒体、游戏软件、固件更新（例如智能电视和消费型电子设备）或物联网 (IoT) 终结点（例如汽车和传感器）的网站或移动应用时，内容交付网络 (CDN) 有助于减少加载时间、节省带宽和提升响应能力。

本地提供商在中国提供 CDN 服务。 （相比之下，适用于全球 Azure 的 [Azure CDN](https://azure.microsoft.com/documentation/articles/cdn-overview/) 服务通过 Akamai 和 Verizon 提供选项。）目前，可以选择六家本地 CDN 提供商，其中包括 ChinaCache Webluker、[ChineNetCenter](http://www.chinanetcenter.com/) 和 [Fastweb](http://www.fastweb.com.cn/)，将来会有更多的提供商。

若要使用 Azure CDN 服务，请执行以下操作： 
- [从世纪互联获取支持](/azure/china/china-resources-support)。世纪互联是所有 Azure 服务（包括 Azure CDN）的单一联系点。
- 在 Azure [统一 CDN 管理门户](https://www.azure.cn/documentation/articles/cdn-management-portal-how-to-use/)（不是全球 Azure CDN 门户）上登录。
