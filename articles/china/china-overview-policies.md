# <a name="china-cloud-policies-and-regulations"></a>中国云政策和法规
若要在中国推出 Internet 服务，组织必须遵守适用的中国云政策和法律。 根据这些规定，Internet 内容提供商 (ICP) 必须向中国工业和信息化部 (MIIT) 申请以下类型的 [ICP 号码](https://en.wikipedia.org/wiki/ICP_license)之一：
- **商业 ICP 服务需要 ICP 执照。** 在中国提供商业 ICP 服务的所有运营商必须从 MIIT 或其分支机构获取运营执照。 商业服务定义为增值电信服务类型，包括通过在线广告、文本、图像、音频、视频和应用衍生直接收入的任何网站。 提供信息搜索服务、信息社区平台、即时消息服务和信息保护服务也被视为商业服务。 ICP 执照号遵循“京ICP证12345678号”格式（其中，“京”表示“北京”）。
- **非商业 ICP 服务必须提交 ICP 备案。** 若要运营非商业 ICP 服务，必须向 MIIT 或其分支机构提交一份称为“ICP 备案”的记录。 ICP 备案适用于提供免费内容的非商业性、纯信息性网站。 ICP 备案号遵循“京ICP备12345678号”格式（其中，“京”表示“北京”）。 对于非商业网站，应先购买主机空间和域名，再提交 ICP 备案，然后才能让网站上线。

ICP 备案号以小号字体显示在网站主页的页脚中，并且应一直显示在网站上。 如果未获得 ICP 执照或提交 ICP 备案记录，请不要推出网站， 否则会关停该网站。

内部运营（包括云运营）在中国受到高度管制。 组织必须遵守会对购买体验（包括注册网站和注册公共帐户）产生影响的法规。 

以下机构参与 Azure 主机空间相关的决策：
- 具体 ICT 和网络安全法律与法规的重要政府制定者包括全国人民代表大会常务委员会、国务院、中国网络空间管理局、MIIT、公安部 (MPS) 和国家安全部 (MNS)。
- 全国人民代表大会和全国人民代表大会常务委员会是审批中国法律（例如中国反恐法和网络安全法）的重要组织。 国务院及其相关部门（例如 MIIT）通常负责制定并审批 ICT 和新兴云服务领域方面的法规。
- MPS 和 MNS 是执法机构，重心工作是执法和防范非法行为。 MPS 也会颁布自身的法规。 

这些基本的背景信息旨在帮助大家做好准备工作。 有关更新的和官方的法律与法规，请参考每家中国政府组织的办事处公告。 

## <a name="step-1-icp-license-and-filing"></a>步骤 1：ICP 执照和备案
若要在位于中国大陆的服务器上托管网站，必须申请国家签发的注册号：适用于商业实体的 ICP 执照号，或适用于非商业实体的 ICP 备案号。

共用 ICP 执照是违反中国法律和法规的行为。 违反此法律的提供商随时可能会连同其托管的网站一起被关停，且事先不会收到通知。 此外，如果组织共用 ICP 执照，则不再受允许在中国的网络空间中运营。 这包括网站、主机空间甚至域名服务。 在这种情况下，如果提供商破产或停业，享有的法律追索权极少甚至没有。

申请要求和审查流程根据[省/自治区/直辖市](https://wacndevelop.blob.core.chinacloudapi.cn/marketing-resource/documents/special-request.pdf)的不同而异。 例如，在上海和重庆，该流程需要大约三天时间。 在甘肃和云南，该流程需要三周时间。 我们建议提前两个月做好规划，因为在提交申请文件后，ICP 执照的审批通常至少需要 20 个工作日。

审查申请和结果会发布在[世纪互联申请网站](http://icp.cloud.21vianet.com/)上，同时，世纪互联会通过电子邮件或电话通知你。 以下示例概述了适用于上海市注册的企业的流程步骤。

### <a name="apply-for-icp"></a>申请 ICP
1.  使用电子邮件地址注册新帐户，并登录到[申请网站](http://icp.cloud.21vianet.com/)。 
2.  遵照说明填写 ICP 备案的必要信息，包括有关网站和 ICP 的详细信息。
3.  以彩色扫描件或照片的形式上传所需材料的软拷贝（电子版）。
4.  在两个工作日内，会将首次审查反馈发送给你。 如果申请通过了首次审查，你会收到一封电子邮件，要求前往北京或上海的世纪互联办事处参加现场访谈，地址如下：
    - 北京市 朝阳区 酒仙桥东路 1 号<br>
    电话：400-089-0365
    - 上海市浦东区 科苑路 88 号 3 栋 657 房<br>
    电话：400-089-0365 <br>
    如果无法亲自前往，世纪互联可以使用其支持电话号码开展场外访谈。
5.  提交的材料将会发送给世纪互联开展第二次审查，此过程大约需要 1.5 个工作日。 通过此次审查后，所有材料将提供给 MIIT 做最后一次审查。 如果文件被 MIIT 拒绝，可以修订并重新提交。

### <a name="qualifications-for-icp"></a>ICP 合格要求
若要获取 ICP 执照或 ICP 备案号，可能需要在中国建立合法身份。
 
若要符合 ICP 备案要求，可能需要满足以下条件：
- 在中国注册的、持有中国公司执照的实体可以申请 ICP 备案，这包括持有任何类型的中国公司执照的中外合资或外商独资（非中国）公司实体，例如合资企业或外商独资企业 (WFOE)。 
- 中国公民可以使用国家签发的身份证申请 ICP 备案。 外国（非中国）个人可以使用其护照作为身份证申请，但在中国的实际居留时间必须足够长，才能满足法律要求。

以下实体**不可以**申请 ICP：
- 在中国没有合法商业身份的外国公司。
- 没有护照的外国个人（以及无法实际居留在中国的个人）。

## <a name="step-2-public-security-registration"></a>步骤 2：公安注册
在收到 ICP 执照后的 30 天内，还必须向当地派出所申请备案。 可以直接使用[注册门户](http://www.beian.gov.cn/portal/index)申请，而无需通过世纪互联申请。
1. 若要开始提交在线申请材料，请在[注册门户](http://www.beian.gov.cn/portal/index)中注册新帐户（中文版，可能需要求助英文翻译）。
2. 遵照说明填写备案的必要信息，包括网站和 ICP 详细信息。 
3. 通过在线申请审查后，相关机构会告知准备[所需材料](https://www.azure.cn/support/announcement/public-security-registration/process/)，以继续参加现场审查。 审查地点取决于实体的注册地址，或身份证上的用于网站主机空间的个人家庭注册地址。 此项审查大约需要两周时间。
4. 通过材料审查后，相关机构会分配授权代码并在 [www.beian.gov.cn](http://www.beian.gov.cn) 上公布。
