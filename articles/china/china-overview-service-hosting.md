# <a name="service-hosting-for-websites-in-china"></a>中国的网站服务托管
组织经常会问到，是要将网站和服务托管在中国大陆的境内还是境外。 答案取决于中国是否是你的主要目标市场之一。 如果网站必须要让访客快速方便地访问并托管在本地，则应完成政府法规申请流程。

可将网站托管在中国境外， 优点是无需在中国设立公司，且无需经历 ICP 备案过程。 但是，也要考虑到这种做法的几个缺点：
- 在中国不一定每次都能顺利地访问该网站。 即使网站已翻译成中文，也可能很难在百度（中国最大的搜索引擎）中找到它。
- 不是在中国大陆托管的网站可能会被封锁。 受禁网站的列表随时可能变化。 
- 如果网站未托管在专用服务器上，而另一个网站托管在受禁的共享服务器上，则整个服务器（包括网站）可能会被封锁。 
- 用户无法获得最佳体验。 网络延迟可能会减慢中国境外托管的服务的访问速度，即使服务已专门针对低带宽和移动访问进行优化。

### <a name="compliance"></a>合规性
为了帮助客户及其公司遵守有关收集和使用个人数据的国家、地区和行业特定要求，世纪互联提供了全面的[认证和证明](https://www.trustcenter.cn/en-us/compliance/)。

世纪互联专家团队与 Microsoft 工程和运营团队以及外部监管机构密切合作，跟进现有的标准和法规，并针对在本公司的云服务中构建解决方案的产品团队制定了数百个管制条例。 法规和标准不断演变，因此，我们的法规专家会尽力预判这些变化，帮助确保持续合规。 他们会研究法规草案，评估潜在的新要求，并制定相应的管制条例。

在信任中心网站（[英文](https://www.trustcenter.cn/en-us/)或[中文](https://www.azure.cn/zh-cn/support/trust-center/)版）上了解世纪互联运营的 Microsoft Azure（Azure 中国世纪互联）的隐私声明、管制条例与合规性。
