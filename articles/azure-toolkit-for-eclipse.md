# <a name="azure-toolkit-for-eclipse"></a>用于 Eclipse 的 Azure 工具包
使用 Azure Toolkit for Eclipse 提供的模板和功能，可轻松地利用 Eclipse 开发环境创建、开发、测试和部署 Azure 应用程序。 它是一个开放源代码项目，其源代码可从项目站点的 MIT 许可证下获取，网址为：

* <https://github.com/microsoft/azure-tools-for-java>

> [!NOTE]
> 另外还有 Azure Toolkit for IntelliJ IDE。 有关详细信息，请参阅[用于 IntelliJ 的 Azure 工具包]。
>
