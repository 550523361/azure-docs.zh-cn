#<a name="application-gateway-documentation"></a>应用程序网关文档

Azure 应用程序网关以服务形式提供应用程序传送控制器 (ADC)。  它使客户能够通过将 CPU 密集型 SSL 终端的负载卸载到应用程序网关上来优化 Web 场生产率。  它是应用程序专用的虚拟设备，具有可伸缩性和高可用性，提供丰富的诊断和日志记录功能来提高可管理性。
