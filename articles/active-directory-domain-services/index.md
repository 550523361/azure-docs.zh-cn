# <a name="active-directory-domain-services-documentation"></a>Active Directory 域服务文档

Azure AD 域服务 (DS) 提供与 Windows Server Active Directory 完全兼容的托管域服务，例如域加入、组策略、LDAP、Kerberos/NTLM 身份验证。  用户可借助此服务灵活部署各种计算解决方案。
