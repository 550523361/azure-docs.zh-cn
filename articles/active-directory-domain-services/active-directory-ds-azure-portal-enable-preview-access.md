# <a name="enable-the-preview-experience-to-administer-your-managed-domain-from-the-azure-portal"></a>启用预览版体验，从 Azure 门户管理托管域

## <a name="your-existing-managed-domain-is-not-shown-in-the-azure-portal-preview-experience"></a>现有托管域不会在 Azure 门户（预览版）体验中显示
你的现有 [Azure AD 域服务](active-directory-ds-overview.md)托管域是使用 Azure 经典门户创建的。 导航到 Azure 门户时，Azure AD 域服务边栏选项卡（预览版）不会列出此现有托管域。

![现有托管域不会在 Azure 门户中显示](./media/getting-started/missing-domain-services-instance.png)

这是因为现有托管域尚未迁移到新的 Azure 门户中。 可以联系我们为现有托管域启用新的 Azure 门户（预览版）体验。


## <a name="join-the-preview-for-the-azure-portal-experience-for-azure-ad-domain-services"></a>为 Azure AD 域服务加入预览版 Azure 门户体验
按照[联系我们](active-directory-ds-contact-us.md)页上的说明操作并向我们发送电子邮件。 我们很高兴为你的现有托管域启用 Azure 门户预览版。

> [!NOTE]
> 正式推出 (GA) Azure AD 域服务的新 Azure 门户体验后，将自动迁移所有现有托管域。 届时便能从 Azure 门户管理现有托管域。
