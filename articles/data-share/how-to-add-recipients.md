# <a name="how-to-add-a-recipient-to-an-existing-data-share-in-azure-data-share"></a>如何将收件人添加到 Azure 数据共享中的现有数据共享

本文将介绍如何将收件人添加到预先存在的数据共享。

有关如何在创建共享时添加收件人的信息，请参阅[共享数据](share-your-data.md)教程。

## <a name="navigate-to-a-sent-data-share"></a>导航到已发送的数据共享

在 Azure 数据共享中，导航到已发送的共享并选择 **"邀请"** 选项卡。此处列出的是此数据共享邀请的所有收件人。 要添加新收件人，请单击"**添加收件人**"。

![添加收件人](./media/how-to/how-to-add-recipients/add-recipient.png)

页面右侧将弹出一个面板。 单击"**添加收件人**"，然后在空白行上填写新收件人的电子邮件。 根据需要添加尽可能多的收件人。

![添加收件人](./media/how-to/how-to-add-recipients/add-recipient-side.png)

单击"**添加并发送邀请**"。 新收件人将发送邀请电子邮件到此数据共享。
