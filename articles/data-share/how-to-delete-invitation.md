# <a name="how-to-delete-an-invitation-to-a-recipient-in-azure-data-share"></a>如何删除对 Azure 数据共享中的收件人的邀请

本文介绍如何在收件人接受邀请之前删除邀请。 这将阻止他们创建共享订阅，并且他们永远不会接收任何共享数据。 如果删除对已创建共享订阅的消费者的邀请，则删除其邀请将没有任何影响。

## <a name="navigate-to-a-sent-data-share"></a>导航到已发送的数据共享

在 Azure 数据共享中，导航到已发送的共享并选择 **"邀请"** 选项卡。此处列出的是此数据共享邀请的所有收件人。 单击要删除邀请的收件人旁边的框，然后单击"**删除**"。
