# <a name="set-up-support-information-lab-account-owner-in-azure-lab-services"></a>设置支持信息（Azure 实验室服务中的实验室帐户所有者）
本文介绍了如何（作为实验室帐户所有者）如何指定支持信息，如果用户遇到使用实验室帐户或实验室的任何技术问题，就可以使用这些信息来获取帮助。 

支持信息包括：

- 代码
- 电子邮件
- 电话
- 其他说明 

## <a name="specify-support-information"></a>指定支持信息
1. 登录到 [Azure 门户](https://portal.azure.com)。
2. 在搜索栏中输入 "**实验室服务**"，并在搜索结果中选择 "**实验室服务**"。 
3. 从实验室帐户列表中选择实验室帐户。 
4. 切换到 "**内部支持**" 页，执行以下步骤：
    1. 输入**支持 URL**。 
     2. 输入**支持电子邮件**。 
     3. 输入**支持电话**。
     4. 输入详细**支持说明**（可选）。 实验室所有者和用户将看到此文本以及支持联系人信息。 Url 将自动转换为链接。 
     5. 在工具栏上选择“保存”。
