# <a name="view-support-information-lab-user-in-azure-lab-services"></a>查看支持信息（Azure 实验室服务中的实验室用户）
本文介绍了如何（作为实验室用户）查看以下支持信息：

- 代码
- 电子邮件
- 电话
- 其他说明

当你在实验室帐户中使用实验室时，你可以使用此信息来获取技术问题的帮助。

 
## <a name="view-support-information"></a>查看支持信息
1. 登录到[Azure 实验室服务门户](https://labs.azure.com)。
2. 选择需要帮助的**实验室或虚拟机**，并选择 **"** 在页面的右上角。 
3. 确认你看到指向 "**查看支持网站**"、"**电子邮件支持**" 和 "**支持电话号码**" 的链接。

    ![查看支持信息](../media/lab-user-support-information/support-information.png)
4. 可以通过在下拉列表中切换到该实验室来查看其他实验室的支持联系人信息。 

    ![切换到另一个实验室](../media/lab-user-support-information/switch-another-lab.png)
5. 现在，你将看到另一实验室的支持联系信息。 

    ![其他实验室支持信息](../media/lab-user-support-information/second-lab-support-information.png)
