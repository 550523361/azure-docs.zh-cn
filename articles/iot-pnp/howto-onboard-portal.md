# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>载入到 Azure IoT 认证门户

您可以使用 Azure[认证的 IoT 门户](https://aka.ms/ACFI)：

- 完成 IoT 设备的认证流程。
- 查找 IoT 即插即用预览设备功能模型。 您可以使用这些模型快速构建物联网就绪设备并将其与解决方案集成。
- 将设备功能模型发布到公共存储库。

## <a name="sign-in"></a>登录

使用 Microsoft_工作帐户_登录到[适用于 IoT 门户的 Azure 认证](https://aka.ms/ACFI)门户。 登录时，门户会通过 Microsoft 合作伙伴中心检查您的成员资格：

- 如果您的组织已经是[Microsoft 合作伙伴中心](https://docs.microsoft.com/partner-center/)的成员，则授予您访问门户中所有功能的权限。
- 如果您的组织不是[Microsoft 合作伙伴中心](https://docs.microsoft.com/partner-center/)的成员，则可以运行认证测试并浏览设备功能模型存储库。 您无法在 IoT 认证设备目录中发布设备。
- 如果您看到一条通知消息，指出您的 Microsoft 合作伙伴网络 ID 丢失，则应[创建合作伙伴中心帐户](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account)。 您的帐户获得批准后，您可以返回到门户并发布设备。 有关详细信息，请参阅[合作伙伴中心常见问题解答](https://support.microsoft.com/help/4340639/partner-center-account-faqs)。

在将设备发布到 IoT 认证设备目录之前，请阅读并接受程序协议。 您可以在 Azure 认证 IoT 门户中的设置页上找到协议。
