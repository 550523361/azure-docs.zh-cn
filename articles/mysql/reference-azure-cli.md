# <a name="commands"></a>命令
|||
|---|---|
|az mysql server|管理服务器。 服务器包含数据库、数据仓库和弹性池。|
|az mysql server create|创建 MySQL 服务器。|
|az mysql server delete|删除 MySQL 服务器。|
|az mysql server list|获取资源组中所有 MySQL 服务器的列表。|
|az mysql server show|显示 MySQL 服务器的详细信息。|
|az mysql server update|更新服务器。|
|az mysql server restore|将服务器还原到以前的时间点。|
|az mysql server firewall-rule|管理服务器的防火墙规则。|
|az mysql server firewall-rule create|创建防火墙规则。|
|az mysql server firewall-rule delete|删除防火墙规则。|
|az mysql server firewall-rule list|列出防火墙规则。|
|az mysql server firewall-rule show|显示防火墙规则的详细信息。|
|az mysql server firewall-rule update|更新防火墙规则。|
|az mysql db|管理数据库。|
|az mysql db list|列出服务器下的所有 MySQL 数据库。|
|az mysql server-logs list|列出服务器的日志文件。|
|az mysql server-logs download|从服务器将日志下载到本地。|
|az mysql server configuration list|列出服务器配置参数。|
|az mysql server configuration show|显示服务器配置参数详细信息。|
|az mysql server configuration update|修改服务器配置参数值。|
|||
