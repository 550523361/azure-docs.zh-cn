# <a name="supported-azure-database-for-mysql-server-versions"></a>支持的 Azure Database for MySQL 服务器版本
开发人员是使用 InnoDB 引擎通过 MySQL Community Edition [MySQL Community Edition](https://www.mysql.com/products/community/) 开发 Azure Database for MySQL。  Azure Database for MySQL 目前支持以下版本：

## <a name="mysql-version-5635"></a>MySQL 5.6.35 版
若要详细了解 MySQL 5.6.35 中的改进和修复，请参阅 MySQL [文档] (https://dev.mysql.com/doc/relnotes/mysql/5.6/en/news-5-6-35.html)。

## <a name="mysql-version-5718"></a>MySQL 5.7.18 版
若要详细了解 MySQL 5.7.18 中的改进和修复，请参考 MySQL [文档] (https://dev.mysql.com/doc/relnotes/mysql/5.7/en/news-5-7-18.html)。

## <a name="managing-updates-and-upgrades"></a>管理更新和升级
Azure Database for MySQL 自动管理次要版本更新的修补。 目前，公共预览版中不支持主版本升级。 例如，不支持从 MySQL 5.6 升级到 MySQL 5.7。

## <a name="next-steps"></a>后续步骤

有关基于服务层的具体资源配额和限制的信息，请参阅[服务层](./concepts-service-tiers.md)
