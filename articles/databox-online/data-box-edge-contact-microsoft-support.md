# <a name="open-a-support-ticket-for-azure-data-box-edge-and-azure-data-box-gateway"></a>打开 Azure 数据框边缘和 Azure 数据框网关的支持票证

本文适用于 Azure 数据框边缘和 Azure 数据框网关，它们都由 Azure 数据框边缘/Azure 数据框网关服务管理。 如果服务遇到任何问题，可以创建技术支持服务请求。 本文介绍：

* 如何创建支持请求。
* 如何从门户内部管理支持请求生命周期。

## <a name="create-a-support-request"></a>创建支持请求

可以执行以下步骤来创建支持请求：

1. 转到数据框边缘或数据框网关订单。 导航到**支持 + 故障排除**部分，然后选择 **"新建支持请求**"。
   
2. 在 **"基本"** 选项卡上**的新支持请求**中，执行以下步骤：
    
    1. 从“颁发类型”下拉列表中，选择“技术”********。
    2. 选择您的**订阅**。
    3. 在“服务”下，检查“我的服务”。******** 从下拉列表中，选择 **"数据框边缘"和"数据框网关**"。
    4. 选择您的**资源**。 这对应于您的订单名称。
    5. 简要**概述**您遇到的问题。 
    6. 选择您的**问题类型**。
    7. 根据您选择的问题类型，选择相应的**问题子类型**。
    8. 选择 **"下一步"：解决方案 >>**。

        ![基础](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-1.png)

3. 在"**详细信息"** 选项卡上，执行以下步骤：
    
    1. 提供问题的开始日期和时间。
    2. 提供您的问题**描述**。
    3. 在 **"文件上传**"中，选择文件夹图标以浏览要上载的任何其他文件。
    4. 选中“共享诊断信息”。****
    5. 根据您的订阅，将自动填充**支持计划**。
    6. 从下拉列表中，选择**严重性**。
    7. 指定**首选联系人方法**。
    8. **响应时间**会根据您的订阅计划自动选择。
    9. 提供您喜欢支持的语言。
    10. 在**联系信息**中，提供您的姓名、电子邮件、电话、可选联系人、国家/地区。 Microsoft 支持将使用此信息与你联系，以获得进一步信息、诊断和解决方法。 
    11. 选择 **"下一步"：查看 = 创建 >>**。

        ![问题](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-2.png)

4. 在"**审核 + 创建**"选项卡上，查看与支持票证相关的信息。 选择 **“创建”**。 

    ![问题](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-3.png)

    创建支持票证后，支持工程师将尽快与您联系以继续您的请求。

## <a name="get-hardware-support"></a>获取硬件支持

此信息仅适用于数据盒边缘设备。 报告硬件问题的过程如下：

1. 为硬件问题从 Azure 门户打开支持票证。 在 **"问题类型**"下，选择 **"数据框边缘硬件**"。 选择 **"问题子类型**"作为**硬件故障**。 

    ![硬件问题](./media/data-box-edge-contact-microsoft-support/data-box-edge-hardware-issue-1.png)

    创建支持票证后，支持工程师将尽快与您联系以继续您的请求。 

2. 如果 Microsoft 支持确定这是一个硬件问题，则发生以下操作之一： 

    - 将发送故障硬件部件的现场更换单元 （FRU）。 目前，电源单元是唯一受支持的 FRU。 
    - 对于任何其他部件故障，Microsoft 会执行完整的系统更换 （FSR） 或设备交换。

3. 如果在当地时间下午 4：30（周一至周五）之前升起支持票证，则现场技术人员将派遣至您所在的位置执行 FRU 或完整设备更换。

## <a name="manage-a-support-request"></a>管理支持请求

创建支持请求之后，可以在门户内管理请求的生命周期。

#### <a name="to-manage-your-support-requests"></a>管理支持请求

1. 若要访问“帮助和支持”页，请导航到“浏览”>“帮助 + 支持”****。

    ![管理支持请求](./media/data-box-edge-contact-microsoft-support/data-box-edge-manage-support-request-1.png)   

2. “帮助 + 支持”中会显示“最近的支持请求”表格。********

    <!--[Manage support requests](./media/data-box-edge-contact-microsoft-support/data-box-edge-support-request-1.png)--> 

3. 选择并单击某个支持请求。 可以查看此请求的状态和详细信息。 如果想要跟进此请求，请单击“+ 新建消息”。****
