# <a name="itar-overview-for-azure-government"></a>Azure Government 的 ITAR 概述
## <a name="overview"></a>概述
此信息适用于在遵守国际武器贸易条例 (ITAR) 的前提下，希望使用 Azure Government 服务进行存储、处理或传输管控信息的 Azure Government 客户。 所提供的信息描述了 Microsoft Azure Government 服务的功能，并提供了适用于 ITAR 客户的一般性指南。 在 Azure Government 订阅中包含受 ITAR 管控的数据之前，应先熟悉 Azure Government 服务的各项功能，并在有任何疑问时咨询你的客户团队。

你应参考 <a href="http://www.microsoft.com/en-us/TrustCenter/Compliance/default.aspx/">Microsoft Azure 信任中心法规遵从页</a>，了解在 ITAR 管控下关于 Azure Government 服务的当前信息。 虽然可能也提供了其他 Microsoft 服务，但不在 Azure Government 涵盖服务的范围内。 Azure Government 服务可能还允许你使用不同的其他资源、应用程序或由第三方 — 或根据单独的使用条款和隐私策略由 Microsoft 提供的服务。 你有责任审阅所有此类“外接”产品（如应用商店中的产品/服务）的条款，以确保它们满足你在 ITAR 法规遵从方面的需要。

<a href="https://azure.microsoft.com/en-us/features/gov/">Azure Government</a> 适用于受特定政府法规和要求（如 ITAR）约束来进行数据处理的实体。这些实体在使用 Azure Government 时需要遵守这些法规。 Azure Government 客户必须接受资格验证。 由 Microsoft 执行的资格验证将包括：根据 ITAR 的防御条款，按照您在美国国务部注册的信息确认你是制造商、出口商还是代理人，或与对处理数据具有特定要求的政府实体通过了赞助协议。

对 Azure Government 的资格验证有疑问的实体应咨询其客户团队。
