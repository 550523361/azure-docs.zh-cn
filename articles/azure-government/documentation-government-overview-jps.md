# <a name="justice-and-public-safety-jps-in-azure-government"></a>Azure 政府版中的司法和公共安全 (JPS)
## <a name="overview"></a>概述
司法和公共安全 (JPS) 机构在保护社区安全、降低犯罪率和提高响应能力方面承受的压力越来越大。 从智能警务意识系统，到全国各地的便携式照相机系统，再到日常移动警务协作，云计算正在改变执法机构的工作方式。

如果云服务规划合理、安全可靠，它将给 JPS 提供很多强大的新功能。 这些功能包括数字证据管理、数据分析和实时决策支持 -- 在最新的移动设备上传送解决方案。 但云提供商还是有差别的。 执法机构使用云时，需要可信任的云服务提供商。 执法使命的核心要求合作伙伴要尽力满足安全性、合规性和运营的全方位需求。

从设备到云， Microsoft 将隐私和信息安全放在首位，同时提高现场和整个部门人员的工作效率。 通过将高度安全的移动设备与“随时随地”云访问相结合，JPS 机构可以协助正在进行的调查、分析数据、管理证据并保护公民免受威胁。

其他云提供商将刑事司法信息系统 (CJIS) 合规性视为可有可无的选项，而不是一项承诺。 Microsoft 致力于提供满足当前和未来 CJIS 适用控制要求的解决方案。 此外，我们还通过<a href="http://news.microsoft.com/presskits/dcu/#sm.0000eqdq0pxj4ex3u272bevclb0uc#KwSv0iLdMkJerFly.97">反数字犯罪部门</a><a href="https://channel9.msdn.com/Blogs/Taste-of-Premier/Satya-Nadella-on-Cybersecurity">网络防御运营中心</a>和<a href="https://enterprise.microsoft.com/en-us/industries/government/public-safety/">全球司法和公共安全组织</a>进一步致力于司法公正和公共安全。
