# <a name="azure-marketplace-for-government"></a>Azure 政府版应用商店
Azure 应用商店具有来自应用商店发布服务器的已更新映像列表，可用于 Azure 政府版。 

## <a name="variations"></a>变体
下面是使用 Azure 政府版应用商店时的一些注意事项：

* 仅自带许可 (BYOL) 可用。 不能访问任何需要通过 Azure 应用商店进行购买事务的映像
* 与公共应用商店相比，当前仅有映像的一个子集可用。 可在[此处](../azure-government-image-gallery.md)找到可用映像列表 
* 在预配映像之前，企业管理员必须对 Azure 订阅启用应用商店购买
  * 以企业管理员身份登录到门户
  * 导航到管理
  * 在“注册详细信息”下，单击“Azure 应用商店”明细项目旁边的铅笔图标
  * 根据需要，切换“启用/禁用”
  * 单击“保存”

> [!NOTE]
> 如果有兴趣向 Azure 政府版中提供你的映像，请参阅[合作伙伴加入指南](documentation-government-manage-marketplace-partners.md)，了解详细信息。
>
