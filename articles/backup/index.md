#<a name="backup-documentation"></a>备份文档

Azure 备份是一个云服务，可用于在 Microsoft 云中备份（或保护）和还原数据。  Azure 备份将现有的本地或异地备份解决方案替换为安全可靠、性价比高的云端解决方案。
