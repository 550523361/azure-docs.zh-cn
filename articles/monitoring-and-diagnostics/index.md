# <a name="monitoring-and-diagnostics-documentation"></a>监视和诊断文档

监视和诊断是一种分布式系统平台，适用于打包、部署和管理可缩放的可靠微服务，以及处理在开发和管理云应用程序的过程中遇到的重大难题。  监视和诊断代表着用于生成和管理这些企业级第 1 层云规模应用程序的下一代中间件平台。
