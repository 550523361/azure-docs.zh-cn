# 如何取消 Azure 订阅

>[AZURE.NOTE] Azure 支持计划的期限为 6 个月，且 Microsoft 要求有持续的资金承诺才会延续期限。在期限到期前取消，不会免除剩余的承诺，也不会就任何预支付金额提供退款。如果不小心取消订阅并需要重新激活，请联系 [Azure 支持](/documentation/articles/billing-how-to-create-billing-support-ticket)寻求帮助。

如果你取消订阅，将立即删除你的所有服务。取消之前，请务必先备份你要保存的数据。

1. 以帐户管理员身份登录到 [Azure 帐户中心](https://account.windowsazure.com/subscriptions)。
2. 在“单击订阅以查看详细信息和使用情况”下面，选择要取消的订阅。</br> ![selectsub](.\media\billing-how-to-cancel-azure-subscription\Selectsub.png)
3. 在页面右侧，选择“取消订阅”。</br> ![cancelsub](.\media\billing-how-to-cancel-azure-subscription\cancelsub.png)
4. 选择“是，取消我的订阅”。</br> ![cancelbox](.\media\billing-how-to-cancel-azure-subscription\cancelbox.png)
5. 单击 ![checkbutton](.\media\billing-how-to-cancel-azure-subscription\checkbutton.png) 关闭对话框窗口并返回订阅页。
