# <a name="sql-database-release-notes"></a>SQL 数据库发行说明

本文列出了目前在公共预览版中的 SQL 数据库功能。 有关 SQL 数据库更新和改进功能，请参阅[SQL 数据库服务更新](https://azure.microsoft.com/updates/?product=sql-database)。 更新和其他 Azure 服务的改进，请参阅[服务更新](https://azure.microsoft.com/updates)。
