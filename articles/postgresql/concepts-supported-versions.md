# <a name="supported-postgresql-database-versions"></a>支持的 PostgreSQL Database 版本
Azure Database for PostgreSQL 目前支持以下版本：

## <a name="postgresql-version-962"></a>PostgreSQL 9.6.2 版
若要详细了解 PostgreSQL 9.6.2 中的改进和修复，请参阅 [PostgreSQL 文档](https://www.postgresql.org/docs/9.6/static/release-9-6-2.html)。

## <a name="postgresql-version-957"></a>PostgreSQL 版本 9.5.7
若要了解 PostgreSQL 9.5.7 中的改进和修复，请参阅 [PostgreSQL 文档](https://www.postgresql.org/docs/9.5/static/release-9-5-7.html)。

## <a name="managing-updates-and-upgrades"></a>管理更新和升级
Azure Database for PostgreSQL 自动管理次要版本更新的修补。 目前，公共预览版中不支持主版本升级。 例如，不支持从 PostgreSQL 9.5 升级到 PostgreSQL 9.6。

## <a name="next-steps"></a>后续步骤
有关不同 PostgreSQL 扩展支持的信息，请参阅 [PostgreSQL 扩展](concepts-extensions.md)
