# <a name="about-azure-networking-msp-partners"></a>关于 Azure 网络 MSP 合作伙伴

[Azure 网络托管服务提供商 （MSP） 合作伙伴计划](https://azure.microsoft.com/blog/enhancing-the-customer-experience-with-the-azure-networking-msp-partner-program/)支持以网络服务为中心的 MSP、电信公司和系统集成商 （SI） 提供以 Azure 网络产品和服务组合为中心的云和混合网络服务。

Azure 网络 MSP 是一组专门的托管服务提供商，可满足云和混合网络各个方面的企业云网络需求和挑战。 托管网络服务包括以下一个或多个服务：网络体系结构、规划、部署、操作、维护和优化。

![Msp][0]

客户可以在 Azure 应用商店中购买 MSP 托管的服务产品，也可以直接从 MSP 购买服务。 MSP 可以使用[Azure 灯塔](https://azure.microsoft.com/services/azure-lighthouse/)在客户的订阅和资源组中部署和管理 Azure 网络资源。
