# <a name="power-bi-embedded-documentation"></a>Power BI Embedded 文档

Power BI Embedded 是一项 Azure 服务，应用程序开发人员可通过它将交互式 Power BI 报表添加到自己的应用程序中。 Power BI Embedded 可配合现有应用程序使用，无需重新设计或更改用户登录方式。
