# <a name="how-to-code-and-test-azure-functions-locally"></a>如何在本地对 Azure 函数进行编码和测试
> [!NOTE]
> Azure Functions 本地开发和工具体验目前处于预览状态，将在最终发行前进行显著改进。 在本地运行 Azure Functions 主机的最简单的方法是使用 [Azure Functions CLI](https://go.microsoft.com/fwlink/?linkid=832752)，并从 npm 安装它。 目前仅支持 Windows。 即将发布完整文档。 若要提供有关 Azure Functions CLI 的反馈或提出存在的 Bug，请在 [azure-webjobs-sdk-script](https://github.com/Azure/azure-webjobs-sdk-script) GitHub 存储库中提出问题。
