#<a name="azure-functions-documentation"></a>Azure Functions 文档

Azure Functions 是用于在云中轻松运行小段代码（“函数”）的一个解决方案。 用户可以编写解决现有问题所需的代码，无需担心要运行该代码的整个应用程序或基础结构。 用户还可以使用所选的语言（例如 C#、F#、Node.js、Python 或 PHP）开发软件。
