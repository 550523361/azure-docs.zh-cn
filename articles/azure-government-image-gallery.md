# <a name="azure-government-marketplace-images"></a>Azure 政府版应用商店映像
## <a name="overview"></a>概述
Microsoft Azure 政府版应用商店与公共 Azure 门户有着相似体验。  客户可以选择从 Microsoft 和我们的合作伙伴部署预构建的映像或上载自己的 VHD。  这为按需要部署自己的标准化映像提供了灵活性。

以下是 Azure 政府版应用商店内的可用映像列表。  一些预构建映像包含特定软件的即付即授权。  请认真阅读<a href="http://azure.microsoft.com/pricing/details/virtual-machines/">虚拟机定价</a>页面获得更多指南，并和您的 Microsoft 帐户团队或分销商共同合作，对 Azure Government 进行具体定价。

## <a name="quick-start-templates"></a>快速入门模板
### <a name="step-1"></a>步骤 1
转到 Azure 快速入门模板 GitHub 存储库并从下面的解决方案模板列表中进行选择：

| 快速入门模板 |
| --- |
| <a href="https://github.com/Azure/azure-quickstart-templates/tree/master/ethereum-consortium-blockchain-network"> Ethereum Consortium Blockchain Network </a> |
| <a href="https://github.com/Azure/azure-quickstart-templates/tree/052db5feeba11f85d57f170d8202123511f72044/splunk-on-ubuntu"> Splunk on Ubuntu </a> |
| <a href="https://github.com/Azure/azure-quickstart-templates/tree/058f18cbb43165b043e5bb56a82a406290c02dac/cisco-csr-1000v"> Cisco CSR1000v (2 NIC) </a> |
| <a href="https://github.com/Azure/azure-quickstart-templates/tree/058f18cbb43165b043e5bb56a82a406290c02dac/cisco-csr-1000v-4-nic"> Cisco CSR1000v (4 NIC) </a> |
| <a href="https://github.com/Azure/azure-quickstart-templates/tree/058f18cbb43165b043e5bb56a82a406290c02dac/cisco-csr-1000v-existing-vnet-4-nic"> Cisco CSR1000v（4 NIC 现有 Vnet）</a> |

### <a name="step-2"></a>步骤 2
在自述文件中，通过单击页面顶部的“部署到 Azure 政府版”按钮/链接，启动模板。

### <a name="step-3"></a>步骤 3
浏览页将在 Azure 政府版门户中打开模板部署边栏选项卡。 在参数边栏选项卡，完成必填的字段，然后单击“确定”。

### <a name="step-4"></a>步骤 4
接受法律条款和条件，并勾选“固定到仪表板”框，然后单击“创建”。

> [!NOTE]
> 为了继续，需要有效的 Azure 政府版帐户。 若要为组织体验 Azure 政府版的强大功能，请注册 [Azure 政府试用版](https://azuregov.microsoft.com/trial/azuregovtrial)。
> 
> 

### <a name="step-5"></a>步骤 5
将关闭边栏选项卡，并返回显示映像部署的门户仪表板视图。 这需要几分钟。 VM 解决方案开始运行后，它将显示在仪表板中。 如果未在步骤 4 中选择“固定到仪表板”选项，则需要在“资源”中搜索 VM。

> [!NOTE]
> 解决方案模板由 Microsoft 和第三方提供商提供，这才仅仅是开始而已。 你全权负责开发、实施和管理在 Microsoft Azure 和 Azure 政府版上运行的应用程序和/或订阅，且需遵守适用的法律、法规和合同义务。  这些模板原样提供，且没有任何形式的保证（无论是明示、暗示还是法定），且 Microsoft 明确表示，不保证其适销性、针对特定用途的适用性或非侵权性。
>
