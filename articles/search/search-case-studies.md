# <a name="whos-using-azure-search-case-studies-and-customer-stories"></a>谁在使用 Azure 搜索：案例研究和客户案例
了解以下组织如何通过将丰富的 Azure 搜索体验集成到其联机应用程序来开展其业务。

## <a name="autotraderhttpscustomersmicrosoftcompagescustomerstoryaspxrecid18596"></a>[autoTrader](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18596)
autoTRADER.ca 正在根据 Microsoft Azure 的平台即服务 (PaaS) 功能创建 IT 基础结构。 该公司已开始使用 Azure 托管和处理其数以百万计的映像。 它现在将其应用程序搜索功能移动到 Azure，并且随时可以将其所有 Web 属性迁移到 Azure。 因此，该公司不需要管理自己的基础结构，即可更好地改进服务并创建新产品。 [了解详细信息...](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18596)

## <a name="gjirafahttpscustomersmicrosoftcompagescustomerstoryaspxrecid18633"></a>[gjirafa](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18633)
如何像阿尔巴尼亚演讲者所言，使用较小的搜索引擎处理当今庞大的搜索量？ 通过了解本地语言、创建目标用户体验以及依赖于 Microsoft 云搜索技术。 [了解详细信息...](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18633)

## <a name="jllhttpscustomersmicrosoftcompagescustomerstoryaspxrecid18662"></a>[JLL](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18662)
JLL（以前称为 Jones Lang LaSalle）希望通过充分利用云，更改其应用程序的设计和托管方式。 使用 Microsoft Azure，JLL 可以更好地以更敏捷且更灵活的方式服务于全球员工和客户，它可侧重于解决方案而不是基础结构。 [了解详细信息...](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18662)

## <a name="xomnihttpscustomersmicrosoftcompagescustomerstoryaspxrecid18667"></a>[XOMNI](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18667)
这是一个经典的技术创业公司案例：有想法的开发人员看到了市场，想要抢占它。 市场？ 零售应用场景。 想法？ 面向 Web 和移动应用程序开发的后端服务，用于使日渐衰弱的商店内零售体验重现生机。 XOMNI 已使用 Microsoft Azure 快速实现 NoSQL 数据库和全文搜索服务，因此客户现在具有一个可缩放平台，用于开发客户端软件应用、与社交网络集成并分析数据以提升丰富的交互式用户体验。 [了解详细信息...](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=18667)

## <a name="medihoohttpscustomersmicrosoftcompagescustomerstoryaspxrecid19540"></a>[Medihoo](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=19540)
作为一家着重计划医疗保健搜索网站的新兴公司，Medihoo 需要强大的快速搜索功能，而不需要对新基础结构进行大量投资。 公司已选择在 Microsoft Azure 云平台上生成其站点，其中包含 Azure 搜索解决方案，可提供快速性能、全面功能以及多种语言的地理空间信息和支持。 它们现在可帮助每个人（尤其是海外侨胞和出差人员）找到优质的医疗保健，无论他们身处世界的何处。 [了解详细信息...](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=19540)

## <a name="weichert-realtorshttpscustomersmicrosoftcompagescustomerstoryaspxrecid21252"></a>[Weichert Realtors](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=21252)
Weichert Realtors 正在移动到云，这会使公司开展其业务的方式发生重大转变。 通过迁移到 Microsoft Azure，公司可以降低 60 %的运营成本（因为不再需要昂贵的本地硬件和软件）并使用强大的搜索功能更新其 CRM 解决方案，还生成了各种对将来业务至关重要的新数据服务。 [了解详细信息...](https://customers.microsoft.com/Pages/CustomerStory.aspx?recid=21252)

## <a name="whattopediacom"></a>Whattopedia.com
了解一家瑞典创业公司如何构建一个通过高度相关的范围搜索体验将消费者与零售商联系在一起的信息门户，类似于旅游门户如何将处于陌生地区的旅行者与酒店、餐馆和娱乐匹配在一起。
