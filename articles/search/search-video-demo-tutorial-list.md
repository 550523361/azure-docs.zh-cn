# <a name="azure-search-code-samples-demos-videos-and-tutorials"></a>Azure 搜索：代码示例、演示、视频和教程
借助以下视频、演示、教程和示例应用更快入门。

## <a name="code-samples"></a>代码示例
* [**演示站点的代码示例：NYC Jobs 网站**](https://github.com/Azure-Samples/search-dotnet-asp-net-mvc-jobs)使用 ASP.NET MVC 通过 NYC 作业数据提供直播演示 Azure 搜索网站的源。
* [**演示站点的代码示例：Tate Art Gallery**](https://github.com/liamca/azure-search-tate-art-gallery/) 通过图库数据提供直播演示 Azure 搜索网站的源，显示集合数据类型和嵌套数据结构、平面和筛选器、人员姓名和缩略图。
* [**.NET 客户端库的代码示例**](https://github.com/Azure-Samples/search-dotnet-getting-started)是使用 .NET 客户端库进行搜索操作的 ASP.NET MVC 5 应用程序。
* [**REST API 的代码示例**](https://github.com/Azure-Samples/search-rest-api-getting-started)提供阐述如何使用 REST API 的示例代码。

以上所有的代码示例及介绍如何在特定场景中使用 Azure 搜索的其他示例均可在 Azure 示例 Github 存储库中找到：“[**Azure 示例 > Azure 搜索**](https://github.com/azure-samples?utf8=%E2%9C%93&query=search)”。

## <a name="online-demo-sites"></a>在线演示站点
这些站点是使用 Azure 进行搜索体验的功能完整的 Web 应用程序。 

[**New York City (NYC) Jobs 门户对 Azure 搜索的演示**](http://aka.ms/azjobsdemo)是使用 NYC Open Data Initiative 中的数据来演示 Azure 搜索功能的实时 ASP.NET Web 网站。 该应用使用[简单查询语法](https://msdn.microsoft.com/library/azure/dn798920.aspx)。 你可以修改代码来试用 [Lucene 语法](https://msdn.microsoft.com/library/azure/mt589323.aspx)。 

[**Azure 搜索演示站点**](https://searchsamples.azurewebsites.net/#/)是 Azure 搜索的演示应用集合。 使用 Azure 搜索试用媒体搜索、地理位置搜索和产品库存应用。

## <a name="top-five"></a>前五项
以下是一些评价较好的 Azure 搜索教程和视频。

* [视频：Microsoft Virtual Academy 对 Azure 搜索的简介](https://mva.microsoft.com/en-us/training-courses/adding-microsoft-azure-search-to-your-websites-and-apps-10540?l=ADkxnd97_9304984382)是对 Azure 搜索加强了解的极好资源。
* [视频：AzureCon 2015：Azure 搜索最佳实践](https://azure.microsoft.com/documentation/videos/azurecon-2015-azure-search-best-practices-for-web-and-mobile-applications/)提供有关扩展、性能优化及在为 Web 或移动场景构建应用程序时如何实现高效编码的实用建议。 
* [.NET 客户端库的代码示例](https://github.com/Azure-Samples/search-dotnet-getting-started)是使用 .NET 客户端库进行搜索操作的 ASP.NET MVC 5 应用程序。
* [教程：在 .NET 中如何使用 Azure 搜索](search-howto-dotnet-sdk.md)|配置并运行集成 Azure 搜索的示例 Visual Studio Web 应用程序。
* [教程：门户中的 Azure 搜索入门](search-get-started-portal.md)是完全基于门户的演练，使你不必编写任何代码即获取索引和数据。 该教程有助于了解如何在门户中执行常见的任务。
