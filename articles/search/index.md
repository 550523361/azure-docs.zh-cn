# <a name="azure-search-documentation"></a>Azure 搜索文档

Azure 搜索是一种托管的云搜索服务，可帮助在自定义应用中提供强大的搜索体验，以及通过门户视图监视服务、索引和查询活动。 主要功能包括可扩展的多语言全文搜索、地理空间搜索、筛选和多面导航、提前键入查询、命中突出显示以及自定义分析器。
