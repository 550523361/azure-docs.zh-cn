#<a name="notification-hubs-documentation"></a>通知中心文档

Azure 通知中心提供易用的多平台扩展式推送基础结构，让用户从任何后端（云中或本地）向任何移动平台发送移动推送通知。

借助通知中心，可以轻松发送跨平台的个性化推送通知，提取不同平台通知系统 (PNS) 的详细信息。 只需 API 调用一次，就可以将通知发送到单个用户或包含数百万用户的整个受众群的所有设备上。
