## <a name="working-with-images"></a>使用映像
可以使用适用于 Mac、Linux 和 Windows 的 Azure 命令行接口 (CLI) 模块管理可供 Azure 订阅使用的映像。 也可以使用 Azure 经典门户完成某些映像任务，但命令行可提供更多选项。

有关这些工具的示例，请参阅 [Common Azure CLI commands on Linux and Mac](virtual-machines-linux-cli-manage.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json)（Linux 和 Mac 上的常用 Azure CLI 命令）。
