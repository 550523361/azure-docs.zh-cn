#<a name="linux-virtual-machine-documentation"></a>Linux 虚拟机文档

使用 Azure 虚拟机创建 Linux 虚拟机，并在 Azure 计算机、网络和存储基础结构上运行它。  扩展至数千台计算机以便进行 Internet 范围内的部署，或在几分钟内运行一台虚拟机以便进行繁重的数字运算：同时保证速度与一致性。
