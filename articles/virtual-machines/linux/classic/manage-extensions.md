# <a name="manage-virtual-machine-extensions"></a>管理虚拟机扩展
介绍如何在 Azure 上查找、添加、修改或删除 Windows 或 Linux 虚拟机的 VM 扩展。

> [!IMPORTANT] 
> Azure 提供两个不同的部署模型用于创建和处理资源：[Resource Manager 和经典模型](../../../resource-manager-deployment-model.md)。 本文介绍如何使用经典部署模型。 Microsoft 建议大多数新部署使用 Resource Manager 模型。 有关将 VM 扩展与 Resource Manager 模型配合使用的信息，请参阅[此处](../extensions-features.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json)。
