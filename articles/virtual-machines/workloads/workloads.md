# <a name="workloads-on-azure-virtual-machines"></a>Azure 虚拟机上的工作负荷

在 Azure 中的虚拟机可以支持很多工作负荷。 每个工作负荷可以专用化中 Azure Vm 上部署的最佳设计指南。 下面是指南的一些适用于专用工作负荷，我们可以提供。


- [SAP](./sap/get-started.md)在 Azure 中可以运行任务关键 SAP 工作负荷和方案的可缩放、 合规且经企业认证的平台上。
- [Oracle](./oracle/oracle-considerations.md)上虚拟机映像在 Azure 中运行 Oracle DB 12.1 标准版和企业版支持基于 Oracle Linux 上。 
- [大型机重新承载](./mainframe-rehosting/overview.md)
