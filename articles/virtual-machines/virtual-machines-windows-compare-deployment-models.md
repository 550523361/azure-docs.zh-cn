# Azure Resource Manager 部署模型中适用于 Windows 应用程序的 Azure 计算、网络和存储提供程序

Azure Resource Manager 中包含的计算、网络和存储功能，将从根本上简化 IaaS 上运行的复杂应用程序的部署和管理。很多应用程序需要虚拟网络、存储帐户、虚拟机和网络接口等资源的组合。Azure Resource Manager 部署模型允许你构建 JSON 模板，以便将所有这些资源作为单个应用程序一起部署和管理。
