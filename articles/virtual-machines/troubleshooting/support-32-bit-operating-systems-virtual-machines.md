# <a name="support-for-32-bit-operating-systems-in-azure-virtual-machines"></a>Azure 虚拟机对 32 位操作系统的支持

Microsoft Azure 现在允许用户将其 32 位 Windows 操作系统引入 Azure。 由于其中一些操作系统已经达成其寿命终止支持协议，Microsoft 可能不会为他们提供额外的支持。 对于在 Microsoft Azure 虚拟机 （VM） 上运行的基于 Linux 或基于伯克利软件分发 （BSD） 的操作系统，也不提供支持。
