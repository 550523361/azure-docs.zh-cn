## <a name="create-a-personalizer-azure-resource"></a>创建个性化体验创建服务 Azure 资源

使用 [Azure 门户](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account)或 [Azure CLI](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account-cli) 在本地计算机上创建用于个性化体验创建服务的资源。 

从资源获取密钥后，创建两个[环境变量](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account#configure-an-environment-variable-for-authentication)：

* `PERSONALIZER_RESOURCE_KEY`（代表资源键）。
* `PERSONALIZER_RESOURCE_ENDPOINT`（代表资源终结点）。

在 Azure 门户中，可以从“快速入门”页中获取密钥和终结点值。
