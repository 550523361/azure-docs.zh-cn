# <a name="upload-images"></a>上传图像

若要探索用户界面，一次上传最多 5 张图像，每张图像的大小不超过 2 MB 文件大小。 上传之前请务必检查文件大小。

![上传图像](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>提交审核
如果一次上传最后超过了 5 张图像，请使用垃圾桶图标删除不需要的图像。 然后，提交图像进行自动审核。

![提交图像](images/1-try-image-2.png)
