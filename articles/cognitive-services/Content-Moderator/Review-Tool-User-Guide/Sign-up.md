# <a name="sign-up-for-content-moderator"></a>注册内容审查器 #
转到[内容审查器注册](http://contentmoderator.cognitive.microsoft.com/Account/Signup)页。 可以使用 Microsoft 帐户注册，也可以创建内容审查器网站帐户。

![注册](images/0-signup-1.PNG)

## <a name="name-your-team"></a>命名团队 ##
命名审阅团队。 可以视需要邀请同事加入团队。

![创建团队](images/1-Team-Name.PNG)
