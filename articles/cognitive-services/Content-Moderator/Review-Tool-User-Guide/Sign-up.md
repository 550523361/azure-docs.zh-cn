# <a name="sign-up-for-content-moderator"></a>注册内容审查器

在内容审查器上注册[审阅工具](https://contentmoderator.cognitive.microsoft.com)网站。 可以使用你的 Microsoft 帐户注册，或在站点上创建一个帐户。

![注册](images/0-signup-1.PNG)

## <a name="name-your-team"></a>命名你的团队

命名审阅团队。 可以视需要邀请同事加入团队。
