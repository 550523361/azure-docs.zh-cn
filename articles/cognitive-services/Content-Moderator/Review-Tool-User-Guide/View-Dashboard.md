# <a name="view-dashboard"></a>查看仪表板

单击“仪表板”选项卡，查看在该工具中完成的所有内容审阅的关键指标。 可以看到图像和文本的总评审、已完成的评审，和正在进行的评审。 可以看到默认团队和子团队的指标。
