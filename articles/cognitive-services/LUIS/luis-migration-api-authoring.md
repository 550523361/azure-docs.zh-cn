# <a name="authoring-api-changes"></a>创作 API 更改 

为 V3 更新了创作 Api，以允许[通过机器学习的实体进行模型分解](luis-concept-data-extraction.md)。 

可以在 V2 或预览版 V3 API 中创作。 同一应用程序还可以选择使用 V2 或 V3 预测终结点响应。 对于创作和预测终结点响应，无需使用相同版本的 API。

LUIS 门户只使用 V3 创作 Api。
