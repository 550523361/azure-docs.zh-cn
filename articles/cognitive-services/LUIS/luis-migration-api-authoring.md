# <a name="authoring-api-changes"></a>创作 API 更改 

为 V3 更新了创作 API，以便[使用机器学习的实体进行模型分解](luis-concept-data-extraction.md)。 

您可以在 V2 中创作或预览 V3 APIIS。 同一应用也可以选择使用 V2 或 V3 预测终结点响应。 不必具有相同的 API 版本，用于创作和预测终结点响应。

LUIS 门户仅使用 V3 创作 API。
