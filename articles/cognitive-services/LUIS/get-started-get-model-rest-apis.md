# <a name="quickstart-change-model-with-rest-apis"></a>快速入门：使用 REST API 更改模型

在此快速入门中，你要将示例言语添加到披萨应用并训练该应用。 示例话语是映射到意向的对话式用户文本。 通过提供意向的示例话语，可以教 LUIS 识别用户提供的文本类型属于哪种意向。
