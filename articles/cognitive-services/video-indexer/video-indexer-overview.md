# <a name="what-is-video-indexer-preview"></a>什么是视频索引器？ （预览版）

视频索引器是使用 Azure 媒体分析、认知服务（例如人脸 API、Microsoft Translator、计算机视觉 API、自定义语音服务）和 Azure 搜索生成的云应用程序。 可以使用它通过人工智能技术从视频中提取以下见解：

- **自动检测语言**：视频索引器可以自动检测视频的语言。 自动检测语言功能目前支持英语、西班牙语、法语、德语、意大利语、中文（简体）、日语、俄语。 检测不到语言时会回退到英语。
- **音频听录**：视频索引器有语音转文本功能，因此客户可以获得所说内容的脚本。 支持的语言包括英语、西班牙语、法语、德语、意大利语、中文（简体）、葡萄牙语（巴西）、日语和俄语（以后还会支持更多）。 
- **人脸跟踪和标识**：可以使用人脸技术检测视频中的人脸。 检测到的人脸会与名人数据库进行匹配，以便评估视频中存在哪些名人。 客户也可以对与名人不匹配的人脸进行标记。 视频索引器根据这些标签生成人脸模型，可以在以后提交的视频中识别这些人脸。
- **说话人索引**：视频索引器可以映射并理解哪个说话人在何时说哪些话。
- **视觉文本识别**：使用此技术，视频索引器服务可以提取显示在视频中的文本。  
- **语音活动检测**：视频索引器可以通过检测将背景噪音和语音活动隔离开来。 
- **场景检测**：视频索引器可以对视频进行视觉分析，确定视频中的场景何时改变。
- **关键帧提取**：视频索引器自动检测视频中的关键帧。 
- **情绪分析**：视频索引器使用语音转文本和光学字符识别功能对提取的文本进行情绪分析，并以正面、负面或中立情绪的形式提供该信息和时间代码。
- **翻译**：视频索引器可以将音频脚本从一种语言翻译为另一种语言。 支持以下语言：英语、西班牙语、法语、德语、意大利语、中文-简体、葡萄牙语-巴西、日语和俄语。 翻译以后，用户甚至可以在视频播放器中获得另一种语言的字幕。
- **可视内容审核**：此技术用于检测视频中存在的成人和/或不雅内容，并可用于内容筛选。 
- **关键字提取**：视频索引器根据所说内容的脚本以及视觉文本识别器所识别的文本提取关键字。
- **标签**：视频索引器为视觉对象（例如猫、狗、桌子、汽车以及站立、跑动或飞行之类的活动）提供标签。
- **品牌**：视频索引器根据所说内容的脚本以及视觉文本识别器所识别的文本提取商业品牌。

等到视频索引器处理和分析完毕，你就可以查看、策展、搜索和发布视频见解。

不管你的角色是内容管理员还是开发人员，视频索引器服务都可以满足你的需求。 内容管理员可以通过视频索引器 Web 门户使用此服务，不需编写一行代码，详见[视频索引器门户使用入门](video-indexer-get-started.md)。 开发人员可以利用 API 来大规模处理内容，详见[使用视频索引器 REST API](video-indexer-use-apis.md)。 客户还可以通过此服务使用小组件来发布视频流以及在其自己的应用程序中提取的见解，详见[在应用程序中嵌入视觉小组件](video-indexer-embed-widgets.md)。

可以使用现有的 AAD、LinkedIn、Facebook、Google 或 MSA 帐户注册此服务。 有关详细信息，请参阅[入门](video-indexer-get-started.md)。

## <a name="scenarios"></a>方案

下面是一些可以使用视频索引器的方案

- 搜索 - 从视频中提取的见解可以用来增强整个视频库的搜索体验。 例如，对所说内容和人脸进行索引可以启用在视频中查找特定瞬间的搜索体验，视频中的特定的人可能说出了特定的内容，或者两个人出现在一起。 根据视频中的此类见解进行的搜索适用于新闻机构、教育机构、广播公司、娱乐内容所有者、企业 LOB 应用。推而广之，它适用于任何行业，前提是该行业有一个需要用户对其进行搜索的视频库。

- 盈利 - 视频索引器有助于提高视频的价值。 例如，依赖广告收入的行业（如新闻媒体、社交媒体等）可以利用提取出来的见解作为广告服务器的附加信号（在足球比赛中展示运动鞋广告比在游泳比赛中展示更具针对性），从而提供更相关的广告。

- 用户参与度 - 可以通过视频见解来提高用户参与度，方法是为用户提供相关的视频瞬间。 以某个教育视频为例，该视频在头 30 分钟讲述球体，在接下来的 30 分钟讲述锥体。 如果某个学生想要了解锥体，则该视频从 30 分钟处开始播放更合适。

有关详细信息，请参阅此[博客](http://aka.ms/videoindexerblog)。
