# <a name="bing-search-sdk"></a>必应搜索 SDK
必应实体搜索 API 示例包括执行以下操作的方案：
1.  搜索实体（例如 Tom Cruise）并获取丰富的信息。
2.  处理可能具有多个意图的查询的术语消歧。
3.  搜索一个本地实体（例如一家餐馆）并获取有关它的丰富信息。
4.  搜索本地企业（例如餐馆）并获取丰富的信息。
5.  触发一个错误的请求并进行错误处理。

> [!NOTE] 
> 某些 SDK 现在为正式版，并且对文档的更改处于挂起状态。 

必应搜索 SDK 使 Web 搜索功能可采用以下编程语言进行访问：
* 开始使用 [.NET 示例](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7) 
    * [NuGet 包](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.EntitySearch/1.2.0)
    * 有关定义和依赖项，另请参阅 [.NET 库](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingEntitySearch)。
* 开始使用 [Node.js 示例](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * 有关定义和依赖项，另请参阅 [Node.js 库](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/entitySearch)。
* 开始使用 [Java 示例](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * 有关定义和依赖项，另请参阅 [Java 库](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingentitysearch)。
* 开始使用 [Python 示例](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * 有关定义和依赖项，另请参阅 [Python 库](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-entitysearch)。

每种语言的 SDK 示例包括一个自述文件，其中包含有关先决条件和安装/运行示例的详细信息。
