# <a name="recommendations-api-discontinued"></a>建议 API（已停用）

建议 API 预览不再进行有效开发，并已于 2 月 15 日正式停用。 

如需替代方法，请考虑新的开源[建议解决方案模板](http://aka.ms/recopcs)，可使用该模板在自己的 Azure 订阅中承载建议引擎。 该模板支持建议 API 的核心方案，也使你能够自由部署到任何 Azure 区域，并更好地控制用来定型模型和提供建议的计算资源。 

了解有关建议解决方案模板的详细信息，请阅读[文档](https://github.com/Microsoft/Product-Recommendations)。
