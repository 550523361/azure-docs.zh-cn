# <a name="how-to-sign-up-for-the-translator-speech-api"></a>如何注册文本翻译 API

请按照以下步骤注册语音翻译 API。

[!INCLUDE [Deprecation note](../../../includes/cognitive-services-translator-speech-deprecation-note.md)]

## <a name="1-sign-into-azure"></a>1：登录 Azure

- 没有帐户？ [注册 Microsoft Azure 帐户](http://azure.com/)
- 已有帐户？ [登录](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2：订阅 Microsoft Translator

- 登录到 Azure 后，请转到 [Azure 认知服务](https://portal.azure.com/#create/Microsoft.CognitiveServices)部分
- 在“API 类型”下，选择“语音翻译 API”
- 在“定价层”部分中，选择最适合需要的定价层
    - 每个订阅都有一个免费层。 免费层具有与付费计划相同的特征和功能，并且不会过期。
    - 帐户中只能有一个免费订阅
- 填写表单其余部分并选择“创建”按钮
- 订阅已准备就绪，可立即使用

## <a name="3-authentication-key"></a>3：身份验证密钥

- 转到菜单“所有资源”，单击所需订阅，检索身份验证密钥
- 密钥值用于身份验证，请勿泄露此密钥值。 开发应用时，需要此密钥值
