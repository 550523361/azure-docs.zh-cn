# <a name="bing-speech-client-samples"></a>必应语音客户端示例

必应语音提供端到端示例来演示各种用例，例如命令识别、连续识别和意向检测。 所有示例均在 GitHub 上提供，并可通过以下链接下载：每个存储库中的 README.md，此外，[客户端库](GetStarted/GetStartedClientLibraries.md)页提供了有关如何生成和运行这些示例的详细信息。

- [REST API 示例](https://github.com/Azure-Samples/SpeechToText-REST)

- [JavaScript 示例](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [C# 桌面客户端示例](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [C# 服务示例](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Android 上的 Java 示例](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [iOS 上的 ObjectiveC 示例](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

所有 Microsoft 认知服务 SDK 和示例均获得 MIT 许可证的许可。 有关详细信息，请参阅[许可证](https://github.com/Azure-Samples/SpeechToText-REST/blob/master/LICENSE)。
