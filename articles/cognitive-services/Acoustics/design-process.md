# <a name="project-acoustics-design-process-concepts"></a>项目声学设计过程概念

此概念概述解释了项目声学如何将物理声学仿真集成到声音设计过程中。

## <a name="sound-design-with-audio-dsp-parameters"></a>带音频 DSP 参数的声音设计

3D 交互式字幕使用音频引擎中托管的音频数字信号处理 （DSP） 块实现其特定声音。 这些块的复杂性范围从简单的混合，到混响，回声，延迟，均衡，压缩和限制，以及其他效果。 选择、排列和设置这些效果的参数是声音设计师的责任，他构建一个音频图，以实现体验的美学和游戏目标。

在交互式标题中，随着声音和侦听器在整个 3D 空间中移动，这些参数如何适应不断变化的条件？ 声音设计器通常会在整个空间中排列用于触发参数更改的音量，以实现混响效果的变化，或者当侦听器从场景的一个部分移动到另一个部分时，在混音中避开声音。 声学系统也可用，可以自动这些效果。

3D 标题使用照明和运动物理系统，这些系统以物理为动力，但由设计师调整，以实现沉浸式和游戏游戏目标的组合。 视觉设计器不会设置单个像素值，而是调整基于物理的 3D 模型、材料和轻型传输系统，以权衡视觉美观性和 CPU 成本。 音频的等效过程是什么？ 声学项目是探讨这个问题的第一步。 首先，我们将了解通过空间传输声学能量意味着什么。

## <a name="impulse-responses-acoustically-connecting-two-points-in-space"></a>脉冲响应：在声学上连接空间中的两个点

如果您熟悉音频设计，您可能熟悉声学脉冲响应。 声脉冲响应模拟声音从源到侦听器的传输。 因此，脉冲响应可以捕捉房间声学（如遮挡和混响）的每一个有趣的效果。 脉冲响应还具有某些强大的属性，允许音频 DSP 效果缩放。 将两个音频信号相加，并处理脉冲响应，结果与将脉冲响应分别应用于每个信号并添加结果相同。 声学传播和脉冲响应也不依赖于正在处理的音频，仅取决于要建模的场景以及源和侦听器位置。 简而言之，脉冲响应会蒸馏场景对声音传播的影响。

脉冲响应可捕获每个有趣的房间声学效果，并且我们可以通过滤波器有效地将其应用于音频，并且我们可以通过测量或模拟获得脉冲响应。 但是，如果我们不完全希望声学与物理学完全匹配，而是塑造它，以满足场景的情感需求呢？ 但与像素值一样，脉冲响应只是数千个数字的列表，我们怎样才能调整它以满足审美需求呢？ 如果我们想要在穿过门口或障碍物后面时有平稳变化的遮挡/障碍物，我们需要多少脉冲反应才能得到平滑的效果呢？ 如果源移动得快怎么办？ 我们如何插值？

在交互式标题中，很难对声学的某些方面使用仿真和脉冲响应。 但是，如果我们能将模拟的冲动响应与我们熟悉的音频 DSP 效果参数连接，我们仍然可以构建一个音频传输系统，支持设计师的调整。

## <a name="connecting-simulation-to-audio-dsp-with-parameters"></a>使用参数将仿真连接到音频 DSP

脉冲响应包含每一个有趣的（和每一个无趣的）声学效果。 音频 DSP 块在正确设置其参数时，可以呈现有趣的声学效果。 使用声学仿真驱动音频 DSP 块以在 3D 场景中自动传输音频，这仅是使用脉冲响应测量音频 DSP 参数的问题。 对于某些常见和重要的声学效果（包括遮挡、阻塞、入口和混响），此测量值已广为人知。

但是，如果模拟直接连接到音频 DSP 参数，则设计器调整在哪里？ 我们得到什么了？ 嗯，我们通过丢弃脉冲响应和保留一些DSP参数来获得大量的内存。 为了给设计人员一些对最终结果的权力，我们只需要找到一种方法在仿真和音频DSP之间插入设计器。

## <a name="sound-design-by-transforming-audio-dsp-parameters-from-simulation"></a>通过从仿真转换音频 DSP 参数进行声音设计

考虑你的太阳镜对你对世界的看法的影响。 在明亮的一天，眼镜可以减少光泽到更舒适的东西。 在黑暗的房间里，你可能什么也看不见。 眼镜不会在所有情况下设置一定的亮度水平;他们只是让一切都更黑暗。

如果我们使用仿真来驱动我们的音频DSP使用遮挡和混响参数，我们可以在模拟器后添加一个滤波器来调整DSP"看到的"参数。 滤镜不会强制一定级别的遮挡或混响尾长，就像太阳镜不会让每个房间都具有相同的亮度一样。 过滤器可能只是减少每个遮挡。 或遮挡更多。 通过添加和调整一个"变暗"遮挡参数滤波器，大型开放式房间仍然具有很少或没有遮挡效果，而门道将从介质增加到强遮挡效果，同时保持平滑度实际上过渡模拟提供。

在此范例中，设计人员的任务从为每个情况选择声学参数，到选择和调整滤波器以应用于仿真产生的最重要的 DSP 参数。 它将设计师的活动从设置平滑过渡的狭隘关注提升到对遮挡和混响效果强度以及混合中存在源的更高关注。 当然，当情况需要时，始终可用的一个筛选器就是简单地返回在特定情况下为特定源选择 DSP 参数。

## <a name="sound-design-in-project-acoustics"></a>项目声学中的声音设计

Project 声学包集成了上述每个组件：模拟器、提取参数和构建声学资产的编码器、音频 DSP 和一系列滤波器。 使用 Project 声学进行声音设计需要为滤波器选择参数，以调整从仿真派生并应用于音频 DSP 的遮挡和混响参数，并在游戏编辑器和音频引擎中公开动态控件。
