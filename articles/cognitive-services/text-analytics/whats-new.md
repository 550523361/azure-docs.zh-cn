# <a name="whats-new-in-the-text-analytics-api"></a>文本分析 API 中有哪些新功能？

文本分析 API 会持续更新。 为了及时了解最新的发展, 本文提供了有关新版本和功能的信息。

## <a name="sentiment-analysis-v3-public-preview---july-2019"></a>情绪分析 v3 公共预览版-2019 年7月

下一版本的情绪分析现在可用于公共预览版, 并在 API 的文本分类和评分方面提供了显著的改进。 它还提供:

* 为文本中的不同情绪自动添加标签。
* 在文档和句子级别上情绪分析和输出。 

它支持英语, 并在三个区域中提供:加拿大中部、东亚和美国中部。 查找要在后续版本中添加的其他语言和区域。
