> [!TIP]
> 若要在 Azure 门户上查找密钥和终结点，请执行以下操作：
> 1. 导航到位于 https://portal.azure.com/ 的 Azure 资源。
> 2. 单击位于“资源管理”  下的“快速启动”  。
>
> 请务必从终结点中删除 `/text/analytics/<version>` 后缀（如果其中包含）。 与客户端库一起使用不需要此后缀。
