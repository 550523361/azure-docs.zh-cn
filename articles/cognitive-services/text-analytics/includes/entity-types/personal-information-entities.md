## <a name="personal-information-entity-types"></a>个人信息实体类型：

### <a name="person"></a>人员
在文本中识别人员姓名。

语言：
* 公共预览：`English`

| 子类型名称 | 描述                                               | 可从模型版本开始 |
|--------------|-----------------------------------------------------------|----------------------------------------|
| 空值          | 识别的名称，例如`Bill Gates`，`Marie Curie` | `2020-02-01`                           |

### <a name="organization"></a>组织  

认可组织、公司、机构、公司、俱乐部和其他团体。

语言： 

* 公共预览：`English`

| 子类型名称 | 描述                                                                                       | 可从模型版本开始|
|--------------|---------------------------------------------------------------------------------------------------|--------------|
| 空值          | 例如`Microsoft`， `NASA``National Oceanic and Atmospheric Administration` | `2020-02-01` |

### <a name="phone-number"></a>电话号码

电话号码（仅限美国电话号码）。 

语言：

* 公共预览：`English`

| 子类型名称 | 描述                                    | 可从模型版本开始 |
|--------------|------------------------------------------------|----------------------------------------|
| 空值          | 美国电话号码，例如`(312) 555-0176` | `2020-02-01`                           |

### <a name="email"></a>电子邮件

电子邮件地址。 

语言：

* 公共预览：`English`

| 子类型名称 | 描述                                      | 可从模型版本开始 |
|--------------|--------------------------------------------------|----------------------------------------|
| 空值          | 电子邮件地址，例如`support@contoso.com` | `2020-02-01`                           |

### <a name="url"></a>代码

互联网网址。

语言：

* 公共预览：`English`

| 子类型名称 | 描述                                          | 可从模型版本开始 |
|--------------|------------------------------------------------------|----------------------------------------|
| 空值          | 网址到网站，例如`https://www.bing.com` | `2020-02-01`                           |

### <a name="ip-address"></a>IP 地址

互联网协议地址

语言：

* 公共预览：`English`

| 子类型名称 | 描述                              | 可从模型版本开始 |
|--------------|------------------------------------------|----------------------------------------|
| 空值          | 例如网络地址`10.0.0.101` | `2020-02-01`                           |

### <a name="quantity"></a>数量 

数值数量

语言：

* 公共预览：`English`

| 子类型名称 | 描述                   | 可从模型版本开始 |
|--------------|-------------------------------|----------------------------------------|
| 年龄          | `90 days old`, `30 years old` | `2020-02-01`                           |

### <a name="datetime"></a>DateTime

日期和时间实体

语言：

* 公共预览：`English`

| 子类型名称 | 描述                   | 可从模型版本开始 |
|--------------|-------------------------------|----------------------------------------|
| Date         | `May 2nd, 2017`, `05/02/2017` | `2020-02-01`                           |

### <a name="eu-gps-coordinates"></a>欧盟 GPS 坐标

 欧盟境内各地点的 GPS 坐标。 

语言：

* 公共预览：`English`

| 子类型名称 | 描述                               | 可从模型版本开始 |
|--------------|-------------------------------------------|----------------------------------------|
| 空值          | 欧盟内的GPS坐标 | `2019-10-01`                           |

### <a name="azure-information"></a>Azure 信息

可识别的 Azure 信息，包括身份验证信息和连接字符串。 

* 可从模型版本`2019-10-01`开始。

语言：

* 公共预览：`English`

| 子类型名称                          | 描述                                                                 |
|---------------------------------------|-----------------------------------------------------------------------------|
| Azure DocumentDB 身份验证密钥             | Azure 文档 DB 服务器的授权密钥。                           |
| Azure IAAS 数据库连接字符串 | Azure 基础结构作为服务 （IaaS） 数据库的连接字符串。 |
| Azure SQL 连接字符串           | Azure SQL 数据库的连接字符串。                                |
| Azure IoT 连接字符串           | Azure 事物 Internet （IoT） 的连接字符串。                        |
| Azure 发布设置密码        | Azure 发布设置的密码。                                        |
| Azure Redis 缓存连接字符串   | 用于雷瑞斯的 Azure 缓存的连接字符串。                             |
| Azure SAS                             | Azure 软件作为服务 （SAS） 的连接字符串。                     |
| Azure 服务总线连接字符串   | Azure 服务总线的连接字符串。                                |
| Azure 存储帐户密钥             | Azure 存储帐户的帐户密钥。                                   |
| Azure 存储帐户密钥（通用）   | Azure 存储帐户的通用帐户密钥。                           |
| SQL Server 连接字符串          | SQL 服务器的连接字符串。                                         |
