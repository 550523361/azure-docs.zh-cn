# <a name="share-your-custom-search-instance"></a>共享自定义搜索实例
仅使用电子邮件地址即可与任何人共享自定义搜索实例。 通过共享，其他人可修改你的自定义搜索实例。 登录[自定义搜索](https://customsearch.ai)，选择自定义搜索实例，然后单击设置图标（显示为齿轮）。 在“共享实例”部分，输入共享对象的电子邮件地址，然后单击“共享”。 电子邮件地址显示在标题为“实例共享对象”的列表中。 对自定义搜索的每个共享对象重复此操作。 

要将某人的电子邮件添加到列表，并不要求其拥有自定义搜索帐户。 在更改配置前，他们需要先登录自定义搜索。 与某人共享实例后，他们会在自定义搜索实例列表中看到该实例。 一次只能有一人修改实例。 若要尝试修改其他人正在编辑的实例，会出现警告。 最多可与 10 名用户共享实例。

## <a name="stop-sharing"></a>停止共享
若要停止与某人共享实例，请使用“删除”图标从列表删除其电子邮件地址。
