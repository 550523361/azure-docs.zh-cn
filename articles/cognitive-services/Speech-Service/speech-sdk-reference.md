# <a name="speech-sdk-reference"></a>语音 SDK 参考

使用语音软件开发工具包 (SDK)，应用程序可以访问语音服务的功能，这使得开发启用了语音的软件更为容易。 目前，SDK 允许访问**语音到文本**和**语音翻译**。 文本到语音采用 [REST]（基于 HTTP 的 https://docs.microsoft.com/azure/cognitive-services/speech-service/rest-apis) POST 调用）。

下表介绍了当前支持的编程语言和操作系统。

|编程语言|平台|API 参考|
|-|-|-|
|C#|Windows|[Browse](https://aka.ms/csspeech/csharpref)
|C/C++|Windows、Linux|[Browse](https://aka.ms/csspeech/cppref)
|Java\*|设备|[Browse](https://aka.ms/csspeech/javaref)

\* *Java SDK 是[语音设备 SDK](speech-devices-sdk.md) 的一部分。*
