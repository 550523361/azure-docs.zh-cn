> 请先获取订阅密钥。
> 可以从[认知服务订阅](https://azure.microsoft.com/try/cognitive-services/)页面获取免费试用的订阅密钥。 选择语音 API 后，选择“获取 API 密钥”来获取两个密钥（主密钥和辅助密钥）。 可以使用其中任一密钥。 这两个密钥绑定到同一配额。
>
> 付费订阅密钥是通过 Azure 仪表板提供的。
