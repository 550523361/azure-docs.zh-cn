## <a name="reference-docs"></a>参考文档

语音服务提供两个 Sdk。 第一个 SDK 是主要的[语音 sdk](../speech-sdk.md) ，它提供了与语音服务交互所需的大部分功能。 第二个 SDK 特定于设备，正确命名为[语音设备 SDK](../speech-devices-sdk.md)。 这两种 Sdk 都提供多种语言版本。

### <a name="speech-devices-sdk-reference-docs"></a>语音设备 SDK 参考文档

[语音设备 SDK](../speech-devices-sdk.md)是语音 sdk 的超集，具有针对特定设备的扩展功能。 若要下载语音设备 SDK，必须先[选择一个开发工具包](../get-speech-devices-sdk.md#choose-a-development-kit)。
