在本快速入门中，你将使用 REST API 在批处理中识别文件中的语音。 批处理执行语音听录，无需任何用户交互。 它提供了一个简单的编程模型，无需管理并发、自定义语音模型或其他详细信息。 它需要高级控制选项，同时可以有效利用 Azure 语音服务资源。

[批处理听录概述](../../../batch-transcription.md)详细介绍了如何使用此功能。 详细的 API 以 [Swagger 文档](https://westus.cris.ai/swagger/ui/index#/Custom%20Speech%20transcriptions%3A)形式提供，标题为 `Custom Speech transcriptins`。
