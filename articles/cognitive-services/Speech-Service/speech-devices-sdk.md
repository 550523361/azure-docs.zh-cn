# <a name="about-the-speech-devices-sdk-preview"></a>关于语音设备 SDK（预览）

Microsoft 语音服务本身适用于多种设备和音频源。 现在，可以通过匹配的硬件和软件进一步利用语音应用程序。 语音设备 SDK 是与启用麦克风的特定硬件配对使用的一个预优化库。 利用该 SDK，可以轻松集成设备和基于云的 Microsoft 语音服务，还可为客户打造卓越的用户体验。

该 SDK 支持使用自定义唤醒字词生成自己的环境设备，这样，启动用户交互的提示词将为品牌特有。 它使用多声道音频，并通过噪声抑制、回音消除、远场语音和波束赋形来提供更准确的语音识别。

该 SDK 支持多种启用语音的方案，例如免下车点单系统、车载或家用助理和智能扬声器。 可以使用文本回复用户、用默认或自定义语音回答用户、提供搜索结果以及翻译为其他语言等。
