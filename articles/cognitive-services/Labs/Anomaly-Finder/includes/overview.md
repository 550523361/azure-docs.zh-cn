## <a name="overview"></a>概述
通过异常查找器 API 可监视时序数据，并使用机器学习检测异常情况，该机器学习自动应用正确的统计模型来适应独特的数据。 它使组织能够维护数据质量，提供可靠服务，识别业务事件并优化其业务方法，无需解决手动识别异常数据等棘手问题。
