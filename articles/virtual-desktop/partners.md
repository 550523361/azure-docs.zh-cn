# <a name="windows-virtual-desktop-partner-integrations"></a>Windows 虚拟桌面合作伙伴集成

本文列出了按字母顺序的 Windows 虚拟桌面的合作伙伴。

## <a name="active-cypher"></a>Active Cypher

Active 密码是一个 Windows 虚拟桌面增值服务提供程序，它使 IT 与加密技术，帮助保护企业免受漏洞和数据丢失。 活动密钥与创新的加密技术支持在 Azure 上运行 Windows 虚拟桌面的整体文件保护策略。

当用于 Windows 的虚拟桌面，活动密钥提供其他功能，能使可访问性和用户数据的移动性，同时防止数据意外丢失。 与活动密钥，可以利用现有的组安全策略和 Azure 信息保护中的权限来使用 Windows 虚拟桌面的同时加密用户文件。

[转到合作伙伴网站。](https://activecypher.com/)

## <a name="cloudjumper"></a>CloudJumper

CloudJumper 是 Windows 虚拟桌面的增值服务提供程序，它使解决方案提供商和企业 IT 使用软件来全面预配和管理 Windows 虚拟桌面环境。 借助 CloudJumper 软件，it 部门可以管理 Windows 虚拟桌面部署的每一层。 被自动化的工作负荷和应用程序的交付，确保用户可以快速访问他们的桌面的任何设备上的任何位置。

CloudJumper 的软件，云工作区管理套件通过简化部署和在 Azure 中正在进行的管理任务来扩展 Windows 虚拟机的值。 从单一控制台，it 部门可以预配、 管理和优化用户工作区的基础结构。 CloudJumper 的简单脚本触发引擎集成与 IT 服务平台自动预配 Windows 虚拟桌面中涉及的任务。 此外，CloudJumper Api 允许进一步的可扩展性和与 ServiceNow 和 BMC 准备等其他企业系统的集成。

[转到合作伙伴网站。](https://cloudjumper.com/wvd/)

## <a name="lakeside-software"></a>Lakeside 软件

Lakeside 软件是用于监视性能和评估 Azure 迁移就绪性的用户工作负荷提供软件的 IT 团队的 Windows 虚拟桌面增值服务提供程序。 使用此软件，IT 收益更清晰的可见性应用程序使用情况和资源消耗来简化迁移过程。 Lakeside 软件收集数据的每个工作区以在用户环境中，创建一份综合报告实现快速的故障排除和优化的资产。

Lakeside 软件的数字体验监视解决方案，SysTrack，可以帮助跟踪性能和标识迁移的理想工作负荷提供出色的用户体验。 SysTrack 致力于扩展的 Windows 虚拟桌面，到适度调整评估值和连续监视的用户环境。

[转到合作伙伴网站。](https://www.lakesidesoftware.com/assessments/wvd)

## <a name="liquidware"></a>Liquidware

Liquidware 是提供的管理和优化 Windows 虚拟桌面部署的软件的 Windows 虚拟桌面增值服务提供程序。 Liquidware Essentials 套件提供了通过分层、 用户环境管理和主要的用户体验可见性和诊断应用程序交付。 评估已做好迁移准备和分析使用情况指标的解决方案，Liquidware 为最终用户提供无缝的虚拟桌面体验。

Liquidware Essentials 扩展通过有效地收集用户配置文件，并收集关键用户数据，以简化迁移到 Azure 的用户环境的 Windows 虚拟机的值。 此外，Liquidware Essentials 通过统一的用户配置文件简化了映像管理和分层应用基于可配置权限管理设置。

[转到合作伙伴网站。](https://www.liquidware.com/solutions/solutions-platform/microsoft)

## <a name="nerdio"></a>Nerdio

Nerdio 是 Azure IT 自动化平台，它可以更轻松地部署和管理 Windows 虚拟桌面。 Nerdio 提供的知识和技术来部署，价格、 包、 管理和优化客户的 Azure 部署 — 使用 Windows 虚拟桌面前端和中心。

Nerdio 扩展轻松地预配 Azure 资源和简化部署了 Windows 虚拟机的值。 使用 Nerdio 为 Azure，it 部门可以自动部署和管理完整的 Azure 环境，包括 Windows 虚拟桌面中在两个小时内完成。

[转到合作伙伴网站。](https://getnerdio.com/academy/windows-virtual-desktop-explained-for-msps/)

## <a name="thinprint"></a>ThinPrint

ThinPrint 是一个 Windows 虚拟桌面增值服务提供程序，可提供简单且安全的云打印从 Windows 虚拟桌面。 其服务和软件，可以利用现有的打印基础结构从云打印文档。 ThinPrint 使连接到这两个本地和网络打印机，方便用户在办公室或远程工作打印。

ThinPrint 的 ezeep 解决方案与现有企业打印基础结构的连接，从而扩展 Windows 虚拟机的值。 ezeep 使用户无论身在何处企业中对打印的控制。 使用 ezeep，用户可以桥接 Windows 虚拟桌面和打印硬件之间的差距。

[转到合作伙伴网站。](http://www.ezeep.com/wvd-printing)

## <a name="workspot"></a>Workspot

Workspot 是提供具有高性能的台式计算机和在 Azure 中的工作站的企业的 Windows 虚拟桌面增值服务提供程序。 使用 Workspot，基础结构预配自动化的这意味着用户可以从任何位置访问其 Windows 虚拟桌面环境的高可用性在世界各地。

Workspot 通过简化的云桌面基础结构预配过程来扩展 Windows 虚拟机的值。 与 Workspot，资源可以轻松地进行扩展和缩减以满足不同的用户，并使用案例的需求。 Workspot 可以优化的高性能 GPU 工作站 CAD 和工程用户，以及 Windows 应用程序所需的部署和所有业务用户的 Windows 10 桌面。

[转到合作伙伴网站。](https://www.workspot.com/wvd)
