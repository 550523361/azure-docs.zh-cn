# <a name="connect-with-the-web-client"></a>使用 Web 客户端进行连接

Web 客户端允许您从 Web 浏览器访问 Windows 虚拟桌面资源，而无需冗长的安装过程。

>[!NOTE]
>Web 客户端当前不支持移动操作系统。

## <a name="supported-operating-systems-and-browsers"></a>支持的操作系统和浏览器

虽然任何支持 HTML5 的浏览器都应该工作，但我们正式支持以下操作系统和浏览器。

| 浏览者           | 支持的 OS                     | 说明               |
|-------------------|----------------------------------|---------------------|
| Microsoft Edge    | Windows                          |                     |
| Internet Explorer | Windows                          |                     |
| Apple Safari      | macOS                            |                     |
| Mozilla Firefox   | Windows、macOS、Linux            | 版本 55 或更高版本 |
| Google Chrome     | 视窗、macOS、Linux、Chrome操作系统 |                     |

## <a name="access-remote-resources-feed"></a>访问远程资源源

在浏览器中，导航到[Windows 虚拟桌面 Web 客户端](https://rdweb.wvd.microsoft.com/webclient)，然后使用用户帐户登录。

>[!NOTE]
>如果您已经使用与要用于 Windows 虚拟桌面的帐户不同的 Azure 活动目录帐户登录，则应注销或使用专用浏览器窗口。

登录后，现在应看到资源列表。 您可以通过在"**所有资源"** 选项卡中选择资源来启动资源，就像选择普通应用一样。
