# <a name="sign-in-and-start-apps-from-the-my-apps-portal"></a>从"我的应用"门户登录并开始应用

您可以将工作或学校帐户与基于 Web**的"我的应用"** 门户一起使用，以便：

- 查看和启动组织的许多基于云的应用
- 更新您的个人资料和帐户信息
- 查看**您的群组**信息
- 执行应用和组**的访问审核**

如果无法访问“我的应用”门户，请联系支持人员以获取相关权限。****

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

> [!Important]
> 此内容适用于“我的应用”用户。 如果你是管理员，可以在[应用程序管理文档](https://docs.microsoft.com/azure/active-directory/manage-apps)中查找有关如何设置和管理基于云的应用的详细信息。

## <a name="supported-browsers"></a>支持的浏览器

您可以通过以下任何 Web 浏览器访问 **"我的应用"** 门户：

- Google Chrome

- Mozilla 火狐，版本 26.0 或更高版本

- Microsoft Edge

- 互联网浏览器，版本 11（有限支持）

## <a name="download-and-install-the-my-apps-secure-sign-in-extension"></a>下载并安装"我的应用安全登录扩展"

如果系统提示，请下载并安装"我的应用安全登录扩展"。 此扩展可帮助您启动组织的任何需要您使用单个登录过程的云应用。 如果您的组织已为您设置单一登录，则会自动安装扩展，您可以跳过此部分。

此扩展可帮助您：

- 直接从登录页面登录到应用。

- 使用**快速搜索**功能启动任何应用。

- 请参阅"**最近使用"** 部分中使用的最后一个应用。

- 使用内部公司 URL 时使用[应用程序代理](https://docs.microsoft.com/azure/active-directory/active-directory-application-proxy-get-started)远程。

### <a name="to-download-and-install-the-extension"></a>下载并安装扩展

下载并安装扩展，具体取决于您使用的浏览器：

- **谷歌浏览器。** 从 Chrome 网上商店转到["我的应用安全登录扩展"](https://chrome.google.com/webstore/detail/my-apps-secure-sign-in-ex/ggjhpefgjjfobnfoldnjipclpcfbgbhl)功能，然后选择"添加到**Chrome**"。

- **莫齐拉·火狐**从**Firefox 加载页**，转到["我的应用程序安全登录扩展"](https://addons.mozilla.org/firefox/addon/access-panel-extension/)功能，然后选择"**添加到 Firefox**"。

- **微软边缘**从 Microsoft 应用商店转到["我的应用安全登录扩展"](https://www.microsoft.com/p/my-apps-secure-sign-in-extension/9pc9sckkzk84?rtc=1&activetab=pivot%3Aoverviewtab)功能，然后选择 **"获取"** 以获取 Microsoft 边缘旧版浏览器的扩展。  
如果你正在使用新的微软边缘浏览器（铬的边缘），那么你可以从[微软边缘加载项商店](https://microsoftedge.microsoft.com/addons/category/EdgeExtensionsEditorsPick)获得兴奋。

**地址**栏右侧会添加一个图标，允许您登录并自定义扩展。

## <a name="to-change-your-my-apps-portal-using-the-extension"></a>使用扩展更改"我的应用"门户

您可以在"**最近使用"** 部分中选择要查看的应用数，并决定是否允许组织的内部 URL 重定向。

1. 选择**地址**栏右侧的新 **"我的应用安全登录扩展"**![图标](media/my-apps-portal/my-apps-portal-extension-icon.png)扩展图标，然后选择"**登录以开始**"。

2. 右键单击 **"设置"** 图标!["](media/my-apps-portal/my-apps-portal-extension-settings-icon.png)设置"图标，然后选择 **"设置**"。

3. 在 **"设置"** 框中，选择要在门户上看到的最新应用数，以及是否允许组织的内部 URL 重定向，以便您可以远程使用它们。

    ![扩展的设置页，显示可用的自定义项](media/my-apps-portal/my-apps-portal-extension-settings-page.png)

## <a name="access-and-use-the-my-apps-portal-by-device"></a>按设备访问和使用"我的应用"门户

您可以通过 Intune 管理的浏览器或 iOS 或 Android 移动设备访问和使用计算机上的"我的应用"门户。

!["我的应用"门户中的应用页面](media/my-apps-portal/my-apps-portal-apps-page.png)

### <a name="access-and-use-the-my-apps-portal-on-your-computer"></a>访问和使用计算机上的"我的应用"门户

如果您有权访问和使用组织的基于云的应用，则可以通过 **"我的应用"** 门户访问这些应用。

1. 登录到计算机上的工作或学校帐户。

2. 在受支持的 Web 浏览器中，打开https://myapps.microsoft.com并转到 ，或者使用组织提供的链接（如果他们将您定向到自定义页面，如`https://myapps.microsoft.com/contoso.com`）。

    将显示 **"应用"** 页面，显示组织拥有的所有可供您使用的所有基于云的应用。

3. 在 **"应用"** 页面中，选择要开始使用的应用。

    将打开应用的新页面，您可以在其中登录（如有必要）或开始使用该应用程序。

### <a name="access-and-use-the-my-apps-portal-on-an-intune-managed-browser"></a>在 Intune 管理的浏览器上访问和使用"我的应用"门户

在 iOS 和 Android 设备上从 Intune 托管浏览器查看和使用组织的应用。

1. 在移动设备上，从 Apple 应用商店和 Google Play 商店下载并安装 Intune 托管浏览器应用。

2. 打开 Intune 托管浏览器应用，转到https://myapps.microsoft.com，或使用组织提供的链接（如果他们将您定向到自定义页面，如https://myapps.microsoft.com/contoso.com）。

    将显示 **"应用"** 页面，显示组织拥有并可供您使用的所有基于云的应用。

3. 在 **"应用"** 页面中，选择要开始使用的应用。

    将打开应用的新页面，您可以在其中登录（如有必要）或开始使用该应用程序。

### <a name="access-and-use-the-my-apps-portal-on-an-ios-device"></a>在 iOS 设备上访问和使用"我的应用"门户

从 iPhone 或 iPad 设备查看和使用 **"我的应用"** 门户，运行 iOS 版本 7 或更高版本。 您还可以安装["我的应用"移动应用](https://itunes.apple.com/us/app/my-apps-azure-active-directory/id824048653?mt=8)，以在 iOS 设备上访问组织的应用。

1. 在移动设备上，启动 Web 浏览器应用，如 Safari。

2. 转到https://myapps.microsoft.com，或使用组织提供的链接（如果他们将您定向到自定义页面，如https://myapps.microsoft.com/contoso.com。

    将显示 **"应用"** 页面，显示组织拥有的所有可供您使用的所有基于云的应用。

3. 在 **"应用"** 页面中，选择要开始使用的应用。

    将打开应用的新页面，您可以在其中登录（如有必要）或开始使用该应用程序。

### <a name="access-and-use-the-my-apps-portal-on-an-android-device"></a>在 Android 设备上访问和使用"我的应用"门户

在 Android 设备上查看和使用 **"我的应用"** 门户。

1. 在移动设备上，启动 Web 浏览器应用，如 Google Chrome。

2. 转到https://myapps.microsoft.com，或使用组织提供的链接（如果他们将您定向到自定义页面，如https://myapps.microsoft.com/contoso.com。

    将显示 **"应用"** 页面，显示组织拥有并可供您使用的所有基于云的应用。

3. 在 **"应用"** 页面中，选择要开始使用的应用。

    将打开应用的新页面，您可以在其中登录（如有必要）或开始使用该应用程序。

## <a name="add-a-new-app-to-the-my-apps-portal"></a>向"我的应用"门户添加新应用

如果管理员已授予您权限，则可以向 **"应用"** 页添加新应用。

1. 在 **"应用"** 页面中，执行以下操作之一：
    - 如果您处于原始"我的应用"体验中，请选择"**添加应用**"，如下所示。

      ![在"我的应用"门户中添加应用页面](media/my-apps-portal/my-apps-portal-add-apps-page.png)

    - 如果您正在更新的"我的应用"体验中，请选择"**添加自助服务应用**"。

       ![在"我的应用"门户中添加应用页面，myapplications.microsoft.com](media/my-apps-portal/my-apps-portal-add-app-link.png)

2. 选择要从提供列表中添加的应用，然后选择 **"添加**"。

3. 应用将添加到 **"应用"** 页上的列表。

    某些应用在添加之前可能需要管理员批准。 发生这种情况时，在管理员批准应用之前，不会将应用添加到**应用**页面。

## <a name="start-a-cloud-based-app"></a>启动基于云的应用

可以从 **"我的应用"** 门户启动任何可用的基于云的应用。 您只会看到您有权使用的应用。

- 在 **"应用"** 页面中，选择要开始使用的应用。

    将打开应用的新页面，您可以在其中登录（如有必要）或开始使用该应用程序。

## <a name="activities-in-the-my-apps-portal"></a>"我的应用"门户中的活动

进入 **"我的应用"** 门户后，您可以：

- 将应用整理到组织创建和提供的各种类别中。 有关详细信息，请参阅[在“我的应用”门户中访问和使用集合](my-applications-portal-workspaces.md)。

- 查看、更新或撤销授予应用程序的权限。 有关详细信息，请参阅[在“我的应用”门户中编辑或撤销应用程序权限](my-applications-portal-permissions-saved-accounts.md)。

>[!Note]
>如果看不到任何可用的集合或类别，则表示管理员尚未设置或共享任何集合。 若要查看共享的集合，请联系组织的支持人员以获得其他帮助或权限。
