# <a name="change-your-work-or-school-account-password-from-the-change-password-page"></a>从"更改密码"页更改您的工作或学校帐户密码

"**我的帐户"** 门户的 **"更改密码**"页面可帮助您更新工作或学校帐户的现有密码，前提是您记住密码，并且您未锁定在帐户之外。 如果您不记得密码，如果您被锁定在帐户之外，或者从未从组织获得密码，则可以使用安全信息和移动设备重置密码。

>[!Important]
>本文适用于尝试更新现有工作或学校帐户的已知密码的用户。 如果您是试图进入个人帐户的用户（如 Xbox、Hotmail 或Outlook.com，请尝试在"[何时无法登录 Microsoft 帐户](https://support.microsoft.com/help/12429/microsoft-account-sign-in-cant)"文章中提供建议。 如果您是管理员，试图查找有关如何为员工或其他用户测试自助服务密码重置的详细信息，请参阅[自助服务密码重置](https://docs.microsoft.com/azure/active-directory/authentication/quickstart-sspr)。

## <a name="update-a-password-from-the-change-password-page"></a>从"更改密码"页更新密码

1. 使用现有密码登录到您的工作或学校帐户，然后转到 **"我的帐户**"（https://myprofile.microsoft.com/)页面）。

2. 从左侧导航窗格中选择 **"密码"** 或从 **"密码**"块中选择 **"更改密码**"链接。

    ![我的帐户页面，显示突出显示的"更改密码"链接](media/my-account-portal/my-account-portal-change-password.png)

3. 键入旧密码，然后创建和确认新密码。

    ![更改密码页，显示密码字段](media/my-account-portal/my-account-portal-change-password-page.png)

4. 选择“提交”****。

    您的密码将被更改，系统将要求您再次登录您的工作或学校帐户。
