# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>在“我的个人资料(预览)”门户上查看近期的登录活动
可以使用工作或学校帐户从“我的个人资料”门户的“近期活动”页查看最近所有的登录活动。

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>查看近期的登录活动
1. 登录到工作或学校帐户，然后转到 https://myprofile.microsoft.com/ 页。

2. 从左侧导航窗格中选择“近期活动”或从“近期活动”块中选择“查看活动”链接。

    ![“我的个人资料”页，显示突出显示的近期活动链接](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. 展开并查看每个登录项，确保辨认得出每一项。 如果发现不熟悉的登录项，我们强烈建议更改密码以保护帐户安全。

    ![包含展开的登录详细信息的近期活动页面](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >如果认为帐户受到威胁，请参阅[如何访问受到威胁的 Microsoft 帐户](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account)，以获得有关重置密码或解锁帐户的更多帮助。

## <a name="next-steps"></a>后续步骤

- 查看近期登录活动后，可以选择查看或管理[安全信息](user-help-security-info-overview.md)、连接[设备](myprofile-portal-devices-page.md)或[组织](myprofile-portal-organizations-page.md)，或者查看组织[使用数据](myprofile-portal-privacy-page.md)的方式。
