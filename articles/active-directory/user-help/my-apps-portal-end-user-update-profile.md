# <a name="update-your-profile-and-account-info-from-the-my-apps-portal"></a>更新我的应用门户从你配置文件和帐户信息
可以使用基于 web 的使用工作或学校帐户**我的应用**门户中，来查看和启动很多组织的基于云的应用，以更新您的部分配置文件和帐户信息，若要查看你**组**信息，并执行**访问评审**应用和组。 如果您不能访问**我的应用**门户中，您必须与支持人员联系的权限。

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

>[!Important]
>此内容适用于用户。 如果你是管理员，可以找到有关如何设置和管理基于云的应用程序中的详细信息[应用程序管理文档](https://docs.microsoft.com/azure/active-directory/manage-apps)。

## <a name="view-your-organization-related-profile-information"></a>查看你的组织相关的配置文件的信息
具体取决于什么你的组织已选择要包括在你**配置文件**页**我的应用**门户中，您可能会看到你特定与工作相关的详细信息，以及你的设备和活动，和任何您所属的其他组织。

### <a name="to-view-your-profile-information"></a>若要查看个人资料信息
1.  登录到你的工作或学校帐户[转到**我的应用**门户](my-apps-portal-end-user-access.md)。

2. 从**应用程序**页上，从右上角，在那里显示你的名称和你的组织，选择你的个人资料图片，然后选择**配置文件**。

    **配置文件**页出现后，显示个人资料信息。

    ![显示所有可用信息的配置文件页](media/my-apps-portal/my-apps-portal-profile-page.png)

3. 从**配置文件**页上，你可以：

    - **查看你的组织相关的详细信息。** 查看照片、 名称、 标题、 关联的电子邮件地址和 office 信息。 此信息由您的组织，你将无法对其进行更改。 如果看到错误，请联系支持人员联系。

    - **查看设备和活动**。 请确保每个设备已熟悉并已正确连接到你的组织。 如果您不能识别设备，选择**禁用设备**若要删除与你的帐户关联。 禁用设备后，它将删除在此页。

    - **查看你的组织。** 请确保仍处理与每个指定的组织。 如果您不能再用于组织，我们强烈建议您选择**登录以离开组织**。 离开组织后，它将删除在此页。

## <a name="manage-your-work-or-school-account-information"></a>管理你的工作或学校帐户信息
更新并管理你的工作或学校帐户信息从**配置文件**页**我的应用**门户。 此页可帮助你：

- 更改工作或学校帐户密码。

- 启用密码重置 （如果你的管理员已启用此功能）。

- 提供了其他安全验证信息。

- 查看你组织的使用条款。

- 注销所有位置。

## <a name="change-your-password"></a>更改密码
如果你想要更改工作或学校帐户的密码，则可以选择**更改密码**从**管理帐户**区域**配置文件**页。

### <a name="to-change-your-password"></a>若要更改你的密码
1. 从**管理帐户**区域中的**配置文件**页上，选择**更改密码**。

2. 上**更改密码**页上，请确保你的用户 ID 是否正确，，然后的框中键入旧密码和新密码。

    ![更改密码页](media/my-apps-portal/my-apps-portal-change-password-page.png)

3. 选择“提交”。

    更改你的密码。 你将需要登录到任何应用之前登录到使用你的工作或学校帐户。

## <a name="set-up-and-use-password-reset"></a>设置和使用密码重置
如果你忘记了密码、 永远不会获得的是来自公司支持人员，或锁定你的帐户，可以重置自己的密码。

>[!Important]
>你的管理员必须启用此功能，必须注册参加。 有关如何注册以及如何重置密码的详细信息，请参阅[注册自助服务密码重置](active-directory-passwords-reset-register.md)并[重置工作或学校密码](user-help-reset-password.md)。

## <a name="change-your-security-verification-information"></a>更改你的安全验证信息
如果你的组织要求你使用双重验证，您可以添加、 更新和删除关联的安全信息从**其他安全性验证**页。

双重验证要求使用两条验证信息，如密码和 pin 码，然后才能获取你的帐户或组织的信息。 有关详细信息的双重验证，请参阅[设置我的帐户进行双重验证](multi-factor-authentication-end-user-first-time.md)。

### <a name="to-change-your-security-information"></a>若要更改你的安全信息
1. 从**管理帐户**区域中的**配置文件**页上，选择**其他安全性验证**。

    ![其他安全性验证页](media/my-apps-portal/my-apps-portal-additional-verification-page.png)

2. 上**其他安全性验证**页上，您可以添加、 更改或删除以下信息：

    - **默认验证选项。** 选择要用于双重验证的默认辅助方法。 输入用户名和密码后，都需要双因素验证任何时间自动使用此方法。

    - **添加、 更新或删除的验证方法。** 您可以添加新、 更新现有的、 或删除不再使用的旧信息。

    - **设置 Microsoft Authenticator 应用。** 您可以选择设置 Microsoft Authenticator 应用作为验证方法进行处理。 有关 Microsoft Authenticator 应用的详细信息，请参阅[Microsoft Authenticator 应用是什么？](user-help-auth-app-overview.md)

3. 选择“保存”以保存更改。

## <a name="review-your-organizations-terms-of-use-statement"></a>查看你组织的使用条款声明
您可以选择查看你组织的使用条款声明，如果可用。

1. 从**管理帐户**区域中的**配置文件**页上，选择**查看使用条款**。

2. 查看你组织的使用条款和选择**接受**以确认你已阅读并理解您的组织的使用条款。

    ![使用从我的应用门户页面中的条款](media/my-apps-portal/my-apps-portal-tou-page.png)

    如果你的组织不具有的使用条款，则可以选择**完成**回到**配置文件**页。

## <a name="sign-out-of-everywhere"></a>所有位置注销
您可以选择注销无处不在你当前登录到使用你的工作或学校帐户。 这包括所有应用和设备。

### <a name="to-sign-out-of-everywhere"></a>若要注销所有位置
1. 从**管理帐户**区域中的**配置文件**页上，选择**从所有位置注销**。

2. 在中**从所有位置注销**确认框中，选择**是**以确认你想要注销所有会话和设备。 选择**否**如果改变主意。
