# <a name="update-your-profile-and-account-information-on-the-my-apps-portal"></a>在"我的应用"门户上更新您的个人资料和帐户信息

您可以将工作或学校帐户与基于 Web**的"我的应用"** 门户一起使用， 

- 查看和启动组织的许多基于云的应用
- 更新您的一些个人资料和帐户信息
- 查看**您的群组**信息
- 执行应用和组**的访问审核**。 

如果无法访问“我的应用”门户，则必须联系支持人员以获取相关权限。****

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

> [!Important]
> 此内容适用于“我的应用”用户。**** 如果你是管理员，可以在[应用程序管理文档](https://docs.microsoft.com/azure/active-directory/manage-apps)中查找有关如何设置和管理基于云的应用的详细信息。

## <a name="view-your-organization-related-profile-information"></a>查看与组织相关的配置文件信息

根据您的组织选择在 **"我的应用"** 门户的**个人资料**页面上包含的内容，您可能会看到特定与工作相关的详细信息以及您的设备和活动以及您所属的任何其他组织。

### <a name="to-view-your-profile-information"></a>查看您的个人资料信息

1. 登录到您的工作或学校帐户，[转到 **"我的应用"** 门户](my-apps-portal-end-user-access.md)。

2. 在 **"应用"** 页面中，从右上角选择您的个人资料图片，其中显示您的姓名和组织，然后选择 **"配置文件**"。

    将显示 **"配置文件"** 页，显示您的个人资料信息。

    ![配置文件页，显示所有可用信息](media/my-apps-portal/my-apps-portal-profile-page.png)

3. 在 **"配置文件"** 页面中，您可以：

    - **查看与组织相关的详细信息。** 查看您的照片、姓名、标题、关联的电子邮件地址和办公室信息。 此信息由您的组织管理，您将无法更改它。 如果您看到错误，请联系您的帮助台。

    - **查看您的设备&活动**。 确保每个设备都熟悉并正确连接到您的组织。 如果您无法识别设备，请选择 **"禁用设备"** 以删除与您的帐户的关联。 禁用设备后，设备将从此页面中删除。

    - **查看您的组织。** 确保您仍与每个指定的组织一起工作。 如果您不再与组织合作，我们强烈建议您选择 **"登录"以离开组织**。 离开组织后，它将从此页面中删除。

## <a name="manage-your-work-or-school-account-information"></a>管理您的工作或学校帐户信息

从 **"我的应用"** 门户的 **"配置文件"** 页面更新和管理您的工作或学校帐户信息。 此页面可帮助您：

- 更改工作或学校帐户密码。

- 打开密码重置（如果您的管理员已打开此功能）。

- 提供其他安全验证信息。

- 查看组织的使用条款。

- 从任何地方注销。

## <a name="change-your-password"></a>更改密码

如果要更改工作或学校帐户的密码，可以从 **"配置文件"** 页的 **"管理帐户**"区域选择 **"更改密码**"。

### <a name="to-change-your-password"></a>更改密码

1. 从 **"配置文件"** 页的 **"管理帐户**"区域中，选择 **"更改密码**"。

2. 在 **"更改密码"** 页上，请确保您的用户 ID 正确，然后在框中键入旧密码和新密码。

    ![更改密码页](media/my-apps-portal/my-apps-portal-change-password-page.png)

3. 选择“提交”****。

    您的密码已更改。 您需要登录到以前登录使用工作或学校帐户的任何应用。

## <a name="set-up-and-use-password-reset"></a>设置和使用密码重置

如果您忘记了密码，从未从公司支持获得密码，或者帐户被锁定，您可以重置自己的密码。

>[!Important]
>管理员必须打开此功能，您必须注册才能参与。 有关如何注册以及如何重置密码的详细信息，请参阅[注册自助服务密码重置](active-directory-passwords-reset-register.md)和[重置工作或学校密码](active-directory-passwords-update-your-own-password.md)。

## <a name="change-your-security-verification-information"></a>更改安全验证信息

如果您的组织要求您使用双重验证，则可以从 **"其他安全验证**"页添加、更新和删除关联的安全信息。

双重验证要求您使用两段验证信息，如密码和 PIN，然后才能访问您的帐户或组织的信息。 有关双因素验证的详细信息，请参阅[设置我的帐户进行两步验证](multi-factor-authentication-end-user-first-time.md)。

### <a name="to-change-your-security-information"></a>更改安全信息

1. 从 **"配置文件"** 页的 **"管理帐户**"区域中，选择 **"其他安全验证**"。

    ![其他安全验证页](media/my-apps-portal/my-apps-portal-additional-verification-page.png)

2. 在 **"其他安全验证**"页上，您可以添加、更改或删除以下信息：

    - **默认验证选项。** 选择用于双因素验证的默认辅助方法。 输入用户名和密码后，任何时间都需要双重验证时，都会自动使用此方法。

    - **添加、更新或删除验证方法。** 您可以添加新、更新现有或删除不再适用的旧信息。

    - **设置 Microsoft 身份验证器应用。** 您可以选择将 Microsoft 身份验证器应用设置为验证方法。 有关 Microsoft 身份验证器应用的详细信息，请参阅什么是[Microsoft 身份验证器应用？](user-help-auth-app-overview.md)

3. 选择 **"保存"** 以保存更改。

## <a name="review-your-organizations-terms-of-use-statement"></a>查看组织的使用条款声明

您可以选择查看组织的使用条款声明（如果可用）。

1. 从 **"配置文件"** 页的 **"管理帐户**"区域中，选择 **"查看使用条款**"。

2. 查看组织的使用条款**并选择"接受"** 以确认您已阅读并理解组织的使用条款。

    !["我的应用"门户中的使用条款页面](media/my-apps-portal/my-apps-portal-tou-page.png)

    如果您的组织没有使用条款，则可以选择 **"完成"** 以返回到 **"配置文件"** 页。

## <a name="sign-out-of-everywhere"></a>注销无处不在

您可以选择在您当前登录到您的工作或学校帐户的任何地方注销。 这包括所有应用和设备。

### <a name="to-sign-out-of-everywhere"></a>到处注销

1. 从 **"配置文件"** 页的 **"管理帐户**"区域中，选择 **"随处注销**"。

2. 在"**到处注销"** 确认框中，选择 **"是**"以确认要注销所有会话和设备。 如果您改变主意，请选择 **"否**"。
