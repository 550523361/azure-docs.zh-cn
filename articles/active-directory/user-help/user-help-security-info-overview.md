# <a name="security-info-preview-overview"></a>安全信息（预览版）概述

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-security-info.md)]

>[!Important]
>此内容适用于用户。 如果你是管理员，可以在 [Azure Active Directory 文档](https://docs.microsoft.com/azure/active-directory)中查找有关如何设置和管理 Azure Active Directory (Azure AD) 环境的详细信息。

在“安全信息”中设置双重验证和密码重置信息。 在初次设置安全信息后，不需再次设置。 但是，可以随时编辑该信息并更改默认方法。

可以添加下述任何方法，其中包括：

|文章 |说明 |
|------|------------|
|[设置安全信息以使用电话呼叫](security-info-setup-phone-number.md)| 介绍如何将电话号码设置为方法。|
|[设置安全信息以使用短信](security-info-setup-text-msg.md)|介绍如何将电话号码设置为方法。|
|[设置安全信息以使用 Authenticator 应用](security-info-setup-auth-app.md)|介绍如何将 Authenticator 应用设置为方法。|
|[设置安全信息以使用电子邮件](security-info-setup-email.md)|介绍如何设置电子邮件地址，以便重置密码。|
|[设置安全信息以使用预定义的安全性问题](security-info-setup-questions.md)|介绍如何使用安全性问题来重置密码。|
|[管理应用密码](security-info-app-passwords.md)|介绍如何使用安全信息来设置应用密码。|
|[管理安全信息](security-info-manage-settings.md)|介绍如何更新安全信息并将其与应用密码协同使用。|
|[如何登录](user-help-sign-in.md)|介绍如何使用指定方法登录。|
