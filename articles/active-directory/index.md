# <a name="azure-active-directory-documentation"></a>Azure Active Directory 文档

Azure Active Directory (Azure AD) 是 Microsoft 提供的基于多租户云的目录和标识管理服务。  Azure AD 为运行在 Azure 和本地的应用程序提供标识和访问功能。
