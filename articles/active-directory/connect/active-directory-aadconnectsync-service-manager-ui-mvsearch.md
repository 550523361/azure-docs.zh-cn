# <a name="sync-service-manager-metaverse-search"></a>Sync Service Manager Metaverse 搜索

![Sync Service Manager](./media/active-directory-aadconnectsync-service-manager-ui/mvsearch.png)

排查与数据相关的问题时，“Metaverse 搜索”选项卡非常有用。 在上半部分，可以基于属性组合创建查询。 如果对查询结果满意，请单击“搜索”。 结果显示在底部网格中。 可以利用“列设置”来选择应显示的列。

在搜索结果中，选择对象，并选择“属性”查看 [Metaverse 对象属性](active-directory-aadconnectsync-troubleshoot-object-not-syncing.md#metaverse-object-properties)。
