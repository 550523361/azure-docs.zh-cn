# <a name="how-do-azure-active-directory-admins-add-b2b-collaboration-users"></a>Azure Active Directory 管理员如何添加 B2B 协作用户？

全局管理员和受限制的管理员可以使用 [Azure 门户](https://portal.azure.com)邀请 B2B 协作用户加入目录、加入任何组或任何应用程序。
