# <a name="view-requests-for-an-access-package-in-azure-ad-entitlement-management"></a>查看 Azure AD 权限管理中访问包的请求

在 Azure AD 授权管理中，可以看到谁已请求访问包、其策略和状态。 本文介绍如何查看访问包的请求。

## <a name="view-requests"></a>查看请求

**必备角色：** 全局管理员、用户管理员、目录所有者或访问包管理器

1. 在 Azure 门户中，依次单击“Azure Active Directory”、“标识监管”。

1. 在左侧菜单中，单击 "**访问包**"，然后打开访问包。

1. 单击 "**请求**"。

1. 单击特定的请求以查看更多详细信息。

    ![访问包的请求列表](./media/entitlement-management-access-package-requests/requests-list.png)
