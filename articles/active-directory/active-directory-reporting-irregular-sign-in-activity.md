# <a name="irregular-sign-in-activity"></a>异常登录活动
异常登录是指根据“不可能前往”条件与异常登录位置和设备，由机器学习算法标识的登录。 这可能指示黑客已使用此帐户成功登录。
如果我们在 30 天或更短时间内遇到 10 个或以上异常登录事件，我们将向全局管理员发送一个电子邮件通知。 请务必将 aad-alerts-noreply@mail.windowsazure.com 包含在安全发件人列表中。
