# <a name="windows-10-for-the-enterprise-ways-to-use-devices-for-work"></a>面向企业的 Windows 10：在工作中使用设备的方式
Windows 10 使你可以利用 Azure Active Directory (Azure AD)。 可以将 Windows 10 设备连接到 Azure AD，以便用户可以通过使用 Azure AD 帐户或添加其 Azure ID 登录到 Windows，获取对业务应用和资源的访问权限。
