# <a name="azure-active-directory-b2b-collaboration-invitation-redemption"></a>Azure Active Directory B2B 协作邀请兑换

## <a name="azure-ad-and-microsoft-account-users"></a>Azure AD 和 Microsoft 帐户用户
对于拥有现有 Azure AD 帐户或 Microsoft 帐户用户，兑换体验非常简单，只需为签名。

## <a name="social-id-user-first-time-redemption"></a>社交 ID 用户首次兑换
Azure AD B2B 协作可轻松用于兑换任何电子邮件地址。 非 Microsoft 电子邮件地址用于 B2B 协作时，应考虑兑换体验。 此兑换流是更多地涉及，因为你可能需要在兑换时创建帐户。 下面的视频中签出它：
