# <a name="azure-active-directory-b2b-collaboration-invitation-redemption"></a>Azure Active Directory B2B 协作邀请兑换

## <a name="azure-ad-and-microsoft-account-users"></a>Azure AD 和 Microsoft 帐户用户
对于拥有现有 Azure AD 帐户或 Microsoft 帐户的用户，兑换体验就像登录一样简单。

## <a name="social-id-user-first-time-redemption"></a>社交 ID 用户第一时间兑换
Azure AD B2B 协作可轻松地将任何电子邮件地址用于兑换。 请看一下将非 Microsoft 电子邮件地址用于 B2B 协作时的兑换体验。 这是一个更复杂的兑换流程，因为你可能需要在兑换时创建帐户。 在以下视频中一探究竟：
