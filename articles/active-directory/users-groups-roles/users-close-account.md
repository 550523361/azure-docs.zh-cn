# <a name="close-your-work-or-school-account-in-an-unmanaged-directory"></a>关闭工作或学校帐户的非托管目录

如果您是在非托管的 Azure Active Directory (Azure AD) 组织，并且用户不再需要使用该组织的应用或维护是与任何关联，可以随时关闭帐户。 非托管的目录没有全局管理员。 非托管目录中的用户可以关闭其帐户代表自身，而无需联系管理员。

非托管目录中的用户是通常在创建自助服务注册。 一个示例可能是信息工作者的组织中的一项免费服务注册。 有关自助服务注册的详细信息，请参阅[什么是自助服务注册 Azure Active Directory？](directory-self-service-signup.md)。

[!INCLUDE [GDPR-related guidance](../../../includes/gdpr-intro-sentence.md)]

## <a name="before-you-begin"></a>开始之前

可以关闭你的帐户之前，应确认以下各项：

* 请确保您是用户的非托管 Azure AD 目录。 如果您属于托管目录，无法关闭你的帐户。 如果你属于托管目录，并想要关闭你的帐户，则必须联系你的管理员。 有关如何确定您是否属于非托管目录的信息，请参阅[从非托管租户中删除用户](https://docs.microsoft.com/flow/gdpr-dsr-delete#delete-the-user-from-unmanaged-tenant)。

* 保存你想要保留任何的数据。 有关如何提交导出请求的信息，请参阅[访问和导出系统生成的日志的非托管租户](https://docs.microsoft.com/power-platform/admin/powerapps-gdpr-dsr-guide-systemlogs#accessing-and-exporting-system-generated-logs-for-unmanaged-tenants)。

> [!WARNING]
> 关闭你的帐户是不可逆的。 当关闭你的帐户时，将删除所有个人数据。 不再将有权访问你的帐户和与你的帐户关联的数据。

## <a name="close-your-account"></a>关闭你的帐户

若要关闭的非托管的工作或学校帐户，请按照下列步骤：

1. 登录到[关闭你的帐户](https://go.microsoft.com/fwlink/?linkid=873123)，使用你想要关闭的帐户。

1. 上**我的数据请求**，选择**关闭帐户**。

    ![我的数据请求-关闭帐户](./media/users-close-account/close-account.png)

1. 查看确认消息，然后选择**是**。

    ![我的数据请求-关闭确认](./media/users-close-account/confirm-close.png)
