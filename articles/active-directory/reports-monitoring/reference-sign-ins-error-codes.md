# <a name="sign-in-activity-report-error-codes"></a>登录活动报告错误代码 

使用[用户登录报告](concept-sign-ins.md)提供的信息，您可以找到问题的答案，例如：

- 谁登录到了我的应用程序？
- 已登录到哪些应用程序？
- 哪些登录失败？为什么？

当登录失败时，您将看到与失败对应的错误代码。 [查找错误代码的说明](https://login.microsoftonline.com/error)。
