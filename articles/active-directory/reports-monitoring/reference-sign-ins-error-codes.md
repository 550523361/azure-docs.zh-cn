# <a name="sign-in-activity-report-error-codes"></a>登录活动报告错误代码 

参考[用户登录报告](concept-sign-ins.md)提供的信息，可以找到一些问题的答案，例如：

- 谁登录到了我的应用程序？
- 已登录到哪些应用程序？
- 哪些登录失败？

登录失败时，将显示与失败对应的错误代码。 [查找错误代码的说明](https://login.microsoftonline.com/error)。
