# 来自未知源的登录

| 说明 | 报告位置 |
| :-------------     | :-------        |
| <p>此报表指示已分配 Microsoft 已识别为匿名代理 IP 地址的客户端 IP 地址时，已成功登录到你的目录的用户。这些代理通常由隐藏了自己计算机的 IP 地址的用户使用，可能用于恶意目的 – 有时黑客使用这些代理。</p><p> 来自此报表的结果将显示用户从该地址和代理的 IP 地址成功登录到你的目录的次数。</p> | “目录”>“报告”选项卡 |
