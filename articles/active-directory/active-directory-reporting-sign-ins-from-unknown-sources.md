# <a name="sign-ins-from-unknown-sources"></a>来自未知源的登录
此报告指示已分配 Microsoft 已识别为匿名代理 IP 地址的客户端 IP 地址（例如，Tor IP 地址）时，已成功登录到你的目录的用户。 这些代理通常由隐藏了自己计算机的 IP 地址的用户使用，可能用于恶意目的。

来自此报告的结果将显示用户从该地址和代理的 IP 地址成功登录到你的目录的次数。
