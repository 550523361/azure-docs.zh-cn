# <a name="quickstart-secure-a-web-api-with-azure-active-directory"></a>快速入门：使用 Azure Active Directory 保护 Web API

[!INCLUDE [active-directory-develop-applies-v1](../../../includes/active-directory-develop-applies-v1.md)]

[Microsoft 标识平台](v2-overview.md)由 Azure Active Directory (Azure AD) 开发人员平台演变而来。 开发人员可以通过它来生成应用程序，从而可以采用所有 Microsoft 标识登录，以及获取令牌来调用 Microsoft Graph 等 Microsoft API 或开发人员生成的 API。

## <a name="next-steps"></a>后续步骤

请参阅 [Azure Active Directory OIDC node.js Web 示例](https://github.com/azureadquickstarts/appmodelv2-webapp-openidconnect-nodejs)，其中演示了如何在通过 Express 使用 Node.js 构建的 Web 应用程序中使用 Azure AD v2.0 设置 OpenId Connect 身份验证。 它设计为在任何平台上运行。
