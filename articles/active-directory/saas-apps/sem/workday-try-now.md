# <a name="workday-and-azure-active-directory"></a>Workday 和 Azure Active Directory

使用预先构建的集成将 Workday 与 Azure Active Directory (Azure AD) 相连接, 并在员工加入、搬迁或休假时自动预配或取消预配用户帐户。

![显示 Workday 应用徽标](./media/workday.png)

> [!div class="nextstepaction"]
> [尝试 Azure AD 的 Workday >](https://aka.ms/WorkdayTryItNow)

## <a name="what-can-you-do-with-workday-and-azure-ad"></a>您可以对 Workday 和 Azure AD 执行哪些操作？

- **自动执行用户预配**-基于 Workday 的组、角色或其他属性, 自动将应用程序分配给员工。
- **同步员工配置文件**-确保员工数据在 Azure AD 和 Workday 之间保持最新。
- **简化加入和脱离**-自动为新员工预配用户帐户, 以便他们可以在第一天有效地工作, 并通过在员工离职时自动取消预配用户帐户来降低安全风险。
- **简单单一登录**消除用户名和密码, 并允许用户使用其在 Azure AD 中托管的组织帐户进行登录。

## <a name="get-started"></a>开始使用

通过简单的配置指南设置 Workday 的[用户预配](https://aka.ms/WorkdayProvisioning)和[单一登录](https://aka.ms/WorkdaySSODoc)。
