# <a name="servicenow-and-azure-active-directory"></a>ServiceNow 和 Azure Active Directory

在整个组织中加快采用 ServiceNow, 并通过预生成的集成提高用户采用。 通过将 ServiceNow 连接到 Azure Active Directory (Azure AD) 简化你的 IT 服务管理。

![显示 ServiceNow 应用徽标](./media/servicenow.png)

> [!div class="nextstepaction"]
> [尝试 Azure AD 的 ServiceNow >](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>您可以对 ServiceNow 和 Azure AD 执行哪些操作？

- **简单单一登录**-通过允许用户使用其 Azure AD 帐户登录到 ServiceNow, 消除了多个用户名和密码, 并简化了访问。
- **自动用户预配**-当用户加入、移动或离开你的组织时, 自动创建、维护和删除用户帐户到 ServiceNow 中。
- 通过应用条件访问策略, 控制对 ServiceNow 的访问安全访问, 通过访问评审控制访问权限, 并使用登录日志监视访问。

## <a name="get-started"></a>开始使用

通过简单的配置指南, 为 ServiceNow 设置[用户预配](https://aka.ms/ServiceNowProvisioning)和[单一登录](https://aka.ms/ServiceNowSSO)。
