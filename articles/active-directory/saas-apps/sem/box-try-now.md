# <a name="box-and-azure-active-directory"></a>Box 和 Azure Active Directory

在整个组织中加快采用机箱, 并通过预生成的集成提高用户采用。 使用户能够通过将 Box 连接到 Azure Active Directory (Azure AD) 快速进行协作。

![显示 Box 应用徽标](./media/box.png)

> [!div class="nextstepaction"]
> [Azure AD > 的 Try 方框](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Box 和 Azure AD 有什么作用？

- **简单单一登录**-通过允许用户使用其 Azure AD 帐户登录到 Box, 消除了多个用户名和密码, 并简化了访问。
- **自动用户预配**-当用户加入、移动或离开你的组织时, 自动创建、维护和删除用户帐户到 Box。
- 通过应用条件性访问策略, 控制对 Box 的访问安全访问, 通过访问评审控制访问权限, 并使用登录日志监视访问权限。

通过简单的配置指南, 为 Box 设置[用户](https://aka.ms/BoxSSODoc)设置和[单一登录](https://aka.ms/BoxProvisioningDoc)。
