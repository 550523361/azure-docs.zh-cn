# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet 和 Azure Active Directory

在整个组织中加快采用 Smartsheet, 并通过预生成的集成提高用户采用。 使用户能够通过将 Smartsheet 连接到 Azure Active Directory (Azure AD) 来快速协作。

![显示 Smartsheet 应用徽标](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [试用 Azure AD > 的 Smartsheet](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>Smartsheet 和 Azure AD 有哪些用途？

- **简单单一登录**-通过允许用户使用 Azure AD 帐户登录到 Smartsheet, 消除了多个用户名和密码, 并简化了访问。
- **自动执行用户预配**-当用户加入、移动或离开你的组织时, 自动创建、维护和删除用户帐户 Smartsheet。
- 通过应用条件访问策略, 控制对 Smartsheet 的访问安全访问, 通过访问评审控制访问权限, 并使用登录日志监视访问。

## <a name="get-started"></a>开始使用

为 Smartsheet 设置[用户预配](https://aka.ms/SmartsheetProvisioningDoc)和[单一登录](https://aka.ms/SmartsheetSSODoc), 并提供简单的配置指南。
