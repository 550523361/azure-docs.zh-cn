# <a name="sign-ins-from-possibly-infected-devices"></a>从可能受感染的设备登录
此报告尝试标识已受感染并且现已成为僵尸网络一部分的用户设备。 我们针对已知连接到僵尸网络服务器的 IP 地址关联用户登录的 IP 地址。

建议：此报告标记 IP 地址，而不是用户设备。 我们建议你联系用户并扫描用户的所有设备进行确认。 用户的个人设备也可能受感染，或者以用户身份使用了相同 IP 地址的人（用户除外）具有受感染的设备。

有关如何解决恶意软件感染的详细信息，请参阅[恶意软件防护中心](http://go.microsoft.com/fwlink/?linkid=335773)。
