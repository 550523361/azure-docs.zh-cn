# <a name="users-with-anomalous-sign-in-activity"></a>具有异常登录活动的用户
这是一份聚合报告，其中合并了以下报告中的可疑登录：

<ul><li>来自未知源的登录</li>
<li>多次失败后登录</li>
<li>从多个地理区域登录</li>
<li>从具有可疑活动的 IP 地址登录</li>
<li>从可能受感染的设备登录</li>
<li>异常登录活动</li>
</ul>
