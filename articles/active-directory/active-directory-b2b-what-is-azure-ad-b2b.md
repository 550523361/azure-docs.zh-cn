# <a name="azure-ad-b2b-collaboration-preview-simple-secure-cloud-partner-integration"></a>Azure AD B2B 协作预览版：简单、安全的云合作伙伴集成
随着公司更多地专注于其核心业务，必然会增加与其他企业的合作。 公司需要方便、安全地与其合作伙伴共享资源（如对公司应用程序的访问），来开展有效协作。 Azure Active Directory B2B 协作可让合作伙伴使用其自行管理的标识有选择性地访问本方的企业应用程序和数据，为跨公司合作关系提供支持。 Azure AD B2B 协作的体验如下所示：

* **简单**：每个合作伙伴用户使用现有的 Azure AD 帐户或在邀请接受过程中轻松创建的 Azure AD 帐户。 可以允许该用户通过“应用访问面板”直接访问选定的公司应用或一组应用程序。
* **安全**：管理员通过 Azure AD 目录控制对公司应用的所有访问。 协作终止后，可以从 Azure AD 中删除合作伙伴用户，将立即撤销这些用户对你的应用的访问权限。 此外，在合作伙伴用户退出所属合作伙伴组织时，相应的访问权限自动失去。
* **无缝**：需要访问公司应用的合作伙伴公司**无需拥有 Azure AD**。 Azure AD B2B 协作简化了用户注册体验，使这些合作伙伴可以立即访问你的应用。

请查看[宣布推出公共预览版的博客文章](http://blogs.technet.com/b/ad/archive/2015/09/15/learn-all-about-the-azure-ad-b2b-collaboration-preview.aspx)和演示 Azure AD B2B 协作的[本深入探讨视频](https://channel9.msdn.com/Series/Azure-Active-Directory-Videos-Demos/Azure-Active-Directory-B2B-collaboration-demo)。
