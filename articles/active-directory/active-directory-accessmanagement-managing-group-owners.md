# 管理组的所有者

##将所有者分配到组
**通过 Azure 管理门户向组中添加所有者**

1. 在管理门户中，单击“Active Directory”，然后单击你所在组织的目录的名称。 
2. 单击“组”选项卡。
3. 在“组”页上，单击要将所有者添加到的组的名称。
4. 在该组的页上，单击“添加所有者”。
5. 在“添加所有者”页上，单击要添加为该组所有者的用户名，并确保该名称已添加到“选定”窗格中。


**通过 Azure 管理门户删除组中的所有者**

1. 在管理门户中，单击“Active Directory”，然后单击你所在组织的目录的名称。
2. 单击“组”选项卡。
3. 在“组”页上，单击要从中删除所有者的组的名称。
4. 在该组的页上，单击“所有者”选项卡。
5. 在该组的页上，单击要从该组中删除的所有者的名称，然后单击“删除”。
