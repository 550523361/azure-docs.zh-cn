# <a name="azure-data-factory-whitepapers"></a>Azure 数据工厂白皮书 

阅读白皮书，若要了解 Azure 数据工厂在更深的层次。 本文提供一系列可用白皮书用于 Azure 数据工厂。

| **白皮书** | **说明** |
| --- | --- |
|[Azure 数据工厂-数据 in the Cloud 的集成](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | 本白皮书介绍 Azure 数据工厂如何使您能够构建现代数据仓库，启用高级的分析，从而提高智能 SaaS 应用程序并将 SQL Server 集成服务包迁移到 Azure。|
|[数据从本地关系数据仓库迁移到 Azure 中使用 Azure 数据工厂](https://azure.microsoft.com/mediahandler/files/resourcefiles/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/Data_migration_from_on-prem_RDW_to_ADLS_using_ADF.pdf) | 本白皮书中讨论迁移的复杂性数十 TB 数据从现有本地关系数据仓库 (例如，Netezza、 Oracle、 Teradata，SQL server) 到 Azure （例如，Blob 存储或 Azure Data Lake 存储） 使用 Azure 数据工厂。 围绕恢复能力、 性能、 可伸缩性、 管理和大数据引入迁移到 Azure 的 Azure 数据工厂的安全说明了挑战和最佳做法。 |
|[Azure 数据工厂：在云中的 SSIS](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| 本白皮书都要通过为什么想要将现有 SSIS 工作负荷迁移到 Azure 数据工厂和地址常见的注意事项和问题。 然后将引导你完成创建 Azure SSIS IR 的技术详细信息，然后演示如何上载、 执行和监视 Azure 数据工厂使用的工具可能是使用者熟悉类似 SQL Server Management Studio (SSMS) 通过各个包。 |
