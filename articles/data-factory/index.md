# <a name="azure-data-factory-documentation"></a>Azure 数据工厂文档

Azure 数据工厂是一项基于云的数据集成服务，可以自动移动和转换数据。 数据工厂服务可以创建数据集成解决方案，从各种数据存储引入数据、转换和处理数据，以及将结果数据发布到数据存储。
