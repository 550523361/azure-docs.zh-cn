# <a name="sql-data-warehouse-documentation"></a>SQL 数据仓库文档

Azure SQL 数据仓库将 SQL Server 关系数据库与 Azure 云向外扩展功能相结合。  SQL 数据仓库在大规模并行处理 (MPP) 体系结构的基础上构建，可以处理你的企业工作负荷。
