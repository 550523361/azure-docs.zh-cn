# <a name="module-twin-json-schema"></a>模块孪生 JSON 架构

IoT Edge 上的事件网格与 IoT Edge 生态系统集成，并支持通过模块孪生创建主题和订阅。 它还报告模块孪生上报告的所有主题和事件订阅的当前状态。

> [!WARNING]
> 由于 IoT Edge 生态系统中的限制，以下 json 示例中的所有数组元素都编码为 json 字符串。 请参阅`EventSubscription.Filter.EventTypes`下面的`EventSubscription.Filter.AdvancedFilters`示例中的键。

## <a name="desired-properties-json"></a>所需属性 JSON

* 主题部分中每个键值对的值与创建主题时 API 上使用的`Topic.Properties`JSON 架构完全相同。
* **事件订阅**部分中每个键值对的值与创建主题时 API 上用于`EventSubscription.Properties`的 json 架构完全相同。
* 要删除主题，请将其值设置为`null`所需的属性。
* 不支持通过所需属性删除事件订阅。

```json
{
    "topics": {
        "twinTopic1": {},
        "twinTopic2": {
            "inputSchema": "customEventSchema"
        }
    },
    "eventSubscriptions": {
        "twinTopic1WebhookSub": {
            "topic": "twinTopic1",
            "retryPolicy": {
                "eventExpiryInMinutes": 120,
                "maxDeliveryAttempts": 30
            },
            "destination": {
                "endpointType": "WebHook",
                "properties": {
                    "endpointUrl": "https://localhost:4438"
                }
            },
            "filter": {
                "subjectBeginsWith": "^",
                "subjectEndsWith": "$",
                "isSubjectCaseSensitive": false,
                "includedEventTypes": "[\"et1\",\"et2\"]",
                "advancedFilters": "[{\"value\":true,\"operatorType\":\"BoolEquals\",\"key\":\"data.b\"},{\"values\":[\"\\\"\",\"c\"],    \"operatorType\":\"StringContains\",\"key\":\"data.s\"}]"
            }
        },
        "twinTopic2EdgeHubSub": {
            "topic": "twinTopic2",
            "deliveryPolicy": {
                "approxBatchSizeInBytes": 200000,
                "maxEventsPerBatch": 25
            },
            "destination": {
                "endpointType": "EdgeHub",
                "properties": {
                    "outputName": "twinTopic2EdgeHubSub"
                }
            },
            "filter": {
                "advancedFilters": "[{\"value\":true,\"operatorType\":\"BoolEquals\",\"key\":\"dAt\\\"A.a\"},{\"values\":[\"\\\"\",    \"c\"],\"operatorType\":\"StringContains\",\"key\":\"dAt\\\"A.a\"}]"
            }
        }
    }
}
```

## <a name="reported-properties-json"></a>报告的属性 JSON

模块孪生的已报告属性部分包括以下信息：

* 模块存储中存在的主题和订阅集
* 创建所需主题/事件订阅时遇到的任何错误
* 任何启动错误（如所需的属性 JSON 解析失败）
