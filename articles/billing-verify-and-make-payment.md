# 如何验证付款方式并为 Azure 订阅付费

>[AZURE.NOTE] 如果你在提交信用卡信息后收到错误，请参阅以下文章：[尝试注册 Azure 时信用卡没有通过验证](billing-credit-card-fails-during-azure-sign-up.md)。

[AZURE.INCLUDE [support-disclaimer](../includes/support-disclaimer.md)]

## 验证当前付款方式
1. 以帐户管理员身份登录到 [Azure 帐户中心](https://account.windowsazure.com/Subscriptions)。

2. 在“单击订阅以查看详细信息和使用情况”下面，选择要验证付款方式的订阅。

     ![selectsub](./media/billing-verify-and-make-payment/selectsub.png)

3. 选择“更改付款方式”。

    ![changepayment](./media/billing-verify-and-make-payment/changepayment.png)

4. 你将看到列出了当前付款方式。选择它以验证所需信息。

    ![changecard](./media/billing-verify-and-make-payment/changecard.png)

## 付款

如果你已将信用卡或借记卡设置为付款方式，则将自动进行付款。若要在任何时候更改卡片，请参阅[如何更改 Azure 订阅的付款方式或信用卡](billing-how-to-change-credit-card.md)。

如果你采用发票付款方式，可将付款寄送到发票底部列出的地点。有关更多帮助，请联系[Microsoft 支持部门](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade)。
