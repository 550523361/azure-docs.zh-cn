# <a name="connect-citrix-analytics-security"></a>连接 Citrix Analytics（安全） 

Citrix 分析（安全）与 Azure Sentinel 集成可帮助您将针对风险事件分析的数据从 Citrix 分析（安全）导出到 Azure Sentinel。 您可以创建自定义仪表板、分析来自其他来源的数据以及 Citrix 分析（安全性）的数据，并使用逻辑应用创建自定义工作流来监视和缓解安全事件。 



## <a name="prerequisites"></a>先决条件

- 读取和写入 Azure Sentinel 工作区的权限。

- 在 Citrix 云中授权 Citrix 安全分析。


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>将 Azure 哨兵连接到 Citrix 分析（安全性）

1. 在 Azure 门户中，导航到**Azure 哨兵** > **数据连接器**，然后选择**Citrix 分析（安全）** 连接器。

2. 选择 **"打开连接器"页**。

3. 按照**Citrix 分析（安全）** 页面上的说明进行操作。
