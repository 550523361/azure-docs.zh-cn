# <a name="connect-extrahop-revealx"></a>连接 ExtraHop Reveal(x)

ExtraHop 显示（x） 数据连接器可让您轻松地将"显示"系统与 Azure Sentinel 连接，以查看仪表板、创建自定义警报并改进调查。 通过此集成，您可以深入了解组织的网络并改进安全操作功能。


## <a name="prerequisites"></a>先决条件

- 读取和写入 Azure Sentinel 工作区的权限。

- 具有固件版本 7.8 或更高版本的 ExtraHop 发现或命令设备，具有具有无限（管理员）权限的用户帐户。

## <a name="connect-azure-sentinel-to-extrahop-revealx"></a>将 Azure 哨兵连接到外点显示（x）

1. 在 Azure 门户中，导航到**Azure 哨兵** > **数据连接器**，然后选择 **"额外显示（x）"** 连接器。

2. 选择 **"打开连接器"页**。

3. 按照 **"额外霍普显示（x）"** 页面上的说明进行操作。
