# <a name="connect-trend-micro-deep-security"></a>连接趋势微深安全

趋势微深度安全连接器可让您轻松地将深层安全日志与 Azure Sentinel 连接，以查看仪表板、创建自定义警报并改进调查。 这使您可以更深入地了解组织的网络和系统，并改进您的安全操作功能。


## <a name="prerequisites"></a>先决条件

- 读取和写入 Azure Sentinel 工作区的权限。

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>将 Azure 哨兵连接到趋势微深度安全

1. 在 Azure 门户中，导航到**Azure 哨兵** > **数据连接器**，然后选择**趋势微深层安全**连接器。

2. 选择 **"打开连接器"页**。

3. 按照说明和后续步骤操作。
