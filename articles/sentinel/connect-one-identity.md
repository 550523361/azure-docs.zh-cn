# <a name="connect-one-identity-safeguard"></a>连接一个标识保护
"一个标识保护"数据连接器使用特权会话特定仪表板的防护功能增强标准公共事件格式 （CEF） 连接器。 使用此连接器可以轻松地开始使用设备生成的事件进行可视化、警报、调查等。


## <a name="prerequisites"></a>先决条件

- 读取和写入 Azure Sentinel 工作区的权限。

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>将 Azure 哨兵连接到一个标识保护

1. 在 Azure 门户中，导航到**Azure 哨兵** > **数据连接器**，然后选择 **"一个标识保护"** 连接器。

2. 选择 **"打开连接器"页**。

3. 按照"**一个身份保护"** 页面上的说明进行操作。
