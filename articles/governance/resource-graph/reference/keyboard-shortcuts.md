# <a name="keyboard-shortcuts-for-azure-resource-graph-explorer"></a>Azure 资源 Graph 资源管理器的键盘快捷方式

本文列出了在 Azure 门户的 "Azure 资源图资源管理器" 页中可以使用的键盘快捷方式。 若要查看全局键盘快捷方式列表或其他页面可用的键盘快捷方式列表，请访问 [Azure 门户中的键盘快捷](../../../azure-portal/azure-portal-keyboard-shortcuts.md)方式。

|---|---|---|
|注释/撤销注释 |<kbd>CTRL</kbd>+<kbd>/</kbd> | <kbd>Port</kbd>+<kbd>/</kbd> |

|增加缩进 |<kbd>CTRL</kbd> +<kbd>]</kbd> |<kbd>Cmd</kbd> +<kbd>]</kbd> |
|全选 |<kbd>CTRL</kbd> +<kbd>一个</kbd> |<kbd>Cmd</kbd> +<kbd>一个</kbd> |
|下移行 |<kbd>ALT</kbd> +<kbd>↓</kbd> |<kbd>选项</kbd> +<kbd>↓</kbd> |
|向上移动一行 |<kbd>ALT</kbd> +<kbd>↑</kbd> |<kbd>选项</kbd> +<kbd>↑</kbd> |
|Replace |<kbd>CTRL</kbd> +<kbd>F</kbd> |<kbd>Cmd</kbd> +<kbd>ALT</kbd> +<kbd>F</kbd> |

## <a name="keyboard-shortcuts-for-the-query-editor"></a>查询编辑器的键盘快捷方式

| 操作 | Windows 快捷方式 | Mac 快捷方式 |
|---|---|---|
|运行当前选定的查询 |<kbd>移位</kbd> +<kbd>输入</kbd> | <kbd>移位</kbd> +<kbd>输入</kbd> |
|取消正在运行的查询 |<kbd>Esc</kbd> | <kbd>Esc</kbd> |
|打开 "新建查询" 选项卡 |<kbd>CTRL</kbd> +<kbd>ALT</kbd> +<kbd>N</kbd> | <kbd>Cmd</kbd> +<kbd>选项</kbd> +<kbd>N</kbd> |
|关闭活动查询选项卡 |<kbd>CTRL</kbd> +<kbd>ALT</kbd> +<kbd>W</kbd> | <kbd>Cmd</kbd> +<kbd>选项</kbd> +<kbd>W</kbd> |
|保存当前查询 |<kbd>CTRL</kbd> +<kbd>S</kbd> | <kbd>Cmd</kbd> +<kbd>S</kbd> |
|打开已保存的查询 |<kbd>CTRL</kbd> +<kbd>O</kbd> | <kbd>Cmd</kbd> +<kbd>O</kbd> |
|切换到查询选项卡 (1-9)  |<kbd>CTRL</kbd> +<kbd> (1-9) </kbd> | <kbd>Cmd</kbd> +<kbd> (1-9) </kbd> |
|跳转到下一个查询选项卡 |<kbd>CTRL</kbd> +<kbd>↓</kbd> | <kbd>Cmd</kbd> +<kbd>↓</kbd> |
|跳转到上一个查询选项卡 |<kbd>CTRL</kbd> +<kbd>↑</kbd> | <kbd>Cmd</kbd> +<kbd>↑</kbd> |
|将当前查询复制到新选项卡 |<kbd>CTRL</kbd> +<kbd>K</kbd>， <kbd>CTRL</kbd> + <kbd>F</kbd> | <kbd>Cmd</kbd> +<kbd>K</kbd>， <kbd>Cmd</kbd> + <kbd>F</kbd> |
