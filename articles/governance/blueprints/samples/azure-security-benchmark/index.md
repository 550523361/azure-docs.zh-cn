# <a name="overview-of-the-azure-security-benchmark-blueprint-sample"></a>Azure 安全基准蓝图示例的概述

Azure 安全基准蓝图示例提供了监管防护措施，其中使用 [Azure Policy](../../../policy/overview.md) 来帮助评估特定 [Azure 安全基准](../../../../security/benchmarks/overview.md)控制要求。 此蓝图帮助客户部署了一组核心策略，以用于客户要在其中实现 Azure 安全基准控制要求的、由 Azure 部署的任何体系结构。

## <a name="control-mapping"></a>控制映射

控制映射部分提供了有关包含在此蓝图内的策略的详细信息，以及有关这些策略如何满足 Azure 安全基准中的各种控制要求的详细信息。 分配给一个体系结构时，资源由 Azure Policy 评估是否不符合已分配的策略。 有关详细信息，请参阅 [Azure Policy](../../../policy/overview.md)。
