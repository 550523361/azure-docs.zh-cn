# <a name="manage-your-application-preferences"></a>管理应用程序首选项

*本文适用于操作员、构建者和管理员。*

使用 IoT Central 可以灵活地自定义应用程序来满足需求。 我们还允许每个用户灵活定义其自己的视图。 本文介绍了用户可以应用于其配置文件的各种自定义选项。

## <a name="changing-language"></a>更改语言

IoT 中心支持 17 种语言。 可以使用顶部导航栏上的**语言选取器**切换你的首选语言。 更改语言后，会记住你的选择并将其应用于你的所有应用程序中。 请注意，应用程序中的自定义项（例如仪表板图像）不会本地化。 

![IoT Central 语言选取器](media/howto-manage-profile/language-picker.png)

## <a name="changing-theme"></a>更改主题

我们同时支持深色主题和浅色主题。 虽然深色主题为默认主题，但是可以通过单击右上角中你的名称并切换主题来进行更改。

![IoT Central 主题选取器](media/howto-manage-profile/theme-picker.png)
