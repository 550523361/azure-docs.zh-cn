# <a name="commercial-marketplace-policies-and-terms"></a>商业市场政策和条款

>[!Note]
>"微软商业市场"是指 Microsoft 拥有或运营的平台，无论该平台如何命名，通过该平台，产品/服务可能会向客户展示或获得。 除非另有说明，商业市场包括微软 Azure 应用商店和微软 AppSource。

感谢您对商业市场上发布优惠的兴趣。 我们致力于与您合作，打造丰富的云解决方案和业务系列产品，让全球客户满意，并帮助您建立业务。

市场上的报价必须符合我们的政策和条款。 我们不时更新这些政策和条款，以帮助确保良好的客户体验，并为客户提供成功的机会。 如果您对我们的政策或条款有反馈，请在[Azure 应用商店和 Microsoft AppSource 论坛](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/bd-p/2222)中发表评论，告知我们。
