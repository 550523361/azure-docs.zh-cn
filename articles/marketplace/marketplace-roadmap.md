# <a name="microsoft-commercial-marketplace-roadmap"></a>微软商业市场路线图

本文档提供了 Microsoft 为发布商、卖家和客户提供商业市场下一步内容的视图。 它捕获了我们承诺的一些重要功能，这是一个粗略的时间框架，当您可以预期看到它们，并将每月刷新。 它不是所有新功能的完整列表，但旨在提供我们关键投资的可见性。 具体功能和上市时间可能随时发生更改。

欢迎提供建议！ 加入 [Microsoft 合作伙伴社区](https://www.microsoftpartnercommunity.com/)中的对话，获取新功能、计划和事件的最新更新。
