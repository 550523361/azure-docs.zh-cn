# <a name="microsoft-commercial-marketplace-roadmap"></a>Microsoft 商业市场路线图

本文档提供了针对发布者、卖方和客户的 Microsoft 商业市场的下一步。 它会捕获一些已提交的重要功能，这是你可能希望看到这些功能并按月进行刷新的大致时间范围。 它并不是所有新功能的完整列表，但旨在提供我们的主要投资。 具体功能和上市时间可能随时发生更改。

欢迎提供建议！ 加入 [Microsoft 合作伙伴社区](https://www.microsoftpartnercommunity.com/)中的对话，获取新功能、计划和事件的最新更新。
