# <a name="office-365-dynamics-365-power-apps-and-power-bi-offer-publishing-guide"></a>Office 365、动态 365、电源应用和 Power BI 提供发布指南

Microsoft AppSource 包括基于动态 365、Office 365、Power BI 和电源应用构建或扩展的优惠。 这些产品中的每一个都有特定的文档，可指导您进行发布过程。 有关发布每个产品/服务类型的发布流程和指南的链接，请参阅下文。

AppSource 已付费、列出和试用优惠，并调用"立即获取"、"立即联系"和"立即试用"。

## <a name="product-specific-guides"></a>特定于产品的指南

| **产品**    | **发布指南**  |
| :------------------- | :-------------------|
| Office 365 | 查看[发布流程和指南](/office/dev/store/submit-to-the-office-store) |
| Dynamics 365 Business Central | 要在合作伙伴中心发布产品/服务，请参阅[创建新的 Dynamics 365 商业中心产品/服务](./partner-center-portal/create-new-business-central-offer.md)。 云合作伙伴门户不再支持创建和管理 Dynamics 365 业务中心产品/ |
| Dynamics 365 for Finance and Operations | 构建企业版时，请查看[发布过程和指南](/dynamics365/unified-operations/dev-itpro/lcs-solutions/lcs-solutions-app-source)。<br/>要在合作伙伴中心发布产品/服务，请参阅[为运营产品/服务创建新的 Dynamics 365。](./partner-center-portal/create-new-operations-offer.md) 云合作伙伴门户不再支持创建和管理 Dynamics 365，用于财务和运营产品。 |
| Dynamics 365 for Customer Engagement | 查看[发布过程和指导](/dynamics365/customer-engagement/developer/publish-app-appsource)。<br/>要在合作伙伴中心发布产品/服务，请参阅[为客户互动创建新的动态 365 & PowerApps 产品/](./partner-center-portal/create-new-customer-engagement-offer.md) 云合作伙伴门户不再支持创建和管理 Dynamics 365，用于财务和运营产品。 |
| Power Apps | 了解如何[让客户在 AppSource 上测试您的应用程序](https://powerapps.microsoft.com/blog/appsource-test-drive/)。 |
| Power BI | 查看[发布过程和指导](/power-bi/developer/office-store)。 |
