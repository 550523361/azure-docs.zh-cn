# <a name="co-sell-partner-engagement"></a>联合销售合作伙伴参与

微软的联合销售计划使合作伙伴能够直接与微软合作，共同寻找销售机会。 您可以覆盖庞大的客户社区，创造新的销售机会，帮助加快业务增长。

## <a name="partner-selling-motions"></a>合作伙伴销售动议

有三个合作伙伴销售动议：

* *通过合作伙伴销售*：您可以通过自己的卖家转售微软技术。
* *服务共同销售*：卖家和合作伙伴协作销售您的项目或托管服务与微软技术嵌入。
* *IP 共同销售*： 卖家/合作伙伴协作销售您的可重复打包的 IP 解决方案*。

\*当与 Microsoft 的现场销售团队成功销售并部署时，其中一些 IP 共同销售解决方案可能会获得额外的奖励。
