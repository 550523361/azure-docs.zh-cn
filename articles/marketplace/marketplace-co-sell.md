# <a name="co-sell-partner-engagement"></a>联合销售合作伙伴参与

如果你是合作伙伴，则可以通过 Microsoft 联合销售计划直接与 Microsoft 合作，获得联合销售机会。 该计划可以帮助我们的合作伙伴赢得广泛的客户群并创造新的销售机会，帮助推动业务增长。

## <a name="partner-selling-motions"></a>合作伙伴销售活动

Microsoft 提供三种合作伙伴销售活动：

* 通过合作伙伴销售 - 可以通过自己的卖方转售 Microsoft 技术。
* 服务联合销售 - 卖方与合作伙伴合作销售嵌入了 Microsoft 技术的项目或托管服务。
* IP 联合销售 - 卖方与合作伙伴合作销售可重复的打包 IP 解决方案。*

\* 其中一些 IP 联合销售解决方案在通过 Microsoft 现场销售团队成功销售和部署后可能会收到额外的奖励。
