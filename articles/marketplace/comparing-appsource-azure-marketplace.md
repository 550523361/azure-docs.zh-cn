# <a name="comparing-appsource-and-azure-marketplace"></a>比较 AppSource 和 Azure 市场  
店面服务于独特的客户需求。 商店按角色启用目标设置。 你可以根据客户提供正确的解决方案或服务。  

## <a name="understanding-the-differences-between-the-storefronts"></a>了解店面之间的差异  
要选择店面，首先从确定产品/服务的目标受众开始： 如果面向两种受众，那么发布一次即可在两个店面中销售。  
 
考虑店面的其他优势。   

### <a name="appsource"></a>AppSource  

| 优点 | 详细信息 | 
|:--- |:--- | 
| ***灵活计费*** | 设置试用体验。 目前暂不提供启用商务功能的发布选项。 可以利用当前的订购和计费基础结构，且无需进行额外的投资或更改。 | 
| ***推进与其他合作伙伴的连接*** | 可将独立软件供应商、系统集成商和托管服务提供商链接到特定的实现方案。 你能以协作方式销售给新客户。 | 
| ***自动化*** | 可将独立软件供应商、系统集成商和托管服务提供商链接到特定的实现方案。 你能以协作方式销售给新客户。 |  
| ***多个云类型*** | 当前不支持 Azure Stack、Azure 政府或区域云。 | 
| ***向客户进行上下文演示*** | 通过 Dynamics 365、Power BI 和 Office 365 等 Microsoft 产品的应用内体验来接触更多的客户。 |  

### <a name="azure-marketplace"></a>Azure 市场  

| 优点 | 详细信息 | 
|:--- |:--- | 
| ***灵活计费*** | 对于 VM，采用“即用即付”计费选项，使用 Microsoft 企业协议 (EA) 或 Web 直销模型。 定价选项包含“免费层”订阅（即产品永久免费）。 定价选项还包括“立即试用”订阅。 “立即试用”订阅因促销而在有限时间内免费，之后 VM 转换为付费订阅。 BYOL 激活也是用于支持你的选项。<br />对于计费选项和以下 VM 产品/服务，所有已预配的 Azure 资源都直接向客户收费。<ul> <li>Azure 应用：托管应用</li> <li>Azure 应用：解决方案模板</li> </ul> | 
| ***推进与其他合作伙伴的连接*** | 目前无法将服务提供商或交付合作伙伴链接到你的产品/服务。 此功能在 2018 年启动。 | 
| ***自动化*** | 目前无法将服务提供商或交付合作伙伴链接到你的产品/服务。 通过附加产品预配，利用自动化 SaaS。 使用 Azure 应用：解决方案模板自动实现基于 SaaS 的数据收集和部署场景。 |  
| ***多个云类型*** | 使用以下类型在公有云和本地解决方案上发布。<ul> <li>Azure Stack</li> <li>Azure Government </li> <li>区域云，包括中国和德国</li> </ul> | 
| ***向客户进行上下文演示*** | 让用户能够在 Azure 门户中对解决方案进行上下文搜索（使用虚拟机和 Azure 应用：解决方案模板）。 |
