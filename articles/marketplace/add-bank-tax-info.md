# <a name="add-bank-and-tax-information"></a>添加银行和税务信息  

使用事务处理列表类型的所有商业产品/服务都必须添加银行和税务信息。  

## <a name="partner-center"></a>合作伙伴中心

现在，在合作伙伴中心内管理银行和税务信息。 有关配置这些设置的说明，请参阅[管理合作伙伴中心帐户](https://docs.microsoft.com/azure/marketplace/partner-center-portal/manage-account#payout-account)的 "商用 Marketplace 概述" 页。
