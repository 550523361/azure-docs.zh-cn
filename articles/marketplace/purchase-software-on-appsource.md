# <a name="how-to-purchase-software-on-microsoft-appsource"></a>如何在微软应用程序上购买软件来源

Microsoft [AppSource](https://appsource.microsoft.com/)现在允许客户订阅由 Microsoft 合作伙伴提供的 SaaS 应用程序。 客户可以在商店中找到经过认证的 Web 应用程序，并且可以使用 Microsoft 管理中心在一个位置管理费用、升级、降级和取消。 本文介绍如何从应用商店购买应用。

AppSource 为您提供了一系列应用供您选择，包括免费试用版和付费订阅版。 本文介绍如何通过 AppSource 购买付费订阅。

## <a name="requirements"></a>要求

要购买 SaaS 优惠，您需要：

- 工作或学校帐户。 如果您没有工作或学校帐户，请按照购买过程中的链接"注册免费帐户"。

- 购买优惠的国家/地区的有效信用卡。

    > [!Note]
    > 信用卡计费国家/地区可能锁定到 Azure 活动目录实例的位置。

## <a name="steps-for-making-purchases-on-appsource"></a>在 AppSource 上进行购买的步骤

1. 在[AppSource](https://appsource.microsoft.com/)上选择您的帐单/国家/地区。
1. 查找您要购买的应用程序。
1. 选择**立即获取它**。
1. 选择计划：

    1. 设置座位数
    1. 设置计费期限。
    
1. 输入您的帐单地址。
1. 输入您的付款方式：
    * 支持信用卡：Visa、万事达卡、美国运通卡、发现卡。
    
1. 选择 **"地点顺序**"。

## <a name="how-to-configure-software-post-purchase"></a>如何配置购买后的软件

收到您的订单后，可能需要几秒钟才能确认。 您将收到一个链接，用于在页面上配置 SaaS 订阅，以及确认购买的电子邮件以及完成配置的链接。

1. 在 **"订单确认"** 页上，选择 **"配置**"。 或者，您可以使用在确认电子邮件中发送给您的"配置"链接。
1. 在软件提供商的网站上完成预配和设置。

    > [!Note]
    > 此步骤是强制性的，可以随时完成。

## <a name="contact-support"></a>联系支持人员

有关商务产品，[请在此处联系帮助](https://docs.microsoft.com/office365/admin/contact-support-for-business-products?view=o365-worldwide&tabs=phone)。
