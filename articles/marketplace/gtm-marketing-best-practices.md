# <a name="marketing-best-practices"></a>市场营销最佳做法

经过优化的 marketplace 市场营销活动可帮助你加快客户的购置增长，因此可以在云业务策略和引擎中发挥重要作用。 准备好列出产品/服务、提供客户试验，并与 Microsoft 客户和合作伙伴社区进行联系。 下载[AppSource & Azure Marketplace 营销最佳实践指南](https://aka.ms/marketplacebestpractices)以了解如何充分利用在线营销工作。

**后续步骤**

有关市场 GTM 服务的详细信息，请转到[市场推广服务](https://partner.microsoft.com/reach-customers/gtm)。

登录到[云合作伙伴门户](https://cloudpartner.azure.com)，以便创建并配置产品/服务。
