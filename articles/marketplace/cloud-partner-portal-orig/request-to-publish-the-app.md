# <a name="request-to-publish-the-app"></a>请求发布应用

![单击此项以发布](./media/CRMScreenShot18.png)

1. 单击“发布”按钮将你的产品/服务提交给我们的认证团队。 可以在“状态”选项卡下查看产品/服务的状态。

![查看发布状态](./media/CRMScreenShot19.png)

## <a name="republish-the-app"></a>重新发布应用

 如果需要提交重新发布应用的请求，请先单击“状态”选项卡下的“取消发布”按钮（如果请求不再处于活动状态，请跳过此步骤）。 对应用进行更改，相应地更新产品/服务，然后单击“发布”按钮以重新提交。 # Microsoft Dynamics 365 商业版的特殊注意事项。请注意，为Microsoft Dynamics 365 商业版构建的解决方案将在认证期间进行额外检查，以确保遵守商业版限制。 此处记录了商业版限制。 # 获取帮助。 如果遇到问题，可以通过单击右上角并转到其中来记录支持票证。
