# <a name="dynamics-365-for-operations-help-wiki"></a>Dynamics 365 for Operations 帮助 Wiki

Microsoft Dynamics 365 for Operations 是面向企业的 Microsoft 商务应用程序。 使用它，用户可以通过智能用户界面访问实时见解和数据，从而快速做出更明智的决策。 借助它，企业可以交付经过验证的业务逻辑，以便更快速地重新设计业务流程，从而进行转型和创新。 此外，企业还能灵活地按照自己的步调发展，并几乎随时随地开展业务，从而在全球范围内扩展业务，以满足业务需求。

本用户指南有助于了解如何使用 Dynamics 365 for Operations 满足自己的业务需求。

有关详细信息，请参阅 [Dynamics 365 for Operations 帮助 Wiki](https://docs.microsoft.com/dynamics365/operations/)。
