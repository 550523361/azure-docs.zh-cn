# <a name="cloud-partner-portal-migration-to-partner-center"></a>云合作伙伴门户迁移到合作伙伴中心

为了简化你的帐户并提供管理体验，Microsoft 正在将产品/服务从云合作伙伴门户迁移到合作伙伴中心。 如果你当前在一个商业伙伴投放市场（OCP 向下）目录中管理共同销售的宣传品，你将能够在合作伙伴中心尽快更新共同销售的宣传资料。

| **产品/服务类型** | **从云合作伙伴门户迁移到合作伙伴中心的产品/服务管理** | **能够在合作伙伴中心更新共同销售的宣传品** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| 软件即服务（SaaS） |     现可用       |   大约2020年2月公开上市      |
| Dynamics 365 for Customer Engagement & PowerApps      |     现可用        |   现可用      |
| Dynamics 365 for Operations     |     现可用       |   现可用     |
| Dynamics 365 Business Central      |     现可用        |   大约2020年2月公开上市     |
| Azure 应用程序     |     公开上市的年12月。2019       |   大约2020年2月公开上市     |
| Azure 虚拟机      |     待公布地区       |   待公布地区    |
| Azure 容器     |     待公布地区       |   待公布地区     |
| 咨询服务      |     待公布地区       |   待公布地区     |
| IoT Edge 模块      |     待公布地区       |   待公布地区     |
| Power BI 服务应用     |     待公布地区       |   待公布地区     |
|  |  |
