# <a name="cloud-partner-portal-migration-to-partner-center"></a>通过云合作伙伴门户迁移到合作伙伴中心

为了简化您的帐户并提供管理经验，Microsoft 正在将产品从云合作伙伴门户迁移到合作伙伴中心。 如果您目前管理"一个商业合作伙伴上市（OCP GTM）"目录中的共售抵押品，您将很快能够在合作伙伴中心更新您的共同销售抵押品。

| **优惠类型** | **产品/服务管理从云合作伙伴门户迁移到合作伙伴中心** | **能够更新合作伙伴中心中的共同销售抵押品** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| 软件即服务 （SaaS） |     目前可用       |   2020 年 2 月左右一般供货      |
| 面向客户互动&电源应用的动态 365      |     目前可用        |   目前可用      |
| Dynamics 365 for Operations     |     目前可用       |   目前可用     |
| Dynamics 365 Business Central      |     目前可用        |   2020 年 2 月左右一般供货     |
| Azure 应用程序     |     2019 年 12 月通用       |   2020 年 2 月左右一般供货     |
| Azure 虚拟机      |     即将公布       |   即将公布    |
| Azure 容器     |     即将公布       |   即将公布     |
| 咨询服务      |     即将公布       |   即将公布     |
| IoT Edge 模块      |     即将公布       |   即将公布     |
| 电源 BI 服务应用     |     即将公布       |   即将公布     |
