# <a name="cloud-partner-portal-migration-to-partner-center"></a>通过云合作伙伴门户迁移到合作伙伴中心

为了简化你的帐户并提供管理体验，Microsoft 正在将产品/服务从云合作伙伴门户迁移到合作伙伴中心。 如果你当前在一个商业伙伴投放市场（OCP 向下）目录中管理共同销售的宣传品，你将能够在合作伙伴中心尽快更新共同销售的宣传资料。

| **产品类型** | **从云合作伙伴门户迁移到合作伙伴中心的产品/服务管理** | **能够在合作伙伴中心更新共同销售的宣传品** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| 软件即服务（SaaS） |     目前可用       |   公开上市2020年4月      |
| Dynamics 365 for Customer Engagement & PowerApps      |     目前可用        |   目前可用      |
| Dynamics 365 for Operations     |     目前可用       |   目前可用     |
| Dynamics 365 Business Central      |     目前可用        |   公开上市2020年4月     |
| Azure 应用程序     |     目前可用       |   公开上市2020年4月     |
| Azure 虚拟机      |     正式上市-2020 年5月       |   即将公布    |
| Azure 容器     |     正式上市-2020 年5月       |   即将公布     |
| 咨询服务      |     正式上市-2020 年5月       |   即将公布     |
| IoT Edge 模块      |     正式上市-2020 年5月       |   即将公布     |
| Power BI 服务应用     |     正式上市-2020 年5月       |   即将公布     |
