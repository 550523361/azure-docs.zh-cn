# <a name="build-your-landing-page"></a>生成登陆页  
公司网站的登录页面是用于访问需求激励活动和市场中商品信息的门户。  

从确定目标开始。 对于你的解决方案，请选择目标客户和客户采取的操作。  解决方案可能包括下列操作。  
*   测试运行解决方案  
*   立即获取解决方案  
*   等等...  

你可以通过多种市场营销策略引用登陆页。 可以使用市场营销策略。  
*   活动  
*   网络广播  
*   社交媒体  
*   白皮书  
*   技术培训课程  
*   新闻发布  

你的宣传内容和行动号召越一致，客户就越容易找到你的解决方案。  

如果准备好了活动计划，请创建登陆页。 查看下表以最大限度地提高登陆页的有效性。  

| 最佳做法 | 要避免的内容 |  
|:--- |:--- |  
| 说明解决方案能解决的客户问题，以及 Azure 如何实现其功能 |  未能说明解决方案和 Azure 协同工作的原理。 |  
| ***创建简短易记的 URL*** | 使用长 URL。 URL 太长，难记又难找。 |  
| ***添加相关的视觉内容，例如客户口碑宣传和代言视频或解决方案体系结构*** | 使用太多文本会导致客户忽略内容并停止浏览。 |  
| 
  ***创建明晰的行动号召，将访问者吸引到市场目录页*** | 页面上有过多链接和操作。 |  
| 将行动号召设置在顶部或非常显眼的位置 |  将链接嵌入至市场商品信息中的一段文本中。 |  
| ***研究最佳关键字并针对搜索优化页面*** | 想当然地认为产品名称可以获得更高的搜索排名。 |  
| ***使用广告推广活动中的关键字*** |  在 Web 属性中使用许多不同的关键字会削弱广告投资的效果。 |  
| 将相关的产品名称和关键字放在`above the fold` | 使用户需要滚动才能看到所展示的产品或解决方案。 |  
| 使用品牌图像来验证解决方案，同时遵循 Microsoft 品牌准则<br />，例如 Azure 认证 | 使用未获批准的 Microsoft 品牌图像。 |  

*   有关 Azure 认证徽章的详细信息，请访问位于 [azure.microsoft.com/support/legal/marketplace/certified-guidelines](https://azure.microsoft.com/support/legal/marketplace/certified-guidelines) 的 Microsoft Azure 认证指南页面。  
*   有关品牌打造的其他资源，请访问位于 [microsoft.sharepoint.com/teams/brandcentral](https://microsoft.sharepoint.com/teams/brandcentral) 的品牌中心页面。  
*   有关徽标生成器工具的详细信息，请访问位于 [logobuilder.partner.microsoft.com](https://logobuilder.partner.microsoft.com) 的徽标生成器页面。  
*   有关详细信息，请访问位于 [partner.microsoft.com/membership/how-it-works](https://partner.microsoft.com/membership/how-it-works) 的工作方式页面。
