# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Azure 市场和 AppSource 中推荐的解决方案

Microsoft 推荐的解决方案是一个云应用程序，入选理由是它具备在某一特定垂直行业或解决方案领域中满足客户需求的质量、性能和功能。 Microsoft 行业和销售专家小组从 Microsoft 合作伙伴网络[金会员资格](https://partner.microsoft.com/en-us/membership/core-benefits)合作伙伴中选择解决方案。 这些解决方案是我们 Azure 市场和 AppSource 云市场店面中的精选解决方案。 垂直行业可以发现 AppSource 上推荐的解决方案，而在 Azure 市场上，可以在“解决方案范围”类别中单击查看 

如果他们的一个或多个解决方案已被选为推荐解决方案，Microsoft 将联系解决方案提供商。
