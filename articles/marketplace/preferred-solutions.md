# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Azure 市场和 AppSource 中推荐的解决方案

Microsoft 推荐的解决方案是一个云应用程序，入选理由是它具备在某一特定垂直行业或解决方案领域中满足客户需求的质量、性能和功能。 一组 Microsoft 专家从具有特定经过验证的能力和功能的合作伙伴处验证解决方案。 这些解决方案在我们的云市场网店、Azure 应用商店和 AppSource 以及 Azure 门户中都有特色。 AppSource 上的首选解决方案可以通过行业垂直部发现。 Azure 应用商店和 Azure 门户上的首选解决方案可以跨水平解决方案类别发现。
