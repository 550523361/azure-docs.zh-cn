# <a name="update-an-existing-offer-in-the-commercial-marketplace"></a>在商业市场中更新现有的套餐

您可以在合作伙伴中心[商业市场门户](https://partner.microsoft.com/dashboard/commercial-marketplace/offers)的 **"概述"** 选项卡上查看现有优惠。

要更新当前市场上现有的产品/服务：：

1. 选择要更新的优惠名称。 报价的状态可能被列为**实时**，**发布在进行**中，**草稿**，**需要注意**，或**不在市场上出售**（如果你以前选择停止销售报价）。 选中后，该优惠的 **"产品/服务"概览**页面将打开。
2. 选择"从概览卡片**中更新**"，或从左侧导航中选择菜单项，以选择要更新的区域。 您可能需要更新**产品/服务设置**、**属性**、**优惠列表**、**预览**、**技术配置**、**计划概述**或**试驾**。 
3. 进行更改，然后选择 **"保存**"。 重复此过程，直到所有更改完成。

## <a name="review-and-publish-an-updated-offer"></a>查看并发布更新的报价

准备好发布更新的优惠时，从任何页面选择 **"发布**"。 将打开 **"审阅和发布**"页。 在此页面上，您可以：

- 请参阅您更新的报价部分的完成状态： 
    - **未发布的更改**：该部分已更新并已完成。 提供了所有必需的数据，并且更新中没有引入任何错误。
    - **不完整**：对该部分所做的更新引入了需要修复或需要提供更多信息的错误。
- 向认证测试团队提供其他信息，以确保测试顺利进行。
- 通过选择 **"提交**"提交更新的发布产品/服务。  当您有更新优惠的预览版本可供您查看和注销时，我们会向您发送电子邮件。

> [!IMPORTANT]
> 一旦优惠预览可用，您必须查看该优惠预览，并选择 **"Go-live"** 将更新的优惠发布到预期受众（公共或私人）。

## <a name="update-a-plan-within-an-existing-offer"></a>在现有产品/服务范围内更新计划

要更改已发布的现有产品/服务范围内的计划，：

- 打开现有**优惠的"产品/服务概览**"页后，请选择要更改的计划。 如果计划无法从 **"计划概览"列表中**访问，请选择"**查看所有计划**"。
- 您可以选择计划**名称**、**定价模型**或**可用性**。 *目前计划仅提供英文（美国）版本*。
- 对计划名称、说明、定价或访问群体可用性进行任何必要的更改后选择 **"保存**"。 
- 准备好发布更新后选择 **"发布**"。 "**[审阅"和"发布](#review-and-publish-an-updated-offer)**"页将打开，并为您的更新提供完成状态。

## <a name="compare-changes-to-marketplace-offers"></a>将更改与市场优惠进行比较

您可以使用 比较 对[已发布](#compare-changes-to-published-offer)或[预览](#compare-changes-to-a-preview-offer)产品/服务进行实时更改，然后对其进行**审核**。

>[!NOTE]
> 已发布的产品/服务是已成功发布到预览版或实时状态的优惠。

有关一般审核信息，请参阅下文：

- 您可以在编辑过程中的任何点使用**比较**。
- 在 **"比较**"页上选择一个字段以导航到要修改的值。
- 如果您已准备好发布更新，请选择 **"审阅"并发布**。
- 要查看所有字段的值，即使字段未更新，请选择"**所有字段**"筛选器。 您可以通过选择 **"修改"字段**来修改这些字段中的筛选器，然后选择以下筛选器之一：
    - **删除的值**筛选器显示您发布的字段，现在您正在完全删除。
    - **添加的值**筛选器显示您最初未发布且正在添加的字段。
    - **编辑的值**筛选器显示已发布但现已更新内容的字段。

      >[!NOTE]
      > 如果这些筛选器之一不可用，则表示您没有更新该类型。

- 要仅查看尚未更新的值，请选择"**未更改字段**"筛选器。 为已发布版本和草稿版本显示的字段值将相同。

  ![用于将更新与已发布或预览产品/服务进行比较的筛选器](./media/compare-changes-marketplace.png)

>[!NOTE]
> 以下页面当前不支持**比较**：
>- CSP 经销商受众
>- 试驾技术配置
>- 试驾市场列表
>- 共同销售
>- 补充文件

### <a name="compare-changes-to-published-offer"></a>将更改与已发布产品/服务进行比较

Follow the instructions below to compare your changes from the published offer:

1. 在页面的命令栏中选择 **"比较**"。 **比较**页面提供此产品/服务已保存的更改和已发布的市场产品/服务并行版本。
2. 当您从产品/服务的特定页面访问**比较时**，默认值仅显示该页面上所做的更改。
    - 如果要比较所有页面上的更改，请从 **"选择要比较的页面"更改页面**。  
    - 如果要比较所有页面对产品/服务所做的更改，请选择 **"所有页面**"。

默认情况下，**比较**将您对实时市场产品/服务的新更改进行比较。

请记住，在更新更改以生效后重新发布产品/服务。

### <a name="compare-changes-to-a-preview-offer"></a>将更改与预览优惠进行比较

如果预览版中的更改不实时，则可以将新更改与预览市场产品/服务进行比较。

按照以下说明将新更改与您的预览市场产品/服务进行比较：

1. 在页面的命令栏中选择 **"比较**"。
2. 选择 **"带下**拉列表"并将其从**实时优惠**更改为**预览优惠**。
3. "**比较**"页提供显示更改的并行版本。

>[!NOTE]
>如果您的报价尚未变为实时，但您已发布预览版，则您无法选择与实时优惠进行比较。

请记住，在更新更改以生效后重新发布产品/服务。

## <a name="delete-a-draft-offer"></a>删除要约草稿

您可以通过从 **"优惠概览**"页中选择 **"删除"草稿**来删除草稿产品/服务（尚未发布）。 如果您以前发布过产品/服务，此选项将不可用。

确认要删除优惠草稿后，优惠将不再在合作伙伴中心可见或访问，**并且"所有优惠"** 页面将打开。

## <a name="delete-a-draft-plan"></a>删除计划草稿

要删除尚未发布的计划，请从 **"计划概述"** 页**中选择"删除草稿**"。 如果您以前发布过产品/服务，此选项将不可用。

确认要删除计划草稿后，该计划将不再在合作伙伴中心可见或访问。

## <a name="cancel-publishing"></a>取消发布

要取消处于 **"发布"** 状态的优惠：

- 选择产品/服务名称以打开 **"产品/服务概览**"页。
- 选择 **"从页面右上角取消发布**"。
- 确认要停止发布产品/服务。

如果要在以后发布产品/服务，则需要重新开始发布过程。

> [!NOTE]
> 仅当产品/服务尚未向发布者签署步骤时，才能阻止发布产品/服务。 选择 **"上线"** 后，您将无法选择取消发布。

## <a name="stop-selling-an-offer-or-plan"></a>停止销售要约或计划

要在发布产品/服务后停止销售产品/服务，请从 **"优惠概览**"页面选择 **"停止销售**"。 要停止销售计划，请从 **"计划概览"** 页中选择 **"停止销售**"。 （只有在产品/服务单中有多个计划时，才提供停止销售计划的选项。 您可以选择停止销售一个计划，而不影响产品/服务范围内的其他计划。

一旦您确认要停止销售产品/服务或计划，在几个小时内，它将不再在市场上可见，并且没有新客户可以下载它。

任何以前获得产品/服务或计划的客户仍可以使用。 他们可以再次下载它，但如果您稍后更新并重新发布产品/服务或计划，他们将不会获得更新。 

完成停止销售产品/计划的请求后，您仍将在合作伙伴中心的商业市场门户中看到它。 要删除草稿，请从 **"优惠概览**"或"**计划概览"** 页中选择 **"删除草稿**"。 

如果您决定列出或出售此要约或再次计划，请按照说明[更新现有报价](#update-an-existing-offer-in-the-commercial-marketplace)。 不要忘记，在进行任何更改后，您需要**发布**产品/服务或再次计划。

## <a name="remove-offers-from-existing-customers"></a>删除现有客户的产品/服务

要删除现有客户的产品/服务，[请记录支持请求](https://support.microsoft.com/supportforbusiness/productselection?sapId=48734891-ee9a-5d77-bf29-82bf8d8111ff)。 从 **"问题类型"** 列表中选择您的特定产品/服务类型，然后在 **"类别"** 列表中选择 **"删除已发布的产品/** 服务"。 提交请求。 支持团队将指导您完成产品/服务删除过程。

## <a name="sync-private-plan-audiences"></a>同步私人计划访问群体

如果您的优惠包含一个或多个计划，这些计划配置为仅对私人受限受众可用，则可以仅更新可以访问该私人计划的受众，而无需发布对产品/服务的其他更改。 

要更新和同步计划的私人访问群体：

- 使用 +**添加 ID**或**导入客户 （csv）** 按钮修改一个或多个私人计划中的受众，然后保存更改。
- 从 **"计划概述"** 页面选择 **"同步私人受众**"。

**同步私人访问群体**将仅向您的私人访问群体发布更改，而不发布您可能对草稿产品/服务所做的任何其他更新。
