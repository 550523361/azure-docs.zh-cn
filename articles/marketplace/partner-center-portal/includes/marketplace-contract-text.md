Microsoft 为商业市场提供了标准合同模板。

- **使用微软商业市场的标准合同？**

为了简化客户的采购流程并降低软件供应商的法律复杂性，Microsoft 为 Microsoft 商业市场提供了标准合同，以帮助促进市场交易。 商业市场出版商可以选择根据标准合同提供他们的软件，而不是制定自定义条款和条件，客户只需审查并接受一次。 标准合同可以在这里找到： https://go.microsoft.com/fwlink/?linkid=2041178.

您可以选择使用标准合同，而不是通过选择"为商业市场使用标准合同"复选框来提供您自己的自定义条款和条件。

![使用标准合同复选框](./media/use-standard-contract.png)

> [!NOTE]
> 使用 Microsoft 商业市场的标准合约发布产品/服务后，您将无法使用自己的自定义条款和条件。 这是一个"或"方案。 您要么根据标准合同**或**您自己的条款和条件提供您的解决方案。 如果您想修改标准合同的条款，您可以通过标准合同修订来修改。

**标准合同修订**

标准合同修订允许发布商选择标准合同条款，以便简单，并自定义其产品或业务的条款。 客户只需审核并接受 Microsoft 标准合同，就只需查看合同的修订。

商业市场出版商有两种修正案：

- 普遍修改：这些修订适用于所有客户的标准合同。 通用修订显示在采购流程中向产品/服务的每个客户。 客户必须接受标准合同的条款和修改，然后才能使用您的优惠。
- 自定义修订：这些修订是标准合同的特别修订，仅通过 Azure 租户 ID 针对特定客户。 发布者可以选择要定位的租户。 只有租户的客户才会在优惠的采购流程中收到自定义修改条款。  客户必须接受标准合同的条款和修改，然后才能使用您的优惠。

>[!NOTE]
> 这两种类型的修正相互叠加。 以定制修订为目标的客户在购买过程中还将获得标准合同的普遍修订。

**微软商业市场标准合同的通用修正条款**：在此框中输入通用修订条款。 您可以提供每个报价的单一通用修订。 您可以在此框中输入无限数量的字符。 在发现和购买流期间，这些术语在 AppSource、Azure 应用商店和/或 Azure 门户中向客户显示。

**微软商业市场标准合同的自定义修改条款**：从选择**添加自定义修改条款**开始。 每个优惠最多可以提供 10 个自定义修订条款。

- **自定义修改条款**：在自定义修改条款框中输入您的自定义修订条款。 您可以在此框中输入无限数量的字符。 只有您为这些自定义条款指定的租户 ID 的客户才会在 Azure 门户中向产品/服务的采购流中显示自定义修改条款。  
- **租户 ID（** 必需）：每个自定义修订的目标可多达 20 个租户 ID。 如果添加自定义修订，则必须至少提供一个租户 ID。 租户 ID 标识 Azure 中的客户。 您可以向客户询问此 ID，并且他们可以通过导航到portal.azure.com> Azure 活动目录>属性来找到它。 目录 ID 值是租户 ID（例如，50c464d3-4930-494c-963c-1e951d15360e）。 您还可以使用我的[Microsoft Azure 和 Office 365 租户 ID](https://www.whatismytenantid.com)上的域名 URL 查找组织的客户的租户 ID。
- **说明**（可选）：可选地为租户 ID 提供友好说明，帮助您通过修订确定目标客户。

**条款和条件**

如果要提供自己的自定义条款和条件，可以选择在条款和条件字段中输入它们。 您可以在此字段中输入最多 10，000 个字符的文本。 如果您的条款和条件需要较长的描述，请在此字段中输入单个 URL 链接，其中可以找到您的条款和条件。 它将作为活动链接向客户显示。

客户在试用您的优惠之前，必须接受这些条款。
