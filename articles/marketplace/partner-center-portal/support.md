# <a name="support-for-the-commercial-marketplace-program-in-partner-center"></a>支持合作伙伴中心的商业市场计划

Microsoft 支持各种产品和服务。 找到合适的支持团队对于确保做出适当和及时的反应非常重要。 请考虑使用以下有助于将查询路由到相应团队的方案： 

- 如果您是发布者，并且对客户有疑问，请要求客户使用 [Azure 门户](https://portal.azure.com/)中的支持链接请求支持。 

- 如果您是发布者，并且对应用或服务有疑问，请查看以下支持选项。

## <a name="support-options-for-publishers"></a>发布者支持选项

1. 使用工作帐户登录[合作伙伴中心的商业市场计划](https://partner.microsoft.com/dashboard/commercial-marketplace/overview)。 （如果您尚未这样做，则需要[创建合作伙伴中心帐户](./create-account.md)。

2. 在页面右侧的上部菜单中，选择 **"支持"** 图标。 
 
3. **"帮助和支持**"窗格将显示在页面右侧。 
 
   ![支持下拉菜单](./media/commercial-marketplace-support-pane.png)

    或者转到**主页**窗格并选择 **"帮助和支持**"。

   ![主页的帮助和支持](./media/homepage-help-support.png)

4. 选择**[文档](../index.yml)** 以查看对问题和资源的全面答案。 

5. 选择**["市场合作伙伴"社区论坛](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/bd-p/2222)**，利用其他 Microsoft 发布者的知识来回答您的问题。 

6. 选择**["其他帮助](https://partner.microsoft.com/support/v2/?stage=1)**"以打开 **"新建支持请求**票证"。  

## <a name="how-to-open-a-support-ticket"></a>如何开具支持票证

现在，您已准备好在 **"帮助和支持**"屏幕上打开支持票证。

![帮助和支持](./media/help-and-support.png)

>[!Note]
>如果您登录合作伙伴中心，您将获得更好的支持体验。

**选项 1：** 输入关键字，如：应用商店、Azure 应用、SaaS 产品/服务、帐户管理、潜在顾客管理、部署问题、支出等。

**选项 2：** 浏览主题 ->选择**类别**= 商业市场 ->选择适当的**主题**，然后**子主题**。

找到您选择的主题后，选择 **"查看解决方案**"。

![后续步骤](./media/next-step.png)

以下选项将变为可用：

* 要选择其他主题，请在 **"选定问题**"下选择其他主题链接。
* 查看此问题的说明（如果可用）。  它是**建议步骤**上方所示的文本。
* 查看**建议的步骤**（如果可用）。
* 查看**建议的文档**（如果可用）。

![建议的解决方案](./media/recommended-solutions.png)

如果您无法在**推荐的解决方案**中找到答案，请单击 **"提供问题详细信息**"。  填写表单和所有必需的字段，以帮助加快解决过程。  然后单击**提交**。

>[!Note]
>如果您尚未登录合作伙伴中心，并且该主题需要身份验证，则系统将要求您登录，然后才能继续。  对于公共主题，不需要身份验证。

## <a name="track-your-existing-support-requests"></a>跟踪现有支持请求 

要查看所有已打开和关闭的票证，请转到左侧导航栏上的**商业市场**，然后选择**支持**。
