# <a name="orders-dashboard-in-commercial-marketplace-analytics"></a>商业市场分析中的订单仪表板

本文提供有关合作伙伴中心中的 **"订单"仪表板**的信息。 此仪表板以图形和可下载格式显示有关订单的信息。

要访问合作伙伴中心分析工具中的**订单仪表板**，请打开"商业市场下**[分析"仪表板](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)**。

>[!NOTE]
> 有关分析术语的详细定义，请参阅[商业市场分析的常见问题和术语](./faq-terminology.md)。

## <a name="orders-dashboard"></a>订单仪表板

**"分析**"菜单**的订单仪表板**显示所有 SaaS 产品/服务当前的订单。 您可以查看以下项目的图形表示形式：

- [订单摘要](#order-summary)
- [按地理位置排序](#orders-by-geography)
- [按优惠排序](#orders-by-offers)
- [每个站点的订单趋势与每个座位](#orders-trend-per-site-versus-per-seat)
- [按 SKU 的订单](#orders-by-skus)
- [订单和席位趋势](#orders-and-seats-trend)
- [订单详细信息表](#order-details-table)

> [!NOTE]
> 分析报告在云合作伙伴门户 （CPP） 中的显示方式与合作伙伴中心中的新商业市场计划之间存在差异。 一种特定方式是，CPP 中的**卖家见解**具有**订单&使用情况**"选项卡，该选项卡显示基于使用情况的优惠和非基于使用情况的优惠的数据。 在合作伙伴中心，"**订单"** 页具有 SaaS 优惠的单独选项卡。

## <a name="order-dashboard-details"></a>订购仪表板详细信息

本节将更详细地介绍分析报告。

### <a name="order-summary"></a>订单摘要

"订单摘要"部分显示所有已购买订单（不包括取消的订单）、取消的订单和座位的计数。

总订单旁边的百分比值表示所选日期范围的增长量。

![合作伙伴中心分析订单摘要](./media/order-summary.png)

- 指向向上的绿色三角形表示正增长趋势。
- 指向向下的红色三角形表示与上月相比的负增长趋势。
- 增长趋势由微条图表示。 您可以通过将鼠标悬停在图表中的列上来显示每个月的值。
- 取消的订单是以前购买订单，然后在所选日期范围内取消的订单的计数。
- 座位是在所选日期范围内创建的座位数。

### <a name="orders-by-geography"></a>按地理位置排序

**按地理位置划分的订单**热图在世界地图上显示您的订单计数，并显示基于客户国家/地区映射的座位。 此热图**[的功能与按地理热图划分的客户](./customer-dashboard.md#customer-by-geography)** 相同。

![合作伙伴中心按地理位置分析订单](./media/orders-by-geography.png)

### <a name="orders-by-offers"></a>按优惠排序

**"按产品/服务订单**"圆环图根据报价名称组织订单（包括取消的订单）。

- 顶部优惠显示在图表中，其余优惠分组为"全部休息"。
- 您可以在图例中选择特定产品/服务，以仅显示图形中的这些优惠。
- 将鼠标悬停在图中的切片上将显示订单数量和该优惠的百分比，而与所有产品/服务之间的订单总数相比。
- **订单按优惠趋势**显示逐月增长趋势。 "月份"列表示订单数量（按要约名称）。 折线图显示在 z 轴上绘制的增长百分比趋势。
- 您可以使用图表顶部的滑块沿 x 轴向右和向左滚动，并关注特定数据点。
- 您可以通过在图例上选择特定项来显示趋势图表。
- 您还可以选择显示**已取消订单**的趋势和数据。 该图的工作方式与**按报价图排序**的图形相同。

### <a name="orders-trend-per-site-versus-per-seat"></a>每个站点的订单趋势与每个座位

**每个站点与每个座位**的圆环图表示每个站点 SaaS 和客户购买每个座位 SaaS 订单的细分（此图表包括已取消的订单）。 柱形图表示每个站点 SaaS 和客户购买每个座位 SaaS 订单的趋势（此图表包括已取消的订单）。

### <a name="orders-by-skus"></a>按 SKU 的订单

**按 SKU 排序的**图表表示所有报价（包括取消的订单）的库存单位 （SKU） 级别的订单趋势。 圆环图表示前五个 SKU 订单的细目，柱形图表示前五个 SKU 的订单趋势。

### <a name="orders-and-seats-trend"></a>订单和席位趋势

**订单和席位趋势**图表显示订单数量最多的前 50 个报价。 这些显示在排行榜上，并按最高订单计数和订单百分比排序。

- **按 SKU 排序**：选择要约以查看图表中前五个 SKU 的订单计数细分。
- **按 SKU 进行的座位**：前五个 SKU 的每月座位趋势。 如果您选择的报价不是每个座位优惠，您将在此区域图中看不到任何数据。

### <a name="canceled-orders-by-offers"></a>按优惠取消的订单

**"取消订单（按优惠）** 饼图"根据优惠名称组织所有已取消的订单。 顶部优惠显示在图表中，其余优惠分组为"全部休息"。 您可以在图例中选择要在图形中显示的特定产品/服务。

- 将鼠标悬停在图中的切片上将显示所选产品/服务单的数量和百分比，而与所有产品/服务/地区的订单总数相比。
- 柱形图显示逐月趋势。 列按产品/服务单名称表示取消的订单数。 您可以使用图表顶部的滑块沿 x 轴向右和向左滚动，并专注于特定数据点。 您可以通过在图例上选择特定项来显示趋势图表。

### <a name="order-details-table"></a>订单详细信息表

"订单详细信息"表显示按购置日期排序的 1000 个顶级订单的编号列表。

- 网格中的每个列都是可排序的。
- 如果记录计数小于 1000，则可以将数据提取到 TSV 文件。
- 如果记录数超过 1000，导出的数据将在未来 30 天内异步放置在下载页中。
- 筛选器可以应用于 **"订单详细信息"表**，以仅显示您感兴趣的数据。 数据可以按国家/地区、Azure 许可证类型、应用商店许可证类型、产品/服务类型、订单状态、可用路径、应用商店订阅 ID、客户 ID 和公司名称进行筛选。

#### <a name="orders-page-filters"></a>订单页面筛选器

这些筛选器在页面级别应用。

您可以选择多个筛选器来呈现图表，以呈现您选择查看的条件以及希望在 **"详细订单数据"** 网格/导出中显示的数据。 筛选器应用于订单页面右上角为所选数据范围提取的数据。

- 优惠类型和优惠名称仅列出您在所选日期范围内的订单。 列表中的要约名称显示在列表中所选的优惠类型。
- 应用筛选器显示所选每个筛选器的每个选定选择中的总指标。 选择默认选择时，不会显示应用的筛选器。
- 如果为其中一个下拉列表选择了 **"全部"，** 则所选页面中的所有指标都将聚合。 例如："产品/服务类型筛选器选项中的"全部"表示已选择所有产品/服务类型。 这是下拉列表的默认选择。 选择 **"全部**"时，应用的筛选器显示不会显示任何内容。
- **多个值选择**：页面中的所有指标都将聚合为下拉列表下进行的所有选择。 如果进行了多个选择，则应用的筛选器将显示所有选择的计数。 请参阅下图以参考。

    ![合作伙伴中心分析应用多个值进行筛选的顺序](./media/filters-applied.png)

- **单个值选择**：如果选择了一个值，则应用的筛选器将显示所选筛选器的计数。 请参阅下图供参考。

     ![合作伙伴中心分析单一值应用于筛选的顺序](./media/filters-applied-single.png)
