# <a name="analytics-for-the-commercial-marketplace-in-partner-center"></a>合作伙伴中心商业市场分析

了解如何访问 Microsoft 合作伙伴中心的分析报告，以监视销售、评估绩效并优化市场产品/服务。 作为合作伙伴，您可以使用合作伙伴中心支持的数据可视化和洞察图来监控产品/服务列表，并找到最大化销售额的方法。 改进的分析工具使您能够根据性能结果采取行动，并与客户和经销商保持更好的关系。

## <a name="partner-center-analytics-tools"></a>合作伙伴中心分析工具

要访问合作伙伴中心分析工具，请打开"商业市场"下的**["分析](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)**"仪表板。

>[!NOTE]
> 有关分析术语的详细定义，请参阅[商业市场分析的常见问题和术语](./faq-terminology.md)。

- 有关您的订单以图形和可下载格式的信息，请参阅[商业市场分析中的订单仪表板](./orders-dashboard.md)。
