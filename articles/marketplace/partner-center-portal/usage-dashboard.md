# <a name="usage-dashboard-in-commercial-marketplace-analytics"></a>商业市场分析中的使用情况仪表板

本文提供有关合作伙伴中心中的"使用情况"仪表板的信息。 此仪表板在两个单独的选项卡中显示所有 VM 提供的使用情况和按流量计费的计费指标：VM 使用情况和按流量计费的计费使用情况。

要访问使用情况仪表板，请打开"商业市场下**[分析"](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)** 仪表板。

>[!NOTE]
> 有关分析术语的详细定义，请参阅[商业市场分析的常见问题和术语](./faq-terminology.md)。

## <a name="usage-dashboard"></a>使用情况仪表板

使用情况仪表板表示所有虚拟机 （VM） 提供的使用情况和按流量计费的计费使用情况的指标。 这些位于两个单独的选项卡中：VM 使用情况和按流量计费的计费使用情况。

在 VM 使用选项卡中，有以下项的图形表示形式：

- [使用情况摘要](#usage-summary)
- [按地理位置分列的使用](#usage-by-geography)
- [按优惠的使用情况](#usage-by-offers)
- [按优惠和 SKU 表示的使用趋势](#usage-trend-by-offers-and-skus)
- [按产品/服务类型表示的使用情况](#usage-by-offer-type)
- [按 VM 大小表示的使用情况](#usage-by-vm-size)
- [按销售渠道的使用情况](#usage-by-sales-channel)
- [详细的使用数据](#detailed-usage-data)

> [!NOTE]
> 分析报告在云合作伙伴门户 （CPP） 和合作伙伴中心中的显示方式不同。 CPP 中的**卖家洞察**具有订单和使用选项卡，该选项卡显示基于使用情况的优惠和非基于使用情况的优惠的数据。 在合作伙伴中心中，使用指标显示在单独的页面上。

### <a name="usage-summary"></a>使用情况摘要

使用情况摘要表显示他们购买的所有产品/服务的客户使用时间。

- 标准化使用时间定义为标准化的使用小时数，以考虑 VM 内核的数量（[VM 内核数] x [原始使用小时]）。 被指定为"SHAREDCORE"的 VM 使用 1/6（或 0.1666）作为 [VM 内核数] 乘数。
- 原始使用小时定义为 VM 以小时形式运行的时间量。
- 百分比值表示所选日期范围（[上个月使用情况 + 第一个月使用量]）/第一个月使用情况的使用情况增长变化）。
- 向上指向的绿色三角形表示增长变化。
- 指向向下的红色三角形表示与上月相比出现负增长变化。
- 微条图表示每月值。 您可以通过将鼠标悬停在图表中的列上来显示每个月的值。

### <a name="usage-by-geography"></a>按地理位置分列的使用

**按地理热图的标准化使用情况**显示根据客户国家/地区映射的使用时数。 国家/地区颜色变化表示规范化的使用浓度。 按地图上的**主页**按钮，还原到原始视图。

### <a name="usage-by-offers"></a>按优惠的使用情况

- **按优惠饼图的标准化使用情况**显示按所选日期范围按优惠的标准化使用时间细分。 前 5 个优惠以图形显示，其余产品分组在"全部休息"类别中。
- 条形图描绘了所选日期范围的逐月增长趋势。 月份列表示相应月份使用时间最高的优惠的工时。 折线图描绘了在辅助 Y 轴上绘制的增长百分比趋势。
- 使用图表顶部的滑块沿 x 轴向右向左滚动和/或关注特定数据点。

### <a name="usage-trend-by-offers-and-skus"></a>按优惠和 SKU 表示的使用趋势

此图表显示产品/服务所选 SKU 的规范化使用趋势。 产品/服务排行榜显示使用率最高的前 50 个产品/服务，并按使用时间进行排序。 SKU 排行榜显示前 50 个 SKU，所选产品/服务使用率最高。

### <a name="usage-by-offer-type"></a>按产品/服务类型表示的使用情况

- **按产品/服务类型饼图的使用情况**根据产品/服务类型组织使用情况。
- 顶部优惠显示在图表中，其余优惠分组为"全部休息"。
- **趋势**图表显示逐月增长趋势。 "月份"列表示该月按顶级产品/服务类型的使用情况。

### <a name="usage-by-vm-size"></a>按 VM 大小表示的使用情况

此图表表示所有产品/SKU 中所选 VM 大小（最多 5 个）的使用趋势。 柱形图与所选 VM 大小的使用小时一起堆叠。

排行榜显示使用率最高的前 50 个 VM 大小，并按使用时间排序。

### <a name="usage-by-sales-channel"></a>按销售渠道的使用情况

- 按销售渠道饼图的使用情况根据销售渠道组织使用情况
- 使用率最高的顶级销售渠道显示在图表中，销售渠道的其余部分分组为"全部休息"。
- "月份"列表示该月中顶级销售渠道的使用情况。
- 此图表的功能与"按优惠的使用情况"图表相同

### <a name="detailed-usage-data"></a>详细的使用数据

**使用情况详细信息表**显示按使用情况排序的前 1000 个使用记录的编号列表。

- 网格中的每个列都是可排序的。
- 如果记录计数小于 1000，则可以将数据提取到 CSV 文件。
- 如果记录计数超过 1000，导出数据将异步放置在下载页中，该页面将在未来 30 天内可用。
- 筛选器可以应用于**详细的使用情况数据**，以仅显示您感兴趣的数据。 数据可以按国家/地区、销售渠道、市场许可证类型、使用情况类型、产品/服务名称、产品/服务类型、免费试用版、应用商店订阅 ID、客户 ID 和公司名称进行筛选。

> [!NOTE]
> 选择页面筛选器中的 **"使用情况"类型**，在"标准化视图"或"原始视图"中查看页面上的图表。 这些图表的默认视图是"标准化视图"。

"**使用情况"页筛选器**在页面级别应用。 您可以选择多个筛选器来呈现图表，以便查看所需的条件以及"详细使用情况数据"网格/导出中显示的数据。 筛选器应用于订单页右上角为所选数据范围提取的数据。

- **优惠类型**和**优惠名称**仅列出您在所选日期范围内获得的优惠。 对于从列表中选择的报价类型，将显示列表中的要约名称。
- 每个筛选器选项的默认选择为"全部"，但**使用类型**除外。 **使用类型的**默认选择是规范化使用。 要在图表中显示原始用法，请选择"原始用法"。
- 应用筛选器显示所制作的筛选器选择的计数选择。 不显示默认选择的已应用筛选器。

> [!NOTE]
> 在[常见问题解答和术语](link needed)文章中的数据字典部分中定义了"详细订单数据"网格、页面筛选器和所有可能选择中每个字段的详细定义。

"**按流量计费的计费使用情况**"选项卡显示产品/服务类型的使用情况信息，其中使用情况由每仪表维度衡量。 SaaS 优惠类型当前呈现。 该选项卡显示 SaaS 计量计费使用量的超额趋势的图形表示：

- **按仪表维度显示超额趋势**：显示报价所选仪表维度的每月超额趋势。 X 轴表示月份，Y 轴表示使用数量。 自定义仪表的测量单位也显示在 Y 轴上。
- **SKU 的超额趋势**： 表示所选仪表维度的使用数量趋势（按 SKU）。 显示的 SKU 将代表所选产品/服务使用量最高的前 5 个 SKU。
- **前 50 名客户的超龄趋势**：使用时间最高的前 50 个产品在***排行榜***上显示，按自定义仪表的最高使用量进行排名。 在排行榜中选择客户以查看所选仪表维度的使用趋势。
- **顶级客户的超额趋势**：呈现占总体使用量百分比的顶级客户百分位数。 顶部客户百分位数沿 X 轴显示，由客户的使用数量决定。 Y 轴显示使用数量。 您可以通过沿折线图悬停在点上来显示详细信息。

> [!NOTE]
> 对于为页面筛选器选择的仪表维度，将显示此页上的使用详细信息和所有图表。
