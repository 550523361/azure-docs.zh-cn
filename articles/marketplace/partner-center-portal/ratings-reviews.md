# <a name="ratings-and-reviews-dashboard-in-commercial-marketplace-analytics"></a>商业市场分析中的评级和评论仪表板

本文提供有关合作伙伴中心中的评级&审核仪表板的信息。 此仪表板在 Azure 应用商店和 AppSource 上显示客户反馈的综合视图。

要访问合作伙伴中心分析工具中的**评级&审核仪表板**，请打开"商业市场"下的**["分析](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)**"仪表板。

>[!NOTE]
> 有关分析术语的详细定义，请参阅[商业市场分析的常见问题和术语](./faq-terminology.md)。

## <a name="ratings--reviews"></a>评分&评论

当客户在两个市场中浏览、搜索和购买优惠时，他们可以保留他们获得的优惠的评级和评论。

- 客户可以提交新评级或审核、更新或删除已提交的现有评级或审核。 客户只能更改他们拥有的评级和评论。  
- 审核张贴在 Azure 市场或 AppSource 产品展示页上的"审核"选项卡上。 客户可以包括其姓名或匿名发布。  

**"评级&审核仪表板**显示客户此活动的趋势和见解。 您可以查看以下项目的图形表示形式：

- 评分&评论  
- 查看评论

应用商店**网选项卡**允许发布者在 AppSource & Azure 应用商店选项卡中单独查看其产品/服务指标。 要查看特定产品/服务指标，请从产品/服务下拉列表中选择产品/服务。


### <a name="ratings-and-reviews-summary"></a>评分和评论摘要

审核摘要&评级显示以下选定日期范围的指标：

- **平均评级：** 加权平均星级评级，即客户为所选产品/服务提交的所有评级。
- **评级细分：** 按提交评级的客户数细分星级。 条形图与实际和修订的评级（更新的评级计数）一起堆叠在一起。
- **总评分：** 提交的评级总数。 此计数还包括有评论和没有评论的评分。
- **评分与评论：** 提交的审核计数。

![合作伙伴中心分析评分和评论摘要](./media/analyze-ratings-summary.png)

### <a name="review-comments"></a>查看评论

评论按时间顺序显示，以过帐时的顺序显示。 默认视图显示所有评论，您可以使用下拉菜单中的 **"评级"筛选器**按星级筛选评论。 此外，您可以按评论中显示的关键字进行搜索。  

![合作伙伴中心分析评论评论](./media/analyze-reviews.png)
