# <a name="overview-of-the-commercial-marketplace-portal-in-partner-center"></a>合作伙伴中心中的商业 Marketplace 门户的概述

欢迎参加 Microsoft 合作伙伴中心中的商业 Marketplace 门户。 [商业 Marketplace 门户](https://partner.microsoft.com/dashboard/commercial-marketplace/)旨在帮助您将转化为收益产品和服务，吸引客户并发展你的业务。

在本文档中，我们将介绍基本的创建和使用你的合作伙伴中心帐户，创建新的软件与服务 (SaaS) 提供，可以上列出[Azure Marketplace](https://azuremarketplace.microsoft.com/)或[AppSource](https://appsource.microsoft.com/)，管理现有的 SaaS 产品，并提供资源以帮助您的问题，并收集您的反馈。 

## <a name="about-microsofts-commercial-marketplace"></a>关于 Microsoft 的商业市场

企业、 政府以及各种规模的组织查找业务解决方案、 创新的技术，并通过启用服务[Azure Marketplace](https://azuremarketplace.microsoft.com/)并[AppSource](https://appsource.microsoft.com/) web 存储[Azure 门户中产品体验](https://portal.azure.com)，[云解决方案提供商](https://partner.microsoft.com/cloud-solution-provider)，并由 Microsoft 销售团队。 

合作伙伴中心是体验的 Microsoft 合作伙伴的中心。 对于转到以与 Microsoft 进入市场，并充分利用 marketplace 的组织，合作伙伴中心是通过其发布者创建和管理产品/服务、 提供测试驱动器相关演示，以及查看分析对产品/服务性能的门户。 

## <a name="migration-from-cloud-partner-portal"></a>迁移的云合作伙伴门户

合作伙伴中心发展为更好地与软件发布者的 Microsoft 的关系的所有方面。 作为这一改进的一部分，我们要迁移的函数[云合作伙伴门户](https://cloudpartner.azure.com/)到合作伙伴中心。 

**目前，合作伙伴中心仅支持创建和管理软件作为-服务 (SaaS) 产品/服务。**

现有的 marketplace 发布者会发现具有直观的界面的熟悉，但改进了用户体验。 在门户中，通过电子邮件通知将通知发布服务器的迁移。 用户将无缝地定向到相应的位置。 客户不会受到此合作伙伴体验升级。 

### <a name="improvements-on-user-experience-and-billing"></a>改进用户体验和计费

过渡到合作伙伴中心，新功能可供发布服务器，以及对用户体验方面的改进。 SaaS 产品/服务，可以每月或按年度计费，支持每用户授权 （基于席位） 除了每个站点的授权 （固定费率）。 这一新货币化模型非常适合于商业软件。 引入了 SaaS 的每用户许可符合允许完成购买的客户在 AppSource，面向业务或行业解决方案的 Microsoft 商业市场中的新功能。 

### <a name="account-and-user-management-updates"></a>帐户和用户管理更新

在合作伙伴中心帐户和用户管理轻松适用于个人才能访问门户和管理员可以应用适当的权限。 与以前的产品/服务管理门户相比，在合作伙伴中心的基于角色的访问控制功能提供使发布过程更简单的大型组织的安全增强功能。 

### <a name="improved-certification-experience"></a>改进的证书体验

合作伙伴中心提供了改进的证书体验。 证书现在将利用自动化和改进的周转时间的机器学习和更快速的处理。 维护准确性将保持的头等大事，确保云应用商店是安全的且包含有价值的产品/服务为我们的客户。 要求将不会更改，但现在通过合作伙伴中心提供的提交时，你将收到更多详细描述性证书的报告引用[面向公众策略](https://docs.microsoft.com/legal/marketplace/certification-policies)。 

### <a name="analytics-for-performance-reporting"></a>分析性能报告

合作伙伴中心分析帮助用户了解产品/服务的性能和改进的报告工具。 发布服务器可以监视使用的数据可视化和见解关系图来优化列表的销售。 改进的分析工具，可对性能结果和维护与客户和分销商的更好的关系。
