# <a name="summary-dashboard-in-commercial-marketplace-analytics"></a>商业市场分析中的“摘要”仪表板

本文提供合作伙伴中心的“摘要”仪表板的相关信息。 此仪表板显示了聚合值的图表、趋势和值，这些内容汇总了你的产品/服务的市场活动。

要访问“摘要”仪表板，请在商业市场下打开[分析仪表板](https://partner.microsoft.com/dashboard/commercial-marketplace/analytics/summary)。

>[!NOTE]
> 有关分析术语的详细定义，请参阅[适用于商业市场分析的常见问题和术语](./faq-terminology.md)。

## <a name="summary-dashboard"></a>“摘要”仪表板

“摘要”仪表板基于每个产品/服务类型显示相关概述。 “见解”一目了然地显示关键信息，并提供关于你的产品/服务的销售活动的丰富视图。 可使用“摘要”仪表板直观显示这些报表。 本文详细介绍了下面每个元素：

- [日期范围](#date-range)
- [摘要部分](#summary-section)
- [按地域分类的客户](#customers-by-geography)
- [增长趋势图](#growth-trend-charts)
- [客户排行榜](#customer-leaderboard)
- [席位计数趋势](#seat-count-trend)
- [SaaS 免费试用版订单趋势](#free-trials-saas-orders-trend)

## <a name="elements-of-the-summary-dashboard"></a>“摘要”仪表板的元素

下面各部分介绍了如何使用“摘要”仪表板以及如何读取数据。

### <a name="date-range"></a>日期范围

可在每页右上角找到日期范围选择。 通过基于过去 3、6 或 12 个月选择一个日期范围，或者选择最长时段为 12 个月的自定义日期范围，自定义“摘要”页面图表的输出内容。 默认日期范围为 6 个月。

![合作伙伴中心分析仪表板](./media/analyze-dashboard.png)

### <a name="summary-section"></a>摘要部分

摘要部分显示了在所选日期范围内创建的所有订单数、获得的客户数和报告的使用计数。 计算这些指标时将不考虑当前月份的部分时段。 例如：如果选择的时间范围为 6 个月，则计算当前月份之前 6 个月的使用小时数。 如果选择的是自定义日期范围，则计算时不考虑当前月份的部分时段。

![“摘要”仪表板中的增长趋势](./media/summary-summary-section.png)

#### <a name="reading-the-summary-section"></a>读取摘要部分

- **订单数**：你至今为止发布的产品/服务所销售的所有订单的计数（已取消的订单除外）。
- **客户数**：购买过你的产品/服务且至少有一个订阅未取消的所有客户的计数。
- **规范化使用小时数**：定义为根据 VM 核心数规范化的使用小时数（[VM 核心数] x [原始使用小时数]）。 指定为“SHAREDCORE”的 VM 使用 1/6（或 0.1666）作为 [VM 核心数] 的乘数。
- **原始使用小时数**：VM 的运行时间（以小时为单位）。 订单总数、客户总数、规范化使用小时数、原始使用小时数、页面访问次数和行为召唤旁边的百分比值表示所选时间范围的使用增长量，计算公式为 ([上个月的使用量 - 第一个月的使用量])/第一个月的使用量     。 如上所述，从该指标中排除了当前月份的部分时段。
- **增长趋势**：如果将鼠标悬停在图表的列上，条形图会显示每个月的值。
- **朝上的绿色三角形**：表示正增长趋势。
- **朝下的红色三角形**：表示与上个月相比增长趋势为负。

### <a name="customers-by-geography"></a>按地域分类的客户

“按地域分类的客户”热度地图显示了世界地图上的客户计数。

![“摘要”仪表板中按地域分类的客户](./media/summary-customers-by-geography.png)

- 可移动地图来查看确切的位置。
- 可缩放到具体的位置。
- 热度地图有一个补充性的网格，它显示特定位置的客户计数、订单计数和规范化使用小时数的详细信息。
- 可在网格中搜索和选择一个国家/地区，放大地图找到该位置。 按地图上的主页按钮可还原为原始视图。
- 新客户是指在所选日期范围内的月份中首次购买了你的某项产品/服务的客户。

### <a name="growth-trend-charts"></a>增长趋势图

可查看基于购买的订单数（包括取消的订单）、获得的客户数（包括流失的客户）和报告的使用情况的增长而得出的趋势，其中这些信息根据所选日期范围逐月显示  。 可选择图表下方的链接进一步分析这些趋势，这些链接将导航到相应的订单、使用情况、客户或市场见解页面   。

两个选项卡上分别显示了 Azure 市场和 AppSource 的市场产品/服务“页面访问和行为召唤”趋势图。

![“摘要”仪表板中的“页面访问和行为召唤”趋势图](./media/summary-page-visits-and-cta.png)

“订单(按产品/服务)”图表根据产品/服务名称对你的订单进行整理。

“订单(按销售渠道)”饼图按销售渠道对你在选定日期范围内的订单（包括客户取消的订单）进行整理。 销售渠道是客户用来购买 Azure 的许可协议类型，也就是云解决方案提供商 (CSP)、企业、企业通过经销商、GTM 和 即付即用。

“使用情况(按产品/服务)”和“使用情况(按销售渠道)”饼图分别按主要的产品/服务和销售渠道显示了使用情况的明细信息 。 内部饼图显示了原始使用情况，外部饼图显示了规范化的使用情况。

“订单(按市场许可证类型)”和“使用情况(按市场许可证类型)”饼图按各自的许可证类型显示了订单和使用情况的明细信息 。 许可证类型包括：

- **通过 Azure 计费**：如果你选择使用此许可证类型通过 Microsoft 销售你的产品/服务，Microsoft 将代表你对客户计费。 付款类型包括通过信用卡的即用即付或企业发票。
- **自带许可**：Microsoft 不会向使用此类型的市场产品/服务的客户计费。 这种使用在市场中被列为“立即获取（免费）”。
- **免费**：Microsoft 不会向使用此类型的市场产品/服务的客户计费。 这种使用在市场中被列为“免费试用版”。
- **Microsoft 充当经销商**：表示 Microsoft 经销商作为云解决方案提供商 (CSP) 计划的一部分销售的产品/服务。

### <a name="customer-leaderboard"></a>客户排行榜

订单数最多的前 50 名客户按订单数百分比从高到低的顺序显示在排行榜上。

- 选择客户可查看其档案详细信息、按产品/服务整理的订单，或者按 Azure 许可证类型和定价频道整理的订单。
- “产品/服务(按订单)”圆环图显示排名前四的产品/服务（按订单计数），其余产品/服务分组到“其余所有”类别中。
- “规范化使用情况(按产品/服务)”圆环图按使用情况显示了排名前五的产品/服务。

> [!NOTE]
> 仅当客户同意时，才显示客户的个人信息。 如果你已使用所有者角色权限级别登录，则可查看此信息。 具有参与者角色的用户无法查看此信息。 [详细了解用户角色和权限](./manage-account.md#define-user-roles-and-permissions)。

### <a name="seat-count-trend"></a>席位计数趋势

“订单(按每席位/每站点)”图表显示了根据定价模型购买的所有订单的明细信息。 “席位计数趋势”图表显示了席位数与针对每个席位的所有软件即服务 (SaaS) 产品购买的订单数。

### <a name="free-trials-saas-orders-trend"></a>SaaS 免费试用版订单趋势

“SaaS 免费试用版订单趋势”图表显示了提供 30 天试用期的 SaaS 免费试用版产品/服务的订单趋势。
