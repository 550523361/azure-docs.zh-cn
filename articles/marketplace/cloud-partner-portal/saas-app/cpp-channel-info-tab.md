# <a name="saas-application-channel-info-tab"></a>SaaS 应用程序“渠道信息”选项卡

使用“渠道信息”选项卡为你的应用提供渠道信息，包括上市 (GTM) 材料和联系信息。

![“渠道信息”窗体](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>配置渠道信息

1. 在“渠道信息”>“GTM 材料”下，输入托管 GTM 材料的网站的 URL。
2. 在“渠道经理联系人”下，提供经理联系人，Microsoft 可以联系他以获取支持和业务问题。 为以下字段输入信息：“名称”、“电子邮件地址”和“电话号码”。
3. 选择“保存”以完成设置渠道信息。
