# <a name="azure-saas-application-offer"></a>Azure SaaS 应用程序产品/服务

[云合作伙伴门户](https://cloudpartner.azure.com/)不再支持创建和管理的 SaaS 应用程序产品/服务。  相反，此功能已移到 Microsoft[合作伙伴中心门户](https://partner.microsoft.com/pcv/)。  有关详细信息，请参阅[创建新的 SaaS 产品/服务](../../partner-center-portal/create-new-saas-offer.md)。 


## <a name="next-steps"></a>后续步骤

在创建新的 SaaS 应用程序产品之前，应阅读[SaaS 应用程序产品/服务发布指南](../../marketplace-saas-applications-technical-publishing-guide.md)并查看中列出的合作伙伴中心门户步骤[产品/服务创建清单](../../partner-center-portal/offer-creation-checklist.md)。
