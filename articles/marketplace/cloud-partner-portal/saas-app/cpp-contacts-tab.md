# <a name="saas-application-contacts-tab"></a>SaaS 应用程序“联系人”选项卡

本文介绍了如何使用“联系人”选项卡来为使用你的产品/服务的客户标识联系人。

## <a name="identify-contacts"></a>标识联系人

使用下面的屏幕截图作为指南来提供所需的信息。 标有星号 (*) 的所有字段都必须完成。

![“工程和支持联系人”窗体](./media/saas-offer-contacts.png)

1. 在“工程联系人”下，提供以下字段的信息：

   - 姓名 - 输入应用的工程联系人姓名。 此联系人将收到来自 Microsoft 的技术邮件。
   - 电子邮件 - 输入工程联系人的电子邮件地址。
   - 电话 - 输入工程联系人的电话号码。 支持 ISO 电话号码表示法。 有关详细信息，请参阅 [E.123](https://en.wikipedia.org/wiki/E.123)
  
2. 在“支持联系人”下，提供以下字段的信息：

   - 姓名 - 输入应用的支持联系人姓名。 此联系人将收到来自 Microsoft 的支持相关邮件。
   - 电子邮件 - 输入应用的支持联系人的电子邮件地址。
   - 电话 - 输入支持联系人的电话号码。 支持 ISO 电话号码表示法。 有关详细信息，请参阅 [E.123](https://en.wikipedia.org/wiki/E.123)
   - 支持 URL - 输入支持页面的 URL。

3. 在完成窗体时选择“保存”。
