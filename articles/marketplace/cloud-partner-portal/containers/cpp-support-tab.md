# <a name="container-support-tab"></a>容器的“支持”选项卡

> [!IMPORTANT]
> 从 2020 年 4 月 13 日开始，我们将开始将 Azure 容器产品的管理移动到合作伙伴中心。 迁移后，您将在合作伙伴中心创建和管理您的优惠。 按照["创建 Azure 容器产品/服务](https://aka.ms/CreateContainerOffer)"中的说明进行操作，以管理迁移的优惠。

使用“新建产品/服务”**** 页的“支持”**** 选项卡为产品/服务提供技术和用户支持资源。  “支持”窗体上提供了以下输入区：“工程联系人”、“客户支持”和“支持 URL”。************ 名称旁边附有星号 (*) 的字段表示必填字段。


## <a name="support-form-fields"></a>“支持”窗体字段

提供“工程联系人”和“客户支持”字段所需的信息。


### <a name="engineering-contact"></a>工程联系人

工程联系人是你的组织与 Microsoft 之间的技术联系人。 必填字段用星号 (*) 表示。

- **名称\***- 担任技术/工程支持的人员或组的名称。
- **电子邮件\***- 此技术联系人的电子邮件地址。
- **电话\***- 技术支持的电话号码。


### <a name="customer-support"></a>客户支持

客户支持联系人接收 Azure 中的客户开具的支持票证。  必填字段用星号 (*) 表示。

- **名称\***- 担任客户支持的人员或组的名称。
- **电子邮件\***- 此支持联系人的电子邮件地址。
- **电话\***- 客户支持的电话号码。


## <a name="next-steps"></a>后续步骤

提供支持信息后，便可[发布产品/服务](./cpp-publish-offer.md)。
