# <a name="iot-edge-module-offer"></a>IoT Edge 模块套餐

[云合作伙伴门户](https://cloudpartner.azure.com/)不再支持创建和管理 IoT Edge 模块产品。 此功能已移动到 Microsoft[合作伙伴中心](https://partner.microsoft.com/pcv/)。 有关详细信息，请参阅[在 Azure Marketplace 中创建、配置和发布 IoT Edge 模块产品](../../partner-center-portal/azure-iot-edge-module-creation.md)。
