<a name="test-drive-marketing-and-best-practices"></a>体验版营销和最佳做法
=======================================

Azure 市场体验版是面向营销人员的极佳工具，我们建议将它整合到即将推出的面市套餐。 我们制定了许多的最佳做法，建议在发布套餐时立即遵循这些做法，这样可以大幅增加体验版带来的潜在顾客：

- **推荐页面**：在 Web 内容中发布一个页面来全面介绍你的 Azure 体验版，并发布多个推荐链接，以便将流量定向到体验版
- **社交媒体**：创建一个计划，以便定期在社交媒体渠道中发布有关体验版的信息
- **博客**：编写有关 Azure 体验版的博客文章。 强调用户在极短的时间内就能免费体验到的功能和优势。 请确保在博客副本中的多个位置链接到 Azure 体验版页面。 下面是有关 Azure 体验版的示例博客文章的链接。 还有更好的？ 在社交媒体渠道中发布博客文章的链接。
- **电子邮件关系培养**：通过电子邮件联系已在其他渠道中与你的品牌有互动的潜在顾客，告知他们有关体验版的信息，并鼓励他们免费试用。
- **销售培训和工具**：为销售团队安排培训会议，以演示体验版的体验，并探讨销售人员与潜在顾客和客户互动的多种方法。 销售人员可为公司的多个联系人安排演示，然后向每个联系人发送体验版的链接，让他们自己体验。
- **商展和大会**：在潜在客户将会参加的行业活动中预订一个展位。 在展位中摆放几台笔记本电脑并运行体验版，让客户在展位中试用该体验版，并在他们体验期间和之后谈论该产品！

<a name="lead-management"></a>线索管理
---------------

现在，体验版正在产生很适合培养合作关系的潜在顾客，需确保将这些潜在顾客整合到销售和营销渠道并与他们互动，以将他们转化为付费客户。 下面是有关推动潜在顾客参与你的销售周期的一些建议做法：

- 在潜在顾客试用体验版之后的 24 小时内与他们取得联系。 在客户部署体验版之后，立即将潜在顾客登记到所选的 CRM -- 请务必在最初的 24 小时内向他们发送电子邮件，此时他们对体验版仍保持着极高的热情。 请求安排与他们进行电话沟通，以更好地了解他们正在试图解决的问题，并确定你的产品是否有助于解决这些问题。
- 跟进几次，但不要刨根问底。 建议在放弃潜在顾客之前再发几次电子邮件，而不要尝试一次就放弃。 请记住 -- 这些客户已直接体验了你的产品，并花了一些时间进行免费试用。
- 他们是非常适合培养的潜在客户！
- 如果你与体验版的潜在顾客达成了交易，请务必将其注册到合作伙伴销售联络系统中。 另外 -- 我们很乐意听到体验版为客户的成功发挥了作用！
