# <a name="power-bi-app-contacts-tab"></a>Power BI 应用联系人选项卡

>[!Important]
>从 2020 年 3 月 30 日开始，我们将开始将 Power BI 应用优惠的管理转移到合作伙伴中心。 迁移后，您将在合作伙伴中心创建和管理您的优惠。 按照[Power BI 应用创建概述](https://aka.ms/AzureCreatePBIServiceApp)中的说明进行操作，以管理迁移的优惠。

使用“新建产品/服务”**** 页的“联系人”**** 选项卡为产品/服务提供技术和用户支持资源。 它分为两个部分：**工程和****客户支持**。

![“联系人”选项卡](media/contacts-tab.png)


## <a name="contacts-fields"></a>“联系人”字段 

下表描述了每个字段的用途。  必填字段用星号 (*) 表示。 

|   字段               |   说明                                                            |
|-----------------------|--------------------------------------------------------------------------|
| *工程联系人* | 作为 AppSource 与你的组织之间的技术联系人。    |
| **名字\***              | 作为技术或工程支持的人员或组的名称。 |
| **电子邮件\***             | 此技术联系人的电子邮件地址。                                  |
| **电话\***             | 技术支持的电话号码。                                       |
| *支持联系人*     | 接收 AppSource 内客户提交的支持工单。            |
| **名字\***              | 作为客户支持的人员或组的名称。              |
| **电子邮件\***             | 客户支持的电子邮件地址。                                        |
|   |   |


## <a name="next-steps"></a>后续步骤

完成 Power BI 应用产品/服务后，确认已为此产品/服务创建了所有关联的[技术资产](./cpp-create-technical-assets.md)。
