# <a name="create-power-bi-app-technical-assets"></a>创建 Power BI 应用技术资产

此产品/服务类型所需的主要技术资产是 Power BI 应用程序，它是主数据集、报表和（或）仪表板及连接的可选服务和嵌入的数据集的集合。 （Power BI 应用程序是之前称为[内容包](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction)的资产类型的演变结果。）有关开发这些应用程序的完整信息，请参阅 [Power BI 应用](https://go.microsoft.com/fwlink/?linkid=2028636)。


## <a name="obtaining-an-installation-url"></a>获取安装 URL

只能在 [Power BI 服务](https://powerbi.microsoft.com/)内构建 Power BI 应用，且需要使用 [Power BI Pro 许可证](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro)登录。 在 Power BI 中创建并测试服务应用程序后，请立即保存 Power BI 生成的应用程序安装 URL。  必须在云合作伙伴门户提交窗体的[“技术信息”选项卡](./cpp-technical-info-tab.md)中提供此 URL。
 

## <a name="next-steps"></a>后续步骤

创建并测试了 Power BI 应用且创建了关联的产品/服务后，便可[发布 Power BI 产品/服务](./cpp-publish-offer.md)。
