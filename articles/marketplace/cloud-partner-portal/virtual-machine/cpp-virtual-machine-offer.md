# <a name="azure-virtual-machine-offer"></a>Azure 虚拟机套餐

[云合作伙伴门户](https://cloudpartner.azure.com/)不再支持创建和管理 Azure 虚拟机套餐。 此功能已移至 Microsoft [合作伙伴中心](https://partner.microsoft.com/pcv/)。 有关详细信息，请参阅[创建 Azure 虚拟机套餐](../../partner-center-portal/azure-vm-create-offer.md)。
