# <a name="cloud-partner-portal-management"></a>云合作伙伴门户管理

本节介绍可用于创建和管理使用[云合作伙伴门户](https://cloudpartner.azure.com)所需的帐户的操作的类型。  [Azure Marketplace](https://azuremarketplace.microsoft.com)和[AppSource](https://appsource.microsoft.com)需要这些帐户。  


## <a name="background-information"></a>背景信息

使用云合作伙伴门户之前，应熟悉以下文章：

- [Azure marketplace 政策和条款](../../policies-terms.md)列出了适用于和发布者的用户和[Azure marketplace](https://azuremarketplace.microsoft.com)和[AppSource](https://appsource.microsoft.com)的各种法律和技术要求及规则。
- [成为云 Marketplace 发布者](https://docs.microsoft.com/azure/marketplace/become-publisher)概述成为官方 Microsoft 云发布者所需的所有步骤。


## <a name="account-creation-and-management"></a>帐户创建和管理

本部分包含以下文章，这些文章介绍了如何管理 marketplace 发布者帐户。

- [管理帐户的先决条件](./cpp-manage-prerequisites.md)介绍了如何成为已注册的云发布者，以及使用云合作伙伴门户的要求。
- [管理 Azure Marketplace 和 AppSource 发布者配置文件](./cpp-manage-publisher-profile.md)说明了如何使用云合作伙伴门户查看和管理 Microsoft Marketplace 发布者配置文件。 
- [在云合作伙伴门户上管理用户](./cpp-manage-users.md)将用户添加到你的组织的云发布帐户，以及如何管理他们的访问权限，以允许多名人员处理相同的产品/服务。


## <a name="next-steps"></a>后续步骤

在继续操作之前，必须满足[管理帐户的先决条件](./cpp-manage-prerequisites.md)。  之后，还应学习[云合作伙伴门户教程](../portal-tour/cpp-portal-tour.md)来熟悉门户的用户界面。
