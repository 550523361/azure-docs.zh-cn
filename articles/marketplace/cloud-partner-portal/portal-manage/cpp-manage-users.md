# <a name="managing-users-on-the-cloud-partner-portal"></a>管理云合作伙伴门户上的用户

使用[云合作伙伴门户](https://cloudpartner.azure.com)，你可以将具有基于角色的访问权限的用户添加到虚拟机产品/服务。 这有助于在多人使用产品/服务时管理访问和权限。

## <a name="add-users-and-assign-roles"></a>添加用户和分配角色 

使用以下步骤来添加用户和分配用于产品/服务发布的角色。

1. 单击左侧导航窗格中的 "**用户**" 选项卡。

    ![“用户”选项卡](./media/userstab.png)


2. 单击“添加用户”。

    ![选择“添加用户”](./media/adduser.png)


3. 键入用户的电子邮件地址，然后选择角色分配。  您可以将新用户添加为 "所有者" 或 "参与者"。

    **电子邮件**：添加要发布产品的队友的电子邮件地址。 支持 Microsoft 帐户（Outlook、Hotmail 和 Live）和组织 Id。

    - 添加 "团队/组电子邮件别名/安全组" 以防止人员离开组织。
    - 确保为来自 Microsoft 的任何通信监视云合作伙伴门户上提供的电子邮件 Id。
    
    ![分配角色](./media/assignrole.png)

    **角色**：请参阅下表以确定合适的用户角色类型。

    ![角色级别](./media/roleaccesslevel.png)

    只有所有者才能访问[卖方 Insights](../../cloud-partner-portal-orig/si-getting-started.md)中的 azure 付款和 azure 客户选项卡。


4. 选择要添加到新用户的发布者名称，然后单击 "添加" 以完成用户添加。

    > [!NOTE]
    > 电子邮件作为所有者或参与者添加的 "发布者名称" 将显示在 "**发布者**" 下拉列表中以供选择。


    ![分配角色 - 选择发布者](./media/assignselectpublisher.png)


## <a name="delete-an-existing-user"></a>删除现有用户

若要删除现有用户，请在 "**用户**" 选项卡中搜索其电子邮件，然后按右侧的 "**删除**" 按钮。
