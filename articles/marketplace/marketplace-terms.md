# <a name="azure-marketplace-terms"></a>Azure 市场条款 

这些条款约束 Microsoft Azure 市场（简称“市场”）的用途，必须根据这些条款访问或购买经过优化的、可在 Azure 上使用的产品和服务。 这些产品和服务包括虚拟机库和任何其他 Microsoft Azure 功能。 市场中包括 Microsoft 和第三方发布者（每个此类发布者简称为“发布者”）发布的产品或服务（“市场套餐”，简称为“套餐”）。

## <a name="purchasing-and-billing"></a>购买和计费

1. 直接客户。 如果你从 Microsoft 购买了 Microsoft Azure 订阅，则以下条款适用于通过该订阅购买的市场套餐： 

    1. 付款。 你同意支付所有适用的、与市场套餐相关的使用费。 这些费用是另外计收的，是任何 Microsoft Azure 服务使用费的补充。 
    1. 价格。 除非另有说明，否则市场套餐的定价不包括所有适用的税费和货币兑换结算费。 你需要独自负责支付此类税费或其他费用。 价格可能会按照订阅型套餐的重复性费用给出，也可能按照消费型套餐的某个计量单位（例如运行小时数或执行的事务数）给出。 
    1. 付款方式。 向 Microsoft 提供 Microsoft Azure 订阅的付款方式后，即已授权 Microsoft 按照该付款方式收取市场套餐的购买费用。 信用卡就是付款方式的示例。 
    1. 计费。 市场购买活动受 Microsoft Azure 订阅协议的计费和付款条款的约束。  这些条款适用于基于消耗量或即用即付的购买方式。 如果在计费周期内，购买或使用套餐所产生的费用未得到处理，我们可能会在后续计费期间向你计收此类费用。 我们会通过 Microsoft Azure 帐户门户提供费用列表，你可以在其中查看和打印费用。 这是我们提供的唯一费用帐单。 非最终费用帐单中显示的任何用量或支出金额只是估算数字。  估算数字不包括最近的活动，或者可能包括基于以往使用模式的预测。
    1. 涨价。 如果你购买了其有效订阅的市场套餐的价格上调，我们会在涨价之前的 30 天发出通知。 30 天通知期过后，继续使用该市场套餐即表示你已授权按新价格计费。
    1. 退款。 除非法律有规定或者受发布者的允许，否则所有费用不可退还。
    1. 计费更正。 如果你发现帐单有误，必须在该错误首次出现在帐单上之后的 120 天内告诉我们。 如果你未在该期限内告知我们，则我们不会承担该错误引发的任何责任，并且你无法就相关损失索赔。 我们没有义务更正错误或提供退款。 如果我们发现了计费错误，将在 90 天内更正该错误。

1. 间接客户。 如果你从经销商处购买了 Microsoft Azure 订阅，则与你经销商达成的协议中的购买和计费条款将会约束通过该订阅购买的市场套餐。 

1. *通则。* 不管从哪个实体购买了 Microsoft Azure 订阅，以下条款均适用。 

    1. 货币承诺。 除非对特定的市场套餐另外做了规定，否则，Microsoft 订阅信用额度（例如免费试用、MSDN 或 BizSpark）或货币承诺资金不可用于购买市场套餐。 此类购买活动将单独计费。
    1. 预订。 “预订”是指根据指定的期限和区域提前购买符合条件的市场套餐，例如“预留的软件实例”。 预订的指定期限最长为三年。 指定的期限结束后，预订即会过期。 未使用的预订产品不可退款。 除非对特定的市场套餐另外做了规定，否则不可更换和取消预订产品。 预订价格基于购买时的适用价格。 针对软件的预留实例不包含计算成本。
    1. 自动续订。 如果你使用自动续订选项购买了市场订阅，我们或你的经销商（如果适用）可以自动续订你的订阅，并按照任何续订条款向你收费。  若要避免此类费用，可以在续订日期之前取消订阅。
    1. 免费试用。 如果你参与了市场套餐的免费试用订阅，则必须在试用期结束之前取消订阅才能避免产生费用。 如果在免费试用期结束之前未取消订阅，则需要按购买价格支付市场套餐费用。 如果创建了免费试用套餐的多个订阅，则适用于所有此类订阅的试用期将在创建第一个订阅的日期开始计算。  试用期结束后，所有此类订阅将转换为付费套餐。
    1. 取消。 如果你未按时提供全额付款，我们可以暂停或取消你对市场套餐的访问权限。 由于未付款而暂停或取消访问权限可能会导致数据丢失。

## <a name="use-rights"></a>使用权利

1. 发布者使用条款。 你对任何市场套餐的使用权需要遵守市场套餐发布者单独提供的使用条款（“使用条款”）。 我们只对自己发布的市场套餐负责，而不受发布者使用条款的约束。 

1. BYOL 套餐。 如果你有权在市场外部使用特定的市场套餐，发布者可能会供应此类套餐（“自带许可 (BYOL) 套餐”）。 如果你使用任何 BYOL 套餐，需要负责确保有足够的权利使用该套餐。

1. Microsoft 软件产品。 如果第三方发布者在市场中发布了虚拟机套餐，并且其中包括 [http://azure.microsoft.com/pricing/details/virtual-machines/](http://azure.microsoft.com/pricing/details/virtual-machines/) 上所列的任何 Microsoft 软件产品（“Microsoft 软件产品”）：
    1. Microsoft （而不是第三方）是 Microsoft 软件产品的许可方
    1. 除了与第三方套餐相关的任何费用以外，使用 Microsoft 软件产品需要支付适用的 Azure 订阅费用。  经销商可以规定这些费用；并且
    1. Microsoft 的条款（而不是第三方的使用条款）将约束该套餐中包含的任何 Microsoft 软件产品的使用。 这些条款还涉及到在 Microsoft Azure 外部使用 Microsoft 软件产品的权利（如果有）。

1. 应用程序编程接口。 我们可能会提供可与市场配合使用的应用程序编程接口（“市场 API”）。 如果你使用市场 API 购买或访问市场套餐，则需要责任查看并遵守市场用户界面中的适用使用条款。  这些条款随时可能会有变化。 使用任何市场 API 购买市场套餐即表示你同意根据付款条款支付所有适用费用。  购买时，在市场用户界面中可以找到付款条款。

## <a name="privacy-and-data-protection"></a>隐私和数据保护

1. 透露给发布者的信息。 如果你购买或使用市场套餐，我们可能会与此类套餐的发布者共享你的联系信息以及有关交易的详细信息。 我们只会在经得你的允许的情况下，才与任何发布者共享你的客户数据。

1. 发布者隐私政策。 发布者负责提供隐私声明，描述如何处理其套餐所收集的客户数据或者从 Microsoft 接收的任何客户信息的隐私。 除非另有规定，否则，Microsoft 发布的市场套餐以及 Microsoft 的隐私、安全、数据位置和保留策略不适用于任何市场套餐。 此条款同样适用于发布者对任何客户数据或其他客户信息的使用。

1. 个人数据的处理。 Microsoft 是套餐预配相关的个人数据的处理方或附属处理方。 Microsoft 向所有客户承诺遵守联机服务条款的欧盟一般数据保护条例条款，这些条款自 2018 年 5 月 25 日开始生效，详情请访问 [http://go.microsoft.com/?linkid=9840733](http://go.microsoft.com/?linkid=9840733)。

## <a name="miscellaneous"></a>其他

1. 金融数据。 某些市场套餐可能包含金融数据。 根据美国联邦证券法或其他辖区的证券法，Microsoft 并非中介/经销商或注册的投资顾问。  Microsoft 不会向个人提供有关投资、购买或销售证券或其他金融产品或服务的建议。 市场中不会包含任何建议或恳求购买或销售证券的商品。 Microsoft 及其股票报价或索引数据许可方不会认可或推荐任何特定的金融产品或服务。 市场中的任何商品（包括任何数据集或金融应用程序）都不会提供专业建议，包括但不限于投资或税务建议。

1. 限制。 为了保护系统，我们可能会限制你对市场发出的请求数。  Microsoft 还会针对市场的使用实施合理的限制。 此外，发布者可以限制你对其服务发出的请求数（“特定限制”）。 应用的特定限制可能会显示在发布者的内容详细信息页上。 特定限制可在不另行通知的情况下随时更改。
