# <a name="enable-lead-sharing"></a>启用潜在顾客共享  
请务必为你在市场中的一览启用潜在客户管理，以便接收以下受众的联系信息。

| 店铺 | audience |  
|:--- |:--- |  
| AppSource | 业务客户 |  
| Azure 市场 | 客户 |  

潜在客户提供了受众基础，可便于开展以下活动、现场销售人员促销活动，并了解应用或产品/服务的销售情况。 

| 店铺 | campaign |  
|:--- |:--- |  
| AppSource | 巩固关系 |  
| Azure 市场 | 按需生成 |  

针对潜在客户的最佳做法如下。  
*   限定潜在客户，并将潜在客户标记为销售商机
*   巩固与潜在客户的关系，以进入销售管道
*   根据整体市场营销活动策略开展工作 

潜在客户展现了客户对以下一览的兴趣。 

| 店铺 | 一览 |  
|:--- |:--- |  
| AppSource | <table> <tr><td>应用</td></tr> <tr><td>咨询服务</td></tr> </table> |  
| Azure 市场 | <table> <tr><td>产品</td></tr> <tr><td>技术</td></tr> </table> |  

应有潜在客户跟进流程。  

| 店铺 | 潜在客户 |  
|:--- |:--- |  
| AppSource | <table> <tr><td>你的 CRM</td></tr> <tr><td>Microsoft 现场销售人员 CRM</td></tr> </table> |  
| Azure 市场 | <table> <tr><td>Azure 市场</td></tr> <tr><td>Microsoft 现场销售人员 CRM</td></tr> </table> |  

市场店铺中的潜在客户不同于 Microsoft 销售人员限定的潜在客户（联合销售计划的一项功能）。  
*   若要详细了解如何加入联合销售计划，请访问 Microsoft 的“促进业务发展”页 ([promote-your-business-with-microsoft](./promote-your-business-with-microsoft.md))。
