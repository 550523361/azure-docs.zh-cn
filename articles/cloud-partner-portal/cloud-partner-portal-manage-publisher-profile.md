# <a name="managing-azure-marketplace-and-appsource-publisher-profile"></a>管理“Azure 应用商店”和“AppSource”发布者个人资料

本文档演示如何查看和管理注册的发布者个人资料。 

此时，你已完成成为 Azure 应用商店和 AppSource 发布者所需的步骤。合作伙伴请求获得批准后，发布者个人资料在**[云合作伙伴门户](https://cloudpartner.azure.com/)**上注册。发布者个人资料将应用于合作伙伴注册期间使用的帐户所发布的所有产品和 SKU。  

如果尚未将公司注册为云合作伙伴，请单击[此处](./cloud-partner-portal-getting-started-with-the-cloud-partner-portal.md)。


**发布者个人资料**：发布者个人资料用于在 Azure 应用商店和 AppSource 中区分你的公司。 发布者个人资料由发布者 ID、显示名称和所有者电子邮件组成。 管理完善的个人资料可增加你的可见性，有助于应用商店用户轻松识别和选择合适的产品。

> [!NOTE]
>  
> 发布第一个产品后，发布者 ID 和已注册的所有者电子邮件不可编辑。 但是，可从“云合作伙伴门户”查看个人资料和编辑发布者显示名称。 


> [!NOTE]  
> 通过 **[云合作伙伴门户“用户”部分](cloud-partner-portal-manage-users.md)**中左侧导航窗格上的“用户”部分，可将用户（参与者和所有者）添加到发布个人资料 

**若要查看和管理发布者个人资料**，请在右上方的菜单栏中单击“发布者个人资料”选项卡。 

![“发布者个人资料”选项卡](./media/cloud-partner-portal-how-to-manage-publisher-profile/publisherprofilenew.png)



![发布者个人资料详细信息](./media/cloud-partner-portal-how-to-manage-publisher-profile/publisherprofiledetails.png)


**链接开发人员中心帐户**：也可将现有开发人员中心帐户与云合作伙伴门户上的发布者个人资料链接。 首先，使用注册开发人员中心帐户所用的相同电子邮件地址登录到门户。 链接后，开发人员中心帐户状态、开发人员中心帐户所有者电子邮件和开发人员中心帐户名称会显示在发布者个人资料页面上。 

> [!NOTE]
> 开发人员中心帐户注册是发布应用商店付费 SKU 所需的必要操作。

如果没有**[开发人员中心帐户](https://docs.microsoft.com/azure/marketplace-publishing/marketplace-publishing-accounts-creation-registration)**，可创建一个此帐户，然后将已注册的电子邮件地址添加为云合作伙伴门户上你的产品的一个所有者。  所有者仅可从“用户”部分将已注册的电子邮件地址添加到你的发布个人资料。 然后，使用相同的电子邮件地址登录到门户，并链接开发人员中心帐户。
