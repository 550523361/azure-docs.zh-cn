# <a name="what-products-can-be-published-via-the-cloud-partner-portal"></a>可通过云合作伙伴门户发布哪些产品

云合作伙伴门户是将 Azure 认证虚拟机 (VM) 映像发布到 Microsoft Azure 应用商店或将 Cortana Intelligence/Microsoft R 解决方案应用程序发布到 Microsoft AppSource 的中心。  

VM 映像是预配置的映像，它们具有一个完整安装的操作系统以及一个或多个应用程序。 若要将虚拟机映像发布到 Azure 应用商店，必须满足[技术先决条件](https://docs.microsoft.com/azure/marketplace-publishing/marketplace-publishing-vm-image-creation-prerequisites)以验证映像。
