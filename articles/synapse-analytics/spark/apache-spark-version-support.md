# <a name="supported-language-and-runtime-versions-for-apache-spark-and-dependent-components"></a>支持 Apache Spark 和从属组件的语言和运行时版本  

Azure 突触分析服务中的 Apache Spark 支持几个不同的运行时间，本文档列出了这些版本。

## <a name="spark-version"></a>火花版本

Spark 2.4 是当前支持的版本。

## <a name="operating-system-version"></a>操作系统版本

Azure Synapse 中的 Apache Spark 运行在 Ubuntu 版本 16.04 上。

## <a name="java-version"></a>Java 版本

Azure 突触中的 Apache Spark 包括 Java 版本 1.8.0_222

## <a name="scala-version"></a>Scala 版本

Azure 突触中的阿帕奇火花包括 Scala 2.11.12

## <a name="net-core-version"></a>.NET Core 版本

Azure 突触中的 Apache Spark 包括 .NET 核心 3.0

## <a name="net-support-for-apache-spark"></a>.NET 支持阿帕奇火花

Azure 突触包括 .NET 对 Apache Spark 0.5.0 的支持

## <a name="delta-lake"></a>Delta Lake

突触分析包括对三角洲湖 0.5.0 的支持

## <a name="python-version"></a>Python 版本

Azure 突触中的 Apache Spark 包括安装 Python 版本 3.6.1
