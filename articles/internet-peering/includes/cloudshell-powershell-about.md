若要运行 cmdlet，可以使用 Azure Cloud Shell（一个免费的交互式 shell）。 它预安装有常用 Azure 工具并将其配置与帐户一起使用。 请直接单击“复制”对代码进行复制，将其粘贴到 Cloud Shell 中，然后按 Enter 来运行它。**** 可通过多种方式来启动 Cloud Shell：


|  |   |
|-----------------------------------------------|---|
| 在浏览器中打开 Cloud Shell。 | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| 单击门户右上角菜单上的 **"云壳**"按钮。 | [![门户中的 Cloud Shell](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


如果不想使用 Azure 云壳，则可以在本地安装 PowerShell。 如果您选择在本地安装和使用 PowerShell，请确保安装最新版本的资源管理器 PowerShell cmdlet。 PowerShell cmdlet 经常更新，您通常需要更新 PowerShell cmdlet 以获得最新的功能功能，否则可能会遇到问题。

要查找您正在本地运行的 PowerShell 版本，请使用"获取模块 -列表可用 Az"cmdlet。 若要更新，请参阅[安装 Azure PowerShell 模块](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)。 有关详细信息，请参阅[如何安装和配置 Azure PowerShell](https://docs.microsoft.com/powershell/azure/azurerm/overview)。

如果您在 macOS 上使用 PowerShell，请按照在[macOS 上安装 PowerShell 核心](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6)的步骤操作。
