# <a name="get-started-with-the-excel-connector"></a>Excel 连接器入门
逻辑应用中没有 Excel 连接器。 

## <a name="to-use-excel-data"></a>使用 Excel 数据
可将 Excel 数据作为以逗号分隔的值 (CSV) 文件存储在存储文件夹中，例如 [OneDrive](connectors-create-api-onedrive.md)。 还可以将此 CSV 文件与[平面文件连接器](../logic-apps/logic-apps-enterprise-integration-flatfile.md)结合使用。
