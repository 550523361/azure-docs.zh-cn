# 将 Azure 订阅切换到其他产品/服务

你可以将现有订阅切换到更符合要求的产品/服务，而不会有任何服务因此停机。你可能因为以下某个原因考虑切换产品/服务：

-	如果你发现你的 Azure 花费很高，切换到货币承诺产品/服务可能会有折扣。单击[此处](billing-usage-rate-card-overview.md)查看有关如何深入了解 Microsoft Azure 资源消耗的详细信息。
-	如果用户希望在注册即用即付订阅之后，利用成员资格权益（例如 MSDN 或 BizSpark）。

## 转换订阅另一种产品/服务指南

1.	登录 [Azure 帐户中心](https://account.windowsazure.com/Subscriptions)。
2.	选择要更改的订阅。
3.	单击“切换到其他产品/服务”。

	![siwtchbutton](.\media\billing-how-to-switch-azure-offer\switchbutton.png)
4.	此时会看到订阅可以切换到的所有产品/服务。此列表因帐户的关联成员资格以及当前的产品/服务而异。你可以单击每个产品/服务下面的“了解详细信息”链接获取相关详细信息。单击所需的产品/服务，继续前往下一个步骤。

	![selectoffer](.\media\billing-how-to-switch-azure-offer\selectoffer.png)
5.	根据要切换到的产品/服务，你可能会看到一条有关切换所造成的影响的备注。在继续操作之前，请仔细查看此备注并遵循相关说明。

	![thingstonote](.\media\billing-how-to-switch-azure-offer\thingstonote.png)
6.	可以选择重命名订阅。默认情况下，我们会将它设置为新的产品/服务名称。单击“切换产品/服务”以完成该过程。

	![confirmpage](.\media\billing-how-to-switch-azure-offer\confirmpage.png)
7.	成功！ 订阅现在已经切换到新的产品/服务。

## 常见问题 (FAQ)

**支持哪些产品/服务更改？**

请参阅[可以切换到的可用产品/服务](#available-offers-you-can-switch-to)的列表。

**切换产品/服务是否会对我的服务有任何影响？**

不会导致服务停机。但是，你切换到的产品/服务可能会有一些限制。例如，某些产品/服务禁止生产环境使用。因此，必须将生产环境资源移到另一个订阅。

**切换产品/服务是否会对服务管理员有任何影响？**

不会影响与订阅关联的任何用户。

**切换产品/服务时预计有哪些导致计费的更改？**

在切换当天，将为所有未付费用生成发票。接下来，订阅将按新产品/服务的定价条款计费。订阅计费周年会变为更改产品/服务的日期。更改产品/服务之前的使用量与计费数据将不会保留，因此建议你在切换之前先下载这些数据。

**注意：**因为计费相关约束，产品/服务无法在创建订阅之后的第一个计费周期内更改。

**我是否可以使用此机制从 Web 直接产品/服务迁移到[云解决方案提供商](https://partner.microsoft.com/Solutions/cloud-reseller-overview) (CSP) 或[企业协议](https://azure.microsoft.com/pricing/enterprise-agreement/) (EA) 产品/服务？**

此机制仅限 Web 直接产品/服务来回切换。若要将现有订阅移至 EA，请让注册管理员将你的帐户添加到 EA。接着，你会收到一封邀请电子邮件。按照说明接受邀请时，订阅会自动移到企业协议之下。目前没有将现有 Web 直接订阅移到 CSP 的选项。
