#<a name="app-service-documentation"></a>应用服务文档

应用服务是 Microsoft Azure 的平台即服务 (PaaS) 产品。 为任何平台或设备创建 Web 应用和移动应用。 将应用与 SaaS 解决方案集成、与本地应用程序进行连接，以及实现业务流程的自动化。
