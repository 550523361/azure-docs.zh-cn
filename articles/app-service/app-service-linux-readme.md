# <a name="app-service-on-linux-documentation"></a>Linux 应用服务文档
当前，Linux 应用服务为公共预览版，支持用户在其本地的 Linux 平台上运行 Web 应用。 这使得特定类型的应用程序具有更好的应用程序兼容性，可以更加轻松地将别处的 Linux 平台托管的现有 Web 应用迁移到 Azure App Service。

用户只能在启用 Linux 的应用服务计划中的 Linux 上创建 Web 应用，此应用服务计划只能托管基于 Linux 的 Web 应用。 

> Linux 应用服务的公共预览版定价为常规价的 50%，此价格是按照应用服务计划使用的计算小时数的一半来计算的。
> 
> 公共预览期间不提供 SLA。
