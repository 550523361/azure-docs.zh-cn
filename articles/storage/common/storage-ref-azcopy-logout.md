# <a name="azcopy-logout"></a>azcopy logout

注销用户并终止对 Azure 存储资源的访问。

## <a name="synopsis"></a>概要

此命令将删除当前用户的所有缓存登录信息。

```azcopy
azcopy logout [flags]
```
