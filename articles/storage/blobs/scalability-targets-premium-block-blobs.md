# <a name="scalability-targets-for-premium-block-blob-storage-accounts"></a>高级块 Blob 存储帐户的可伸缩性目标

针对使用较小千字节范围对象的应用程序优化了高性能块 Blob 存储帐户。 它非常适合需要高事务速率或一致的低延迟存储的应用程序。 高级性能块 Blob 存储旨在随应用程序一起扩展。 如果方案要求部署每秒需要数十万个请求或 PB 存储容量的应用程序，请在[Azure 门户](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)中提交支持请求与 Microsoft 联系。
