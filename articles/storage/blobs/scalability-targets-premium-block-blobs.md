# <a name="scalability-targets-for-premium-block-blob-storage-accounts"></a>高级块 blob 存储帐户的可伸缩性目标

高级性能块 blob 存储帐户适用于使用小型 kb 范围对象的应用程序。 它非常适合需要高事务速率或一致的低延迟存储的应用程序。 高级性能块 blob 存储旨在随应用程序一起扩展。 如果你的方案要求你部署的应用程序需要每秒数百个请求或存储容量 pb，请通过在[Azure 门户](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade)中提交支持请求来与 Microsoft 联系。
