# <a name="introducing-azure-security-center-for-iot"></a>适用于 IoT 的 Azure 安全中心简介

统一安全管理，跨混合云工作负载和 Azure IoT 解决方案实现端到端威胁检测和分析。 

## <a name="secure-your-entire-iot-solution-from-iot-devices-to-azure-cloud"></a>保护整个 IoT 解决方案从 IoT 设备到 Azure 云

Azure IoT 安全中心从我们的无缝无代理解决方案中进行选择，或利用基于代理的全面安全性，为 IoT 资产的每个设备（IoT Edge 和 IoT 中心）提供威胁预防和分析。

随着数十亿新设备连接到互联网，并集成到我们的日常生活和业务中，您的安全运营团队必须确保其安全策略发展得足够快，以覆盖每个新的攻击面。 与任何其他系统一样，为了全面保护您的 IoT 解决方案，它需要在每个阶段实施时得到保护。 

IoT 的 Azure 安全中心通过在边缘、本地、Azure 和其他工作负荷中提供统一的可见性和控制、自适应威胁防护以及智能威胁检测和响应，简化了混合工作负载保护云。 

![适用于 IoT 的 Azure 安全中心体系结构](./media/architecture/azure-iot-security-architecture.png)

### <a name="unified-visibility-and-control"></a>统一监视和控制

跨所有本地和云工作负载（包括 Azure IoT 解决方案）统一安全视图。 板载新设备，并在工作负载（Leaf 设备、Microsoft Edge 设备、IoT 中心）中应用安全策略，以确保符合安全标准并改进了安全状态。 

### <a name="adaptive-threat-prevention"></a>自适应威胁防御

使用 IoT 的 Azure 安全中心持续监视计算机、网络和 Azure 服务的安全性。 从数百个内置安全评估中进行选择，或在 IoT 中心中心中央 Azure 安全中心仪表板中创建自己的安全评估。 通过虚拟机、网络、应用和数据中的可操作建议，优化安全设置并提高安全得分。 使用新添加的 IoT 功能，现在可以减少 Azure IoT 解决方案的攻击面，并在利用问题之前修复问题。

### <a name="intelligent-threat-detection-and-response"></a>智能威胁检测和响应

使用高级分析和 Microsoft 智能安全图，在不断演变的网络攻击中获得优势。 内置的行为分析和机器学习可识别攻击和零日漏洞利用。 监视 IoT 解决方案，以寻找传入的攻击和入侵后活动。 使用交互式工具和上下文威胁智能简化设备调查和补救。
