# <a name="get-started-with-azure-security-center-for-iot"></a>适用于 IoT 的 Azure 安全中心入门

本文介绍了适用于 IoT 服务的 Azure 安全中心的不同组件，并解释了如何使用两个可能的部署选项开始使用该服务。

## <a name="deployment-options"></a>部署选项

选择最符合 IoT 设备和环境要求的服务方案。

### <a name="built-in-deployment"></a>内置部署

使用无缝的内置部署选项，可以快速将 IoT Azure 安全中心集成到 IoT 中心中，并提供 IoT 中心配置、设备标识和管理以及中心设备通信模式的安全分析。

启动包含 IoT 中心监视和建议[的内置部署](iot-hub-integration.md)。
    <br>

### <a name="enhanced-deployment"></a>增强型部署

为了增强安全功能，为 IoT 代理部署 Azure 安全中心，除了启用 IoT Hub 安全外，还提供基于代理的事件收集、分析和威胁检测 IoT 设备的关键安全数据以及全面的安全状态管理功能。

启动[增强部署](security-agents.md)，采用基于代理的全面威胁保护和安全态势管理解决方案。
