# <a name="updates-to-the-sign-in-experience-for-online-services"></a>更新联机服务的登录体验

我们正在更新 Office 365 和 Azure 等 Microsoft 联机服务的登录体验。 虽然只是外观上发生了变化，但你会发现屏幕变得更整洁，更直接，让你能更加专注于要做的事 - 登录。

## <a name="current-ui-versus-new-ui"></a>当前 UI 与新 UI

比较当前 UI 与新 UI。 新 UI 更加精简、侧重点更明确，外观焕然一新。

![旧的 Microsoft 登录体验](media/microsoft-sign-in-experience-old.jpg)

![新的 Microsoft 登录体验](media/microsoft-sign-in-experience-new.jpg)

## <a name="when-will-the-new-ui-appear"></a>新 UI 何时推出？

以下是预计进行更新的时间表：

| 时间范围 | 更改 |
|-----------|--------|
| 5 月初 | 在此期间现有登录页面上会出现通知横幅。 可从此处了解相关信息并准备好进行更新。 |
| 5 月中旬   | 通知横幅会发生变化，其中包含更新后的 UI 的预览链接。 使用预览时，仍可使用链接切换回现有登录体验。 |
| 6 月中旬  | 提供更新后的登录 UI，且每个用户都默认使用新的 UI 体验。 |

## <a name="related-info"></a>相关信息

如果你是管理员，则可以根据产品查找有关这些更新的详细信息。

- **Office 365**。 获取有关 [Office 365 消息中心](https://portal.office.com/adminportal/home#/MessageCenter?id=MC133397&MCLinkSource=DigestMail)更新的更多详细信息。 必须使用管理员身份验证登录方可访问此网站。

- **其他联机服务**。 阅读这篇[博客文章](https://cloudblogs.microsoft.com/enterprisemobility/2018/04/04/upcoming-improvements-to-the-azure-ad-sign-in-experience/)，获取有关更新的更多详细信息。
