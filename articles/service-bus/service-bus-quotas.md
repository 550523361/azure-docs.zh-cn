# 服务总线配额

本部分列出了 Microsoft Azure 服务总线消息传送的基本配额和限制阈值。

## 常规配额

每个 Azure 订阅允许的最大服务命名空间数为 100。

有关服务总线的定价及其他配额的信息，请参阅 [Azure 定价](http://www.windowsazure.cn/pricing/overview/)概述。
