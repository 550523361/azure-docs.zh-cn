# <a name="azure-advisor-documentation"></a>Azure 顾问文档

Azure 顾问是一种个性化的云顾问，可帮助遵循最佳做法来优化 Azure 部署。 它分析资源配置和使用情况遥测数据。 然后，它推荐解决方案，帮助提高资源的性能、安全性和高可用性，同时寻找机会减少总体 Azure 支出。
