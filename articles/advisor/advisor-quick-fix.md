# <a name="quick-fix-remediation-for-advisor"></a>顾问的快速修复修补程序
通过**快速修复**，可以更快、更轻松地进行修正，以提供对多个资源的建议。 它为资源提供大容量修正的功能，并可帮助你更快地对资源进行修正来优化订阅。
此功能仅适用于某些建议，通过 Azure 门户。


## <a name="steps-to-use-quick-fix"></a>使用 "快速修复" 的步骤

1. 从具有 "**快速修复**" 标签的建议列表中，单击 "建议"。

   ![Advisor 快速修补](./media/quick-fix-1.png)
   
   *映像中的价格仅用于示例目的*

2. 在 "建议详细信息" 页上，你将看到你具有此建议的资源列表。 选择要修正的所有资源以获取建议。

   ![Advisor 快速修补](./media/quick-fix-2.png)
   
   *映像中的价格仅用于示例目的*

3. 选择资源后，请单击 "**快速修复**" 按钮进行批量更正。

   > [!NOTE]
   > 某些列出的资源可能已禁用，因为你没有适当的权限来修改它们。
   
   > [!NOTE]
   > 如果还有其他影响，则除了 Advisor 中提到的权益，你还将在体验中传达，以帮助你做出及时的补救决策。
   
4. 你将收到有关修正完成的通知。 如果资源列表视图中有未修正的资源和处于所选模式的资源，将会出现错误。
