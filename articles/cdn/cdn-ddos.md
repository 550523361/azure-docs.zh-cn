# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS 保护

内容交付网络按设计提供 DDoS 保护。 除了全局容量来吸收批量攻击外，Azure CDN 还具有如下所述的额外 DDoS 保护，无需额外费用。

## <a name="azure-cdn-from-microsoft"></a>Microsoft 推出的 Azure CDN

来自微软的 Azure CDN 受[Azure 基本 DDoS 的保护](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview)。 默认情况下，它将从 Microsoft 平台集成到 Azure CDN 中，无需额外费用。 Microsoft 全球部署网络的 Azure CDN 全面规模和容量通过始终打开的流量监控和实时缓解功能，可抵御常见的网络层攻击。 基本 DDoS 保护还针对针对 CDN 终结点的最常见、经常发生的第 7 层 DNS 查询洪流和第 3 层和第 4 层卷攻击进行防御。 此服务在保护 Microsoft 的企业和消费者服务免受大规模攻击方面也有着良好的记录。

## <a name="azure-cdn-from-verizon"></a>Verizon 的 Azure CDN

来自 Verizon 的 Azure CDN 受 Verzion 专有 DDoS 缓解平台的保护。 默认情况下，它是从 Verizon 集成到 Azure CDN 中，无需额外费用。 它针对针对 CDN 终结点的最常见、经常发生的第 7 层 DNS 查询洪水和第 3 层和第 4 层卷攻击提供基本保护。

## <a name="azure-cdn-from-akamai"></a>Akamai 的 Azure CDN

Akamai 的 Azure CDN 受 Akamai 专有 DDoS 缓解平台的保护。 默认情况下，它由 Akamai 集成到 Azure CDN 中，无需额外费用。 它针对针对 CDN 终结点的最常见、经常发生的第 7 层 DNS 查询洪水和第 3 层和第 4 层卷攻击提供基本保护。
