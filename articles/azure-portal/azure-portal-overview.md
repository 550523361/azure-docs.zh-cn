# <a name="azure-portal-overview"></a>Azure 门户概述

本文介绍了 Azure 门户中，标识门户页元素，并可帮助你熟悉 Azure 门户管理体验。

## <a name="what-is-the-azure-portal"></a>什么是 Azure 门户？

Azure 门户是一个基于 web 的、 统一的控制台，提供命令行工具的替代方法。 使用 Azure 门户中，可以管理 Azure 订阅使用图形用户界面。 可以生成、 管理和监视一切内容，从简单的 web 应用到复杂的云部署、 创建的资源，结构化视图的自定义仪表板和配置辅助功能选项有关的最佳体验。

## <a name="azure-home"></a>Azure 主页

为 Azure 服务的新订阅服务器，第一件事后看到[登录到门户](https://portal.azure.com)是**Azure 主页**。 此页面编译资源，帮助您充分利用你的 Azure 订阅。 我们包含了指向免费在线课程、 文档、 核心服务、 链接和有用的站点保持为当前和管理更改为你的组织。 若要运行正在进行中的快速方便地访问，我们还会显示您最近访问的资源的列表。 不能自定义此页上，但你可以选择是否查看**Azure 主页**或**Azure 仪表板**作为默认视图。 第一次登录时，将提示安装在您可以在其中保存您的首选项页的顶部。

![屏幕快照显示默认视图选择器](./media/azure-portal-overview/azure-portal-default-view.png)

## <a name="azure-dashboard"></a>Azure 仪表板

仪表板为您最提供你的订阅中重要的资源的集中的视图。 我们提供了你的默认仪表板帮助你入门。 您可以自定义此仪表板使用的资源最引入单个视图。 默认视图的任何更改会影响仅你的体验。 但是，可以创建供自己使用的其他仪表板或发布自定义仪表板并在组织中与其他用户共享它们。 有关详细信息，请参阅[在 Azure 门户中的创建和共享仪表板](../azure-portal/azure-portal-dashboards.md)。

## <a name="getting-around-the-portal"></a>解决在门户

它是有助于您了解了基本的门户布局以及如何与其进行交互。 在这里，我们将介绍用户界面的组件以及一些我们使用提供的指导的术语。 在门户的更详细教程，请参阅课程课[在门户中导航](https://docs.microsoft.com/learn/modules/tour-azure-portal/3-navigate-the-portal)。

Azure 门户边栏和页标头始终都存在的全局元素。 这些持久功能是"shell"的用户界面与每个服务或功能和标头提供了访问全局控件。 配置页 （有时称为"边栏选项卡"） 的资源可能还有左侧的窗格中，以帮助您迁移的功能。

下图列出 Azure 门户中，其中每个下表中所述的基本的元素。

![显示全屏门户视图和到 UI 元素的键的屏幕截图](./media/azure-portal-overview/azure-portal-fullscreen-map.png)

|密钥|描述
|:---:|---|
|第|页标头。 显示在每个门户页的顶部，并保存全局元素。|
|2| 全局搜索。 使用搜索栏可以快速找到特定资源、 服务或文档。|
|3|全局控件。 像所有全局元素，这些功能在门户中保持原样，包括：Cloud Shell、 订阅筛选器、 通知、 门户设置、 帮助和支持，并向我们发送反馈。|
|4|你的帐户。 查看有关你的帐户信息、 切换目录，注销，或使用其他帐户登录。|
|5|侧栏。 侧栏是一个全局元素，可帮助你的服务之间导航。 可以折叠侧栏以便更多关注工作窗格。|
|6|要在当前订阅中创建新的资源的主控件。 搜索或浏览 Azure Marketplace 为你想要创建的资源类型。|
|7|您的收藏夹列表。 添加或删除的最爱**所有服务**页。|
|8|左窗格。 许多服务包括了用于帮助你管理服务的左的窗格菜单。|
|9|命令栏。 在命令栏控件是上下文当前焦点关系。|
|10|痕迹导航。 可以使用痕迹导航链接在工作流中后退一个级别。|
|11|工作窗格。  显示有关当前处于焦点的资源的详细信息。|

## <a name="get-started-with-services"></a>开始使用服务

如果你是新的订阅服务器，必须创建资源之前没有任何要管理的内容。 选择 **+ 创建资源**若要在 Azure Marketplace 中查看可用的服务。 您会发现应用程序和服务从数百个提供程序在这里，所有认证，Azure 上运行。

我们预先填充以链接到常用的服务侧栏中的收藏夹。  若要查看所有可用的服务，请选择**所有服务**从侧栏。

> [!TIP]
> 若要查找资源、 服务或文档的最快方法是使用*搜索*全局标头中。 使用痕迹导航链接以返回到以前的页面。
