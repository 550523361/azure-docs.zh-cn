# <a name="keyboard-shortcuts-in-the-azure-portal"></a>Azure 门户中的键盘快捷方式

本文列出了在 Azure 门户中的任何位置使用的键盘快捷方式。 各个服务可能有其自己的键盘快捷方式。

## <a name="actions"></a>操作

|要执行此操作 |请按 |
| --- | --- |
|创建资源|G+N|
|打开**所有服务**|G+B|
|搜索资源、服务和文档|G+/|
|搜索资源菜单项|CTRL+/ |
|上移所选的左侧边栏项 |ALT+Shift+向上键|
|向下移动所选的左侧边栏项目 |ALT+Shift+向下键|

## <a name="navigation"></a>导航

|若要执行此导航 |请按 |
| --- | --- |
|将焦点移到命令栏 |G+, |
|在页眉和左侧边栏之间切换焦点 | G+. |

## <a name="go-to"></a>转到

|若要定位到此位置 |请按 |
| --- | --- |
|中转到**仪表板** |G+D |
|中转到**所有资源**|G+A |
|中转到**资源组**|G+R |
|在此位置打开左侧边栏项 |G+数字|
