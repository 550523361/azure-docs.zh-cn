# <a name="azure-mobile-engagement-web-sdk-release-notes"></a>Azure Mobile Engagement Web SDK 发行说明
## <a name="202-10182016"></a>2.0.2 (10/18/2016)
* 修复了隐私浏览模式崩溃 (Safari)。
* 修复了已禁用 Cookie 的浏览器崩溃。

## <a name="201-6102016"></a>2.0.1 (6/10/2016)
* 禁用了 Internet Explorer 8 和 Internet Explorer 9 中的 SDK。
* 修复了 Opera web 浏览器检测。

## <a name="200-672016"></a>2.0.0 (6/7/2016)
* 发布了 Azure Mobile Engagement。
* 进行了更新，以便 `appId` 配置替换为连接字符串配置。
* 添加了启用和禁用代理的 API。
* 进行了安全性改进。
* 进行了更新以从浏览器使用本机 JSON API。
* 删除了在设备之间发送和接收消息的 API。
* 删除了与弃用的 XMPP 连接相关的回调。
* 删除了对市场宣传功能的支持。
