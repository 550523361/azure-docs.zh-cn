# <a name="sdk-content"></a>SDK 内容
本文档列出并描述了 SDK 存档的内容。

## <a name="engagement-sdk"></a>Engagement SDK
`EngagementSDK`

> 核心 Engagement SDK 包含标头、库，以及一个提供用来启用/禁用 IDFA 集合的源代码的附加类。
> 
> 

## <a name="engagement-reach-sdk"></a>Engagement Reach SDK
`EngagementReach`

> Engagement Reach SDK 包含标头、库和关联的资源。
> 
> 

## <a name="additional-files"></a>其他文件
`doc/index.html`（在 `doc` 文件夹中）

> Engagement SDK 和 Reach SDK Objective-C 文档。
> 
> 

`src` 文件夹

> Engagement 视图控制器和 Reach SDK 的源代码（适用于想要根据自己的需求调整 Engagement 类的超级用户）。
> 
>
