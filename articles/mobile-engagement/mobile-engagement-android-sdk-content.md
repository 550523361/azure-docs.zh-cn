# <a name="sdk-content"></a>SDK 内容
> [!IMPORTANT]
> Azure Mobile Engagement 已在 2018 年 3 月 31 日停用。 此页将在不久之后删除。
> 

本文档列出并描述了 SDK 存档的内容。

`mobile-engagement-VERSION.jar`

> Engagement SDK 类文件。
> 
> 

`res` 文件夹

> Engagement Reach SDK 资源（要包括在应用程序中的可绘制的布局文件）。
> 
> 

`src` 文件夹

> Engagement SDK 源代码。
> 
> 请注意，不包含服务的源（`com.microsoft.azure.engagement.service.EngagementService` 及其相关的源文件）。
> 
> 

`documentation.html`

> 与 SDK 的下载版本相对应的文档站点的快捷方式。
> 
> 

其他文件是软件许可证的副本。
