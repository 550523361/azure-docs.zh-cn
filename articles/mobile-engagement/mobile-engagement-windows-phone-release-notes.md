# <a name="windows-phone-silverlight-sdk-release-notes"></a>Windows Phone Silverlight SDK 发行说明
##<a name="331-11032016"></a>3.3.1 (11/03/2016)
*MicrosoftAzure.MobileEngagement* Nuget 包 **v3.4.1** 的一部分

* 稳定性改进。

## <a name="330-04192016"></a>3.3.0 (04/19/2016)
*MicrosoftAzure.MobileEngagement* Nuget 程序包 **v3.4.0** 的一部分

* 添加了“TestLogLevel”API，以启用/禁用/筛选由 SDK 发出的控制台日志。

## <a name="320-01222016"></a>3.2.0 (01/22/2016)
*MicrosoftAzure.MobileEngagement* Nuget 程序包 **v3.3.0** 的一部分

* 操作 URL 上可用的自定义方案、ms-appx 和 ms-appdata。
* 稳定性改进。

## <a name="310-11202015"></a>3.1.0 (11/20/2015)
*MicrosoftAzure.MobileEngagement* Nuget 程序包 **v3.2.0** 的一部分

* 修复了初始化后立即请求设备 ID 时出现崩溃的问题。
* 控制台日志改进。

## <a name="300-04032015"></a>3.0.0 (04/03/2015)
* 更新了默认的通知图标。
* 单击通知后发送回系统通知操作反馈。
* 修复了系统通知上的争用条件。

## <a name="200-02172015"></a>2.0.0 (02/17/2015)
* Mobile Engagement 的初始版本。
* appId/sdkKey 配置将替换为连接字符串配置。
* 安全性改进。
