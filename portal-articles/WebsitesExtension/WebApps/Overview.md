# <a name="web-apps-documentation"></a>Web 应用文档

使用 Azure Web 应用可以采用所选编程语言构建和托管 Web 应用程序，而无需管理基础结构。 它提供自动缩放和高可用性，支持 Windows 和 Linux，并允许从 GitHub、Team Services 或任何 Git 存储库进行自动部署。 使用我们的快速入门、教程和示例了解如何使用 Azure Web 应用。

了解如何使用 Web 应用将第一个应用程序部署到云：

- [Node.js](/azure/app-service/app-service-web-get-started-nodejs)
- [PHP](/azure/app-service/app-service-web-get-started-php)
