### <a name="prerequisites"></a>必备条件
* A [Dropbox](https://www.Dropbox.com/)帐户 

在逻辑应用中使用你的 Dropbox 帐户之前，你必须向授权逻辑应用，使连接到你的 Dropbox 帐户。 幸运的是，你可以轻松地从在 Azure 门户上的应用程序逻辑。 

下面是授权逻辑应用程序，以连接到你的 Dropbox 帐户的步骤：

1. 若要创建与 Dropbox 的连接逻辑应用程序设计器中，选择**显示 Microsoft 托管 Api**下拉列表然后输入*Dropbox*的搜索框中。 选择触发器或你将希望使用的操作：  
   ![Dropbox 步骤 1](./media/connectors-create-api-dropbox/dropbox-1.png)
2. 如果你尚未创建任何连接到 Dropbox 之前，你将获取提示提供你的 Dropbox 凭据。 这些凭据将用于连接到你逻辑应用程序进行授权和访问 Dropbox 帐户的数据：  
   ![Dropbox 步骤 2](./media/connectors-create-api-dropbox/dropbox-2.png)
3. 提供你的 Dropbox 用户名和密码逻辑应用程序进行授权：  
   ![Dropbox 步骤 3](./media/connectors-create-api-dropbox/dropbox-3.png)   
4. 授权逻辑应用程序以使用你的 Dropbox 帐户：  
   ![Dropbox 步骤 4](./media/connectors-create-api-dropbox/dropbox-4.png)
5. 请注意已创建连接，并且你现在可以自由地继续其他应用程序逻辑中的步骤：  
   ![Dropbox 步骤 5](./media/connectors-create-api-dropbox/dropbox-5.png)
