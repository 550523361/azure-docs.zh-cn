1. 打开 Android SDK 管理器，通过单击 Android studio 工具栏上的图标，或通过单击**工具** > **Android** > **SDK 管理器**菜单上。 单击**启动独立 SDK 管理器**链接。
2. 在项目中，找到使用的 Android sdk 目标版本，并打开它。 如果**Google Api**尚未安装，然后选择它并单击**安装包...**.
3. 在 Android Studio 中，选择**文件** > **项目结构**。 选择**通知**，选择**Google Cloud Messaging**，然后单击**确定**。
