## <a name="sign-in-to-azure"></a>登录 Azure

将使用 Azure CLI 创建在 Azure 中托管应用所需的资源。 如果是在 Cloud Shell 中运行 Azure CLI 命令，则你已登录。 若要在本地运行 Azure CLI 命令，请使用 [az login](/cli/azure/#login) 命令登录到 Azure 订阅，并按屏幕上的说明进行操作。

```azurecli
az login
```
