#### <a name="to-cable-for-power"></a>进行电源布线
1. 请确保的电源开关上每个电源和冷却模块 (Pcm) 处于关位置。
2. 连接电源线连接到每个主机箱中的 Pcm。
3. 下图中所示，请将电源线连接附加到机架配电装置 (Pdu)。 请确保两个 Pcm 使用单独的电源。
   
   > [!IMPORTANT]
   > 若要确保你的系统的高可用性，我们建议你严格遵循的电源布线方案下图中所示。 
   > 
   > 
   
    ![你将 2U 设备进行电源布线](./media/storsimple-cable-8100-for-power/HCSCableYour2UDeviceforPower.png)
   
    **电源布线 8100 设备上**
   
   | Label | 描述 |
   |:--- |:--- |
   | 1 |PCM 0 |
   | 2 |控制器 1 |
   | 3 |控制器 0 |
   | 4 |PCM 1 |
   | 5 |Pdu |
4. 若要打开系统，翻转到开位置的两个 Pcm 上的电源开关。
