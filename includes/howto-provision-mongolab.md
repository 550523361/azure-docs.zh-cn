您可以在 Azure 应用商店订阅由 Azure 承载且完全托管的 MongoDB 数据库。 为此，请执行下列步骤：

1. 登录到 Azure 管理门户。
2. 单击“新建” 。  
   ![New][button-new]
3. 选择“应用商店”。  
   ![应用商店][button-store]
4. 选择“MongoLab”。 可以在“应用服务”类别以及“全部”下找到该选项。  
   ![MongoLab][entry-mongolab]
5. 单击“下一步”。  
   ![下一步][button-next]  
   将显示 MongoLab 应用商店条目。  
   ![NewMongoLab][screen-newmongolab]
6. 选择所需的**订阅**选项。
7. 输入数据库的**名称**。 名称只能包含字母字符、短划线、句点和下划线。 MongoLab 还要求该名称必须唯一，因此可能要求您在名称已被使用时重新提交您的请求。
8. 选择所需的**区域**。
9. 单击“资源组名称” 的 Azure 数据工厂。  
   ![下一步][button-next]
10. 查看应用商店购买信息，然后单击“购买”确认。  
    ![下一步][button-purchase]  
11. 工具栏进度按钮提供了您的设置状态。  
    ![ProgressButton][button-progress]  
    在配置完成时，显示一条成功消息。  
    ![SuccessMessage][message-success]

祝贺你！ MongoLab 已在您所选择的 Azure 区域配置 MongoDB 数据库。 您现在可以访问我们的管理 UI 和全天候支持服务。
