以下限制适用于 Azure 备份。

| 限制标识符 | 默认限制 |
| --- | --- |
| 可针对每个保管库注册的服务器/计算机数量 |Windows Server/客户端 /SCDPM 为 50 <br/> 对于 IaaS Vm 的 200 |
| 数据存储在 Azure 保管库存储中的数据源的大小 |最大 54400 GB<sup>1</sup> |
| 可以在每个 Azure 订阅中创建的备份保管库数量 |25 （备份保管库） <br/> 每个区域的 25 恢复服务保管库 |
| 可以每天计划备份的次数 |对于 Windows Server/客户端为每天 3 个 <br/> 对于 SCDPM 为每天 2 个 <br/> IaaS Vm 一天一次 |
| 数据磁盘附加到 Azure 虚拟机进行备份 |16 |

* <sup>1</sup>54400 GB 限制不适用于 IaaS VM 备份。
