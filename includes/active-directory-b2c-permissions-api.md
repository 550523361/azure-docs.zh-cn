1. 选择“应用程序”  ，然后选择应该有权访问 API 的 Web 应用程序。 例如，“webapp1”  。
1. 选择“API 访问”，然后选择“添加”   。
1. 在“选择 API”  下拉列表中，选择应授予 Web 应用程序访问权限的 API。 例如，“webapi1”  。
1. 在“选择范围”  下拉列表中，选择先前定义的范围。 例如，*demo.read* 和 *demo.write*。
1. 选择“确定”  。
