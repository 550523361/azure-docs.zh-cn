**公共 DNS 区域**

| Resource | 默认限制 |
| --- | --- |
| 每个订阅的公共 DNS 区域 |250 <sup>1</sup> |
| 每个公共 DNS 区域的记录集 |10,000 <sup>1</sup> |
| 在公用 DNS 区域中的每个记录的记录设置 |20 |
| 每个订阅的专用 DNS 区域 |1000|
| 每个专用 DNS 区域的记录集 |25000|
| 每个记录的记录为专用 DNS 区域设置 |20|
| 每个专用 DNS 区域的虚拟网络链接 |1000|
| 每个使用自动注册的专用 DNS 区域的虚拟网络链接已启用 |100|
| 使用自动注册已启用，可以获取虚拟网络链接到的专用 DNS 区域的数目 |第|
| 虚拟网络可以获取链接的专用 DNS 区域数 |1000|

<sup>1</sup>如果需要增加这些限制，请联系 Azure 支持部门。
