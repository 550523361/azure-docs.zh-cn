Function App 托管函数的执行。 Function App 可让你将函数分组为一个逻辑单元，以便更轻松地管理、部署和共享资源。

选择“新建” > “计算” > “Function App”：

![新建 Function App](media/functions-create-function-app-portal2/new_fun_app.png)

输入必填字段：

![新建 Function App](media/functions-create-function-app-portal2/new_fun_app2.png)

| 字段               | 示例值 | 注释 |
| ----------------- | ------------ | ------------- |
| 应用程序名称 | myFunctionApp | 输入唯一名称。 |
| 订阅 | MSDN | 选择你的订阅。 |
| 资源组 | myFunctionApp | 创建新的资源组。 |
| 托管计划 | 消耗量计划 |  根据资源消耗计费。 |
| 位置 | 美国西部  | 选择离你近的区域。 |
| 存储 | myFunctionApp123  | 新建存储帐户。 |

选择“创建”以预配和部署新的 Function App。
