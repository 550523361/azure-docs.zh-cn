1. 登录到 [Azure 门户]。

2. 单击“新建” 。

3. 在搜索框中，键入**Mobile Apps**。

    ![包含突出显示的移动应用程序快速入门的 azure 门户][quickstart]

4. 在结果列表中，选择**移动应用程序快速入门**，然后选择**创建**。
 
5. 下**资源组**，选择现有资源组或创建一个新 （作为您的应用程序使用相同的名称）。

6. 单击“创建” 。 稍等几分钟，等服务部署成功后再继续。
