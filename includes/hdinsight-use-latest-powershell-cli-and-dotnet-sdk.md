> [!IMPORTANT]
> 通过 Azure PowerShell、Azure CLI 使用 Azure Service Manager (ASM) 和 HDInsight .NET SDK 管理 HDInsight 资源的支持**已弃用**，已在 2017 年 1 月 1 日删除。 本文档中的步骤使用的是与 Azure 资源管理器兼容的新的 Azure PowerShell cmdlet、Azure CLI 命令和 .NET SDK。
> 
> 请按照 [Install and Configure Azure CLI](../articles/cli-install-nodejs.md)（安装和配置 Azure CLI）和 [Install and Configure Azure PowerShell](/powershell/azureps-cmdlets-docs)（安装和配置 Azure PowerShell）中的步骤安装最新版的 Azure PowerShell 和 Azure CLI。 若要安装最新版的 HDInsight .NET SDK，请使用 [Microsoft Azure HDInsight .NET SDK](https://www.nuget.org/packages/Microsoft.WindowsAzure.Management.HDInsight/) NuGet 页上提供的信息。 如果现有脚本或应用程序需要修改后才能使用新 cmdlet、新 CLI 命令或 .NET SDK，请参阅[迁移到适用于 HDInsight 群集的基于 Azure 资源管理器的开发工具](../articles/hdinsight/hdinsight-hadoop-development-using-azure-resource-manager.md)，了解详细信息。
