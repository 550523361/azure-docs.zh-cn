## <a name="peering-virtual-networks-from-rm-to-classic"></a>从 RM 到经典的对等虚拟网络
在此方案中，将在两个 VNet（即 **VNET1** 和 **VNET2**）之间创建对等互连，这两个 VNet 分别属于 Azure Resource Manager 部署模型和经典部署模型。

> [!NOTE]
> 虚拟网络必须属于同一订阅。
