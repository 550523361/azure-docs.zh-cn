具有 100 个 TiB 容量限制的标准文件共享在所有 Azure 区域中都在全球可用，但：

- 本地冗余存储：除南非北部、南非西部、德国中西部和德国北部外的所有区域。
- 区域冗余存储：支持支持区域冗余存储的所有区域，日本东部、北欧、南非北部除外。
- 异地冗余/GeoZone 冗余存储：不支持。
