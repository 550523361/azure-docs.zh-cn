Azure Data Lake Analytics makes the complex task of managing distributed infrastructure and complex code easy. It dynamically provisions resources, and you can use it to do analytics on exabytes of data. When the job completes, it winds down resources automatically. You pay only for the processing power that was used. 增加或减少存储数据的大小或计算用量时，不需要重写代码。 To raise the default limits for your subscription, contact support.

| **资源** | **默认限制** | **注释** |
| --- | --- | --- |
| 最大并发作业数 |20 | |
| Maximum number of analytics units (AUs) per account |250 | 使用 AU 的任意组合，在 20 个作业中的 AU 最大可以为 250。 To increase this limit, contact Microsoft Support. |
| 提交作业时的最大脚本大小 | 3 MB | |
| Maximum number of Data Lake Analytics accounts per region per subscription | 5 | To increase this limit, contact Microsoft Support. |
