Azure 数据湖分析使管理分布式基础结构和复杂代码的复杂任务变得简单。 它动态地提供资源，您可以使用它对 PB 的数据进行分析。 作业完成后，它会自动关闭资源。 您只为所使用的处理能力付费。 随着存储的数据大小或使用的计算量的增加或减小，您不必重写代码。 要提高订阅的默认限制，请与支持人员联系。

| **资源** | **限制** | **注释** |
| --- | --- | --- |
| 最大并发作业数 |20 | |
| 每个帐户的最大分析单位 （A） 数量 |250 | 使用 AU 的任意组合，在 20 个作业中的 AU 最大可以为 250。 要增加此限制，请与 Microsoft 支持部门联系。 |
| 提交作业时的最大脚本大小 | 3 MB | |
| 每个订阅每个区域的最大数据湖分析帐户数 | 5 | 要增加此限制，请与 Microsoft 支持部门联系。 |
