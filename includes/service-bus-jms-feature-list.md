|---|---|
| 队列   | 支持 |
| 主题   | 支持 |
| 临时队列 | 支持 |
| 临时主题 | 支持 |
| 消息选择器 | 支持 |
| 队列浏览器 | 支持 |
| 共享持久订阅 | 支持|
| 非共享持久订阅 | 支持 |
| 共享的非持久订阅 | 支持 |
| 非持久订阅 | 支持 |
| 分布式事务 | 不支持 |
| 持久终点 | 不支持 |
