以下要求适用于设备所需的最低 Internet 带宽。

- 设备始终具有专用的 20 Mbps Internet 带宽（或更高带宽）。 此带宽不应与任何其他应用程序共享。
- 使用网络限制时，设备具有专用的 64 Mbps Internet 带宽（或更高带宽）。
