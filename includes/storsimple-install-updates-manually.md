#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>若要安装常规更新通过 Azure 经典门户
1. 上**设备**页上，选择你要在其安装更新的设备。
2. 导航到**设备** > **维护**和向下滚动到**软件更新**。
3. 若要检查更新，请单击**检查更新**位于页的底部。
4. 如果软件更新可用，你将看到一条消息。 单击**安装更新**以开始更新设备。
   
    已成功安装更新时，你将收到通知。
