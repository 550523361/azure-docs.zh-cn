## <a name="vpn-gateway"></a>VPN 网关
VPN 网关资源可用于创建其本地数据中心和 Azure 之间的安全连接。 可以通过三种不同的方式配置 VPN 网关资源：

* **点到站点**– 你可以安全地访问 Azure VNET 中使用的 VPN 客户端从任何计算机的托管资源。 
* **多站点连接**– 你可以安全地连接从本地数据中心到 VNET 中运行的资源。 
* **VNET 到 VNET** – 你可以安全地连接跨同一区域中的 Azure VNET 或者跨区域以构建地域冗余的工作负载。

VPN 网关的关键属性包括：

* **网关类型**-动态路由或静态路由的网关。 
* **VPN 客户端地址池前缀**– 要分配给在点到站点配置连接的客户端的 IP 地址。
