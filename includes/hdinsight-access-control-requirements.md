如果你不是所使用 Azure 订阅的管理员/所有者（如公司拥有的订阅），必须验证以下项才能使用本文档中的步骤：

* Azure 登录名必须至少对在创建 HDInsight（和其他 Azure 资源）时所使用的 Azure 资源组有**参与者**访问权限。
* 至少对 Azure 订阅具有**参与者**访问权限的用户必须之前已注册所使用资源的提供程序。 对订阅具有参与者访问权限的用户首次在订阅上创建资源时，会进行提供程序注册。 不[使用 REST 注册提供程序](https://msdn.microsoft.com/library/azure/dn790548.aspx)创建资源也可完成该操作。
