## <a name="provision-the-solution"></a>设置解决方案

如果你的帐户中，您已经尚未配置远程监视的预配置的解决方案：

1. 登录到[azureiotsuite.com] [ lnk-azureiotsuite]使用你的 Azure 帐户凭据，然后单击 **+** 创建解决方案。
2. 单击**选择**上**远程监视**磁贴。
3. 输入**解决方案名称**为远程监视预配置解决方案。
4. 选择**区域**和**订阅**你想要用于设置解决方案。
5. 单击**创建解决方案**以开始设置过程。 此过程通常需要几分钟才能运行。

### <a name="wait-for-the-provisioning-process-to-complete"></a>等待设置过程以完成
1. 单击与你的解决方案的磁贴**资源调配**状态。
2. 请注意**预配状态**Azure 服务部署在你的 Azure 订阅。
3. 设置完成后，状态将更改为**准备**。
4. 单击该磁贴以查看你的解决方案在右侧窗格中的详细信息。

> [!NOTE]
> 如果你遇到的问题部署预配置的解决方案，请查看[azureiotsuite.com 站点上的权限][ lnk-permissions]和[常见问题][lnk-faq]。 如果问题仍然存在，对创建的服务票证[门户][lnk-portal]。
