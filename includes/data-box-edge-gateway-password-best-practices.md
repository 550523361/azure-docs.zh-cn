请记住以下最佳实践：

- 我们建议您将所有密码存储在安全的地方，这样，如果忘记密码，就不必重置密码。 管理服务无法检索现有密码。 它只能通过 Azure 门户重置它们。 如果重置密码，请确保在重置密码之前通知所有用户。
- 您可以通过 HTTP 远程访问设备的 Windows PowerShell 接口。 作为安全最佳实践，应仅在受信任的网络上使用 HTTP。
- 确保设备密码是强的，并且保护良好。 遵循[密码最佳实践](https://docs.microsoft.com/azure/security/fundamentals/identity-management-best-practices#enable-password-management)。
