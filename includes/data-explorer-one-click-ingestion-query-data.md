## <a name="query-data"></a>查询数据

1. 选择 **v** 按钮以打开查询。 复制到 Web UI 以编辑查询。

1. 在右侧菜单中，选择“快速查询”和“工具”：   

    * “快速查询”  包含指向 Web UI（其中包含示例查询）的链接。
    * “工具”  包含一个指向 Web UI 上的“删除”命令  的链接，因此，你可以通过运行相关的 `.drop` 命令来排查问题。

    > [!TIP]
    > 使用 `.drop` 命令时，可能会丢失数据。 请谨慎使用。
