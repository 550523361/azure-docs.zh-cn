1. 系统提示输入连接信息时，请提供以下详细信息：

   *必需*

   | 属性 | 值 | 说明 | 
   |----------|-------|-------------| 
   | 连接名称 | <connection-name> | 为连接创建的名称 | 
   | 存储帐户 | <*storage-account*> | 从列表中选择存储帐户。 | 
   ||| 

   例如：

   ![创建连接](./media/connectors-create-api-azureblobstorage/create-storage-account-connection.png)  

2. 完成后，选择“创建”。

3. 创建连接后，继续[添加触发器](#add-trigger)或[添加操作](#add-action)。
