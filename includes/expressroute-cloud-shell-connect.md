 如果使用 Azure 云外壳，则在单击"试用"后自动登录到 Azure 帐户。 要在本地登录，请打开具有提升权限的 PowerShell 控制台，然后运行 cmdlet 进行连接。

```azurepowershell
Connect-AzAccount
```

如果有多个订阅，请获取 Azure 订阅的列表。

```azurepowershell-interactive
Get-AzSubscription
```

指定要使用的订阅。

```azurepowershell-interactive
Select-AzSubscription -SubscriptionName "Name of subscription"
```
