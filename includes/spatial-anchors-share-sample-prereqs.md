## <a name="prerequisites"></a>先决条件

若要完成本教程，请确保做好以下准备：

* 已安装 <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017+</a> 并带有 ASP.NET 和 Web 开发工作负荷的 Windows 计算机。
* [.NET Core 2.2 SDK](https://dotnet.microsoft.com/download)。
* 要部署和运行的一个或多个设备（iOS 或 Android）。
  * 如果使用 Android：
    * 在 Windows 计算机上安装了 <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3+</a> 和 <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3+</a>。
    * <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">支持开发人员</a>和 <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">ARCore 功能</a>的 Android 设备。
  * 如果使用 iOS：
    * 安装了 <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10+</a><a href="https://cocoapods.org" target="_blank">CocoaPods</a> 和 <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3+</a> 的 macOS 计算机。
    * 支持开发人员的 <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">ARKit 兼容</a> iOS 设备。
