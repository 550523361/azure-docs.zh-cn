
### <a name="to-sign-up-for-a-blitline-account"></a>若要注册的 Blitline 帐户
1. 登录到[Azure 管理门户](https://manage.windowsazure.com/)。
2. 在管理门户的下方窗格中，单击**新建**。
   
    ![命令栏新][command-bar-new]
3. 单击**存储**。
   
    ![blitline 应用商店][blitline-store]
4. 在**选择外接程序**对话框中，选择**Blitline**然后单击右箭头。
5. 在**个性化外接程序**对话框中，选择**Blitline**你想要注册的计划。
6. 输入一个名称以标识你**Blitline**服务在你的 Azure 设置，或使用的默认值**Blitline**。 名称必须介于 1 和 100 个字符的长度之间，并且包含仅字母数字字符、 短划线、 点和下划线。 名称必须是唯一的订阅的 Azure 应用商店项目列表中。
   
    ![应用商店屏幕 1][store-screen-1]
7. 选择区域值例如，美国西部。 
8. 单击右箭头。
9. 上**查看购买**选项卡上，查看计划和定价信息，并查看法律条款。 如果你同意条款，请单击复选标记。 单击复选标记后，您的 Blitline 帐户将开始设置过程。 

若要使用的 Blitline 你需要在您的应用程序 id。

### <a name="to-find-your-blitline-application-id"></a>若要查找您的 Blitline 应用程序 ID
1. 单击**连接信息**。
   
    ![blitline 连接信息按钮][blitline-connection-info-button]
2. 在*连接信息*对话框中，你可以看到你的应用程序 ID
   
    ![blitline 连接信息][blitline-connection-info]
