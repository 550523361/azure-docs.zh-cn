你可以增加或减少计算通过调整数据仓库单位 (Dwu)。 默认值为 400 Dwu。 计算资源作为 Dwu 增加而呈线性增加。 例如，更改 Dwu 从 400 到 800 double 的计算资源。 

当前的 DWU 设置的列表，请参阅在服务级别目标[服务容量限制](../articles/sql-data-warehouse/sql-data-warehouse-service-capacity-limits.md)。
