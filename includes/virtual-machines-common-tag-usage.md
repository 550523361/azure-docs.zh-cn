


## <a name="viewing-your-tags-in-the-usage-details"></a>用法的详细信息中查看标记
标记放置在计算、 网络、 并在资源管理器部署模型中的存储资源将在你用法的详细信息中填充[计费门户](https://account.windowsazure.com/)。

单击**下载用法的详细信息**若要查看你的订阅中的用法的详细信息。

![在 Azure 门户中的使用情况详细信息](./media/virtual-machines-common-tag-usage/azure-portal-tags-usage-details.png)

选择帐单和**版本 2**用法的详细信息：

![在 Azure 门户中的版本 2 预览使用情况详细信息](./media/virtual-machines-common-tag-usage/azure-portal-version2-usage-details.png)

从用法的详细信息，你可以看到所有中的标记**标记**列：

![在 Azure 门户中的标记列](./media/virtual-machines-common-tag-usage/azure-portal-tags-column.png)

通过分析这些标记以及使用情况，组织将能够获得对消耗数据的新见解。
