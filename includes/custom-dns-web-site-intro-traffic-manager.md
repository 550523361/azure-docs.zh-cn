使用 Microsoft Azure 流量管理器对传往 Azure 网站的流量执行负载均衡时，可以使用 Azure 分配的 **\*.trafficmanager.net** 域名访问该网站。 还可以将自定义域名（例如 www.contoso.com）与网站相关联，为用户提供更易识别的域名。
