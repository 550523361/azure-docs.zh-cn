### <a name="azure-notebooks---free-jupyter-based-notebooks-in-the-azure-cloud"></a>**Azure Notebooks** - Azure 云中基于 Jupyter 的免费笔记本
已在 Azure Notebooks 上为你安装和配置 SDK。
  
1. 完成[开始快速入门](../articles/machine-learning/service/quickstart-get-started.md)以创建工作区并启动 Azure Notebooks。
1. 转到 [Azure Notebooks](https://notebooks.azure.com/)
1. 在快速入门期间创建的 `Getting Started` 库中，转到 `tutorials` 文件夹
1. 打开笔记本。

### <a name="your-own-jupyter-notebook-server"></a>**你自己的 Jupyter 笔记本服务器**
1. 完成[开始 Python SDK 快速入门](../articles/machine-learning/service/quickstart-create-workspace-with-python.md)以安装 SDK 并创建工作区。
1. 克隆 [GitHub 存储库](https://aka.ms/aml-notebooks)。
1. 将在快速入门期间创建的 `aml_config` 目录复制到克隆目录中。
1. 从克隆目录启动 notebook 服务器。
1. 转到 `tutorials` 文件夹。
1. 打开笔记本。
