## <a name="connecting-to-the-azure-portal-with-a-subscription"></a>使用订阅连接到 Azure 门户
如果有 Microsoft Azure 的现有订阅，请遵循以下步骤连接到 Azure 门户。 如果没有现有的 Microsoft Azure 帐户，请单击以下链接之一获取帐户，然后遵循以下步骤连接到 Azure 门户。

1. 打开所选的浏览器并通过 http://portal.azure.com/ 连接到 [Azure 门户](https://portal.azure.com/)。
2. 通过 http://portal.azure.com/ 登录到 [Azure 门户](https://portal.azure.com/)。
3. 登录页出现时，请提供订阅的凭据
