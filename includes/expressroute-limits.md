#### <a name="expressroute-limits"></a>ExpressRoute 限制
以下限制适用于每个订阅的 ExpressRoute 资源。

| 资源 | 默认限制 |
| --- | --- |
| 每个订阅的 ExpressRoute 线路 |10 |
| 每个区域每个 ARM 的订阅的 ExpressRoute 线路 |10 |
| 最大数量的 Azure 专用对等互连具有 ExpressRoute standard 的路由 |4,000 |
| 最大数量的 Azure 专用对等互连具有 ExpressRoute premium 附加设备的路由 |10,000 |
| 最大数量的 Azure 公共对等互连具有 ExpressRoute standard 的路由 |200 |
| 最大数量的 Azure 公共对等互连具有 ExpressRoute premium 附加设备的路由 |200 |
| 为具有 ExpressRoute standard 的 Azure Microsoft 对等互连的路由的最大数目 |200 |
| 有关 Azure Microsoft 对等互连具有 ExpressRoute premium 附加设备的路由的最大数目 |200 |
| 允许每个 ExpressRoute 线路的虚拟网络链接数 |请参见下表 |
