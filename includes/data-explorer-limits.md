下表描述了 Azure 数据资源管理器群集的最大限制。

| 资源 | 限制 |
| --- | --- |
| 每个订阅每个区域的群集 | 20 |
| 每个群集的实例 | 1000 | 
| 群集中的数据库数 | 10,000 |
| 群集中附加的数据库配置数 | 70 |

| 范围 | Operation | 限制 |
| --- | --- | --- |
| 群集 | 读取（例如，获取群集） | 每5分钟500人 |
| 群集 | 写入（例如，创建数据库） | 每小时1000人 |
