**应用程序模板**

| 模板                 | 名称        | 说明 |
| ------------------------ | ----------- | ----------- |
| iotc-pnp-preview@1.0.0   | 自定义应用程序 | 创建一个空的应用程序，以便在其中填充你自己的设备模板和设备。 |
| iotc-default@1.0.0       | 自定义应用程序（旧版） | 创建一个空的旧版应用程序，以便使用自己的设备模板和设备进行填充。
| iotc-condition@1.0.0     | 应用商店内分析-条件监视 | 创建用于连接和监视应用商店环境的应用程序。 |
| iotc-consumption@1.0.0   | 水消耗监视 | 创建用于监视和控制水流量的应用程序。 |
| iotc-distribution@1.0.0  | 数字分发中心 | 创建一个应用程序，通过对关键资产和操作进行数字化来改善仓库输出效率。 |
| iotc-inventory@1.0.0     | 智能库存管理 | 创建应用程序以自动执行接收、产品移动、循环计数和跟踪。 |
| iotc-logistics@1.0.0     | 连接的后勤 | 创建一个应用程序，用于通过位置和条件监视实时跟踪无线、水和陆地的发货情况。 |
| iotc-meter@1.0.0         | 智能计量分析 | 创建一个应用程序来监视能源消耗、网络状态，并识别趋势以改善客户支持和智能计量管理。  |
| iotc-mfc@1.0.0           | 微执行中心 | 创建一个应用程序，用于以数字方式连接和管理完全自动执行中心。 |
| iotc-patient@1.0.0       | 持续患者监视 | 创建应用程序以扩展病人护理，减少 readmissions 并管理疾病。 |
| iotc-power@1.0.0         | 阳历电源监视 | 创建用于监视阳历面板状态和能源生成趋势的应用程序。 |
| iotc-quality@1.0.0       | 水源质量监视 | 创建一个应用程序，用于以数字方式监视水源质量。 |
| iotc-store@1.0.0         | 应用商店内分析–结帐 | 创建一个应用程序，用于监视和管理你的商店内的结帐流。 |
| iotc-waste@1.0.0         | 连接废物管理 | 创建用于监视垃圾桶和调度字段运算符的应用程序。 |
