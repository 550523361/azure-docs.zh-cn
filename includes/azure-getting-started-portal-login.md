## 通过订阅连接到 Azure 门户

若要连接到 Azure 门户，必须有订阅。

### 获取新帐户

如果目前没有 Azure 帐户，选择以下选项之一获取帐户：

- 获取[免费帐户](https://azure.microsoft.com/get-started/)。
- 使用 [MSDN 订阅](https://azure.microsoft.com/pricing/member-offers/msdn-benefits/)。

### 使用现有帐户登录

使用[现有订阅](https://account.windowsazure.com/Home/Index)，按照以下步骤连接到 Azure 门户。

1. 打开所选的浏览器并连接到 [Azure 门户](https://portal.azure.com/)。

1. 登录到 [Azure 门户](https://portal.azure.com/)。

1. 在“登录”页面上，提供订阅的凭据。
