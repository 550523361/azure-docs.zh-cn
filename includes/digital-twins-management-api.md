在以下示例中，`YOUR_MANAGEMENT_API_URL` 代表数字孪生 API 的 URI：

```URL
https://YOUR_INSTANCE_NAME.YOUR_LOCATION.azuresmartspaces.net/management/api/v1.0
```

| 名称 | 替换为 |
| --- | --- |
| YOUR_INSTANCE_NAME | Azure 数字孪生实例的名称 |
| YOUR_LOCATION | 托管实例的区域 |
