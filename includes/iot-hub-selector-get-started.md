## <a name="introduction"></a>简介
Azure IoT 中心是一种完全托管的服务，使可靠，数以百万计的物联网 (IoT) 设备和解决方案之间的双向通信安全后端。 IoT 项目面临的最大挑战之一是如何可靠、 更安全地将设备连接到的解决方案后端。 为应对此挑战，IoT 中心：

* 提供可靠设备到云和云到设备缩放性超高的消息传送。
* 启用安全通信使用每个设备的安全凭据和访问控制。
* 包括最流行的语言和平台的设备的库。

此教程演示如何为：

* 使用 Azure 门户创建 IoT 中心。
* 在你的 IoT 中心中创建设备标识。
* 创建的模拟的设备应用程序将遥测发送到你的解决方案后端。
