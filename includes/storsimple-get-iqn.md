#### <a name="to-get-the-iqn-of-a-windows-host"></a>若要获取 Windows 主机的 IQN
1. 在你的 Windows 主机上启动 Microsoft iSCSI 发起程序。 单击**启动** > **管理工具** > **iSCSI 发起程序**。
2. 在**iSCSI 发起程序属性**窗口，请在**配置**选项卡上，选择并复制从字符串**发起程序名称**字段。
   
    ![iSCSI 发起程序属性](./media/storsimple-get-iqn/HCS_iSCSIInitiatorPropertiesFigureIQN-include.png)
3. 保存此字符串。
