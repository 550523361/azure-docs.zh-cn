### <a name="prerequisites"></a>必备条件
* [RSS](https://wikipedia.org/wiki/RSS)帐户  

你可以在逻辑应用程序中使用 RSS 帐户之前，你必须向授权逻辑应用，使连接到你 RSS 的帐户。幸运的是，你可以轻松地从在 Azure 门户上的应用程序逻辑。  

下面是授权逻辑应用程序，以连接到你的 RSS 帐户的步骤：  

1. 若要创建与 RSS 的连接逻辑应用程序设计器中，选择**显示 Microsoft 托管 Api**下拉列表然后输入*RSS*的搜索框中。 选择触发器或你将希望使用的操作：  
   ![RSS 连接创建步骤](./media/connectors-create-api-rss/rss-1.png)  
2. 选择**创建连接**:  
   ![RSS 连接创建步骤](./media/connectors-create-api-rss/rss-2.png)  
3. 请注意已创建连接，并且你现在可以自由地继续其他应用程序逻辑中的步骤：  
   ![RSS 连接创建步骤](./media/connectors-create-api-rss/rss-3.png)
