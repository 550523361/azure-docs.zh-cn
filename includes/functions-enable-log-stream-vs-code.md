若要打开你在 Azure 中的函数应用的流式处理日志：

1. 按 F1 键打开命令面板中，然后搜索并运行该命令**Azure Functions:启动流式处理日志**。

1. 在 Azure 中，选择函数应用，然后选择**是**若要启用的 function app 的应用程序日志记录。

1. 触发函数在 Azure 中，并请注意，日志数据显示在 Visual Studio Code 中的输出窗口中。

1. 完成后，请务必先运行该命令**Azure Functions:停止流式传输日志**若要禁用的函数应用日志记录。
