对于静止数据：

- 对存储在共享中的数据的访问受到限制。

    - 访问共享数据的 SMB 客户端需要与共享关联的用户凭据。 创建共享时定义这些凭据。
    - 创建共享时需要添加访问共享的 NFS 客户端的 IP 地址。
