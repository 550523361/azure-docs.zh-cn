### <a name="azure-active-directory-authentication"></a>Azure Active Directory 身份验证

Azure 空间的定位点还允许应用程序使用 Azure Active Directory 的用户令牌进行身份验证。 例如，可以使用 AAD 令牌将 ASA 与进行集成。 如果企业维护在 AAD 中的用户，可以提供 Azure 空间的定位点 SDK 中的用户 AAD 令牌。 这样做使您直接向该帐户是属于相同的 AAD 租户的 ASA 服务进行身份验证。
