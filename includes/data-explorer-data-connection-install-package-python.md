## <a name="install-python-package"></a>安装 Python 包

要为 Azure 数据资源管理器 (Kusto) 安装 Python 包，请打开其路径中包含 Python 的命令提示符。 运行下面的命令：

```python
pip install azure-common
pip install azure-mgmt-kusto
```
