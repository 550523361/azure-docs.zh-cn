### <a name="prerequisites"></a>必备条件
* Wunderlist 帐户  

你可以在逻辑应用程序中使用 Wunderlist 帐户之前，你必须向授权逻辑应用，使连接到你 Wunderlist 帐户。 幸运的是，你可以轻松地从在 Azure 门户上的应用程序逻辑。 

下面是授权逻辑应用程序，以连接到你 Wunderlist 帐户的步骤：

1. 若要创建连接到 Wunderlist，逻辑应用程序设计器中，选择**显示 Microsoft 托管 Api**下拉列表然后输入*Wunderlist*的搜索框中。 选择触发器或你将希望使用的操作：  
   ![](./media/connectors-create-api-wunderlist/wunderlist-0.png)
2. 如果你尚未创建任何连接到 Wunderlist 之前，你将让系统提示你提供 Wunderlist 凭据。 这些凭据将用于连接到你逻辑应用程序进行授权和访问 Wunderlist 帐户的数据：   
   ![](./media/connectors-create-api-wunderlist/wunderlist-1.png)  
3. 提供你的凭据，然后选择登录按钮  
   ![](./media/connectors-create-api-wunderlist/wunderlist-2.png)  
4. 你将然后会告知什么逻辑应用，将有权使用 Wunderlist 帐户执行操作。 如果你同意，则选择按钮指示您的协议。 
   ![](./media/connectors-create-api-wunderlist/wunderlist-4.png)  
5. 最后，选择**Authorize**按钮  
   ![](./media/connectors-create-api-wunderlist/wunderlist-5.png)
