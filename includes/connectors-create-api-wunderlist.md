### <a name="prerequisites"></a>先决条件
* 一个 Wunderlist 帐户  

可以在逻辑应用中使用 Wunderlist 帐户之前，必须先授权该逻辑应用可以连接到你的 Wunderlist 帐户。 好在，可以从 Azure 门户中的逻辑应用内轻松地执行此操作。 

下面介绍授权逻辑应用连接到 Wunderlist 帐户的步骤：

1. 若要创建与 Wunderlist 的连接，请在逻辑应用设计器的下拉列表中选择“显示 Microsoft 托管的 API”，然后在搜索框中输入“Wunderlist”。 选择要使用的触发器或操作：  
   ![](./media/connectors-create-api-wunderlist/wunderlist-0.png)
2. 如果以前尚未创建任何与 Wunderlist 的连接，系统会提示你提供 Wunderlist 凭据。 这些凭据将用于授权逻辑应用连接到你的 Wunderlist 帐户，然后访问该帐户中的数据：   
   ![](./media/connectors-create-api-wunderlist/wunderlist-1.png)  
3. 提供你的凭据，然后选择登录按钮  
   ![](./media/connectors-create-api-wunderlist/wunderlist-2.png)  
4. 然后会告知你，逻辑应用具有哪些用于操作你的 Wunderlist 帐户的权限。 如果同意，请选择表示你同意的按钮。 
   ![](./media/connectors-create-api-wunderlist/wunderlist-4.png)  
5. 最后，选择“授权”按钮  
   ![](./media/connectors-create-api-wunderlist/wunderlist-5.png)
