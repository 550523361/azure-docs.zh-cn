## <a name="displaying-the-azure-explorer-view"></a>显示 Azure 资源管理器视图

如果“Azure 资源管理器”视图未显示在 Eclipse 中，请通过以下步骤来显示它：

1. 依次单击“窗口”、“显示视图”、“其他”。

   ![“显示视图”菜单](./media/azure-toolkit-for-eclipse-show-azure-explorer/show-az-exp-01.png)

2. 出现“显示视图”对话框后，请选择“Azure 资源管理器”，然后单击“确定”。

   ![“显示视图”对话框](./media/azure-toolkit-for-eclipse-show-azure-explorer/show-az-exp-02.png)
