若要创建高级缓存，登录到[Azure 门户](https://portal.azure.com)单击**新建** > **数据库** > **Redis 缓存**。

![创建缓存](media/redis-cache-premium-create/redis-cache-new-cache-menu.png)

> [!NOTE]
> 除了在 Azure 门户中创建缓存，你还可以创建使用资源管理器模板、 PowerShell 或 Azure CLI。 有关创建 Azure Redis 缓存的详细信息，请参阅[创建缓存](../articles/redis-cache/cache-dotnet-how-to-use-azure-redis-cache.md#create-a-cache)。
> 
> 

若要配置高级功能，首先选择一个高级定价层中的**定价层**下拉列表。 有关每个定价层的详细信息，请单击**完整定价详细信息视图**和选择定价层从**选择定价层**边栏选项卡。

![选择定价层](media/redis-cache-premium-create/redis-cache-premium-pricing-tier.png)
