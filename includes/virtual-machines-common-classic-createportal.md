

A*自定义*虚拟机只意味着你创建使用的虚拟机**特色应用**从**Marketplace**因为它为您完成大部分工作。 然而，您仍然可配置选项，包括以下各项：

* 连接到虚拟网络的虚拟机。
* 安装的 Azure 虚拟机代理和 Azure 虚拟机扩展，例如反恶意软件。
* 将虚拟机添加到现有云服务。
* 将虚拟机添加到现有的存储帐户。
* 将虚拟机添加到可用性集。
