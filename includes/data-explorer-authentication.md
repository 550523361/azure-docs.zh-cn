## <a name="authentication"></a>身份验证

若要运行以下示例，需要一个可以访问资源的 Azure AD 应用程序和服务主体。 请参阅[创建 Azure AD 应用程序](/azure/active-directory/develop/howto-create-service-principal-portal)以创建免费的 Azure AD 应用程序，并在订阅级别添加角色分配。 还需要 `directory (tenant) ID`、`application ID` 和 `client secret`。
