## <a name="authentication"></a>身份验证

若要运行以下示例，需要一个可以访问资源的 Azure Active Directory （Azure AD）应用程序和服务主体。 若要创建免费的 Azure AD 应用程序并在订阅级别添加角色分配，请参阅[创建 Azure AD 应用程序](/azure/active-directory/develop/howto-create-service-principal-portal)。 还需要目录（租户） ID、应用程序 ID 和客户端密码。
