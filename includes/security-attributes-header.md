质量或功能的一项 Azure 服务参与的服务的功能来防止、 检测和响应安全漏洞，安全特性。

安全属性分为以下几类：
* 预防
* 网络分段
* 检测
* 标识和访问管理支持
* 审核线索
* 访问控制（如果使用）
* 配置管理（如果使用）

在每个类别中，我们会确定是否使用某个属性（是/否）。 对于某些服务，一个属性可能不适用，并显示为不适用。 还可能会提供有关某个属性的注释或详细信息链接。
