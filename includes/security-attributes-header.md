安全属性是促使 Azure 服务能够防范、检测和响应安全漏洞的一种服务质量或功能。

安全属性分为以下几类：
* 预防
* 网络分段
* 检测
* 标识和访问管理支持
* 审核线索
* 访问控制（如果使用）
* 配置管理（如果使用）

在每个类别中，我们会确定是否使用某个属性（是/否）。 对于一些服务，某个属性可能不适用并显示为 N/A。 还可能会提供有关某个属性的注释或详细信息链接。
