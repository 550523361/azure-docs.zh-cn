安全属性是 Azure 服务的质量或功能。 它有助于服务预防、检测和响应安全漏洞。

在每个类别中，我们显示“是”或“否”以指示是否使用属性。 对于某些服务，我们为不适用的属性显示“N/A”。 我们还可能会提供有关属性的更多信息的注释或链接。
