安全特性是质量或功能的一项 Azure 服务。 它有助于防止、 检测和响应安全漏洞的服务的能力。

安全属性分为以下几类：
* 预防
* 网络分段
* 检测
* 对标识和访问管理的支持
* 审核线索
* 访问控制（如果使用）
* 配置管理（如果使用）

在每个类别中，我们将显示"Yes"否"以指示是否使用属性。 对于某些服务，我们将显示"n/A"不适用的属性。 我们还可能提供的注释或属性有关的详细信息的链接。
