> [!TIP]
> 授予对使用 Azure Active Directory (Azure AD) 的 Blob 和队列存储资源的访问权限的 azure 存储支持。 用户或应用程序使用 OAuth 2.0 令牌由 Azure AD 返回授权通过共享密钥的授权提供卓越的安全性和易于使用并共享访问签名 (SAS)。 与 Azure AD，没有必要进行存储帐户访问密钥与你的代码和风险潜在的安全漏洞。
>
> Microsoft 建议使用 Azure AD 与 Azure 存储应用程序在可能的情况。 有关详细信息，请参阅[授权访问 Azure blob 和队列使用 Azure Active Directory](../articles/storage/common/storage-auth-aad.md)。
