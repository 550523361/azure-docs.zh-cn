本示例需要 Azure PowerShell。 运行 `Get-Module -ListAvailable Az` 以查看是否已安装。 如果需要安装，请参阅[安装 Azure PowerShell 模块](/powershell/azure/install-az-ps)。 

运行 `Connect-AzAccount`，创建与 Azure 的连接。
