### <a name="prerequisites"></a>必备条件
* [FTP](https://wikipedia.org/wiki/File_Transfer_Protocol)帐户  

你可以在逻辑应用程序中使用 FTP 帐户之前，你必须向授权逻辑应用，使连接到 FTP 帐户。幸运的是，你可以轻松地从在 Azure 门户上的应用程序逻辑。  

下面是授权逻辑应用程序，以连接到 FTP 帐户的步骤：  

1. 若要创建连接到 FTP，逻辑应用程序设计器中，选择**显示 Microsoft 托管 Api**下拉列表然后输入*FTP*的搜索框中。 选择触发器或你将希望使用的操作：  
   ![FTP 连接创建步骤](./media/connectors-create-api-ftp/ftp-1.png)  
2. 如果你尚未创建任何连接到 FTP 之前，你将获取提示提供你的 FTP 凭据。 这些凭据将用于连接到你逻辑应用程序进行授权和访问你的 FTP 帐户的数据：  
   ![FTP 连接创建步骤](./media/connectors-create-api-ftp/ftp-2.png)  
3. 请注意已创建连接，并且你现在可以自由地继续其他应用程序逻辑中的步骤：  
   ![FTP 连接创建步骤](./media/connectors-create-api-ftp/ftp-3.png)
