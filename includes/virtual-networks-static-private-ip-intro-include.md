虚拟机 (VM) 是自动分配的专用 IP 地址范围中指定的子网中部署。 删除 VM 之前，VM 被保留地址。 Azure 会动态分配创建中的 VM 子网中下一个可用的专用 IP 地址。 如果想要分配给该 VM 子网中的特定 IP 地址将分配静态 IP 地址。
