虚拟机 （VM） 将根据部署 VM 的子网从您指定的范围自动分配专用 IP 地址。 VM 保留地址，直到删除 VM。 Azure 从你创建 VM 时所在的子网动态分配下一个可用的专用 IP 地址。 如果希望从分配给 VM 的子网中分配特定的 IP 地址，请分配静态 IP 地址。
