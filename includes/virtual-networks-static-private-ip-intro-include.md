根据部署 VM 的子网，自动为虚拟机（VM）分配指定范围内的专用 IP 地址。 VM 将保留地址，直到删除 VM。 Azure 从你创建 VM 时所在的子网动态分配下一个可用的专用 IP 地址。 如果需要分配给 VM 的子网中的特定 IP 地址，请分配静态 IP 地址。
