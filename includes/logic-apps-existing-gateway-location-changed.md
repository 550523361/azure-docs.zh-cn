**问**：为什么更改我的现有网关的位置？ <br/>
**答**：2017 年 5 月 3 日已移动到 Azure AD 租户的原始位置的工作或学校帐户之前创建的网关资源创建这些网关。 但是，这些更改应当不会影响当前正在运行的逻辑应用，这些应用应当继续照常运行。 随着该网关在五月份的正式发布，网关资源的位置可能不同于逻辑应用位置。
