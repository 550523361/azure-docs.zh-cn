`Eula` 设置表示已接受容器的许可条款。 必须为此配置设置指定值，并且该值必须设置为 `accept`。

|必选| “属性” | 数据类型 | 描述 |
|--|------|-----------|-------------|
|是| `Eula` | String | 接受许可证<br><br>示例：<br>`Eula=accept` |

根据管理 Azure 使用的[协议](https://go.microsoft.com/fwlink/?linkid=2018657)授予认知服务容器许可。 如果您没有管理使用 Azure 的现有协议，则同意管理 Azure 的协议是[Microsoft 在线订阅协议](https://go.microsoft.com/fwlink/?linkid=2018755)，该协议包含[联机服务条款](https://go.microsoft.com/fwlink/?linkid=2018760)。 对于预览版，还应同意 [Microsoft Azure 预览版补充使用条款](https://go.microsoft.com/fwlink/?linkid=2018815)。 使用容器即表示同意这些条款。
